# <a name="idisperrorgethresult"></a>IDispError::GetHresult
擷取的錯誤程式碼`IDispError`物件。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT GetHresult(  
   HRESULT*  phr  
);  
```  
  
#### <a name="parameters"></a>參數  
 `phr`  
 [out]指定的錯誤碼。  
  
## <a name="return-value"></a>傳回值  
 方法會傳回 `HRESULT`。 可能的值包括 (但不限於) 下表中的這些值。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>備註  
 這個方法會擷取來自錯誤碼`IDispError`物件。  
  
> [!NOTE]
>  這個方法尚未實作。
