# <a name="idebugapplicationnodeclose"></a>IDebugApplicationNode::Close
導致此應用程式釋放所有參考，並進入非作用中狀態。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT Close();  
```  
  
#### <a name="parameters"></a>參數  
 這個方法不接受任何參數。  
  
## <a name="return-value"></a>傳回值  
 方法會傳回 `HRESULT`。 可能的值包括 (但不限於) 下表中的這些值。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>備註  
 一般而言，應用程式的擁有者會在應用程式結束時呼叫這個方法。
