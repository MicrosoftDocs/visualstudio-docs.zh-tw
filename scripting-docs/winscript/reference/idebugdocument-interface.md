# <a name="idebugdocument-interface"></a>IDebugDocument 介面
所有偵錯文件的基底介面。  
  
## <a name="remarks"></a>備註  
 `IDebugDocument`介面會公開僅方法繼承自[IDebugDocumentInfo 介面](../../winscript/reference/idebugdocumentinfo-interface.md)。 它不會公開任何額外的方法。
