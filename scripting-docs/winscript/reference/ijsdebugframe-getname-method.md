# <a name="ijsdebugframegetname-method"></a>IJsDebugFrame::GetName 方法
取得的堆疊框架的使用者易記名稱。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pName`  
 [out]堆疊框架的名稱。  
  
## <a name="return-value"></a>傳回值  
  
## <a name="requirements"></a>需求  
 **標頭：** jscript9diag.h
