# <a name="idebugapplicationthreadsetstatestring"></a>IDebugApplicationThread::SetStateString
設定執行緒狀態的描述。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT SetStateString(  
   LPCOLESTR  pstrState  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pstrState`  
 [in]執行緒狀態的描述。  
  
## <a name="return-value"></a>傳回值  
 方法會傳回 `HRESULT`。 可能的值包括 (但不限於) 下表中的這些值。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>備註  
 這個方法會設定執行緒狀態的描述。
