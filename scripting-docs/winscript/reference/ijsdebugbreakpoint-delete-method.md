# <a name="ijsdebugbreakpointdelete-method"></a>IJsDebugBreakPoint::Delete 方法
刪除中斷點。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT Delete(void);  
```  
  
## <a name="return-value"></a>傳回值  
  
## <a name="remarks"></a>備註  
 如果在已刪除的中斷點上呼叫，則會傳回 S_FALSE。  
  
## <a name="requirements"></a>需求  
 **標頭：** jscript9diag。h
