# <a name="iactivescripttraceinfostopscripttracing-method"></a>IActiveScriptTraceInfo::StopScriptTracing 方法
停止指令碼追蹤。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT StopScriptTracing(void); }  
```  
  
#### <a name="parameters"></a>參數  
 這個方法沒有任何參數。  
  
## <a name="return-value"></a>傳回值  
 可能的傳回值，這個方法如下所示：  
  
1. S_OK:成功。  
  
2. E_NOTIMPL:未實作。
