# <a name="iactivescriptstatsresetstats"></a>IActiveScriptStats::ResetStats
重設此指令碼的統計資料。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT ResetStats();  
```  
  
#### <a name="parameters"></a>參數  
 這個方法會接受任何參數。  
  
## <a name="return-value"></a>傳回值  
 方法會傳回 `HRESULT`。 可能的值包括 (但不限於) 下表中的這些值。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>備註  
 這個方法會重設此指令碼的統計資料。
