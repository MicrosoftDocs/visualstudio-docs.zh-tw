# <a name="scriptepropagate-error-code"></a>SCRIPT_E_PROPAGATE 錯誤碼
指令碼錯誤傳播至呼叫端，這可能是在不同的執行緒。 主應用程式應將錯誤碼傳遞給呼叫者。  
  
## <a name="syntax"></a>語法  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
