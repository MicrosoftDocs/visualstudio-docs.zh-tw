# <a name="script_e_propagate-error-code"></a>SCRIPT_E_PROPAGATE 錯誤碼
腳本錯誤已傳播至呼叫端，可能在不同的執行緒中。 主機應該將錯誤碼傳遞給呼叫者。  
  
## <a name="syntax"></a>語法  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
