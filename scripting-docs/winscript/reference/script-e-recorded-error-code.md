# <a name="script_e_recorded-error-code"></a>SCRIPT_E_RECORDED 錯誤碼
腳本引擎與主機之間已傳遞錯誤。 主機必須將錯誤碼傳遞給呼叫者。  
  
## <a name="syntax"></a>語法  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
