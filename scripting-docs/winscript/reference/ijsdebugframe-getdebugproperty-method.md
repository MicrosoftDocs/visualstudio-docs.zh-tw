# <a name="ijsdebugframegetdebugproperty-method"></a>IJsDebugFrame::GetDebugProperty 方法
傳回此堆疊框架屬性瀏覽器。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetDebugProperty(  
   IJsDebugProperty **ppDebugProperty  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppDebugProperty`  
 [out]物件，表示屬性瀏覽器。  
  
## <a name="return-value"></a>傳回值
