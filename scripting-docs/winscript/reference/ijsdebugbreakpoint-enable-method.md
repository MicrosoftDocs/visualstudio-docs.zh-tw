# <a name="ijsdebugbreakpointenable-method"></a>IJsDebugBreakPoint::Enable 方法
啟用中斷點。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT Enable(void);  
```  
  
## <a name="return-value"></a>傳回值  
  
## <a name="remarks"></a>備註  
 如果在已刪除的中斷點上呼叫，則傳回 E_UNEXPECTED。  
  
## <a name="requirements"></a>需求  
 **標頭：** jscript9diag.h
