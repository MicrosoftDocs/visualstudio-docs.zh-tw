# <a name="idebugpropertygetparent"></a>IDebugProperty::GetParent
取得屬性的父屬性。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT GetParent (  
   IDebugProperty** ppParent  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppParent`  
 [out]傳回`IDebugProperty`介面，表示屬性的父代。  
  
## <a name="return-value"></a>傳回值  
 會傳回有效`HRESULT`，通常是`S_OK`。
