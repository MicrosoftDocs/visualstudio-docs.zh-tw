# <a name="iactivescriptgarbagecollector-interface"></a>IActiveScriptGarbageCollector 介面
提供一種方法來開始垃圾收集。 這個介面應該由想要清除其資源的動態指令碼引擎所執行。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|開始垃圾收集。|
