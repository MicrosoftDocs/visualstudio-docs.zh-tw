# <a name="ijsenumdebugpropertygetcount-method"></a>IJsEnumDebugProperty::GetCount 方法
取得物件的屬性數目。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pCount`  
 [out]屬性的數目。  
  
## <a name="return-value"></a>傳回值  
  
## <a name="requirements"></a>需求  
 **標頭：** jscript9diag.h
