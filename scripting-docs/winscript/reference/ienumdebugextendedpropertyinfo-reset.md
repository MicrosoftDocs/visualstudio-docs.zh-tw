# <a name="ienumdebugextendedpropertyinforeset"></a>IEnumDebugExtendedPropertyInfo::Reset
將列舉型別序列重設到開頭。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT Reset (  
void  
);  
```  
  
## <a name="return-value"></a>傳回值  
 會傳回有效`HRESULT`，通常是`S_OK`。
