# <a name="ivariantchangetype-interface"></a>IVariantChangeType 介面
可讓呼叫者從一種類型的 VARIANT 值，轉換為另一個。  
  
## <a name="methods"></a>方法  
 除了繼承自方法`IUnknown`，則`IVariantChangeType`介面會公開下列方法。  
  
|方法|描述|  
|------------|-----------------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|採用 variant 值，並建立新的變數與指定的型別。|
