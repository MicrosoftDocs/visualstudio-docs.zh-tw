# <a name="iremotedebugapplicationthreadex-interface"></a>IRemoteDebugApplicationThreadEx 介面
代表特定應用程式內執行的執行緒。  
  
 除了繼承自方法`IUnknown`，則`IRemoteDebugApplicationThreadEx`介面會公開下列方法。  
  
## <a name="methods-in-vtable-order"></a>依照 Vtable 順序的方法  
  
|方法|描述|  
|------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|列舉在此執行緒中執行的所有語言通用的運算式內容。|
