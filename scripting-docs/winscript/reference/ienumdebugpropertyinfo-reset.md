# <a name="ienumdebugpropertyinforeset"></a>IEnumDebugPropertyInfo::Reset
將列舉序列重設為開頭。  
  
## <a name="syntax"></a>語法  
  
```cpp
HRESULT Reset (  
void  
);  
```  
  
## <a name="return-value"></a>傳回值  
 傳回有效的 `HRESULT`，通常是 `S_OK`。
