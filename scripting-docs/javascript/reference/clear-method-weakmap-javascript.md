# <a name="clear-method-weakmap-javascript"></a>clear 方法 (WeakMap) (JavaScript)
將所有項目從 `WeakMap` 物件中移除。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
weakmapObj.clear()  
```  
  
#### <a name="parameters"></a>參數  
 `weakmapObj`  
 必要項。 `WeakMap`来清除物件。
