# <a name="mathasin-function-javascript"></a>Math.asin 函式 (JavaScript)
傳回一個數字的反正弦值。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>備註  
 所需`number`引數是數值運算式所需的反正弦值。  
  
 傳回值是的反正弦值`number`引數，以弧度為單位。
