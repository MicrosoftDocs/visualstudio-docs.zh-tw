# <a name="mathexp-function-javascript"></a>Math.exp 函式 (JavaScript)
傳回 e （自然對數的底數） 乘為乘冪。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>備註  
 所需`number`引數是數值運算式，代表的乘冪*e*。  
  
 傳回值為數字。 常數*e*是數字，大約等於 2.71828 euler 和`number`是提供的引數。
