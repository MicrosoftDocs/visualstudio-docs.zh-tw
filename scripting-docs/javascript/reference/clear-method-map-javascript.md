# <a name="clear-method-map-javascript"></a>clear 方法 (Map) (JavaScript)
從對應移除所有項目。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
mapObj.clear()  
```  
  
#### <a name="parameters"></a>參數  
 `mapObj`  
 必要項。 要清除的對應。
