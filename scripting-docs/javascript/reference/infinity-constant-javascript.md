# <a name="infinity-constant-javascript"></a>Infinity 常數 (JavaScript)
數字大於最大浮點數。 傳回初始值為`Number.POSITIVE_INFINITY`。 負的無限值 (-無限大) 小於最小的浮點數。  
  
## <a name="syntax"></a>語法  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>備註  
 `Infinity`常數是屬於`Global`物件，並初始化指令碼引擎時才可以提供。
