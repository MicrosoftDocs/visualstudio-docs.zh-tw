# <a name="tostring-method-map-javascript"></a>toString 方法 (Map) (JavaScript)
傳回對應的字串表示。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
mapObj.toString()  
```  
  
#### <a name="parameters"></a>參數  
 `mapObj`  
 必要項。 `Map` 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 對應的字串表示。
