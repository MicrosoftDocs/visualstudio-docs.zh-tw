# <a name="mathsqrt-function-javascript"></a>Math.sqrt 函式 (JavaScript)
傳回一個數字的平方根。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## <a name="remarks"></a>備註  
 所需要的 `number` 引數是數字運算式。  
  
 如果`number`是負數，則傳回值是`NaN`。
