# <a name="mathcbrt-function-javascript"></a>Math.cbrt 函式 (JavaScript)
傳回數字的立方根。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## <a name="remarks"></a>備註  
 如果 `number` 是 `NaN`，結果是 `NaN`。
