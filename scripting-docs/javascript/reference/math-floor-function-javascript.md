# <a name="mathfloor-function-javascript"></a>Math.floor 函式 (JavaScript)
傳回小於或等於其數字的引數的最大整數。  
  
## <a name="syntax"></a>語法  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>備註  
 所需*數目*引數是數值運算式。  
  
 傳回值會是小於或等於其數字的引數的整數值等於最大整數。
