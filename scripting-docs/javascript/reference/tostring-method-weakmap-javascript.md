# <a name="tostring-method-weakmap-javascript"></a>toString 方法 (WeakMap) (JavaScript)
傳回的字串表示`WeakMap`物件。  
  
## <a name="syntax"></a>語法  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### <a name="parameters"></a>參數  
 `weakmapObj`  
 必要項。 `WeakMap` 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 `WeakMap` 的字串表示。
