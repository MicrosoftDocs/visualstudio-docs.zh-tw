# <a name="expected--javascript"></a>必須是 ')' (JavaScript)
您嘗試將運算式括在一組括弧內，但未包含右括弧。 某些運算式必須括在一組左右括弧內。 請注意，在下列範例中使用括弧。  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在評估運算式中加入右括弧。
