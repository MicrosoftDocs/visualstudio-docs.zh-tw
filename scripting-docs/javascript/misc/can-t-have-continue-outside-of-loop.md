# <a name="cant-have-continue-outside-of-loop"></a>迴圈外不可以有 'continue'
您嘗試使用**繼續**陳述迴圈外的。 **繼續**陳述式只能用於下列項目： 主體  
  
-   `do-while` 迴圈中，  
  
-   `while` 迴圈中，  
  
-   **針對**迴圈中，  
  
-   **/ 在**迴圈。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定**繼續**陳述式出現在主體中，下列項目：  
  
    -   `do-while` 迴圈中，  
  
    -   `while` 迴圈中，  
  
    -   **針對**迴圈中，  
  
    -   **/ 在**迴圈。
