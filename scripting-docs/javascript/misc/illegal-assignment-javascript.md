# <a name="illegal-assignment-javascript"></a>無效的指派 (JavaScript)
您嘗試將值指派給唯讀識別碼。 您無法將值指派給唯讀識別碼。 例如，主機定義的物件和外部 COM 物件。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請勿將值指派給唯讀識別碼。
