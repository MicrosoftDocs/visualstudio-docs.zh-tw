# <a name="illegal-assignment-javascript"></a>無效的指派 (JavaScript)
您嘗試指派值給唯讀模式的識別項。 您無法將值指派為唯寫的識別項。 例如，主應用程式所定義的物件與外部 COM 物件。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   不要指派值給唯讀識別項。
