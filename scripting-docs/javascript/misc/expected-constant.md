# <a name="expected-constant"></a>必須是常數
您嘗試使用條件式編譯測試陳述式中的 （非條件式編譯） 變數。 條件式編譯測試陳述式必須評估為常數。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將變數取代常值。  
  
-   取代的條件式編譯變數中的變數。
