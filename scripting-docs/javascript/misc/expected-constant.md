# <a name="expected-constant"></a>必須是常數
您嘗試在條件式編譯測試語句中使用 (的非條件式編譯) 變數。 條件式編譯測試語句必須評估為常數。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 以常值取代變數。  
  
- 使用條件式編譯變數來取代變數。
