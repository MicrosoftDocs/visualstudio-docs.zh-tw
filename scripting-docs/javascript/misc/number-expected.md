# <a name="number-expected"></a>必須是數字
您嘗試在類型的物件上叫用**valueOf** **方法，但**不是**數位**的值。 這種調用類型的物件必須是**Number**類型。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 只會在**number**類型的物件上叫用**valueOf** **方法。**
