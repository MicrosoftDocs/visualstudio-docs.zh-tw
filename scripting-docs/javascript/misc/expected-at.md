# <a name="expected-"></a>必須是 '\@'
您嘗試建立一個變數來搭配使用的條件式編譯陳述式`@set`陳述式，但未將放置 at 符號 「**@**"變數的名稱前面。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 新增符號 」**@**"之前的變數名稱。 例如:   
  
    ```JavaScript  
    @set @myvar = 1  
    ```
