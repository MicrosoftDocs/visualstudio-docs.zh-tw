# <a name="expected-"></a>應為 '\@'
您嘗試使用語句來建立要搭配條件式編譯語句使用的變數 `@set` ，但未在變數名稱之前放置 @ 符號 **@** 。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在 **@** 變數名稱前面緊接著加上 @ 符號。 例如：  
  
    ```JavaScript  
    @set @myvar = 1  
    ```
