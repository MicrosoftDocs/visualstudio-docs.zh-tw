# <a name="undefined-identifier"></a>未定義的識別項
您嘗試使用 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 編譯器無法辨識的識別碼。 當您使用時，會傳回未定義的值：  
  
- 不存在的變數。  
  
- 已宣告但從未指派值給它的變數。  
  
- 不存在的物件屬性。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 以 (中的 **var** 語句宣告變數，如 `var` x; ) 。
