# <a name="undefined-identifier"></a>未定義的識別項
您嘗試使用識別項，[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]編譯器無法辨識。 每當您使用，則會傳回未定義的值：  
  
- 變數不存在，  
  
- 變數已宣告，但永遠不會有值，指派給它，  
  
- 物件屬性不存在。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用變數宣告**var**陳述式 (如`var`x;)。
