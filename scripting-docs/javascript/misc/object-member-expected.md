# <a name="object-member-expected"></a>必須是物件成員
您嘗試對物件的資料成員執行作業，但該成員不存在。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確定您已正確輸入資料成員的名稱。
