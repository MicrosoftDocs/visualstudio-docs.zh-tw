# <a name="object-member-expected"></a>必須是物件成員
您嘗試對物件的資料成員，執行作業，但沒有成員。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定您輸入正確的資料成員的名稱。
