# <a name="unterminated-comment"></a>未結束的註解
您已開始多行批註區塊，但未正確地將它終止。 多行批註的開頭為 "/*" 組合，結尾為反向的 " \* /" 組合。 以下是一個範例：  
  
```JavaScript  
/* This is a comment  
This is another part of the same comment.*/  
```  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請務必使用 "*/" 來終止多行批註。
