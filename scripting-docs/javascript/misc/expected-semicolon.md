# <a name="expected-"></a>必須是 ';'
您嘗試在一行上放置一個以上的語句，但未以分號分隔。 使用分號來終止語句。 雖然您可以將數個語句放在同一行，但每一個都必須以分號分隔。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 以分號標示每個語句的結尾。  
  
- 請確定您的函式呼叫正確地使用括弧。  
  
- 請確定您並未忘記**for**迴圈標頭內的分號。
