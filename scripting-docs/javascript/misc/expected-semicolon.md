# <a name="expected-"></a>必須是 ';'
您嘗試在一行上放置一個以上的語句，但是未以分號分隔。 分號可用來終止語句。 雖然您可以將數個語句放在同一行上，但每個語句都必須以分號分隔。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將每個語句的結尾標示為分號。  
  
- 請確定您的函式呼叫正確地使用括弧。  
  
- 請確定您未在 **for** 迴圈的標頭內忘記分號。
