# <a name="syntax-error-javascript"></a>語法錯誤 (JavaScript)
您建立的語句違反了一或多個 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 的語法規則。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在指示的行號上，再次檢查程式的語法。  
  
- 尋找有誤的括弧或大括弧。
