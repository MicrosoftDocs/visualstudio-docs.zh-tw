# <a name="invalid-number"></a>無效的數字
不正確數位 (ErrbadNumber 1013)   
  
 您嘗試使用的數位在 JavaScript 中無效。  
  
 如需有效數字的詳細資訊，請參閱 [Number 物件](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number)。
