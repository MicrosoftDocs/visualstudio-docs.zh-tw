# <a name="expected--javascript"></a>必須是 '(' (JavaScript)
您嘗試將運算式括在一組括弧內，但未包含左括弧。 某些運算式必須包含在一組左右括弧內。 請注意下列範例中的括弧用法。  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將左括弧新增至評估運算式。
