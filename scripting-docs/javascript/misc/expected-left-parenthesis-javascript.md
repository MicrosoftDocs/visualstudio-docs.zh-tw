# <a name="expected-3939-javascript"></a>必須是&#39;(&#39; (JavaScript)
您嘗試將括在括號、 一組運算式，但未包含左括號。 某些運算式必須使用一組的開頭和結尾括號括住。 請注意，在下列範例中的括號內使用。  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入評估運算式的左括號。
