# <a name="the-precision-is-out-of-range"></a>整數位數超過範圍
您嘗試將不正確引數傳遞至 **toPrecision**。 **ToPrecision**的引數必須介於1到 21 (內含) 。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定的引數 `toPrecision` 不太大或太小。
