# <a name="the-precision-is-out-of-range"></a>整數位數超過範圍
您嘗試將無效的引數傳遞至函式**Number.prototype.toPrecision**。 引數**toPrecision**必須介於 1 到 21 （含） 之間。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定引數`toPrecision`不太大或太小。
