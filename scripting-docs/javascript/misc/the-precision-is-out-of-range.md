# <a name="the-precision-is-out-of-range"></a>整數位數超過範圍
您嘗試將不正確引數傳遞至函數**號碼。 toPrecision**。 **ToPrecision**的引數必須介於1到21（含）之間。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定 `toPrecision` 的引數不是太大或太小。
