# <a name="expected-"></a>必須是 '{'
您未包含標示函式主體開頭的左大括弧。 組成函式主體的程式碼（即使它是單行），都必須包含在左括弧和右大括弧內。 請注意，在函式主體中使用大括弧比在迴圈中更嚴格。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 加入標示函式主體開頭的左大括弧。
