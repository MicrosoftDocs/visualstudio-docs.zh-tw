# <a name="expected-"></a>必須是 '{'
您未包含標記的函式主體開頭的左大括號。 程式碼，即使它是在同一行，讓函式的主體必須包含在左邊和右邊大括號。 請注意，函式主體中的括號的用法是比在迴圈中更嚴格。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 新增標記的函式主體開頭的左大括號。
