# <a name="label-redefined"></a>已重新定義標籤
您已建立新的標籤，但為其提供現有標籤的名稱。 標籤可以用來標示程式碼區塊，但在指定的範圍內，它們必須是唯一的。  
  
### <a name="to-correct-this-error"></a>若要改正這項錯誤  
  
- 請確定您在程式中使用的所有標籤在其各自的範圍內都是唯一的。
