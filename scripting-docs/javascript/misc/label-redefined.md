# <a name="label-redefined"></a>已重新定義標籤
您已建立新的標籤，但會指定現有標籤的名稱。 標籤可以用來標示程式碼區塊，但指定範圍內，它們必須是唯一。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定您在程式中使用的所有標籤都是其各自範圍內唯一的。
