# <a name="expected--javascript"></a>必須是 '=' (JavaScript)
您嘗試建立要搭配條件式編譯語句使用的變數，但未在變數與您要指派給它的值之間放置等號。  
  
### <a name="to-correct-this-error"></a>若要改正這項錯誤  
  
- 新增等號。 例如：  
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
