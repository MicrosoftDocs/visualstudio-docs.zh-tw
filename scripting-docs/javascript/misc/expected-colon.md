# <a name="expected-"></a>必須是 ':'
您嘗試使用三元條件運算子建立運算式，但未在第二個和第三個運算元之間包含冒號。 三元（三個運算元）條件運算子需要第二個（true）和第三個（false）運算元之間的冒號。  
  
### <a name="to-correct-this-error"></a>若要改正這項錯誤  
  
- 在第二個和第三個運算元之間插入冒號。
