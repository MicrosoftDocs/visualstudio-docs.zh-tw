# <a name="expected-"></a>必須是 ':'
您嘗試建立使用三元條件式運算子的運算式，但不是包括第二個和第三個運算元之間的冒號。 三元 （三個運算元） 條件式運算子要求第二個 (true) 與第三個 (false) 之間的分號運算元。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   插入第二個和第三個運算元之間的分號。
