# <a name="expected-while"></a>必須是 'while'
您未包含`while`狀況`do...while`迴圈。 A**請勿**陳述式必須有對應`while`測試的程式碼區塊的結尾。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   包含`while`測試右大括號之後的陳述式。
