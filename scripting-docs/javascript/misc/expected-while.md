# <a name="expected-while"></a>必須是 'while'
您未在 `do...while` 迴圈中包含 `while` 條件。 **Do**語句在程式碼區塊的結尾必須有對應的 `while` 測試。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在右大括弧後面包含 `while` 測試語句。
