# <a name="expected-while"></a>必須是 'while'
您未 `while` 在迴圈中包含條件 `do...while` 。 **Do** 語句在程式 `while` 代碼區塊的結尾必須有對應的測試。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在 `while` 右大括弧之後包含測試語句。
