# <a name="array-or-arguments-object-expected"></a>必須是陣列或引數物件
您未提供陣列做為引數。 這個錯誤適用于函式 **. 原型。僅適用于 apply**方法。 若已指定，則此函式的第二個引數必須是 `Array` 物件或**引數**物件。  
  
### <a name="to-correct-this-error"></a>若要改正這項錯誤  
  
- 請指定 `Array` 或**引數**物件做為第二個引數。
