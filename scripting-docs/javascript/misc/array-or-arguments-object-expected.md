# <a name="array-or-arguments-object-expected"></a>必須是陣列或引數物件
您並未提供做為引數的陣列。 此錯誤適用於**Function.prototype.apply**僅方法。 如果指定，此函式的第二個引數必須是`Array`物件或**引數**物件。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 指定`Array`該**引數**做為第二個引數的物件。
