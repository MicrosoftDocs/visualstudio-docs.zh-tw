# <a name="expected-"></a>必須是 '}'
您未包含可標示函式主體、迴圈、程式碼區塊或物件初始化運算式結尾的右大括弧。 此錯誤的其中一個範例是 for 迴圈，只使用左大括弧 **來** 標記迴圈的主體。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 加入標示函式、迴圈、區塊或物件初始化運算式結尾的右括弧。
