# <a name="expected-"></a>必須是 '}'
您未包含標記的函式主體、 迴圈、 程式碼區塊結尾的右大括號或物件初始設定式。 此錯誤的範例**針對**迴圈只左大括號標記迴圈的主體。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   新增標記結束函式、 迴圈、 區塊或物件初始設定式的右大括號。
