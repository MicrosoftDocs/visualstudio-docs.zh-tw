# <a name="regular-expression-object-expected"></a>必須是規則運算式物件
您嘗試在的型別的物件上叫用 **valueOf** **方法，而** 不是 `RegExp` 。 這種調用類型的物件必須是型別 `RegExp` 。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 只有在類型的物件上叫用 **RegExp** 或 **valueOf** `RegExp` 方法。
