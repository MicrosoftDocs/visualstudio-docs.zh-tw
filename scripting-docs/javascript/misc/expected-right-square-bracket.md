# <a name="expected-"></a>必須是 ']'
您已建立陣列元素的參考，但未包含右括弧。 參考陣列元素的任何運算式都必須同時包含左右括弧。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將右括弧新增至參考陣列元素的運算式。
