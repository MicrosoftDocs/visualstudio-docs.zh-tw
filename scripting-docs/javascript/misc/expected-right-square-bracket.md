# <a name="expected-"></a>必須是 ']'
您進行陣列元素的參考，但並未包含右方括號。 參考之陣列元素的任何運算式必須包含左括號和右中括號。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入參考陣列項目運算式的右括號。
