# <a name="string-expected"></a>必須是字串
您嘗試在 `String` 以外之類型的物件上叫用**valueOf** **方法，而不是。** 這種調用類型的物件必須是 `String` 的類型。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 只會在 `String` 類型的物件上叫用**valueOf** **方法。**
