# <a name="string-expected"></a>必須是字串
您嘗試叫用**String.prototype.toString**或是**String.prototype.valueOf**以外的類型的物件上的方法`String`。 這種類型的引動過程的物件必須是型別`String`。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   只能叫用**String.prototype.toString**或是**String.prototype.valueOf**類型的物件上的方法`String`。
