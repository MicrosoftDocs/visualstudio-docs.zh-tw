# <a name="string-expected"></a>必須是字串
您嘗試在的型別物件上叫用**valueOf** **方法，而**不是 `String` 。 這種調用類型的物件必須是型別 `String` 。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 只有在類型的物件上叫用**valueOf** **方法，才**會叫用。 `String`
