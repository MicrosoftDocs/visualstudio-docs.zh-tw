# <a name="expected-var"></a>必須是 'var'
預期的 ' var ' (ERRnoVar 1034) )   
  
 必要時，您無法新增關鍵字 var。  
  
 如需 var 關鍵字的詳細資訊，請參閱 [Var 語句](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Statements/var)。
