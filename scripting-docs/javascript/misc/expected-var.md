# <a name="expected-var"></a>必須是 'var'
必須是 ' var ' （ERRnoVar 1034））  
  
 您無法在需要時新增關鍵字 var。  
  
 如需 var 關鍵字的詳細資訊，請參閱[Var 語句](../../javascript/reference/var-statement-javascript.md)。
