# <a name="the-use-of-a-keyword-for-an-identifier-is-invalid"></a>針對識別項使用關鍵字是無效的動作
1048：識別碼的關鍵字使用無效  
  
 您不能使用關鍵字做為變數、函數或其他使用者定義識別碼的名稱。  
  
 如需關鍵字的詳細資訊，請參閱[JAVAscript 保留字](../../javascript/reference/javascript-reserved-words.md)和[JAVAscript 未來的保留字](../../javascript/reference/javascript-future-reserved-words.md)。
