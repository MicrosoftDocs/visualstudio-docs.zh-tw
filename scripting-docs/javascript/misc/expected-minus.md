# <a name="expected--"></a>應為 '-'
您嘗試建立正則運算式常值，但未包含其中一個斜線 (/) 。 就像字串常值是以字元的形式寫入成對的引號中，正則運算式常值是以一組斜線內的字元表示 (/) 個字元。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 插入終止正斜線以標示正則運算式的結尾。
