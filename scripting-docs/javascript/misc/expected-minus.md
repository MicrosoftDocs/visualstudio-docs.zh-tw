# <a name="expected--"></a>必須是 '-'
您嘗試建立正則運算式常值，但未包含其中一個斜線（/）。 就像字串常值是以字元的形式寫入一組引號內，正則運算式常值會以一對斜線（/）字元內的字元來表示。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 插入終止的正斜線，以標示正則運算式的結尾。
