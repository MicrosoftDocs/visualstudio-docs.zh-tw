# <a name="expected-39-39"></a>必須是&#39;-&#39;
您嘗試建立的規則運算式常值，但不是包含斜線 （/） 的其中一個。 就像字串常值會寫入為一對引號內的字元，規則運算式常值會表示為斜線 （/） 字元的一組內的字元。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   插入標記的規則運算式的結尾終止正斜線。
