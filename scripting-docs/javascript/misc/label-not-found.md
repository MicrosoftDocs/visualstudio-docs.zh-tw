# <a name="label-not-found"></a>找不到標籤
您已對不存在的標籤進行參考。 標籤可以用來標示程式碼區塊，但在指定的範圍內，必須是唯一的。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查並確定您未拼錯標籤名稱。  
  
- 請確定已在目前的範圍中定義的標籤（包括轉寄定義），都參考了所有的 label。
