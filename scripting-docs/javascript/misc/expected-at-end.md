# <a name="expected-end"></a>應為 ' \@ end '
您嘗試建立有條件的編譯器代碼區塊，但不包含結尾的 <strong>@end</strong> 語句。 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 您可以透過將語句封入區塊內的方式，以條件方式編譯語句 <strong>@if/@end</strong> 。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 加入對應的 <strong>@end</strong> 語句。
