# <a name="the-number-of-fractional-digits-is-out-of-range"></a>小數點後數字的數目超過範圍
您嘗試將不正確引數傳遞至 **toExponential**。 函數 **toExponential ()** 的引數必須介於0和 20 (包含) 之間。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定 **toExponential ()** 的引數不太大或太小。
