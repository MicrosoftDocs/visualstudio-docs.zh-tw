# <a name="the-number-of-fractional-digits-is-out-of-range"></a>小數點後數字的數目超過範圍
您嘗試將不正確引數傳遞至 **toExponential**。 函式 **toExponential ( # B1 ** 的引數必須介於0到 20 (包含) 。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定 **toExponential ( # B1 ** 的引數不太大或太小。
