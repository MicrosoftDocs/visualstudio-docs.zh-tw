# <a name="expected-identifier-javascript"></a>必須是識別項 (JavaScript)
您在需要其中一個專案的內容中使用了除了識別碼以外的其他專案。 識別碼可以是：  
  
- 變數、  
  
- 屬性，  
  
- 陣列、  
  
- 或函數名稱。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 變更運算式，讓識別碼出現在等號的左邊。
