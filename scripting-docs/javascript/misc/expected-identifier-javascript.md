# <a name="expected-identifier-javascript"></a>必須是識別項 (JavaScript)
您以外的識別項的內容中使用其中一個是必要。 識別項可以：  
  
- 變數，  
  
- 屬性，  
  
- 陣列，  
  
- 或函式名稱。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 因此識別項的等號左邊出現，請變更運算式。
