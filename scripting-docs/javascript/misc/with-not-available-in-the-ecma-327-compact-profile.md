# <a name="with-not-available-in-the-ecma-327-compact-profile"></a>'with' 在 ECMA 327 Compact Profile 的設定檔中無法使用
ECMA 327 Compact Profile 中無法使用 ' with ' （ErrWithNotInCP  
  
 您不能在 Compact 設定檔中使用 with 關鍵字。  
  
 如需 with 關鍵字的詳細資訊，請參閱[With 語句](../../javascript/reference/with-statement-javascript.md)。
