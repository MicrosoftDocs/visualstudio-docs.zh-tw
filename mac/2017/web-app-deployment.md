# <a name="aspnet-core-web-app-deployment"></a>ASP.NET Core Web 應用程式開發

發佈 (部署) 是開發 Web 應用程式的一個重要步驟。 您可以發佈到遠端環境，讓其他人可以使用該應用程式或進行測試。 Visual Studio for Mac 提供兩個選項來發佈您的應用程式。 [發佈到 Azure App Service] 可讓您將 Web 應用程式發佈到 Azure App Service。 [發佈到資料夾] 可讓您將應用程式發佈到資料夾。

## <a name="publish-to-azure-app-service"></a>發佈至 Azure App Service

使用 Visual Studio for Mac 時，您可以將 Web 應用程式直接發佈到 Azure App Service。 若要將您的 ASP.NET Core Web 應用程式發佈到 Azure App Service，請參閱[發佈到 Azure App Service](publish-app-svc.md) 一文。

## <a name="publish-to-folder"></a>發佈到資料夾

您也可以將 ASP.NET Core Web 應用程式發佈到資料夾。 若要深入了解如何將 Web 應用程式發佈到資料夾，請參閱[發佈到資料夾](publish-folder.md)。
