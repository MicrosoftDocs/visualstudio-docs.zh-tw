# <a name="update-visual-studio-2017-for-mac-version-7-to-visual-studio-2019-for-mac-version-80"></a>將 Visual Studio 2017 for Mac (7 版) 更新為 Visual Studio 2019 for Mac (8.0 版)

Visual Studio 2019 for Mac 有許多新改善：全新 C# 編輯器、全新啟動體驗、支援多個執行個體，以及適用於 Xamarin、Unity 和 .NET Core 開發人員的多個改善。 如需 Visual Studio for Mac 2019 中所有項目的詳細資訊，請參閱下列部落格文章：

- [Visual Studio 2019 for Mac 現已推出](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-is-now-available/)
- [Visual Studio 2019 for Mac 8.1 版現已推出 (以及 8.2 的 Preview)](https://devblogs.microsoft.com/visualstudio/visual-studio-2019-for-mac-version-8-1-is-now-available-and-a-preview-for-8-2/)

> [!IMPORTANT]
> Visual Studio 2017 for Mac **7.7.3 版**有已知問題，更新程式會持續顯示 [擷取更新資訊時發生錯誤]。 如果發生此問題，請[下載 Visual Studio for Mac](https://visualstudio.microsoft.com/vs/mac/)，並遵循步驟以[安裝](./installation.md?view=vsmac-2019) Visual Studio 2019 for Mac。 請注意，這會覆寫您的現有 Visual Studio 2017 for Mac 執行個體。

若要取得最新版本，您可以瀏覽至 [Visual Studio] > [檢查更新]****，以使用[更新程式](./update.md?view=vsmac-2017)：

![更新程式對話方塊](media/update-vsmac-updater.png)

或者，如果您想使用新的 2019 安裝程式，或更新程式有問題，請[下載](https://visualstudio.microsoft.com/vs/mac/)新的 2019 安裝程式來安裝並快速執行：

![安裝畫面](media/update-vsmac-installer.png)
