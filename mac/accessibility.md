# <a name="accessibility"></a>協助工具選項

MacOS 內建了許多協助工具工具和功能，可協助您使用與 Visual Studio for Mac。  此外，Visual Studio for Mac 也具有下列功能，可讓不同能力的人員更容易存取：

* 編輯器文字放大
* 填補文字放大
* 程式碼編輯器色彩主題自訂
* 鍵盤快速鍵自訂
* 鍵盤瀏覽

> [!NOTE]
> 本指南是指 Visual Studio for Mac。 如需 Windows 上的 Visual Studio，請參閱[Visual Studio 的協助工具功能](/visualstudio/ide/reference/accessibility-features-of-visual-studio)。
