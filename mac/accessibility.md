# <a name="accessibility"></a>Accessibility

macOS 中內置了許多協助工具工具和功能，可説明您使用適用于 Mac 的 Visual Studio。  除此之外，Mac 的 Visual Studio 還具有以下功能，使不同能力的人能夠更容易訪問：

* 編輯器文本放大
* 墊文本放大
* 代碼編輯器顏色主題自訂
* 鍵盤快速鍵自訂
* 鍵盤導覽

> [!NOTE]
> 本指南參考了 Mac 的視覺工作室。 有關 Windows 上的視覺化工作室，請參閱[視覺化工作室的協助工具](/visualstudio/ide/reference/accessibility-features-of-visual-studio)。
