# <a name="accessibility-features-of-macos"></a>MacOS 的協助工具功能

macOS 是可存取的作業系統。 它有許多內建的功能，可協助使用者進行不同的能力。 這些功能包括高對比模式、鍵盤導覽和 VoiceOver;macOS 螢幕閱讀程式。

## <a name="enabling-accessibility-features"></a>啟用協助工具功能

預設會關閉 Visual Studio for Mac 的輔助技術支援。 若要啟用協助工具支援，請遵循下列步驟：

1. 移至 [Visual Studio] (功能表) > [偏好設定] > [其他] > [協助工具]

1. 檢查 [啟用協助工具] 核取方塊：

   ![[協助工具偏好設定] 核取方塊](media/accessibility-preferences.png)

1. 選取 [重新啟動 Visual Studio] 按鈕以重新啟動 Visual Studio，並啟用對 Apple 輔助技術的支援。

## <a name="increasing-the-contrast-in-macos"></a>增加 macOS 的對比

Visual Studio for Mac 支援增加 macOS 的對比、增加 UI 元素的對比，以及讓大綱更明確。 若要啟用此動作，請遵循下列步驟：

1. 開啟系統喜好設定

1. 流覽至 [**協助工具**]，然後選取 [**顯示**]

1. 選取 [**增加對比**] 核取方塊
