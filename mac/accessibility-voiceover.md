# <a name="using-voiceover-with-visual-studio-for-mac"></a>使用 VoiceOver 搭配 Visual Studio for Mac

需要螢幕讀取器的使用者將受益于 macOS VoiceOver。 Visual Studio for Mac 可使用一般 VoiceOver 命令來存取。

## <a name="how-to-enable-and-use-voiceover"></a>如何：啟用和使用 VoiceOver

若要啟用或停用 VoiceOver，請按 **&#8984;F5**

VoiceOver 命令在本指南中會顯示為 **vo 的 _金鑰_** ，而 **vo** 會參考 **VoiceOver 公用** 程式應用程式中所設定的修飾詞。 預設修飾詞為 **⌃⌥**。 例如，視您的 VoiceOver 修飾詞而定， **VO M** 將表示 **⌃⌥ M**。 資料指標索引鍵會被稱為 **←→↑↓**。

## <a name="useful-voiceover-commands"></a>實用的 VoiceOver 命令

以下是 Visual Studio for Mac 的一些常見 VoiceOver 命令的清單：

|功能|描述|快速鍵|
|-------|-----------|--------|
|Navigate|流覽 UI 的所有元素|**VO →**， **VO ←**|
|Navigate|移入或移出 UI 元素|**VO ⇧↓**， **VO ⇧↑**|
|互動|選取焦點的控制項或與其互動|**VO ␣**|
|互動|與 Visual Studio for Mac 的功能表列互動|**VO M**|

> [!TIP]
> 如需使用 VoiceOver 和完整命令清單的詳細資訊，請參閱 [Apple 的 VoiceOver 消費者入門指南](https://support.apple.com/en-us/guide/voiceover-guide/welcome/web)。
