# <a name="using-voiceover-with-visual-studio-for-mac"></a>將語音與 Mac 視覺工作室一起使用

需要螢幕閱讀器的使用者將受益于 macOS 畫中音。 使用常見的 VoiceOver 命令可以訪問 Mac 的視覺化工作室。

## <a name="how-to-enable-and-use-voiceover"></a>如何：啟用和使用語音覆蓋

要啟用或禁用畫外音 **&#8984;F5**

VoiceOver 命令在本指南中顯示為**VO_鍵_**，**其中 VO**引用**VoiceOver 實用程式**應用程式中設置的修改器。 預設修改器為 **。**。 例如，根據您的 VoiceOver 修飾符 **，VO M**表示 **#M**。 方向鍵將稱為 [ **] 。**

## <a name="useful-voiceover-commands"></a>有用的畫中音命令

以下是適用于 Mac 的視覺化工作室的一些常見 VoiceOver 命令的清單：

|功能|描述|快速鍵|
|-------|-----------|--------|
|Navigate|導航 UI 的所有元素|**VO+**， **VO+**|
|Navigate|步進或退出 UI 元素|**VO =** **， VO #**|
|互動|選擇控制項或與焦點控制項交互|**VO**|
|互動|與 Mac 功能表列的視覺化工作室互動|**VO M**|

> [!TIP]
> 有關使用 VoiceOver 和命令的完整清單的更多資訊，請參閱[Apple 的"語音覆蓋入門指南](https://support.apple.com/en-us/guide/voiceover-guide/welcome/web)"。
