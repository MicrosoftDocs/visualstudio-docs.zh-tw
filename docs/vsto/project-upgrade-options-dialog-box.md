# <a name="project-upgrade-options-dialog-box"></a>專案升級、 選項對話方塊
  您可以使用 [專案升級]  頁面，指定 Visual Studio 是否會自動變更 VSTO 增益集專案，將其目標設定為開發電腦上所安裝的 Office 版本。  
  
 若要存取這個頁面，請在 [工具]  功能表上，按一下 [選項] 。 展開 [Office 工具]  資料夾，然後按一下 [專案升級] 。  
  
## <a name="uielement-list"></a>UIElement 清單  
 **永遠升級為已安裝的 Office 版本**  
 由於 [!INCLUDE[vs_dev12](../vsto/includes/vs-dev12-md.md)] 中的 Office 專案只能以 [!INCLUDE[vs_dev12](../vsto/includes/vs-dev12-md.md)] 或 Microsoft [!INCLUDE[office14_long](../vsto/includes/office14-long-md.md)] 應用程式為目標，因此會在 [!INCLUDE[Office_15_short](../vsto/includes/office-15-short-md.md)] 中停用這個選項。 如需詳細資訊，請參閱 <<c0> [ 升級和移轉 Office 方案](../vsto/upgrading-and-migrating-office-solutions.md)。
