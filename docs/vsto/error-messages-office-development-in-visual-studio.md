# <a name="error-messages-office-development-in-visual-studio"></a>Visual Studio) 中 (Office 開發的錯誤訊息
  每個錯誤訊息主題都涵蓋一個訊息，並提供可能產生錯誤的原因和可行的解決方式或替代方式來解決問題。

 您可使用以下方式來取得錯誤訊息的資訊：

|方法|描述|
|-|-|
|F1|如果您收到錯誤，可以按下 **F1**。 這會開啟線上文件中對應的錯誤訊息主題。|
|全文檢索搜尋|您可使用錯誤訊息文字第一行中的關鍵字來進行全文檢索搜尋。 在 [說明] 功能表上，按一下 [搜尋]。|
