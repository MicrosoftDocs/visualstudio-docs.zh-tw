# <a name="microsoft-office-is-not-installed-on-the-computer"></a>電腦上未安裝 Microsoft Office
  Visual Studio 找不到對應至這種專案類型的 Microsoft Office 應用程式。 如果您要開發 VSTO 增益集專案，這表示您無法進行此專案的偵錯工具或執行。 如果您嘗試建立文件層級專案，這表示您無法建立專案。

## <a name="to-correct-the-error"></a>更正這個錯誤

1. 安裝與專案類型相關聯的 Microsoft Office 應用程式，包括主要 interop 元件。 如需 Microsoft Office 支援版本的詳細資訊，請參閱 [設定電腦以開發 Office 方案](../vsto/configuring-a-computer-to-develop-office-solutions.md)。
