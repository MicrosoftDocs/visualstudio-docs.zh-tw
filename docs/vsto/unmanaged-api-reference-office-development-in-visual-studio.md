# <a name="unmanaged-api-reference-office-development-in-visual-studio"></a>在 Visual Studio) 中 (Office 開發的非受控 API 參考

從 2007 Microsoft Office 系統開始，Office 應用程式會使用 [IManagedAddin 介面](../vsto/imanagedaddin-interface.md) 介面呼叫隨附的 VSTO 增益集載入器元件 [!INCLUDE[vsto_runtime](../vsto/includes/vsto-runtime-md.md)] 。 這個元件可用來協助載入管理的 VSTO 增益集。您可以藉由執行此介面來建立您自己的 VSTO 增益集載入器元件。
