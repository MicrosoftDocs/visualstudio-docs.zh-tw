# <a name="how-to-programmatically-group-rows-in-a-worksheet"></a>HOW TO：以程式設計方式在工作表中的群組資料列
  您可以群組一個或多個完整的資料列。 若要建立群組時的工作表中，使用<xref:Microsoft.Office.Tools.Excel.NamedRange>控制項或原生的 Excel 範圍物件。
