# <a name="c28022"></a>C28022
警告 C28022：此函式上的函式類別 \<classlist1 > 與用來定義它的 typedef 上的函式類別（es） \<classlist2 > 不相符。

 當注釋中有錯誤時，就會回報此警告。 Typedef 和函式本身都有 `_Function_class_` 注釋，但不符合。 如果兩者都使用，則兩者必須相符。
