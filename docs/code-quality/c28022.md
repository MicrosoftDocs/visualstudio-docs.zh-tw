# <a name="c28022"></a>C28022
警告 C28022:函式類別\<classlist1 > 這個函式上不相符的函式類別\<classlist2 > 來定義它所用之 typedef 上。

 註釋中有錯誤時，會報告這個警告。 Typedef 和函式本身都有`_Function_class_`註解，但它們不相符。 如果使用它們必須相符。
