# <a name="c28022"></a>C28022
警告 C28022：這個函式的 @no__t 0classlist1 > 函數類別，與用來定義它的 typedef 上的函式類別 @no__t 1classlist2 > 不相符。

 當注釋中有錯誤時，就會回報此警告。 Typedef 和函式本身都有 @no__t 0 注釋，但不符合。 如果兩者都使用，則兩者必須相符。
