# <a name="warning-c26453-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>警告 C26453:算術溢位：負值帶正負號數字的左的移位是未定義的行為

這則警告表示我們會保留帶正負號整數類資料將負值，這是個好主意，觸發程序的實作定義行為。

## <a name="example"></a>範例

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

 若要更正這個警告，請使用下列程式碼：

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
