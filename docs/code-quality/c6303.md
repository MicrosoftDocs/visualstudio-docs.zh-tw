# <a name="c6303"></a>C6303
警告 C6303：格式字串不符：以參數形式傳遞的寬字元字串 \<數位 > 在呼叫 \<函式時需要字元字串 >

 此警告表示格式字串指定需要字元字串。 不過，會傳遞寬字元字串。 這項缺陷可能會導致某種形式的當機或損毀。

## <a name="example"></a>範例
 下列範例程式碼會產生這個警告：

```cpp
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%s", buff);
}
```

 若要更正這個警告，請使用 `%ls`，如下列範例程式碼所示：

```cpp
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%ls", buff);
}
```

 下列範例程式碼使用安全字串操作函式 `printf_s` 來更正此警告：

```cpp
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf_s("%ls",buff);
}
```
