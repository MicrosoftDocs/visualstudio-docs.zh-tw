# <a name="c6282"></a>C6282
警告 C6282：不正確的運算子：在布林內容中指派常數。 請考慮改為使用 ' = = '

 此警告表示在測試內容中偵測到變數的常數指派。 將常數指派給測試內容中的變數，幾乎都是不正確的。 請以 `==`取代 `=`，或從測試內容中移除指派，以解決這個警告。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```cpp
void f( int i )
{
   while (i = 5)
   {
   // code
   }
}
```

 若要修正這個警告，請使用下列程式碼：

```cpp
void f( int i )
{
   while (i == 5)
   {
   // code
   }
}
```
