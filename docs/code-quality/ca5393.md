# <a name="ca5393-do-not-use-unsafe-dllimportsearchpath-value"></a>CA5393：請勿使用不安全的 DllImportSearchPath 值

|Item|值|
|-|-|
|CheckId|CA5393|
|類別|Microsoft.Security|
|重大變更|非中斷|

## <a name="cause"></a>原因

使用其中一個不安全的值 <xref:System.Runtime.InteropServices.DllImportSearchPath?displayProperty=fullName> ：
- `AssemblyDirectory`
- `UseDllDirectoryForDependencies`
- `ApplicationDirectory`

## <a name="rule-description"></a>規則描述

預設 DLL 搜尋目錄和元件目錄中可能有惡意的 DLL。 或者，根據執行應用程式的位置而定，應用程式的目錄中可能會有惡意的 DLL。

如需詳細資訊，請參閱 [安全載入程式庫](https://msrc-blog.microsoft.com/2014/05/13/load-library-safely/)。

## <a name="how-to-fix-violations"></a>如何修正違規

使用的安全值 <xref:System.Runtime.InteropServices.DllImportSearchPath> 可改為指定明確搜尋路徑：
- `LegacyBehavior`
- `SafeDirectories`
- `System32`
- `UserDirectories`

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

如果有下列情況，可以安全地隱藏此規則：
- 您確定已載入的元件是您要的。
- 匯入的元件是常用的系統元件，例如 user32.dll，且搜尋路徑策略遵循已知的 [dll 機制](https://blogs.msdn.microsoft.com/larryosterman/2004/07/19/what-are-known-dlls-anyway/)。

## <a name="configurability"></a>性

如果您是從 [FxCop 分析器](install-fxcop-analyzers.md) 執行此規則 (而不是使用舊版分析) ，則可設定此規則。

### <a name="unsafe-search-path"></a>Unsafe 搜尋路徑

您可以將哪一個值設定為不 <xref:System.Runtime.InteropServices.DllImportSearchPath> 安全的分析。 例如，若要指定程式碼不應使用 `AssemblyDirectory` `UseDllDirectoryForDependencies` 或 `ApplicationDirectory` ，請將下列機碼值組新增至專案中的 editorconfig 檔案：

```ini
dotnet_code_quality.CA5393.unsafe_DllImportSearchPath_bits = 770
```

您應該指定列舉值的位元組合的整數值。

## <a name="pseudo-code-examples"></a>虛擬程式碼範例

```csharp
using System;
using System.Runtime.InteropServices;

class ExampleClass
{
    [DllImport("The3rdAssembly.dll")]
    [DefaultDllImportSearchPaths(DllImportSearchPath.AssemblyDirectory)]
    public static extern int MessageBox(IntPtr hWnd, String text, String caption, uint type);

    public void ExampleMethod()
    {
        MessageBox(new IntPtr(0), "Hello World!", "Hello Dialog", 0);
    }
}
```

### <a name="solution"></a>解決方法

```csharp
using System;
using System.Runtime.InteropServices;

class ExampleClass
{
    [DllImport("The3rdAssembly.dll")]
    [DefaultDllImportSearchPaths(DllImportSearchPath.UserDirectories)]
    public static extern int MessageBox(IntPtr hWnd, String text, String caption, uint type);

    public void ExampleMethod()
    {
        MessageBox(new IntPtr(0), "Hello World!", "Hello Dialog", 0);
    }
}
```
