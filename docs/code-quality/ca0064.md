# <a name="ca0064"></a>CA0064

未執行任何分析，因為指定的規則集未包含任何 FxCop 規則。

在下列其中一種情況下，可能會發生此警告：

- 如果您同時遇到此警告與 CA0063 警告，載入規則集檔案時發生問題。 如需詳細資訊，請參閱[CA0063](ca0063.md)一文。

- 如果您在使用 CA0062 警告時遇到此錯誤，則程式碼分析可能找不到包含規則集所指定之規則的元件或元件。 如需詳細資訊，請參閱[CA0062](ca0062.md)一文。

- 否則，當您的規則集是空的，或已停用子規則集中啟用的所有規則時，通常就會發生此警告。 使用 Visual Studio[規則集編輯器](../code-quality/working-in-the-code-analysis-rule-set-editor.md)來啟用規則集中的某些規則。
