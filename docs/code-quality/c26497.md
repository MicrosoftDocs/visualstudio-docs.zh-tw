# <a name="c26497-use_constexpr_for_function"></a>C26497 USE_CONSTEXPR_FOR_FUNCTION

如果需要編譯時間評估，可以將這個函數% function% 標記 `constexpr`。 請參閱[ C++核心指導方針 f. 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rf-constexpr)。
