# <a name="c6296"></a>C6296
警告 C6296: 定義錯誤的 for 迴圈: 僅執行迴圈主體一次

 此警告表示 for 迴圈可能無法如預期運作。 當索引不帶正負號，且迴圈從零向下計數時，其主體只會執行一次。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```cpp
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i--)
   {
      // code ...
   }
}
```

 若要修正這個警告，請使用下列程式碼：

```cpp
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i++)
   {
      // code ...
   }
}
```
