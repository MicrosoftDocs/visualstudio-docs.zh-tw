# <a name="ca0063"></a>CA0063

無法載入規則集檔案或它的其中一個相依規則集檔案。

找不到指定的規則集，或找不到您的規則集所包含的其中一個規則集檔案。 請確定您的規則集內含的所有規則集都存在於磁片上，而且已透過 MSBuild 的**CodeAnalysisRuleSetDirectories**屬性在專案中指定適當的規則集目錄。

若要進行錯誤的分析，請在文字編輯器中檢查您的規則集檔案：

1. 若要尋找規則集檔案的路徑，請以滑鼠右鍵按一下**方案總管**中的專案，按一下 [**屬性**]，然後按一下 [程式**代碼分析**]。 請確定已選取規則集檔案作為作用中的規則集。 規則集的路徑會列在 [描述] 欄位中。

2. 在文字編輯器中開啟規則集檔案。

3. 檢查所有**Include**元素中的**Path**屬性值，並確認它們是否有效。 Include 路徑可以使用環境變數、絕對路徑和相對路徑（相對於目前的規則集檔案）。 例如:

   ```xml
   <Include Path="%PUBLIC%\Documents\RuleSets\alldesign.ruleset" Action="Default" />
   <Include Path="..\alldesign.ruleset" Action="Default" />
   <Include Path="C:\Rulesets\alldesign.ruleset" Action="Default" />
   ```

在某些情況下，規則集的位置可以相依于 MSBuild 屬性。 無法從規則集參考 MSBuild 屬性。 若要解決此問題，請在 MSBuild 的**CodeAnalysisRuleSetDirectories**屬性中指定其他搜尋路徑。 在此案例中，請只在**Include**元素的**Path**屬性中，指定規則集的名稱。
