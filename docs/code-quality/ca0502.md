# <a name="ca0502"></a>CA0502
不支援的語言

當使用者嘗試分析的 ASP.NET 專案包含一或多個以C#或 Visual Basic 以外的語言撰寫的頁面或類別時，就會發生這個錯誤。 若要修正這個錯誤，請從專案中移除該檔案，或停用程式碼分析。
