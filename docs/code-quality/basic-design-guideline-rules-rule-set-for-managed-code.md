# <a name="basic-design-guideline-rules-rule-set-for-managed-code"></a>適用於 Managed 程式碼的基本設計方針規則規則集

您可以使用 Microsoft 基本設計指導方針規則規則集，專注于讓您的程式碼更容易瞭解和使用。 如果您的專案包含程式庫程式碼，或如果您想要針對容易維護的程式碼強制執行最佳作法，則應該包含此規則集。
