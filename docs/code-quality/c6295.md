# <a name="c6295"></a>C6295
警告 C6295：For 迴圈定義錯誤： \<variable > 值的範圍是「最小值」到「最大」。 無限期地執行迴圈

 此警告表示 for 迴圈可能無法如預期運作。 For 迴圈會針對零（0）測試不帶正負號的值，並 > =。 結果一律為 true，因此迴圈是無限的。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
void f( )
{
  for (unsigned int i = 100; i >= 0; i--)
  {
    // code ...
  }
}
```

 若要修正這個警告，請使用下列程式碼：

```
void f( )
{
   for (unsigned int i = 100; i > 0; i--)
   {
      // code ...
   }
}
```
