# <a name="c28283"></a>C28283
警告 C28283：若為符號，則尚未支援指定的大小規格

 此警告表示大小規格 "sentinel" 注釋收到零以外的值。 基本上，呼叫者嘗試指出字串是由二進位零以外的字元所終止。
