# <a name="c28302"></a>C28302
警告 C28302：針對C++參考參數 < parameter_name >，在 @no__t 2annotation > 上找到額外的 @no__t 1 運算子。

 當參數上使用額外層級的 `_Deref_` 時，就會回報此警告。

 處理參考參數時，SAL2 不需要使用額外的 `_Deref_` 層級。 這個特定的注釋是明確的，且會正確地解讀，但應加以更正。

 通常只要移除舊版的 @no__t 0 注釋並使用 SAL2 語法，就可以更正這種情況。 有時候可能需要使用 `_At_` 來參考要標注的特定物件。
