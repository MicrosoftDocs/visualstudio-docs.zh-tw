# <a name="c28023"></a>C28023
警告 C28023：要指派或傳遞的函式，在中至少要有一個類別的 `_Function_class_` 注釋： \<classlist >

 當只有一個函式類別在使用中，而且未宣告適當類型的回呼時，通常會報告這個警告。

 當函式位於指派的左邊（如果這是函式呼叫，則為隱含指派的）時，就會發出這個警告，表示它是使用 `_Function_class_` 注釋的驅動程式特定函式類型，或包含這類批註. 指派右邊的函式沒有 @no__t 0 注釋。 右側的函式應該標注為與左邊函式的類型相同。 這通常是藉由在 @no__t 2funcname2 > 的目前第一個宣告之前，將宣告加入 \<class1 > \<funcname1 > 來完成。
