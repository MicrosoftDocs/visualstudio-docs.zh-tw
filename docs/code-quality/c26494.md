# <a name="c26494-var_use_before_init"></a>C26494 VAR_USE_BEFORE_INIT

變數 '% variable% ' 未初始化。 一律初始化物件。 請參閱[ C++核心指導方針類型. 5](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type)。
