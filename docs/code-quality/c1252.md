# <a name="c1252"></a>C1252
警告 C1252：外掛程式之間有迴圈或遺失的相依性：需要 GUID

 當外掛程式相依性發生內部錯誤，而不是要分析的程式碼時，程式碼分析工具會報告這個警告。
