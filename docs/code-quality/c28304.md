# <a name="c28304"></a>C28304
警告 C28304:針對C++參考參數 < p >、 不當放置`_Notref_`找不到運算子套用至\<k >。

 `_Notref_`運算子只能在特殊的情況下，與C++參考的參數，以及僅在系統提供的巨集。 它必須緊接著`_Deref_`運算子或功能的註解。
