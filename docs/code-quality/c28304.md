# <a name="c28304"></a>C28304
警告 C28304：針對C++參考參數 < parameter_name >，找到未正確放置的 `_Notref_` 運算子已套用至 @no__t 2token >。

 @No__t-0 運算子僅適用于涉及C++參考參數的特殊情況，而且僅適用于系統提供的宏。 其後面必須緊接著 `_Deref_` 運算子或功能注釋。
