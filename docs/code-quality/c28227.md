# <a name="c28227"></a>C28227
警告 C28227: 註釋不能也是 PrimOp (參閱之前的宣告)

 此警告表示已將命名的注釋宣告為一般注釋，而且先前已宣告為 PrimOp。 如果標準宏正用於注釋，則不應該這麼做;一般用途不應出現此警告。
