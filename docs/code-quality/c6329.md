# <a name="c6329"></a>C6329
警告 C6329： \<函式 > 的呼叫傳回值不應針對 \<數位進行檢查 >

 程式會比較數值與呼叫 `CreateFile` 的傳回值。 如果 `CreateFile` 成功，則會將開啟控制代碼傳回至物件。 如果失敗，則會傳回 `INVALID_HANDLE_VALUE`。

## <a name="example"></a>範例
 下面這個程式碼可能會造成警告：

```cpp
if (CreateFile() == NULL)
{
  return;
}
```

## <a name="example"></a>範例
 這個程式碼會修正錯誤：

```cpp
if (CreateFile() == INVALID_HANDLE_VALUE)
{
  return;
}
```
