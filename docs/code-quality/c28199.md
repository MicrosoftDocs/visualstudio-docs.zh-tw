# <a name="c28199"></a>C28199
警告 C28199:使用可能未初始化的記憶體

 此訊息表示變數已取得其位址，但已探索到任何指派給它。

 未明確初始化，使用指定的變數，但在某個時間點其位址拍攝地點，表示，它可能會初始化無形的方式為程式碼分析工具。

 如果變數初始化函式以外，這個警告會被誤判。

 程式碼分析工具會報告這個警告函式結束時，如果參數`_Out_`或`_Inout_`註釋和變數未初始化。
