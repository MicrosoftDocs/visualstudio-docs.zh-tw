# <a name="c28199"></a>C28199
警告 C28199：使用可能未初始化的記憶體

 此訊息表示變數已取得其位址，但是未探索到它的指派。

 在不明確初始化的情況下，會使用指定的變數，但在某個時間點，它會在程式碼分析工具中以不可見的方式初始化。

 如果變數是在函式外部初始化，則此警告可能會被誤認為。

 程式碼分析工具會在函式結束時回報此警告（如果參數有 `_Out_` 或 `_Inout_` 注釋，而且未初始化該變數）。
