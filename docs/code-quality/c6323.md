# <a name="c6323"></a>C6323
警告 C6323-在布林類型上使用算術運算子

 如果在布林資料類型上使用算術運算子，就會發生此警告。 使用不正確的運算子可能會產生不正確的結果。 這也表示程式設計人員的意圖不會反映在程式碼中。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```cpp
int test(bool a, bool b)
{
    int c = a + b;     //C6323
    return c;
}
```

 若要更正這個警告，請使用正確的資料類型和運算子。

```cpp
int test(int a, int b)
{
    int c = a + b;
    return c;
}
```
