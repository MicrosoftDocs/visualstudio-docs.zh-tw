# <a name="ca1811-avoid-uncalled-private-code"></a>CA1811：避免使用未呼叫的私用程式碼

|||
|-|-|
|TypeName|AvoidUncalledPrivateCode|
|CheckId|CA1811|
|類別|微軟.效能|
|重大變更|不中斷|

## <a name="cause"></a>原因
私有或內部(程式集級別)成員在程式集中沒有調用方,不由通用語言運行時調用,並且不由委託調用。 以下成員不由以下規則檢查:

- 顯式介面成員。

- 靜態構造函數。

- 序列化構造函數。

- 標有<xref:System.Runtime.InteropServices.ComRegisterFunctionAttribute?displayProperty=fullName><xref:System.Runtime.InteropServices.ComUnregisterFunctionAttribute?displayProperty=fullName>或 的方法。

- 重寫的成員。

## <a name="rule-description"></a>規則描述
如果出現規則邏輯當前未標識的入口點,則此規則可以報告誤報。 此外,編譯器可能會向程式集中發出不可調用的代碼。

## <a name="how-to-fix-violations"></a>如何修正違規
要修復違反此規則的行為,請刪除不可調用的代碼或添加調用它的代碼。

## <a name="when-to-suppress-warnings"></a>何時禁止警告
可以安全地禁止顯示此規則的警告。 此,請參閱[禁止程式碼分析警告](../code-quality/in-source-suppression-overview.md)。
