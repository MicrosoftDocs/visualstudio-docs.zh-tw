# <a name="ca0067"></a>CA0067
找不到目錄。

在 Fxcopcmd.exe 命令列中找不到 **/目錄**選項的值。

如果您使用 MSBuild 的**CodeAnalysisDependentAssemblyPaths**屬性來指定其他參考元件搜尋路徑，而且其中一個路徑不存在，就會發生此警告。

- 如果警告 CA0067 出現[CA0058](ca0058.md)錯誤或[CA0060](ca0060.md)警告，則解決其他錯誤通常會解析 CA0067。

- 如果出現警告 CA0067，但沒有其他錯誤或警告，您通常可以忽略此警告，因為不需要目錄。
