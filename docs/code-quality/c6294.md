# <a name="c6294"></a>C6294
警告 C6294：為-loop 定義錯誤：初始條件不符合測試。 未執行迴圈主體

 此警告表示無法執行 for 迴圈，因為終止條件為 true。 這則警告表示程式設計人員的意圖並未正確地被捕捉。

## <a name="example"></a>範例
 下列範例程式碼會產生此警告，因為 MAX_VALUE 是0：

```
#define MAX_VALUE 0
void f()
{
  int i;
  for (i = 0; i < MAX_VALUE; i++)
  {
    // code
  }
}
```

 下列範例程式碼會將 MAX_VALUE 的值變更為25，以更正此警告

```
#define MAX_VALUE 25
void f()
{
  int i;
  for (i = 0; i < MAX_VALUE; i++)
  {
    // code
  }
}
```
