# <a name="c26464-use_const_pointer_for_elements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS
  陣列 '% array% ' 的專案所指向的值只會指派一次，將元素標示為 `const`的指標。 請參閱[ C++核心指導方針。 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction)。
