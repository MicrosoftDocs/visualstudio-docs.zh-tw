# <a name="c26464-use_const_pointer_for_elements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS
  陣列 '% array% ' 的元素所指向的值只會指派一次，將元素標記為 `const` 的指標。 請參閱[c + + 核心指南 con.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction)。
