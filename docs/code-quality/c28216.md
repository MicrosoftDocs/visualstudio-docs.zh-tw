# <a name="c28216"></a>C28216
警告 C28216：@No__t-0Check @ no__t-1return @ no__t-2 注釋僅適用于特定函式參數的後置條件。

 @No__t-0 注釋已套用在 `__post` 以外的內容中。它可能需要 `__post` （或 `__drv_out`）修飾詞，否則可能會不正確地放置。
