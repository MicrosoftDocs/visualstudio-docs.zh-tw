# <a name="c28163"></a>C28163
警告 C28163：絕不能從 try/except 區塊內呼叫函式

 當函式的類型絕對不能包含在 @no__t 1 區塊中的 @no__t 0 區塊時，就會回報此警告。 程式碼分析工具至少找到一個路徑，其中呼叫的函式是在 @no__t 0 區塊中。
