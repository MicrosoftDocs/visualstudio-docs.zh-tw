# <a name="c28163"></a>C28163
警告 C28163：永遠不應從 try/except 區塊內呼叫函式

 當函式的類型絕對不能包含在 `try/except` 區塊中的 `try/except` 區塊時，就會回報此警告。 程式碼分析工具至少找到一個路徑，其中呼叫的函式在 `try/except` 區塊內。
