# <a name="c26463-use_const_for_elements"></a>C26463 USE_CONST_FOR_ELEMENTS
  陣列 '% array% ' 的元素只會指派一次，mark 元素 `const`。 請參閱[ C++核心指導方針。 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction)。
