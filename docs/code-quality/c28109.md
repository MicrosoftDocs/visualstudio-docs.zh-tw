# <a name="c28109"></a>C28109
警告 C28109: 呼叫函式時無法持有變數

 程式正在保留不應在呼叫此函式時保留的資源。 一般來說，這表示資源意外取得兩次。 當資源在相同的函式中取得和釋放時，程式碼分析工具會報告這個警告。

## <a name="example"></a>範例
 下列程式碼範例會產生此警告：

```cpp
ExAcquireResourceLite(resource, true);
//...
ExAcquireResourceLite(resource, true);
```

 下列程式碼範例可避免此警告：

```cpp
ExAcquireResourceLite(resource, true);
```
