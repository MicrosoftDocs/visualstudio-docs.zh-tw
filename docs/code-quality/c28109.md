# <a name="c28109"></a>C28109
警告 C28109:呼叫函式時無法持有變數

 程式持有應該呼叫此函式時未保留的資源。 通常，它表示不小心兩次取得資源。 取得和發行相同的函式中的資源時，程式碼分析工具會報告這個警告。

## <a name="example"></a>範例
 下列程式碼範例會產生此警告：

```
ExAcquireResourceLite(resource, true);
...
ExAcquireResourceLite(resource, true);
```

 下列程式碼範例可避免此警告：

```
ExAcquireResourceLite(resource, true);
```
