# <a name="ca0057"></a>CA0057
違規無法對應至目前的目標和規則集

如果使用專案檔或/imported XML 報表（其無法對應到目前目標和規則集的違規）以手動方式叫用 Fxcopcmd.exe，就會發生這個錯誤。

只有在使用工具的命令列版本完成分析時，才會發生。
