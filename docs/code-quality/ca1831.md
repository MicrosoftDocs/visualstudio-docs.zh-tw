# <a name="ca1831-use-asspan-instead-of-range-based-indexers-for-string-when-appropriate"></a>CA1831：在適用情況下，請使用 AsSpan 做為字串，不要使用範圍型的索引子

|Item|值|
|-|-|
|CheckId|CA1831|
|類別|Microsoft。效能|
|重大變更|不中斷|

## <a name="cause"></a>原因

在字串上使用範圍索引子，並隱含地將值指派給 ReadOnlySpan &lt; char &gt; 。

## <a name="rule-description"></a>規則描述

在字串上使用範圍索引子並指派給 span 類型：上的範圍索引子 <xref:System.Span%601> 是非複製作業 <xref:System.Span%601.Slice%2A?#System_Span_1_Slice_System_Int32_System_Int32_> ，但在字串的範圍索引子中， <xref:System.String.Substring%2A?#System_String_Substring_System_Int32_System_Int32_> 會使用方法，而不是 <xref:System.Span%601.Slice%2A?#System_Span_1_Slice_System_Int32_System_Int32_> ，這會產生字串的要求部分複本。 當此複製隱含當做或值使用時，通常不需要這麼做 <xref:System.ReadOnlySpan%601> <xref:System.ReadOnlyMemory%601> 。 如果不打算複製，請使用 <xref:System.MemoryExtensions.AsSpan%2A?#System_MemoryExtensions_AsSpan_System_String_AsSpan> 方法來避免不必要的複製。 如果是預定的複本，請先將它指派給區域變數，或加入明確轉換。 只有在範圍索引子作業的結果上使用隱含轉換時，analyzer 才會報告。

### <a name="detects"></a>檢測

隱含轉換：
- `ReadOnlySpan<char> slice = str[a..b];`

### <a name="does-not-detect"></a>未偵測到

明確轉換：
- `ReadOnlySpan<char> slice = (ReadOnlySpan<char>)str[a..b];`

## <a name="how-to-fix-violations"></a>如何修正違規

若要修正此規則的違規：請 <xref:System.MemoryExtensions.AsSpan%2A?#System_MemoryExtensions_AsSpan_System_String_AsSpan> 在字串上使用而不是 <xref:System.Range> 以為基礎的索引子，以避免建立不必要的資料複本。

```csharp
class C
{
    public void TestMethod(string str)
    {
        // The violation occurs
        ReadOnlySpan<char> slice = str[1..3];
        ...
    }
}
```

```csharp
class C
{
    public void TestMethod(string str)
    {
        // The violation fixed with AsSpan extension method
        ReadOnlySpan<char> slice = str.AsSpan()[1..3];
        ...
    }
}
```

> [!TIP]
> 程式碼修正適用于 Visual Studio 中的這項規則。 若要使用它，請將游標放在陣列違規上，然後按**Ctrl** + **。** (句點)。 選擇 [**使用 AsSpan]，而不是字串上以範圍為基礎的索引子。** 從顯示的選項清單中。
>
> ![CA1831 的程式碼修正-在適當時使用 AsSpan 而非以範圍為基礎的索引子](media/ca1831_codefix.png)

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

如果您想要建立複本，則可以放心抑制此規則的違規。 若要隱藏警告，您可以直接加入明確轉換。

```csharp
class C
{
    public void TestMethod(string str)
    {
        // The violation occurs
        ReadOnlySpan<char> slice = str[1..3];
        ...
    }
}
```

```csharp
class C
{
    public void TestMethod(string str)
    {
        // The violation fixed with explicit casting
        ReadOnlySpan<char> slice = (ReadOnlySpan<char>)str[1..3];
        ...
    }
}
```
