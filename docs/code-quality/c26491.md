# <a name="c26491-no_static_downcast"></a>C26491 NO_STATIC_DOWNCAST

請勿使用 `static_cast` 向下轉換。 請參閱[ C++核心指導方針類型。 2](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type)。
