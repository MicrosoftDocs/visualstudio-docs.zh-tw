# <a name="ca5398-avoid-hardcoded-sslprotocols-values"></a>CA5398：避免硬式編碼 SslProtocols 值

|||
|-|-|
|TypeName|SslProtocolsAnalyzer|
|CheckId|CA5398|
|分類|Microsoft.Security|
|重大變更|不中斷|

## <a name="cause"></a>原因

符合下列任一條件時，就會引發此規則：
- 參考了安全但硬式編碼的 <xref:System.Security.Authentication.SslProtocols?displayProperty=nameWithType> 值。
- 代表安全通訊協定版本的整數值已指派給 <xref:System.Security.Authentication.SslProtocols> 變數，做為 <xref:System.Security.Authentication.SslProtocols> 傳回值，或當做 <xref:System.Security.Authentication.SslProtocols> 引數使用。

安全值為：
- Tls12
- Tls13

## <a name="rule-description"></a>規則描述

傳輸層安全性（TLS）可保護電腦之間的通訊，最常見的方式是使用超文字安全傳輸通訊協定（HTTPS）。 Tls 1.0 和 TLS 1.1 的通訊協定版本已被取代，而 TLS 1.2 和 TLS 1.3 是最新的。 未來，TLS 1.2 和 TLS 1.3 可能會被取代。 若要確保您的應用程式保持安全，請避免硬式編碼通訊協定版本。 如需詳細資訊，請參閱[.NET Framework 的傳輸層安全性（TLS）最佳做法](/dotnet/framework/network-programming/tls)。

## <a name="how-to-fix-violations"></a>如何修正違規

不要硬式編碼 TLS 通訊協定版本。

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

您必須連接到無法升級的舊版服務，才能使用未來的 TLS 通訊協定版本。

## <a name="pseudo-code-examples"></a>虛擬程式碼範例

### <a name="enumeration-name-violation"></a>列舉名稱違規

```csharp
using System;
using System.Security.Authentication;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5398 violation
        SslProtocols sslProtocols = SslProtocols.Tls12;
    }
}
```

```vb
Imports System
Imports System.Security.Authentication

Public Class TestClass
    Public Function ExampleMethod() As SslProtocols
        ' CA5398 violation
        Return SslProtocols.Tls12
    End Function
End Class
```

### <a name="integer-value-violation"></a>整數值違規

```csharp
using System;
using System.Security.Authentication;

public class ExampleClass
{
    public SslProtocols ExampleMethod()
    {
        // CA5398 violation
        return (SslProtocols) 3072;    // TLS 1.2
    }
}
```

```vb
Imports System
Imports System.Security.Authentication

Public Class TestClass
    Public Function ExampleMethod() As SslProtocols
        ' CA5398 violation
        Return CType(3072, SslProtocols)   ' TLS 1.2
    End Function
End Class
```

### <a name="solution"></a>解決方案

```csharp
using System;
using System.Security.Authentication;

public class TestClass
{
    public void Method()
    {
        // With .NET Framework 4.7 or later, let the operating system decide what TLS protocol version to use.
        // See https://docs.microsoft.com/dotnet/framework/network-programming/tls
        SslProtocols sslProtocols = SslProtocols.None;
    }
}
```

```vb
Imports System
Imports System.Security.Authentication

Public Class TestClass
    Public Sub ExampleMethod()
        ' With .NET Framework 4.7 or later, let the operating system decide what TLS protocol version to use.
        ' See https://docs.microsoft.com/dotnet/framework/network-programming/tls
        Dim sslProtocols As SslProtocols = SslProtocols.None
    End Sub
End Class
```
