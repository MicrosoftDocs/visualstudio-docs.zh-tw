# <a name="c6334"></a>C6334
警告 C6334: 將 sizeof 運算子套用至已有一個運算子的運算式，可能會產生未預期的結果

 此警告表示 `sizeof` 運算子的誤用。 當 `sizeof` 運算子套用至運算式時，會產生所產生運算式的類型大小。

 例如，在下列程式碼中：

```cpp
char     a[10];
size_t   x;

x = sizeof (a - 1);
```

 `x` 會被指派值4，而不是9，因為產生的運算式不再是陣列 `a`的指標，而只是指標。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```cpp
void f( )
{
  size_t x;
  char a[10];

  x= sizeof (a - 4);
  // code...
}
```

 若要修正這個警告，請使用下列程式碼：

```cpp
void f( )
{
  size_t x;
  char a[10];

  x= sizeof (a) - 4;
  // code...
}
```
