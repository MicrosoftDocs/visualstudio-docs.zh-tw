# <a name="c6306"></a>C6306
警告 C6306：不正確地呼叫 \<函式 >：請考慮使用接受 va_list 做為引數的 \<函數 >

 此警告表示不正確的函式呼叫。 `printf` 系列包含數個可接受引數清單的函式;不過，您無法使用 `va_list` 引數來呼叫這些函式。 有一個對應的 `vprintf` 系列函式，可用於這類呼叫。 呼叫錯誤的列印函式將導致不正確的輸出。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```cpp
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   printf("%s", v); // warning 6306
   va_end(v);
}
```

 若要修正這個警告，請使用下列程式碼：

```cpp
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   vprintf_s("%d",v);
   va_end(v);
}
```
