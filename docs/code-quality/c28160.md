# <a name="c28160"></a>C28160
警告 C28160：Error 註釋

 當遇到 `__drv_error` 注釋時，就會回報此警告。 這個批註是用來標示應該修正的編碼做法，而且可以與 @no__t 0 注釋搭配使用，以指示特定的參數組合。
