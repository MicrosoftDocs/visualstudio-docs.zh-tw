# <a name="c28160"></a>C28160
警告 C28160:Error 註釋

 會報告這個警告時`__drv_error`註釋發生。 此註解來標示程式碼撰寫慣例，應該加以修正，並可搭配`__drv_when`註解可指示參數的特定組合。
