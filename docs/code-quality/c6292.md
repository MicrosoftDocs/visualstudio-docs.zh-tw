# <a name="c6292"></a>C6292
警告 C6292：-loop 定義錯誤：計數上限為

 此警告表示 for 迴圈可能無法如預期運作。

 當迴圈從最大值計算，但終止條件較低時，就會發生此情況。 只有在發生整數溢位之後，才會終止此迴圈。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```cpp
void f( )
{
   int i;

   for (i = 100; i >= 0; i++)
   {
      // code ...
   }
}
```

 若要修正這個警告，請使用下列程式碼：

```cpp
void f( )
{
   int i;

   for (i = 100; i >= 0; i--)
   {
      // code ...
   }
}
```
