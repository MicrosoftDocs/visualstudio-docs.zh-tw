# <a name="c6411"></a>C6411
警告 C6411：可能會從緩衝區讀取無效資料。

 此警告表示用來從緩衝區讀取的索引值可能會超過緩衝區的可讀取大小。 因為程式碼分析工具在無法減少代表緩衝區大小的複雜運算式或用來存取緩衝區的索引時，會報告這個警告，所以可能會在錯誤中報告這個警告。

## <a name="example"></a>範例
 下列程式碼會產生這個警告。

```cpp
char *a = new char[strlen(InputParam)];
delete[] a;
a[10];
```

 下列程式碼會修正這個錯誤。

```cpp
int i = strlen(InputParam);
char *a = new char[i];
if (i > 10) a[10];
delete[] a;
```
