# <a name="c28229"></a>C28229
警告 C28229：註釋不支援參數

 此警告表示在未使用參數宣告注釋時，批註已與參數搭配使用。 如果標準宏正用於注釋，則不應該這麼做;一般用途不應出現此警告。
