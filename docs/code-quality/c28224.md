# <a name="c28224"></a>C28224
警告 C28224：注釋需要參數

 此警告表示在沒有參數的情況下使用了命名的注釋，而且至少需要一個參數。 如果標準宏正用於注釋，則不應該這麼做;一般使用時，不應該看到這項警告。
