# <a name="ca1501-avoid-excessive-inheritance"></a>CA1501:避免在物件間過度繼承

|Item|值|
|-|-|
|CheckId|CA1501|
|類別|Microsoft 的可維護性|
|重大變更|中斷|

## <a name="cause"></a>原因

類型在其繼承階層架構 (Inheritance Hierarchy) 中超過四個層級的深度。

根據預設，此規則只會排除命名空間中的所有類型 `System` ，但這是 [可](#configurability)設定的。

## <a name="rule-description"></a>規則描述

太深的巢狀類型階層架構可能會難以依循、了解和維護。 此規則會限制對相同模組中階層的分析。

## <a name="how-to-fix-violations"></a>如何修正違規

若要修正此規則的違規情形，請從繼承階層中較深層的基底類型衍生型別，或消除某些中繼基底類型。

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

隱藏此規則的警告是安全的。 不過，程式碼可能更難維護。 請注意，根據基底類型的可見度，解決此規則的違規可能會產生重大變更。 例如，移除公用基底類型是一項重大變更。

## <a name="configurability"></a>性

### <a name="inheritance-excluded-type-or-namespace-names"></a>繼承排除的類型或命名空間名稱

您可以設定類型或命名空間的名稱， (以 ' | ' 分隔) ，使類型或類型的命名空間不計入繼承階層樹狀結構中。 預設 `System.*` 會排除命名空間中的所有類型。 請注意，無論您設定的值為何，都會新增此預設值。

| 選項值 | 摘要 |
| --- | --- |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = MyType` | 符合名為 ' MyType ' 的所有型別，或其包含命名空間包含 ' MyType ' 的類型，以及 ' System ' 命名空間中的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = MyType1\|MyType2` | 符合所有名稱為 ' MyType1 ' 或 ' MyType2 ' 的型別，或其包含命名空間包含 ' MyType1 ' 或 ' MyType2 ' 的類型，以及 ' System ' 命名空間中的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS.MyType` | 符合命名空間 ' NS ' 中的特定類型 ' MyType '，以及 ' System ' 命名空間中的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS1.MyType1\|T:NS2.MyType2` | 會比對特定類型的 ' MyType1 ' 和 ' MyType2 ' 與 ' System ' 命名空間中的個別完整名稱和所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = N:NS` | 符合 ' NS ' 命名空間中的所有類型，以及 ' System ' 命名空間中的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = My*` | 符合所有名稱開頭為 ' My ' 或其包含命名空間部分以 ' My ' 開頭的型別，以及 ' System ' 命名空間中所有類型的類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS.My*` | 符合命名空間 ' NS ' 中名稱開頭為 ' My ' 的所有型別，以及 ' System ' 命名空間中的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = N:My*` | 比對包含命名空間的所有型別，其所有型別都是以 ' My ' 和 ' System ' 命名空間 |
