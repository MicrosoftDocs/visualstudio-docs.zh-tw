# <a name="ca1501-avoid-excessive-inheritance"></a>CA1501:避免在物件間過度繼承

|Item|值|
|-|-|
|CheckId|CA1501|
|類別|Microsoft。可維護性|
|重大變更|中斷|

## <a name="cause"></a>原因

類型在其繼承階層架構 (Inheritance Hierarchy) 中超過四個層級的深度。

根據預設，規則只會排除命名空間中的所有類型 `System` ，但這是[可](#configurability)設定的。

## <a name="rule-description"></a>規則描述

太深的巢狀類型階層架構可能會難以依循、了解和維護。 此規則會將分析限制為相同模組中的階層。

## <a name="how-to-fix-violations"></a>如何修正違規

若要修正此規則的違規情形，請從在繼承階層架構中較不深層的基底類型衍生類型，或排除部分中繼基底類型。

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

您可以放心地隱藏此規則的警告。 不過，程式碼可能比較不容易維護。 請注意，根據基底類型的可見度，解決此規則的違規可能會建立中斷性變更。 例如，移除公用基底類型是一種中斷變更。

## <a name="configurability"></a>可設定性

### <a name="inheritance-excluded-type-or-namespace-names"></a>繼承排除的類型或命名空間名稱

您可以設定類型或命名空間的名稱（以 ' | ' 分隔），讓類型或類型的命名空間不會在繼承階層樹狀結構中計算。 根據預設， `System.*` 會排除命名空間中的所有類型。 請注意，無論您設定哪一個值，都會加入此預設值。

| 選項值 | 摘要 |
| --- | --- |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = MyType` | 比對名為 ' MyType ' 或其包含命名空間的所有類型都包含 ' MyType ' 和 ' System ' 命名空間中的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = MyType1\|MyType2` | 比對名為 ' MyType1 ' 或 ' MyType2 ' 的所有類型，或其包含的命名空間包含 ' MyType1 ' 或 ' MyType2 '，以及來自 ' System ' 命名空間的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS.MyType` | 符合命名空間 ' NS ' 中的特定類型 ' MyType '，以及來自 ' System ' 命名空間的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS1.MyType1\|T:NS2.MyType2` | 將特定類型 ' MyType1 ' 和 ' MyType2 ' 與個別的完整名稱和 ' System ' 命名空間中的所有類型進行比對 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = N:NS` | 符合 ' NS ' 命名空間中的所有類型，以及來自 ' System ' 命名空間的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = My*` | 比對名稱開頭為 ' My ' 或其包含命名空間部分開頭為 ' My ' 的所有類型，以及來自 ' System ' 命名空間的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = T:NS.My*` | 比對名稱開頭為 ' My ' 且命名空間為 ' NS ' 的所有類型，以及來自 ' System ' 命名空間的所有類型 |
|`dotnet_code_quality.CA1501.additional_inheritance_excluded_symbol_names = N:My*` | 比對其包含命名空間開頭為 ' My ' 的所有類型，以及來自 ' System ' 命名空間的所有類型 |
