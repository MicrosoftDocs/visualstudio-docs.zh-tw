# <a name="c26460-use_const_reference_arguments"></a>C26460 USE_CONST_REFERENCE_ARGUMENTS

  函數 '% function% ' 的參考引數 '% 引數% ' 可以標記為 `const`。 請參閱[ C++核心指導方針。 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref)。
