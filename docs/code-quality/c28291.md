# <a name="c28291"></a>C28291
警告 C28291：在 deref 層級0的 Post null/notnull 對於函數 \<x > 在參數 \<number > 沒有意義

 將取值層級零參數的後置條件指定為擁有 null/非 null 屬性時，程式碼剖析工具就會報告這個警告。 因為在取值層級零的值無法變更，所以會發生這個錯誤。
