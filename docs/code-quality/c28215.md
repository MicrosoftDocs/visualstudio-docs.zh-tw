# <a name="c28215"></a>C28215
警告 C28215:typefix 不能套用到已經有 typefix 的參數

 套用`__typefix`參數已經有該註釋的註解會發生錯誤。 `__typefix`只在幾個特殊的情況下使用註解，而且這個警告不應該出現在正常使用。
