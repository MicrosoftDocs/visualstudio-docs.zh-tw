# <a name="c28215"></a>C28215
警告 C28215：typefix 不能套用到已經有 typefix 的參數

 將 `__typefix` 注釋套用至已經有該注釋的參數會是錯誤。 只有在幾個特殊情況下，才會使用 `__typefix` 注釋，而且不應該在一般使用中看到這個警告。
