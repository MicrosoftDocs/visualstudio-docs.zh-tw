# <a name="c6064"></a>C6064
警告 C6064：遺漏對應于轉換規範之 @no__t 0function > 的整數引數 \<number >

 此警告表示提供的引數不足以符合格式字串，而其中一個遺漏的引數是整數。 此缺失可能會導致不正確的輸出。

## <a name="example"></a>範例
 下列程式碼會產生這個警告，因為對 `sprintf` 的呼叫中使用了不正確的引數數目，而且遺漏的引數是一個整數：

```
#include <string.h>
void f( )
{
  char buff[15];
  char *string="Hello, World";

  sprintf(buff,"%s %d", string);
}
```

 若要更正這個警告，請指定遺漏的引數，如下列程式碼所示：

```
#include <string.h>
void f( )
{
  char buff[15];
  char *string = "Hello, World";

  sprintf(buff,"%s %d",string, strlen(string));
}
```

 下列程式碼使用安全字串操作函式，`sprintf_s` 來更正此警告：

```
#include <string.h>
void f( )
{
  char buff[15];
  char *string="Hello World";

  sprintf_s(buff,sizeof(buff),"%s %d", string, strlen(string));
}
```
