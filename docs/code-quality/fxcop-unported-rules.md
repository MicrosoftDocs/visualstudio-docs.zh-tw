# <a name="unported-rules"></a>未移植的規則

## <a name="overview"></a>概觀

尚未實作為 Roslyn 分析器的 FxCop 舊版分析規則集包含下列兩個值區：
- 尚未移植但仍 [可能移植](fxcop-unported-rules-may-get-ported.md)的規則。
- 已淘汰且 [不會移植](fxcop-unported-deprecated-rules.md)的規則。
