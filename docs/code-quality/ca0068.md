# <a name="ca0068"></a>CA0068

找不到目標群組件 *' AssemblyName '* 的偵錯工具資訊（符號）。 若要獲得最佳分析結果，請在與目標群組件相同的目錄中，包含具有 *' AssemblyName '* 之 debug 資訊的 .pdb 檔案。

在 .pdb 檔案中找到的偵錯工具資訊可改善某些程式碼分析檢查的正確性。 遺失的 .pdb 檔案可能會導致誤報的增加。 若要啟用 .pdb 產生，請開啟 [專案屬性] 頁面。 在 [**組建**] 索引標籤上，尋找 [**調試資訊**] 清單。 位置會因專案類型而異。 請確定 debug 組建的 [偵錯工具]**資訊**已設為 [**完整**]，或針對 [發行] 組建設定為 [**僅限 pdb** ]。 不應設定為 [**無**]。
