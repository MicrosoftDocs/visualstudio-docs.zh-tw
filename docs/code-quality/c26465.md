# <a name="c26465-no_const_cast_unnecessary"></a>C26465 NO_CONST_CAST_UNNECESSARY

請勿使用 `const_cast` 來轉換掉 `const`。 `const_cast` 不是必要的資訊;常數性或揮發性不是會移除這項轉換。 請參閱[c + + 核心指南 Type.3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-constcast)。
