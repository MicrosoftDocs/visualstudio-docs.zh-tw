# <a name="c26498-use_constexpr_for_functioncall"></a>C26498 USE_CONSTEXPR_FOR_FUNCTIONCALL

如果需要編譯時間評估，這個函式呼叫% function% 可以使用 `constexpr`。 請參閱[ C++核心指導方針。 5](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-constexpr)。
