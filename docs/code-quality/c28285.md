# <a name="c28285"></a>C28285
警告 C28285:對於函式，參數中有語法錯誤

 程式碼分析工具會報告這個警告，可能的錯誤發生的模型檔案中時。 少數的來源檔案錯誤也可能導致這類錯誤。
