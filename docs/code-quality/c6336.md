# <a name="c6336"></a>C6336
警告 C6336：算術運算子的優先順序高於問號運算子，請使用括弧來澄清意圖

 此警告表示可能的運算子優先順序問題。 ' + '、'-'、' * ' 和 '/' 運算子的優先順序高於 '？ ' 運算子。 如果運算式中的優先順序不正確，請使用括弧來變更運算子優先順序。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
int Count();

void f(int flag)
{
  int result;
  result = Count() + flag ? 1 : 2;
  // code...
}
```

 若要更正這個警告，請新增括弧，如下列程式碼所示：

```
int Count();

void f(int flag)
{
  int result;
  result = Count() + (flag ? 1 : 2);
  // code...
}
```
