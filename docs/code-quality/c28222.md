# <a name="c28222"></a>C28222
警告28222： \_Yes @ no__t-1，\_No @ no__t-3，或 \_Maybe @ no__t-5 必須用於注釋

 此警告表示批註的參數必須是 `_Yes_`、`_No_` 或 `_Maybe_` 的其中一個符號，而且遇到一些其他符號。 這通常表示批註宏的編碼不正確。
