# <a name="c26461-use_const_pointer_arguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS:
  函數 '% function% ' 的指標引數 '% argument% ' 可以標記為 `const` 的指標。 請參閱[ C++核心指導方針。 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref)。
