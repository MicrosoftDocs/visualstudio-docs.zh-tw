# <a name="c6510"></a>C6510
警告 C6510：不正確注釋：' NullTerminated ' 屬性只能用於其元素為整數或指標類型的緩衝區：函數 ' @no__t 0function > ' \<parameter >。

 此警告表示**NullTerminated**屬性（結尾為 ' `_z` '）的使用不正確。 您只能在指標或陣列類型上使用這種類型的屬性。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 若要修正這個警告，請使用下列程式碼：

```
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
