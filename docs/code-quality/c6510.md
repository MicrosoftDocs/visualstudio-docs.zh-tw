# <a name="c6510"></a>C6510
警告 C6510:無效的附註：'NullTerminated' 屬性只可用的緩衝區，其項目上的整數或指標類型：函式 '\<函式 >'\<參數 >。

 這則警告表示使用不正確**NullTerminated**屬性 (結尾 '`_z`')。 您只能在指標或陣列類型上使用這種類型的屬性。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 若要更正這個警告，請使用下列程式碼：

```
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
