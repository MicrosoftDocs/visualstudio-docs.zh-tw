# <a name="c28112"></a>C28112

警告 C28112：透過連鎖函數存取的變數必須一律透過連鎖函式存取

請參閱行 *[數位]* ：存取透過連鎖 @ no__t-0 系列函式存取的變數並不一定安全。

使用連鎖執行者支援常式（例如 InterlockedCompareExchangeAcquire）存取的變數之後，會使用不同的函式來存取。 雖然您可以使用不同的函式來安全地存取連鎖 @ no__t-0 常式所使用之變數的某些一般指派、存取和比較，但此風險足以證明每個實例的檢查。

## <a name="example"></a>範例

下列程式碼範例會產生此警告：

```cpp
inter_var --;
...
InterlockedIncrement(&inter_var);
```

下列程式碼範例可避免此警告：

```cpp
InterlockedDecrement(&inter_var);
...
InterlockedIncrement(&inter_var);
```
