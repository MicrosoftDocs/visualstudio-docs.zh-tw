# <a name="c28223"></a>C28223
警告 C28223：找不到注釋的預期標記/識別碼，參數

 此警告表示批註的參數必須是識別碼，而且遇到一些其他符號。 這通常表示批註宏的編碼不正確。 在此位置使用 C C++或關鍵字會導致此錯誤。
