# <a name="c28194"></a>C28194
警告 C28194：函式已宣告為變數中的值加上別名並結束，而不執行此動作

 此警告表示正在分析之函式的函式原型具有 `__drv_isAliased` 注釋，這表示它將會為*指定的自*變數（也就是指派值，使其無法從函式傳回）。 不過，函式不會在批註所指示的路徑中加上引數的別名。 所有別名變數的函式會將其值儲存至全域資料結構。
