# <a name="c28024"></a>C28024
警告 C28024：要指派給的函式指標會以 > 的函式類別 @no__t 0class，而不包含在 1classlist > 的函式 @no__t 類別中。

 當兩個函式都是以函式類別標注，但類別不相符時，就會回報此警告。

 當函式指標具有 @no__t 0 注釋時，會發出這個警告，指定只應將特定功能類別的函式指派給它。 在函式呼叫的指派或隱含指派中，來源和目標必須是相同的函式類別，但函數類別不相符。
