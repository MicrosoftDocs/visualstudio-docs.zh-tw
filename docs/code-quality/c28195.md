# <a name="c28195"></a>C28195
警告 C28195：函式已宣告為取得變數中的記憶體並結束而不這麼做

 此警告表示正在分析之函式的函式原型具有 @no__t 0 注釋。 @No__t-0 注釋表示函式會在指定的結果位置中取得記憶體，但在至少一個路徑中，函式並未取得記憶體。 請注意，程式碼分析工具將無法辨識記憶體配置器的實際執行（涉及位址算術），而且不會辨識記憶體是否已配置（雖然會識別許多包裝函式）。 在此情況下，程式碼分析工具無法辨識記憶體已配置併發出此警告。 若要隱藏誤報，請在函式主體的左大括弧 `{` 前面的那一行使用 `#pragma` 警告
