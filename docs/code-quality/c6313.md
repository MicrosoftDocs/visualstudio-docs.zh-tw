# <a name="c6313"></a>C6313
警告 C6313：不正確的運算子：不能使用位 and 來測試零值旗標。 使用等號比較測試來檢查是否有零值旗標

 此警告表示將常數值零當做引數提供給測試內容中的位 and （`&`）運算子。 產生的運算式是常數，並會評估為 false。結果與預期不同。

 這通常是使用位 and 來測試值為零的旗標所造成。 若要測試零值旗標，必須執行相等的測試，例如，使用 `==` 或 `!=`。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
#define FLAG 0

void f(int Flags )
{
  if (Flags & FLAG)
  {
    // code
  }
}
```

 若要修正這個警告，請使用下列程式碼：

```
#define FLAG 0

void f(int Flags )
{
  if (Flags == FLAG)
  {
    // code
  }
}
```
