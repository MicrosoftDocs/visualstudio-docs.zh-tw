# <a name="c6102"></a>C6102
警告 C6102：在行 \<location > 使用 @no__t 0variable > 失敗的函式呼叫。

 未設定變數時，會報告這個警告，而不是[C6001](../code-quality/c6001.md) ，因為在先前的函式呼叫中，已將其標示為 @no__t 1 參數。

 問題可能在於之前呼叫的函式沒有完整標註。 它可能需要 `_Always_`、`_Outptr_result_nullonfailure_` (COM 程式碼的 `_COM_Outptr_`) 或相關註釋。
