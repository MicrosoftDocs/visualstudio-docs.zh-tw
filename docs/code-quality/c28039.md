# <a name="c28039"></a>C28039
警告 C28039：實際參數 \<operand > 的類型應完全符合 \<typename 的類型 >

 當列舉型式未傳遞列舉的成員，但也可以用於其他類型時，通常會報告這個警告。

 因為 C 可交換使用列舉，並與常數交換，所以很容易就能將錯誤的列舉值傳遞給函式，而不會發生錯誤。

 針對列舉類型，如果列舉參數的類型是以 `_Enum_is_bitflag_` 來標注，則在參數上允許使用算術。 否則參數的類型必須完全正確。 如果絕對需要常數，則警告 C28137 也可能適用。

 此規則也可以用於其他參數類型;如需類型必須完全符合的原因，請參閱函數檔。
