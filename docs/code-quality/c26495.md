# <a name="c26495-member_uninit"></a>C26495 誤判 MEMBER_UNINIT

變數 '% variable% ' 未初始化。 一律初始化成員變數。 請參閱[ C++核心指導方針類型 6](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type)。
