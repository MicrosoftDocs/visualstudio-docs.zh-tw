# <a name="c28238"></a>C28238
警告 C28238:參數上的註釋具有一個以上的值：stringValue 和 longValue。 請使用 paramn=xxx

 PREfast 的模型檔案中發生內部錯誤。 在一般使用時，應該不會發生這個警告。
