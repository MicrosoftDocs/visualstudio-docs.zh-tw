# <a name="c1254"></a>C1254
警告 C1254：外掛程式版本不符：版本不符合 PREfast 驅動程式的版本

 當外掛程式版本發生內部錯誤，而不是要分析的程式碼時，程式碼分析工具會報告這個警告。
