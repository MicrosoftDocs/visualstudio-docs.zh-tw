# <a name="c26466-no_static_downcast_polymorphic"></a>C26466 NO_STATIC_DOWNCAST_POLYMORPHIC
  請勿使用 `static_cast` 向下轉換。 從多型類型的 cast 應使用 dynamic_cast。 請參閱[c + + 核心指南 Type.2](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-downcast)。
