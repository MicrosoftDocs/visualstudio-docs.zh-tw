# <a name="c6995"></a>C6995
警告 C6995：無法儲存 XML 記錄檔

 此警告表示程式碼分析工具無法建立瑕疵記錄檔，這是程式碼分析的輸出。

 此錯誤可能表示磁片錯誤，或指出您沒有在指定目錄中建立檔案的許可權。
