# <a name="c6995"></a>C6995
警告 C6995:無法儲存 XML 記錄檔

 這則警告表示程式碼分析工具無法建立缺失記錄，也就是程式碼分析的輸出。

 此錯誤可能表示磁碟發生錯誤，或表示您沒有在指定的目錄中建立檔案的權限。
