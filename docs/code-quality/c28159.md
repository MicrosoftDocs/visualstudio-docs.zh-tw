# <a name="c28159"></a>C28159
警告 C28159：請考慮改為使用其他函式。

 系統會針對驅動程式回報此警告，建議您使用在語義上相當於驅動程式所呼叫之函數的慣用函式呼叫。 這是一般警告訊息;已使用批註 `__drv_preferredFunction` （可能具有條件式的 `__drv_when` （）注釋）來標示不良的編碼實務。

## <a name="example"></a>範例
 下列程式碼範例會產生此警告：

```
char buff[MAX_PATH];

OemToChar(buff, input);

    // if strlen(input) > MAX_PATH
....// leads to buffer overrun
```

 下列程式碼範例可避免此警告：

```
char buff[MAX_PATH];

OemToCharBuff(buff, input, MAX_PATH);
```
