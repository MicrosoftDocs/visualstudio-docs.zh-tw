# <a name="c28239"></a>C28239
警告 C28239：參數上的注釋同時具有 value、stringValue 或 longValue;and paramn = xxx。 請只使用 paramn=xxx

 PREfast 模型檔案中發生內部錯誤。 一般用途不應發生此警告。
