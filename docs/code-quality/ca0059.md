# <a name="ca0059"></a>CA0059
命令列參數時發生錯誤。

 如果必要的命令列參數遺漏，比方說，會發生這個錯誤

 /out 或 /console 參數未指定。 無效的參數，例如不正確的逾時值時，也會發生此錯誤。
