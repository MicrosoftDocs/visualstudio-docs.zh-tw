# <a name="ca5378-do-not-disable-servicepointmanagersecurityprotocols"></a>CA5378:請勿停用 ServicePointManagerSecurityProtocols

|||
|-|-|
|TypeName|DoNotSetSwitch|
|CheckId|CA5378|
|Category|Microsoft.Security|
|重大變更|不中斷|

## <a name="cause"></a>原因

方法呼叫會將`Switch.System.ServiceModel.DisableUsingServicePointManagerSecurityProtocols`設`true`為。 <xref:System.AppContext.SetSwitch%2A?displayProperty=nameWithType>

## <a name="rule-description"></a>規則描述

設定`Switch.System.ServiceModel.DisableUsingServicePointManagerSecurityProtocols` 以`true`限制使用 TLS 1.0 的 Windows Communication Framework （WCF）傳輸層安全性（tls）連線。 該 TLS 版本將會被取代。 如需詳細資訊，請參閱[.NET Framework 的傳輸層安全性（TLS）最佳做法](/dotnet/framework/network-programming/tls#switchsystemservicemodeldisableusingservicepointmanagersecurityprotocols)。

## <a name="how-to-fix-violations"></a>如何修正違規

- 如果您的應用程式是以 .NET Framework 4.7 或更新版本為目標， <xref:System.AppContext.SetSwitch%2A?displayProperty=nameWithType>您可以移除方法呼叫，或將參數的`false`值設定為。
- 如果您的應用程式是以 .NET Framework v 4.6.2 或更早版本為目標，並在 .NET Framework 4.7 或更新版本`false`上執行，請將參數的值設定為。
- 否則，請參閱[傳輸層安全性（TLS）最佳做法，其中包含](/dotnet/framework/network-programming/tls)用於緩和措施的 .NET Framework。

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

如果您需要連接到無法升級為使用安全 TLS 設定的舊版服務，則可以隱藏此警告。

## <a name="pseudo-code-examples"></a>虛擬程式碼範例

### <a name="violation"></a>違規

```csharp
using System;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5378 violation
        AppContext.SetSwitch("Switch.System.ServiceModel.DisableUsingServicePointManagerSecurityProtocols", true);
    }
}
```

```vb
Imports System

Public Class ExampleClass
    Public Sub ExampleMethod()
        ' CA5378 violation
        AppContext.SetSwitch("Switch.System.ServiceModel.DisableUsingServicePointManagerSecurityProtocols", true)
    End Sub
End Class
```

### <a name="solution"></a>方案

```csharp
using System;

public class ExampleClass
{
    public void ExampleMethod()
    {
        AppContext.SetSwitch("Switch.System.ServiceModel.DisableUsingServicePointManagerSecurityProtocols", false);
    }
}
```

```vb
Imports System

Public Class ExampleClass
    Public Sub ExampleMethod()
        AppContext.SetSwitch("Switch.System.ServiceModel.DisableUsingServicePointManagerSecurityProtocols", false)
    End Sub
End Class
```
