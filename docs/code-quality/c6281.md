# <a name="c6281"></a>C6281
警告 6281-不正確的作業順序：關係運算子的優先順序高於位運算子

 此警告表示運算子優先順序中可能發生錯誤。 這可能會產生不正確的結果。 您應該檢查優先順序，並使用括弧來澄清意圖。 關係運算子（\<、>、\< =、> =、= =、！ =）的優先順序高於位運算子（& &#124; ^）。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
#include <stdlib.h>
#define FORMAT 1
#define TYPE 2

void f(int input)
{
  if (FORMAT & TYPE != input)
  {
    // code...
  }
}
```

 下列程式碼會使用括弧來更正此警告：

```
#include <stdlib.h>
#define FORMAT 1
#define TYPE 2

void f(int input)
{
  if ((FORMAT & TYPE) != input)
  {
    // code...
  }
}
```
