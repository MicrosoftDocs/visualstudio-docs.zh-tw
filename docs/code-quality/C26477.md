# <a name="c26477-usenullptrnotconstant"></a>C26477 USE_NULLPTR_NOT_CONSTANT

`nullptr` 有一種特殊類型`nullptr_t`，可讓多載的特殊的 null 處理。 使用`0`或是`NULL`代替`nullptr`會略過型別安全和推斷，`nullptr`提供。
