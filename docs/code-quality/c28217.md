# <a name="c28217"></a>C28217
警告 C28217： 函式的註釋的參數數目不會不符合找到的檔案  
  
 目前這一行，並在訊息中的列上的註釋不一致。 這不能進行註解; 使用標準巨集這個警告不會出現在正常使用。
