# <a name="c28235"></a>C28235
警告 C28235:函式不能獨立成為註釋

 這個警告表示嘗試使用未宣告為註解註釋的內容中的函式。 這包括在獨立的內容中使用的基本作業 (PrimOp)。 這不應該是可能的標準巨集所使用的註解。 這個警告不應該看到典型的使用中。
