# <a name="c28235"></a>C28235
警告 C28235：函式不能獨立為注釋

 此警告表示嘗試使用未宣告為注釋內容中注釋的函式。 這包括在獨立內容中使用基本操作（PrimOp）。 如果標準宏正用於注釋，這就不可能發生。 一般用途不應出現此警告。
