# <a name="c28221"></a>C28221
警告 C28221：注釋中的參數需要字串運算式

 此警告表示批註的參數必須是字串，而且遇到一些其他類型。 這通常表示批註宏的編碼不正確，而且預期不會出現在正常使用中。
