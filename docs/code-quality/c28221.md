# <a name="c28221"></a>C28221
警告 C28221:註釋中的參數需要字串運算式

 這則警告表示註釋的參數必須是字串，而且發現某些其他類型。 這通常表示的編碼錯誤註釋巨集，而且不應該出現在正常使用。
