# <a name="c28282"></a>C28282
警告 C28282：格式字串必須在前置條件中

 此警告表示找到 @no__t 0 的注釋，這不在 `_Pre_` （`__drv_in`）注釋中（函數參數預設為 `_Pre_`）。 檢查是否在具有 `_Post_` （`__drv_out`）注釋的明確區塊中使用的注釋。 若是如此，請從已放入 @no__t 0 內容的任何封閉區塊中移除批註。
