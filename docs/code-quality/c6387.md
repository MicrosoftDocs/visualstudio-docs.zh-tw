# <a name="c6387"></a>C6387
警告 C6387： \<argument > 可能 \<value >：這不符合函式的規格 \<function 名稱 >：行： x、y

 如果將未預期的值傳遞給注釋的函式參數，就會引發此警告。 例如，將可能的 null 值傳遞給標記為 `_In_` 注釋的參數會產生此警告。

## <a name="example"></a>範例
 下列程式碼會產生這個警告，因為 null 參數會傳遞給 `f(char *)`：

```cpp

#include <sal.h>

_Post_ _Null_ char * g();

void f(_In_ char *pch);

void main()
{
    char *pCh = g();
    f(pCh); // Warning C6387
}
```

 若要修正這個警告，請使用下列程式碼：

```cpp

#include <sal.h>

_Post_ _Notnull_ char * g();

void f(_In_ char *pch);

void main()
{
    char *pCh = g();
    f(pCh);
}
```
