# <a name="how-to-run-code-analysis-manually-for-managed-code"></a>作法：針對受控碼手動執行程式碼分析
程式碼分析工具會為您提供原始程式碼中可能瑕疵的相關資訊。 您可以使用程式碼專案的每個組建自動執行程式碼分析, 也可以手動執行程式碼分析。 執行程式碼分析時所檢查的規則會在專案屬性頁的 [程式碼分析] 頁面上指定。 如需詳細資訊，請參閱[如何：設定受控碼專案](../code-quality/how-to-configure-code-analysis-for-a-managed-code-project.md)的程式碼分析。

### <a name="to-run-code-analysis-manually"></a>手動執行程式碼分析

1. 在**方案總管**中, 按一下專案。

2. 在 [**分析**] 功能表中，按一下**上執行程式碼分析** *專案名稱*。
