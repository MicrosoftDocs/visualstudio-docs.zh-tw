# <a name="c28279"></a>C28279
警告 C28279：找到符號的 'begin'，但沒有相符的 'end'

 注釋語言支援 begin 和 end （在 C 中為 `{` 和 `}`）結構，而配對已取得不平衡。 如果使用宏，就可以避免這種情況。
