# <a name="c28279"></a>C28279
警告 C28279： 符號，找到 'begin'，但沒有相符的 'end'  
  
 註釋語言支援 begin 和 end (`{`和`}`c) 建構，並發現不對稱。 您可以避免這種情況下，如果使用巨集。
