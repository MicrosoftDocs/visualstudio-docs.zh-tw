# <a name="c6274"></a>C6274
警告 C6274： \<function 的呼叫中需要字元時，以參數形式傳遞的非字元 \<number > >

 此警告表示格式字串指定了需要的字元（例如，`%c` 或 @no__t 1 規格），但不會傳遞非整數，例如 float、string 或 struct。 這種瑕疵可能會導致不正確的輸出。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 若要修正這個警告，請使用下列程式碼：

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 下列程式碼使用安全字串操作函式 `sprintf_s` 來更正此警告：

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
