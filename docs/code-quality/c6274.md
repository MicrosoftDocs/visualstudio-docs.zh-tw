# <a name="c6274"></a>C6274
警告 C6274： \<函式的呼叫中需要字元時，傳遞做為參數 \<數位 > 的非字元 >

 此警告表示格式字串指定需要字元（例如，`%c` 或 `%C` 規格），但傳遞的是非整數（例如 float、string 或 struct）。 這種瑕疵可能會導致不正確的輸出。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```cpp
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 若要修正這個警告，請使用下列程式碼：

```cpp
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 下列程式碼使用安全字串操作函式（`sprintf_s`）來更正此警告：

```cpp
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
