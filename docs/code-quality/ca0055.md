# <a name="ca0055"></a>CA0055

無法載入檔案

程式碼分析工具無法載入為分析指定的目標檔案。 如果找不到檔案或已拒絕存取檔案，就會發生這種情況。 一般來說，如果使用者在 Visual Studio 中選擇覆寫輸入元件屬性，或以手動方式叫用 fxcopcmd.exe，就會發生這種情況。
