# <a name="ca0055"></a>CA0055
無法載入檔案：\<路徑 >

 程式碼分析工具無法載入進行分析所指定的目標檔案。 會發生這種情況是如果找不到檔案或檔案的存取被拒。 通常，這發生於如果使用者選擇覆寫在 Visual Studio 中，輸入組件屬性，或以手動方式叫用 fxcopcmd.exe。
