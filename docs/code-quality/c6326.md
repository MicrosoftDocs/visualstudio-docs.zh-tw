# <a name="c6326"></a>C6326
警告 C6326: 可能將常數和其他常數做比較

 此警告表示可能會比較常數與另一個常數，也就是多餘的程式碼。 您必須檢查以確定您的意圖已正確地在程式碼中捕捉。 在某些情況下，您可以簡化測試條件來達到相同的結果。

## <a name="example"></a>範例
 下列程式碼會產生這個警告，因為會比較兩個常數：

```cpp
#define LEVEL
const int STD_LEVEL = 5;

const int value =
#ifdef LEVEL
  10;
#else
  5;
#endif

void f()
{
  if( value > STD_LEVEL)
  {
    // code...
  }
  else
  {
    // code...
  }
}
```

 下列程式碼顯示使用 c + + 17 `if constexpr`更正此警告的一種方式。

```cpp
#define LEVEL
const int STD_LEVEL = 5;

const int value =
#ifdef LEVEL
  10;
#else
  5;
#endif

void f()
{
  if constexpr( value > STD_LEVEL)
  {
    // code...
  }
  else
  {
    // code...
  }
```

 下列程式碼顯示使用 #ifdef 語句來更正這個警告的其中一種方法，以判斷如果 c + + 17 無法使用時，應該執行的程式碼：

```cpp
#define LEVEL
const int STD_LEVEL = 5;

const int value =
#ifdef LEVEL
  10;
#else
  5;
#endif

void f ()
{
#ifdef LEVEL
  {
    // code...
  }
#else
  {
    // code...
  }
#endif
}
```
