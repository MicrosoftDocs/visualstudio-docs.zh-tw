# <a name="c28309"></a>C28309
警告28309： < parameter_name >：注釋運算元必須是整數/列舉/指標類型。 不支援 Void C++運算元和多載運算子。 浮動會近似為整數。 類型： \<typelist >。

 您已嘗試在批註運算式中使用 void 或函式，而程式碼分析無法處理它。  當使用實作為函式的 @no__t 0 時，通常會發生此錯誤，但也可能會發生其他情況。 檢查 \<typelist > 中的類型，以取得錯誤的線索。
