# <a name="c28228"></a>C28228
警告 C28228：注釋參數：不能在注釋中使用類型

 此警告表示參數的類型不受支援。 批註只能使用一組有限的類型做為參數。 如果標準宏正用於注釋，則不應該這麼做;一般用途不應出現此警告。
