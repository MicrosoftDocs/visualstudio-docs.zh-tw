# <a name="c6386"></a>C6386
警告 C6386：緩衝區溢位：存取 > 的 \<buffer 名稱，可寫入的大小 \<size1 > 個位元組，但可能會寫入 @no__t 2size2 > 位元組：行： x、y

 此警告表示指定緩衝區的可寫入範圍可能小於用來寫入它的索引。 這可能會造成緩衝區溢位。

## <a name="example"></a>範例
 下列程式碼會產生這個警告和[C6201](../code-quality/c6201.md)：

```
#define MAX 25

void f ( )
{
  char ar[MAX];
  //Code ...
  ar[MAX] = '\0';
}
```

 若要更正這兩個警告，請使用下列程式碼：

```
#define MAX 25

void f ( )
{
   char a[MAX];
   // code...
   a[MAX - 1] = '\0';
}
```
