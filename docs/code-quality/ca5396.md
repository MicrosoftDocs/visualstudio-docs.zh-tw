# <a name="ca5396-set-httponly-to-true-for-httpcookie"></a>CA5396：將 HttpCookie 的 HttpOnly 設定為 true

|||
|-|-|
|CheckId|CA5396|
|類別|Microsoft.Security|
|重大變更|不中斷|

## <a name="cause"></a>原因

<xref:System.Web.HttpCookie.HttpOnly?displayProperty=fullName> 設定為 `false`。 請注意，此屬性的預設值為 `false` 。

## <a name="rule-description"></a>規則描述

做為深層防禦措施，請確定安全性敏感的 HTTP cookie 已標示為 HttpOnly。 這表示 web 瀏覽器應該不允許腳本存取 cookie。 插入的惡意腳本是竊取 cookie 的常見方式。

## <a name="how-to-fix-violations"></a>如何修正違規

將 <xref:System.Web.HttpCookie.HttpOnly?displayProperty=fullName> 設定為 `true`。

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

- 如果已設定的全域值 <xref:System.Web.HttpCookie.HttpOnly> ，如下列範例所示：

```
<system.web>
    ...
    <httpCookies httpOnlyCookies="true" requireSSL="true" />
</system.web>
```

- 如果您確定 cookie 中沒有任何敏感性資料。

## <a name="pseudo-code-examples"></a>虛擬程式碼範例

### <a name="violation"></a>違規

```csharp
using System.Web;

class ExampleClass
{
    public void ExampleMethod()
    {
        HttpCookie httpCookie = new HttpCookie("cookieName");
        httpCookie.HttpOnly = false;
    }
}
```

### <a name="solution"></a>解決方法

```csharp
using System.Web;

class ExampleClass
{
    public void ExampleMethod()
    {
        HttpCookie httpCookie = new HttpCookie("cookieName");
        httpCookie.HttpOnly = true;
    }
}
```
