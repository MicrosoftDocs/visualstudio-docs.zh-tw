# <a name="c6302"></a>C6302
警告 C6302：格式字串不相符：當呼叫 \<函式需要寬字元字串時，傳遞做為參數 \<數位的字元字串 > >

 此警告表示格式字串指定需要寬字元字串。 不過，傳遞的是字元字串。 這項缺陷可能會導致損毀或某種形式的損毀。

## <a name="example"></a>範例
 下列範例程式碼會產生這個警告，因為字元字串會傳遞給 `wprintf` 函式：

```cpp
#include <stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%s", buff);
}
```

 下列範例程式碼會使用 `%hs` 來指定具有 `wprintf` 函數的單一位元組字元字串：

```cpp
#include <stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%hs", buff);
}
```

 下列範例程式碼使用安全字串操作函式 `wprintf_s` 來更正此警告：

```cpp
#include <stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf_s(L"%hs", buff);
}
```
