# <a name="ca5364-do-not-use-deprecated-security-protocols"></a>CA5364:請勿使用已取代的安全性通訊協定

|||
|-|-|
|TypeName|DoNotUseDeprecatedSecurityProtocols|
|CheckId|CA5364|
|Category|Microsoft.Security|
|中斷變更|非中斷|

## <a name="cause"></a>原因

符合下列任一條件時, 就會引發此規則:
- 參考的<xref:System.Net.SecurityProtocolType?displayProperty=nameWithType>是已被取代的值。
- 代表已被取代值的整數值已指派<xref:System.Net.SecurityProtocolType>給變數。

已取代的值為:
- Ssl3
- Tls
- Tls10
- Tls11

## <a name="rule-description"></a>規則描述

傳輸層安全性 (TLS) 可保護電腦之間的通訊, 最常見的方式是使用超文字安全傳輸通訊協定 (HTTPS)。 舊版的 TLS 通訊協定版本比 TLS 1.2 和 TLS 1.3 低, 而且更有可能會有新的弱點。 避免較舊的通訊協定版本, 將風險降至最低。 如需識別和移除已淘汰之通訊協定版本的指引, 請參閱[解決 TLS 1.0 問題第2版](/security/solving-tls1-problem)。

## <a name="how-to-fix-violations"></a>如何修正違規

請勿使用已淘汰的 TLS 通訊協定版本。

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

您可以在下列情況抑制此警告:
- 未使用已被取代之通訊協定版本的參考來啟用已淘汰的版本。
- 您必須連接到無法升級的舊版服務, 才能使用安全的 TLS 設定。

## <a name="pseudo-code-examples"></a>虛擬程式碼範例

### <a name="enumeration-name-violation"></a>列舉名稱違規

```csharp
using System;
using System.Net;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5364 violation
        ServicePointManager.SecurityProtocol = SecurityProtocolType.Tls11 | SecurityProtocolType.Tls12;
    }
}
```

```vb
Imports System
Imports System.Net

Public Class TestClass
    Public Sub ExampleMethod()
        ' CA5364 violation
        ServicePointManager.SecurityProtocol = SecurityProtocolType.Tls11 Or SecurityProtocolType.Tls12
    End Sub
End Class
```

### <a name="integer-value-violation"></a>整數值違規

```csharp
using System;
using System.Net;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5364 violation
        ServicePointManager.SecurityProtocol = (SecurityProtocolType) 768;    // TLS 1.1
    }
}
```

```vb
Imports System
Imports System.Net

Public Class TestClass
    Public Sub ExampleMethod()
        ' CA5364 violation
        ServicePointManager.SecurityProtocol = CType(768, SecurityProtocolType)   ' TLS 1.1
    End Sub
End Class
```

### <a name="solution"></a>方案

```csharp
using System;
using System.Net;

public class TestClass
{
    public void TestMethod()
    {
        // Let the operating system decide what TLS protocol version to use.
        // See https://docs.microsoft.com/dotnet/framework/network-programming/tls
    }
}
```

```vb
Imports System
Imports System.Net

Public Class TestClass
    Public Sub ExampleMethod()
        ' Let the operating system decide what TLS protocol version to use.
        ' See https://docs.microsoft.com/dotnet/framework/network-programming/tls
    End Sub
End Class
```
