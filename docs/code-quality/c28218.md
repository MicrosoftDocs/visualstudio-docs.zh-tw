# <a name="c28218"></a>C28218
警告 C28218：針對函式參數，注釋的參數不符合檔案中找到的

 在訊息的目前行和程式程式碼上的注釋不一致。 如果標準宏正用於注釋，則不應該這麼做;一般使用時，不應該看到這項警告。
