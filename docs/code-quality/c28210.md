# <a name="c28210"></a>C28210
警告28210：@No__t-0 內容的注釋不得在明確的前置內容中

 批註 `_On_failure_` 必須在 `__post` 內容中明確或隱含指出，也就是在函數傳回之後套用。  請使用 `_drv_out` 來確保這一點。
