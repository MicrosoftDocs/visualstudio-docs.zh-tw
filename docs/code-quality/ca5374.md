# <a name="ca5374-do-not-use-xsltransform"></a>CA5374：不要使用 XslTransform

|Item|值|
|-|-|
|CheckId|CA5374|
|類別|Microsoft.Security|
|重大變更|不中斷|

## <a name="cause"></a>原因

具現化 <xref:System.Xml.Xsl.XslTransform?displayProperty=nameWithType> ，這不會限制可能危險的外部參考或防止腳本。

## <a name="rule-description"></a>規則描述

<xref:System.Xml.Xsl.XslTransform>在不受信任的輸入上操作時，會受到攻擊。 攻擊可能會執行任意程式碼。

## <a name="how-to-fix-violations"></a>如何修正違規

將 <xref:System.Xml.Xsl.XslTransform> 取代為 <xref:System.Xml.Xsl.XslCompiledTransform?displayProperty=nameWithType>。 如需詳細指引，請參閱 [/dotnet/standard/data/xml/migrating-from-the-xsltransform-class]。

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

<xref:System.Xml.Xsl.XslTransform>物件、XSLT 樣式表單和 XML 來源資料全都來自信任的來源。 

## <a name="pseudo-code-examples"></a>虛擬程式碼範例

### <a name="violation"></a>違規

目前，下列虛擬程式碼範例說明此規則偵測到的模式。

```csharp
using System;
using System.Xml;
using System.Xml.XPath;
using System.Xml.Xsl;

namespace TestForXslTransform
{
    class Program
    {
        static void Main(string[] args)
        {
            // Create a new XslTransform object.
            XslTransform xslt = new XslTransform();

            // Load the stylesheet.
            xslt.Load("https://server/favorite.xsl");

            // Create a new XPathDocument and load the XML data to be transformed.
            XPathDocument mydata = new XPathDocument("inputdata.xml");

            // Create an XmlTextWriter which outputs to the console.
            XmlWriter writer = new XmlTextWriter(Console.Out);

            // Transform the data and send the output to the console.
            xslt.Transform(mydata, null, writer, null);
        }
    }
}
```

### <a name="solution"></a>解決方案

```csharp
using System.Xml;
using System.Xml.Xsl;

namespace TestForXslTransform
{
    class Program
    {
        static void Main(string[] args)
        {
            // Default XsltSettings constructor disables the XSLT document() function
            // and embedded script blocks.
            XsltSettings settings = new XsltSettings();

            // Execute the transform.
            XslCompiledTransform xslt = new XslCompiledTransform();
            xslt.Load("https://server/favorite.xsl", settings, new XmlUrlResolver());
            xslt.Transform("inputdata.xml", "outputdata.html");
        }
    }
}
```
