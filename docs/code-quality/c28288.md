# <a name="c28288"></a>C28288
警告 C28288：針對函式，\_At\_（）注釋的語法錯誤（不正確參數名稱）

 使用 `SAL_at` （`__drv_at`）注釋時，程式碼分析工具會報告這個警告，而且無法在目前內容中解讀參數運算式。 這可能包括使用拼錯的參數或成員名稱，或錯誤的 "return" 或 "this" 關鍵字。
