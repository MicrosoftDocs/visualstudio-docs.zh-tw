# <a name="c28288"></a>C28288
警告 C28288：函式的 \_At\_() 註釋中有語法錯誤 (無效的參數名稱)

 當使用 `SAL_at` （`__drv_at`）注釋，而且無法在目前內容中解讀參數運算式時，程式碼分析工具會報告這個警告。 這可能包括使用拼錯的參數或成員名稱，或錯誤的 "return" 或 "this" 關鍵字。
