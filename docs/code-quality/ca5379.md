# <a name="ca5379-do-not-use-weak-key-derivation-function-algorithm"></a>CA5379：請勿使用弱式金鑰衍生函數演算法

|||
|-|-|
|CheckId|CA5379|
|類別|Microsoft.Security|
|重大變更|不中斷|

## <a name="cause"></a>原因

在具現 <xref:System.Security.Cryptography.MD5?displayProperty=nameWithType> <xref:System.Security.Cryptography.SHA1?displayProperty=nameWithType> 化時使用或 <xref:System.Security.Cryptography.Rfc2898DeriveBytes?displayProperty=nameWithType> 。

## <a name="rule-description"></a>規則描述

<xref:System.Security.Cryptography.Rfc2898DeriveBytes>類別預設為使用 <xref:System.Security.Cryptography.HashAlgorithmName.SHA1> 演算法。 當具現化 <xref:System.Security.Cryptography.Rfc2898DeriveBytes> 物件時，您應該指定或更新版本的雜湊演算法 <xref:System.Security.Cryptography.HashAlgorithmName.SHA256> 。 請注意， <xref:System.Security.Cryptography.Rfc2898DeriveBytes.HashAlgorithm?displayProperty=nameWithType> 屬性只有 `get` 存取子。

## <a name="how-to-fix-violations"></a>如何修正違規

因為 <xref:System.Security.Cryptography.MD5> 或 <xref:System.Security.Cryptography.SHA1> 很容易發生衝突，請 <xref:System.Security.Cryptography.SHA256> 針對類別使用或更高版本 <xref:System.Security.Cryptography.Rfc2898DeriveBytes> 。

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

不建議您隱藏此規則，但應用程式相容性原因除外。

## <a name="pseudo-code-examples"></a>虛擬程式碼範例

### <a name="specify-hash-algorithm-in-constructor-violation"></a>指定違反函數的雜湊演算法

```csharp
using System.Security.Cryptography;

class ExampleClass
{
    public void ExampleMethod(byte[] password, byte[] salt, int iterations, HashAlgorithmName hashAlgorithm)
    {
        var rfc2898DeriveBytes = new Rfc2898DeriveBytes(password, salt, iterations, HashAlgorithmName.MD5);
    }
}
```

### <a name="specify-hash-algorithm-in-derived-class-constructor-violation"></a>指定衍生類別中的雜湊演算法違反了函數

```csharp
using System.Security.Cryptography;

class DerivedClass : Rfc2898DeriveBytes
{
    public DerivedClass (byte[] password, byte[] salt, int iterations, HashAlgorithmName hashAlgorithm) : base(password, salt, iterations, hashAlgorithm)
    {
    }
}

class ExampleClass
{
    public void ExampleMethod(byte[] password, byte[] salt, int iterations, HashAlgorithmName hashAlgorithm)
    {
        var derivedClass = new DerivedClass(password, salt, iterations, HashAlgorithmName.MD5);
    }
}
```

### <a name="set-hash-algorithm-property-in-derived-classes-violation"></a>衍生類別違規中的設定雜湊演算法屬性

```csharp
using System.Security.Cryptography;

class DerivedClass : Rfc2898DeriveBytes
{
    public DerivedClass (byte[] password, byte[] salt, int iterations, HashAlgorithmName hashAlgorithm) : base(password, salt, iterations, hashAlgorithm)
    {
    }

    public HashAlgorithmName HashAlgorithm { get; set;}
}

class ExampleClass
{
    public void ExampleMethod(byte[] password, byte[] salt, int iterations, HashAlgorithmName hashAlgorithm)
    {
        var derivedClass = new DerivedClass(password, salt, iterations, HashAlgorithmName.MD5);
        derivedClass.HashAlgorithm = HashAlgorithmName.SHA256;
    }
}
```

### <a name="solution"></a>解決方法

```csharp
using System.Security.Cryptography;

class ExampleClass
{
    public void ExampleMethod(byte[] password, byte[] salt, int iterations, HashAlgorithmName hashAlgorithm)
    {
        var rfc2898DeriveBytes = new Rfc2898DeriveBytes(password, salt, iterations, HashAlgorithmName.SHA256);
    }
}
```
