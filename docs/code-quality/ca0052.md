# <a name="ca0052"></a>CA0052

未選取要分析的目標。

有兩種情況可能會產生此錯誤：

- 如果 CA0001 或 CA0055 是當分析工具準備好要進行分析時未選取任何目標的根本原因，在 IDE 中使用 managed 程式碼分析工具可能會造成此錯誤。

- 如果在未使用/f 或/file 參數的情況下叫用 Fxcopcmd.exe，或是以/p 或/project 參數指向未列出任何目標的 FxCop 專案檔，則從命令列使用可能會導致此錯誤。 如需 Fxcopcmd.exe 的說明，請輸入 Fxcopcmd.exe/？ 在命令列上。

所有其他錯誤案例都應該伴隨其他訊息，指出錯誤的根本原因。
