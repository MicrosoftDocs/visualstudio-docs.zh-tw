# <a name="ca0503"></a>CA0503
屬性已被取代。 使用取代屬性

當已被取代的程式碼分析屬性包含組建或專案設定檔中的值時，就會產生警告 CA0503。 若要解決這個警告，請使用取代它的指定屬性。
