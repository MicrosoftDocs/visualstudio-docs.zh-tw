# <a name="c26496-use_const_for_variable"></a>C26496 USE_CONST_FOR_VARIABLE
  變數 '% variable% ' 只指派一次，將它標示為 `const`。 請參閱[ C++核心指導方針。 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction)。
