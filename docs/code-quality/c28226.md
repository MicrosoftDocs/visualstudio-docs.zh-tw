# <a name="c28226"></a>C28226
警告 C28226：注釋也不能是 PrimOp （在目前的宣告中）

 此警告表示已將命名的注釋宣告為 PrimOp，而且先前已宣告為一般批註。 如果標準宏正用於注釋，則不應該這麼做;一般使用時，不應該看到這項警告。
