# <a name="c6271"></a>C6271
警告 C6271：格式字串未使用傳遞至 \<function >： parameter \<number > 的額外引數

 此警告表示提供的其他引數超出格式字串所指定的參數。 此瑕疵本身本身並不會有任何明顯的效果，但這表示程式設計人員的意圖不會反映在程式碼中。

## <a name="example"></a>範例
 下列範例程式碼會產生這個警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d",1,2);
}
```

 若要修正這個警告，請使用下列範例程式碼：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d, %d",1,2);
}
```

 下列範例程式碼會呼叫安全字串操作函式 `sprintf_s`，以更正此警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf_s( buff, 5,"%s %d", 1,2 ); //safe version
}
```
