# <a name="ca0505"></a>CA0505

因為已定義取代屬性，所以會忽略被取代的屬性。

當已被取代和取代的程式碼分析屬性在專案或組建設定檔中具有值時，就會產生警告 CA505。 若要解決此問題，請移除已被取代的屬性。
