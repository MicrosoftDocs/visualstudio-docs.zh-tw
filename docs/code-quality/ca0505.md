# <a name="ca0505"></a>CA0505

因為 superceding 屬性定義，將會忽略已被取代的屬性。

這兩個已被取代，取代程式碼分析屬性在專案中有值，或建置組態檔時，會產生警告 CA505。 若要解決此問題，請移除已被取代的屬性。
