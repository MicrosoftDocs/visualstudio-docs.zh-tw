# <a name="c6289"></a>C6289
警告 C6289:不正確的運算子： 互斥的情況&#124;&#124;一律為非零的常數。 您是否想要使用 & & 改為？

 這則警告指出，測試運算式中的變數正在測試針對兩個不同的常數，而且結果取決於其中一個條件為真。 這一律評估為 true。

 此問題通常起因於使用`||`代替`&&`，但也使用，會造成`!=`其中`==`預定。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
void f(int x)
{
  if ((x != 1) || (x != 3))
  {
    // code
  }
}
```

 若要更正這個警告，請使用下列程式碼：

```
void f(int x)
{
  if ((x != 1) && (x != 3))
  {
    // code
  }
}

/* or */
void f(int x)
{
  if ((x == 1) || (x == 3))
  {
    // code
  }
}
```
