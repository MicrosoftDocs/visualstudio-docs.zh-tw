# <a name="c6289"></a>C6289
警告 C6289：不正確的運算子：對&#124; &#124;的相互排除一律為非零的常數。 您是否想要改用 & &？

 此警告表示在測試運算式中，變數正針對兩個不同的常數進行測試，而結果取決於任一條件為 true。 這一律會評估為 true。

 此問題通常是使用 `||` 來取代 `&&` 所造成，但也可能是使用 `==` 的 `!=` 所造成。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
void f(int x)
{
  if ((x != 1) || (x != 3))
  {
    // code
  }
}
```

 若要修正這個警告，請使用下列程式碼：

```
void f(int x)
{
  if ((x != 1) && (x != 3))
  {
    // code
  }
}

/* or */
void f(int x)
{
  if ((x == 1) || (x == 3))
  {
    // code
  }
}
```
