# <a name="c6315"></a>C6315
警告 C6315：不正確的作業順序：位 and 的優先順序高於位 or。 新增括弧以明確意圖

 此警告表示測試內容中的運算式同時包含位 and （`&`）和位 or （`|`）運算，但造成常數，因為位 or 運算最後發生。 應新增括弧以澄清意圖。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```cpp
void f( int i )
{
  if ( i & 2 | 4 ) // warning
  {
    // code
  }
}
```

 若要更正這個警告，請新增括弧，如下列程式碼所示：

```cpp
void f( int i )
{
  if ( i & ( 2 | 4 ) )
  {
    // code
  }
}
```
