# <a name="c6315"></a>C6315
警告 C6315:不正確的作業順序： 位元-and 的優先順序高於位-或。 加上括號釐清目的

 這則警告指出，在測試內容中的運算式同時包含位元-和 (`&`) 和位元-或 (`|`) 作業，但會產生常數，因為位元-or 運算是最後。 應該加入括號釐清目的。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
void f( int i )
{
  if ( i & 2 | 4 ) // warning
  {
    // code
  }
}
```

 若要更正這個警告，請加入括號，如下列程式碼所示：

```
void f( int i )
{
  if ( i & ( 2 | 4 ) )
  {
    // code
  }
}
```
