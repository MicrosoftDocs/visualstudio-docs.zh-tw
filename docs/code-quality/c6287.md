# <a name="c6287"></a>C6287
警告 C6287： 多餘的程式碼： 左邊和右邊的子運算式完全相同

 這則警告指出在運算式中偵測到重複的項目。

 很難判斷此問題的嚴重性而不需檢查程式碼。 重複的測試本身是無害的但刪除第二項測試的結果可能會很嚴重。 應檢查程式碼，以確保測試未被省略。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
void f(int x)
{
  if ((x == 1) && (x == 1))
  {
    //logic
  }
  if ((x != 1) || (x != 1))
  {
    //logic
  }
}
```

 下列程式碼示範各種方式若要更正這個警告：

```
void f(int x, int y)
{
  /* Remove the redundant sub-expression: */
  if (x == 1)
  {
     // logic
  }
  if (x != 1)
  {
    // logic
  }
  /* or test the missing variable: */
  if ((x == 1) && (y == 1))
  {
     // logic
  }
  if ((x != 1) || (y != 1))
  {
     // logic
  }
}
```
