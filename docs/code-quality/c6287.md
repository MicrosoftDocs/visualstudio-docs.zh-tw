# <a name="c6287"></a>C6287
警告 C6287: 多餘的程式碼: 左邊和右邊的子運算式完全相同

 此警告表示在運算式中偵測到多餘的元素。

 在不檢查程式碼的情況下，很容易判斷這個問題的嚴重性。 本身的重複測試是無害的，但刪除第二個測試的結果可能很嚴重。 應檢查程式碼，以確保不會省略測試。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```cpp
void f(int x)
{
  if ((x == 1) && (x == 1))
  {
    //logic
  }
  if ((x != 1) || (x != 1))
  {
    //logic
  }
}
```

 下列程式碼顯示各種更正此警告的方法：

```cpp
void f(int x, int y)
{
  /* Remove the redundant sub-expression: */
  if (x == 1)
  {
     // logic
  }
  if (x != 1)
  {
    // logic
  }
  /* or test the missing variable: */
  if ((x == 1) && (y == 1))
  {
     // logic
  }
  if ((x != 1) || (y != 1))
  {
     // logic
  }
}
```
