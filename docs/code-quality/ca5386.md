# <a name="ca5386-avoid-hardcoding-securityprotocoltype-value"></a>CA5386:避免將 SecurityProtocolType 值寫入程式碼

|||
|-|-|
|CheckId|CA5386|
|類別|Microsoft.Security|
|重大變更|不中斷|

## <a name="cause"></a>原因

符合下列任一條件時，就會引發此規則：
- 參考了安全但硬式編碼的 <xref:System.Net.SecurityProtocolType?displayProperty=nameWithType> 值。
- 代表已指派給變數之安全通訊協定版本的整數值 <xref:System.Net.SecurityProtocolType> 。

安全值為：
- Tls12
- Tls13

## <a name="rule-description"></a>規則描述

傳輸層安全性（TLS）可保護電腦之間的通訊，最常見的方式是使用超文字安全傳輸通訊協定（HTTPS）。 Tls 1.0 和 TLS 1.1 的通訊協定版本已被取代，而 TLS 1.2 和 TLS 1.3 是最新的。 未來，TLS 1.2 和 TLS 1.3 可能會被取代。 若要確保您的應用程式保持安全，請避免硬式編碼通訊協定版本，並以至少 .NET Framework v 4.7.1 為目標。 如需詳細資訊，請參閱[.NET Framework 的傳輸層安全性（TLS）最佳做法](/dotnet/framework/network-programming/tls)。

## <a name="how-to-fix-violations"></a>如何修正違規

不要硬式編碼 TLS 通訊協定版本。

## <a name="when-to-suppress-warnings"></a>隱藏警告的時機

如果您的應用程式是以 .NET Framework v 4.6.2 或更早版本為目標，而且可以在具有不安全預設值的電腦上執行，您可以隱藏此警告

## <a name="pseudo-code-examples"></a>虛擬程式碼範例

### <a name="enumeration-name-violation"></a>列舉名稱違規

```csharp
using System;
using System.Net;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5386 violation
        ServicePointManager.SecurityProtocol = SecurityProtocolType.Tls12;
    }
}
```

```vb
Imports System
Imports System.Net

Public Class TestClass
    Public Sub ExampleMethod()
        ' CA5386 violation
        ServicePointManager.SecurityProtocol = SecurityProtocolType.Tls12
    End Sub
End Class
```

### <a name="integer-value-violation"></a>整數值違規

```csharp
using System;
using System.Net;

public class ExampleClass
{
    public void ExampleMethod()
    {
        // CA5386 violation
        ServicePointManager.SecurityProtocol = (SecurityProtocolType) 3072;    // TLS 1.2
    }
}
```

```vb
Imports System
Imports System.Net

Public Class TestClass
    Public Sub ExampleMethod()
        ' CA5386 violation
        ServicePointManager.SecurityProtocol = CType(3072, SecurityProtocolType)   ' TLS 1.2
    End Sub
End Class
```

### <a name="solution"></a>解決方法

```csharp
using System;
using System.Net;

public class TestClass
{
    public void TestMethod()
    {
        // Let the operating system decide what TLS protocol version to use.
        // See https://docs.microsoft.com/dotnet/framework/network-programming/tls
    }
}
```

```vb
Imports System
Imports System.Net

Public Class TestClass
    Public Sub ExampleMethod()
        ' Let the operating system decide what TLS protocol version to use.
        ' See https://docs.microsoft.com/dotnet/framework/network-programming/tls
    End Sub
End Class
```
