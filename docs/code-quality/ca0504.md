# <a name="ca0504"></a>CA0504
已忽略規則目錄，因為它不存在

當找不到專案或組建設定檔案中的**CodeAnalysisRuleDirectories**屬性指定的目錄時，就會產生警告 CA504。 若要解決此問題，請輸入有效的目錄路徑。
