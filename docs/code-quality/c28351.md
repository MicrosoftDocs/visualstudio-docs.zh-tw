# <a name="c28351"></a>C28351
警告 C28351：注釋 \<批註 > 描述無法在條件中使用動態值（變數）的位置。

 當套用了要檢查 C/C++類型的注釋時，就會回報此警告。

 `_When_` 中的運算式應該評估為常數。 會忽略 `_When_`。
