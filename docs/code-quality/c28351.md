# <a name="c28351"></a>C28351
警告 C28351：批註 \<annotation > 描述在條件中無法使用動態值（變數）的位置。

 當套用了要檢查 C/C++類型的注釋時，就會回報此警告。

 @No__t 0 中的運算式應該評估為常數。 @No__t-0 會被忽略。
