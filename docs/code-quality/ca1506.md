# <a name="ca1506-avoid-excessive-class-coupling"></a>CA1506：應避免使用結合過度的類別

|||
|-|-|
|TypeName|AvoidExcessiveClassCoupling|
|CheckId|CA1506|
|類別|微軟.可維護性|
|重大變更|中斷|

## <a name="cause"></a>原因

類型或方法與許多其他類型耦合。 編譯器生成的類型將在此指標中排除。

## <a name="rule-description"></a>規則描述

這個規則會測量類別的耦合，方法是計算類型或方法包含的唯一類型參考數目。

具有高度類耦合的類型和方法可能難以維護。 最好有低耦合和高內聚的類型和方法。

## <a name="how-to-fix-violations"></a>如何修正違規

要修復此衝突,請嘗試重新設計類型或方法,以減少其耦合的類型數。

## <a name="when-to-suppress-warnings"></a>何時禁止警告

如果類型或方法被視為可維護的,儘管其依賴於其他類型的大量,則排除此警告。
