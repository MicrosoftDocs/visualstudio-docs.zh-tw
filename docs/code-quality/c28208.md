# <a name="c28208"></a>C28208
警告 C28208：函數 \<function > 先前是使用不同的參數清單在 \<file > （\<line >）定義。 有些分析工具會產生不正確的結果

 當函式的已知定義不符合另一個發生的情況時，就會回報此警告。
