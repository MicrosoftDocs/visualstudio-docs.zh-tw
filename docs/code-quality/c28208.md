# <a name="c28208"></a>C28208
警告 C28208:函式\<函式 > 先前已定義在不同的參數清單\<檔案 > (\<行 >)。 有些分析工具會產生不正確的結果

 函式的已知的定義不符合另一個相符項目時，會報告這個警告。
