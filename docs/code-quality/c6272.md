# <a name="c6272"></a>C6272
警告 C6272： \<function 的呼叫中需要 float 時，做為引數傳遞的非浮點數 \<number > >

 此警告表示格式字串指定需要 float，例如，`printf,` 的 `%f` 或 @no__t 1 規格，但傳遞的是整數或字串之類的非浮點數。 這項缺陷可能會導致不正確的輸出;不過，在某些情況下，可能會導致損毀。

## <a name="example"></a>範例
 下列程式碼會產生這個警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %f","a",i);
}
```

 若要更正這個警告，請使用 `%i`，而不是 `%f` 規格，如下列程式碼所示：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %i","a",i);
}
```

 下列程式碼使用安全字串操作函式（`sprintf_s`）來更正此警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf_s(buff,5,"%s %i","a",i); // safe version
}
```
