# <a name="c28213"></a>C28213
警告 C28213： `_Use_decl_annotations_` 注釋必須在先前的宣告中用來參考，而不需要修改。 \<why >

 參考先前的宣告，但沒有必要的 `_Use_decl_annotations_` 注釋時，就會回報此警告。
