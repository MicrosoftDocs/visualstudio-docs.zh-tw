# <a name="develop-apps-with-the-workflow-designer"></a>使用工作流程設計工具開發應用程式

工作流程設計工具是一種視覺化設計工具和偵錯工具，可在 Visual Studio 中 [Windows Workflow Foundation](/dotnet/framework/windows-workflow-foundation/index) (WF) 應用程式的圖形結構和偵測。 它可讓您透過使用範本和活動設計工具，來撰寫複合工作流應用程式、活動程式庫或 Windows Communication Foundation (WCF) 服務。

## <a name="install-windows-workflow-foundation"></a>安裝 Windows Workflow Foundation

若要在 Visual Studio 中使用工作流程專案範本，請先安裝 **Windows Workflow Foundation** 元件。

1. 開啟 Visual Studio 安裝程式。 若要開啟它，您可以選取 [**工具**  >  **取得工具和功能**] Visual Studio 中的快速方法。

1. 在 Visual Studio 安裝程式中，選取 [ **個別元件** ] 索引標籤。

1. 向下滾動至 [ **開發活動** ] 分類，然後選取 **Windows Workflow Foundation** 元件。

   ![Visual Studio 的 Windows Workflow Foundation 元件](media/windows-workflow-foundation-component.png)

1. 選取 [修改]。

   Visual Studio 安裝 **Windows Workflow Foundation** 元件。
