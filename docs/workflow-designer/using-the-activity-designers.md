# <a name="using-the-activity-designers"></a>使用活動設計工具

工作流程設計工具包含數個系統提供建構您的活動和工作流程時，您可以使用的活動。 本節描述系統供應的活動分類與其可用的設計工具。 這些類別內的個別活動與其個別設計工具的主題，均描述這些活動的功能，並提供如何使用活動的指引。
