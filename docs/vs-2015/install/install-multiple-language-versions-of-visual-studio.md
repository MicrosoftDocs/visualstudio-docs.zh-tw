# <a name="install-multiple-language-versions-of-visual-studio"></a>安裝 Visual studio 的多個語言版本
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

如需 Visual Studio 2017 最新文件，請參閱 < [Visual Studio 2017 中安裝語言套件](https://docs.microsoft.com/visualstudio/install/install-visual-studio#install-language-packs)。

藉由安裝一或多個語言套件，您可以在相同電腦上以其他語言執行 Visual Studio 2015。 您必須取得您想要安裝的每一種語言之個別語言套件。  
  
## <a name="important-considerations"></a>重要考量  
  
-   當您安裝語言套件時，已安裝功能的資源檔會自動為每個語言版本安裝。  
  
-   Visual Studio 服務套件套用至所有已安裝語言版本的 Visual Studio。 您不再需要為已安裝的 Visual Studio 每種語言版本安裝服務套件。  
  
-   您可以使用 Visual Studio 安裝程式的維護模式來解除安裝語言套件。 開啟 [控制台] ，開啟 [程式和功能] ，選擇您的 Visual Studio 版本，然後選擇 [解除安裝/變更] 。
