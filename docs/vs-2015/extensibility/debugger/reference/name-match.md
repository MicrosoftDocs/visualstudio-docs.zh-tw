# <a name="name_match"></a>NAME_MATCH
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

選取相符名稱的案例選項。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```  
  
## <a name="members"></a>成員  
 nmNone  
 未指定任何選項。  
  
 nmCaseSensitive  
 指出要比對的名稱會區分大小寫。  
  
 nmCaseInsensitive  
 指出要比對的名稱不區分大小寫。
