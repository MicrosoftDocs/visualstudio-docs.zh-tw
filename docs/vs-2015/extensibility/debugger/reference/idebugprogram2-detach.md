# <a name="idebugprogram2detach"></a>IDebugProgram2::Detach
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

從程式卸離 debug 引擎。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT Detach(   
   void   
);  
```  
  
```csharp  
int Detach();  
```  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回， `S_OK` 否則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 卸離的程式會繼續執行，但不再是 debug 會話的一部分。 卸離偵錯工具之後，就不會再傳送任何程式的偵錯工具事件。
