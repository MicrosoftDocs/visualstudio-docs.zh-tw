# <a name="stepkind"></a>STEPKIND
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

指定用於逐步執行步驟種類。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
enum enum_STEPKIND {   
   STEP_INTO      = 0,  
   STEP_OVER      = 1,  
   STEP_OUT       = 2,  
   STEP_BACKWARDS = 3  
};  
typedef DWORD STEPKIND;  
```  
  
```csharp  
public enum enum_STEPKIND {   
   STEP_INTO      = 0,  
   STEP_OVER      = 1,  
   STEP_OUT       = 2,  
   STEP_BACKWARDS = 3  
};  
```  
  
## <a name="members"></a>成員  
 STEP_INTO  
 逐步執行函式。  
  
 STEP_OVER  
 不是進入函式。  
  
 STEP_OUT  
 跳離函式。  
  
 STEP_BACKWARDS  
 向後逐步執行函式。  
  
## <a name="remarks"></a>備註  
 作為引數[步驟](../../../extensibility/debugger/reference/idebugprocess3-step.md)方法。
