# <a name="idebugprogramhost2gethostid"></a>IDebugProgramHost2::GetHostId
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

本主題的最新的版本可從[IDebugProgramHost2::GetHostId](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugprogramhost2-gethostid)。  
  
取得裝載此程式的處理序的處理序識別碼。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT GetHostId(   
   AD_PROCESS_ID* pdwId  
);  
```  
  
```csharp  
int GetHostId(   
   AD_PROCESS_ID[] pdwId  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pdwId`  
 [in、 out][AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md)會填入的處理序識別項資訊的結構。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回錯誤碼。
