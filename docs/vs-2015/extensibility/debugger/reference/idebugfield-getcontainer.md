# <a name="idebugfieldgetcontainer"></a>IDebugField::GetContainer
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

本主題的最新的版本可從[IDebugField::GetContainer](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugfield-getcontainer)。  
  
這個方法會取得容器的欄位。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT GetContainer(   
   IDebugContainerField** ppContainerField  
);  
```  
  
```csharp  
int GetContainer(  
   out IDebugContainerField ppContainerField  
);  
```  
  
#### <a name="parameters"></a>參數  
 `ppContainerField`  
 [out]表示所傳回的容器[IDebugContainerField](../../../extensibility/debugger/reference/idebugcontainerfield.md)介面。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 如果此欄位沒有容器，傳回的`ppContainerField`將會是 null 的值。
