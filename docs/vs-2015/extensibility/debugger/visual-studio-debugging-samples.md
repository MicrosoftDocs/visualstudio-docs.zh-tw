# <a name="visual-studio-debugging-samples"></a>Visual Studio 偵錯範例
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

兩個範例示範如何使用 [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)]擴充 Visual Studio 偵錯工具。  
  
- Managed 語言的偵錯工具運算式評估工具範例包含在 [!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)]中。 在 [[!INCLUDE[vsipsdk](../../includes/vsipsdk-md.md)]] 視窗中，按一下 [**範例**] 索引卷**C#** 標，按一下語言清單中的 [**全部**] 或，然後按一下 [**偵錯工具**]。  
  
- 您可以從 MSDN 程式碼庫取得適用于原生語言的 Visual Studio Debug Engine 範例。 如需詳細資訊，請參閱 Microsoft 網站上的這個網頁： [Visual Studio Debug Engine Sample](https://docs.microsoft.com/samples/browse/?redirectedfrom=MSDN-samples)。
