# <a name="extending-connected-services"></a>擴充已連線的服務
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio 已連線的服務 SDK 可讓服務提供者建立 Visual Studio 擴充功能，讓應用程式開發人員可以輕鬆地將其新增至專案，而不需要離開 Visual Studio IDE。
