# <a name="getting-started-with-source-control-vspackages"></a>原始檔控制 VSPackage 的使用者入門
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

原始檔控制 VSPackage 是更進階的替代方案，以原始檔控制外掛程式。 (請參閱[建立原始檔控制外掛程式](../../extensibility/internals/creating-a-source-control-plug-in.md)如需有關原始檔控制外掛程式。)原始檔控制 VSPackage 可讓原始檔控制模型、 功能和使用者介面 (UI) 的完整控制權，且它整合至[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)]為 VSPackage 的環境。
