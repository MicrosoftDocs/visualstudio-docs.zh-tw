# <a name="getting-started-with-source-control-vspackages"></a>原始檔控制 VSPackage 的使用者入門
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

原始檔控制 VSPackage 是更先進的原始檔控制外掛程式替代方案。  (如需原始檔控制外掛程式的詳細資訊，請參閱 [建立原始](../../extensibility/internals/creating-a-source-control-plug-in.md) 檔控制外掛程式。 ) 原始檔控制 VSPackage 可讓您完整控制原始檔控制模型、功能和使用者介面 (UI) ，並將其整合至環境中， [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] 以做為 VSPackage。
