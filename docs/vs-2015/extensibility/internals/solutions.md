# <a name="solutions"></a>方案
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

方案是一或多個專案的群組，這些專案會一起運作來建立應用程式。 與方案有關的專案和狀態資訊會儲存在兩個方案檔中：以文字為基礎的方案 ( .sln) 檔案，以及 ( .suo) 檔的二進位方案使用者選項。
