# <a name="managing-configuration-options"></a>管理組態選項
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

當您建立新的專案類型時，您必須管理專案和方案組態設定，以決定如何您將會建立專案、 封裝、 部署及執行。 下列主題將討論專案和方案組態。  
  
## <a name="in-this-section"></a>本節內容  
 [概觀](../../extensibility/internals/configuration-options-overview.md)  
 描述如何在專案[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)]可支援多個組態。  
  
 [屬性頁](../../extensibility/internals/property-pages.md)  
 說明使用者可以檢視和變更 使用屬性頁的 專案組態相依屬性與獨立屬性。  
  
 [方案組態](../../extensibility/internals/solution-configuration.md)  
 提供在 方案組態中儲存的內容和方案組態如何直接行為的相關資訊**開始**並**建置**命令。  
  
 [專案組態物件](../../extensibility/internals/project-configuration-object.md)  
 說明專案組態物件的管理 ui 的組態資訊的顯示方式。  
  
 [建置的專案組態](../../extensibility/internals/project-configuration-for-building.md)  
 說明如何為特定方案的方案組態的清單由管理**方案組態** 對話方塊。  
  
 [用以管理部署的專案組態](../../extensibility/internals/project-configuration-for-managing-deployment.md)  
 定義動作的部署和兩種方式[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)]支援支援部署的專案。  
  
 [輸出的專案組態](../../extensibility/internals/project-configuration-for-output.md)  
 建置程序可支援每個組態的介面和方法所輸出的項目可供說明。  
  
## <a name="related-sections"></a>相關章節  
 [專案類型](../../extensibility/internals/project-types.md)  
 提供專案的概觀，做為基本建置組塊的[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)]整合式的開發環境 (IDE)。 說明專案如何控制建置和編譯程式碼的其他主題會提供連結。
