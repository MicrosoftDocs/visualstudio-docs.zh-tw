將 Vspackage 及其對應的命令新增至 [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] 可能會導致擁擠的 UI。 [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] 提供可協助將 UI 命令混淆降至最低的方式。

 根據命令集的大小，提供執行 VSPackage 的特定指導方針。
