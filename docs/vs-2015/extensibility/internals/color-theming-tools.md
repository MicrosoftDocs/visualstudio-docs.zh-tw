# <a name="color-theming-tools"></a>色彩佈景主題設定工具
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

VS SDK 提供數個工具，可協助您建立和管理的色彩佈景主題。  
  
-   [VSIX 色彩編輯器](../../extensibility/internals/vsix-color-editor.md)工具可以建立和編輯 Visual studio 的 自訂色彩。  
  
-   [VSIX 色彩編譯器](../../extensibility/internals/vsix-color-compiler.md)工具將現有的 Visual Studio 佈景主題.xml 檔案轉換成您的擴充功能的.pkgdef。
