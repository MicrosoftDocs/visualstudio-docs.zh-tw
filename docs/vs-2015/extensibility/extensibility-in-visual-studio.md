# <a name="extensibility-in-visual-studio"></a>Visual Studio 的擴充性
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio 提供許多擴充性選項。 您可以建立自己的 Sdk、使用 Windows、Windows Phone 和 Azure Sdk (安裝為 Visual Studio) 的一部分，並擴充 Visual Studio 本身。

## <a name="extend-visual-studio"></a>擴充 Visual Studio
 您可以使用 Visual Studio SDK 來擴充幾乎 Visual Studio 的每個部分：命令、功能表、視窗、編輯器和專案。 若要深入瞭解，請參閱 [VISUAL STUDIO SDK](../extensibility/visual-studio-sdk.md)。

## <a name="create-your-own-sdks"></a>建立您自己的 Sdk
 瞭解如何建立、封裝及部署您自己的平臺和延伸模組 Sdk： [建立軟體發展工具組](../extensibility/creating-a-software-development-kit.md)。

## <a name="roslyn-extensibility"></a>Roslyn 擴充性
 瞭解如何使用 Roslyn 進行擴充性： [.NET Compiler Platform ( "Roslyn" ) ](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md)擴充性。

## <a name="azure-sdk"></a>Azure SDK
 瞭解如何啟用 azure SDK： [啟用 AZURE sdk](../extensibility/enabling-the-azure-sdk.md)。
