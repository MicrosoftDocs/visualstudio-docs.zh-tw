# <a name="setting-language-specific-editor-options"></a>設定語言特定編輯器選項
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] 提供多種 [文字編輯器] 選項，可以套用至特定的程式語言。

## <a name="in-this-section"></a>本節內容
 [選項、文字編輯器、基本 (Visual Basic) ](../../ide/reference/options-text-editor-basic-visual-basic.md) 描述程式碼的結束結構、程式碼重新格式化、大綱和錯誤修正建議等等的設定 [!INCLUDE[vbprvb](../../includes/vbprvb-md.md)] 。

 [選項、文字編輯器、c/c + +、格式](../../ide/reference/options-text-editor-c-cpp-formatting.md) 描述 C 和 c + + 程式碼的大綱、縮排、快速資訊和其他設定。

 [選項、文字編輯器、C/c + +、Advanced](../../ide/reference/options-text-editor-c-cpp-advanced.md) 描述在使用 C 和 c + + 時，IntelliSense 和資料庫檔案的設定。

 [選項、文字編輯器、c #、格式](../../ide/reference/options-text-editor-csharp-formatting.md) 描述 c # 和 j # 的縮排、新行格式化及換行文字等設定。

 [選項、文字編輯器、c #、Advanced](../../ide/reference/options-text-editor-csharp-advanced.md) 描述 c # 和 j # 的大綱、錯誤識別和 XML 檔批註設定。

 [選項、文字編輯器、c #、IntelliSense](../../ide/reference/options-text-editor-csharp-intellisense.md) 描述當您使用 c # 和 j # 程式碼工作時，指定 IntelliSense 完成清單行為的設定。

 [選項對話方塊、文字編輯器、CSS、驗證](https://msdn.microsoft.com/library/5afe0808-16bb-420f-b620-7ca1a4d9f2cc) 描述級聯樣式表 (CSS) 的語句完成和錯誤修正設定。

 [選項對話方塊、文字編輯器、HTML、驗證](https://msdn.microsoft.com/library/9c24ecfe-263e-4bf1-88de-d01be3992863) 描述根據您選取的架構檢查 HTML 語法的設定。

 [選項、文字編輯器、XAML、格式化](../../ide/reference/options-text-editor-xaml-formatting.md) 描述 XAML 檔中的元素和屬性排列設定。
