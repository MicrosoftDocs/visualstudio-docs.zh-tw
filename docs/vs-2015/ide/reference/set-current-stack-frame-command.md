# <a name="set-current-stack-frame-command"></a>設定目前堆疊框架命令
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

可讓您設定特定堆疊框架。

## <a name="syntax"></a>語法

```
Debug.SetCurrentStackFrame index
```

## <a name="arguments"></a>引數
 `index` 必要項。 依索引選取堆疊框架。

## <a name="example"></a>範例

```
>Debug.SetCurrentStackFrame 1
```
