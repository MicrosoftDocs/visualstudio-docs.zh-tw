# <a name="set-current-thread-command"></a>設定目前執行緒命令
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

  
將指定的執行緒設為目前執行緒。  
  
## <a name="syntax"></a>語法  
  
```  
Debug.SetCurrentThread index  
```  
  
## <a name="arguments"></a>引數  
 `index`  
 必要項。 依索引選取執行緒。  
  
## <a name="example"></a>範例  
  
```  
>Debug.SetCurrentThread 1  
```
