# <a name="extending-the-toolbox"></a>擴充工具箱
[!INCLUDE[vsprvs](../includes/vsprvs-md.md)] **工具箱**提供透過 IDE 的拖放機制，來為編輯器和設計工具提供功能的物件的集合。  
  
 有兩種基本方式，VSPackage 適用於[!INCLUDE[vsprvs](../includes/vsprvs-md.md)]**工具箱**:  
  
-   VSPackage 可以將新的資料項目和控制項加入 **工具箱**中。  
  
-   VSPackage 可以是現有 **工具箱** 功能的目標或取用者，並支援拖放作業及設定 **工具箱**的外觀。
