# <a name="build-cross-platform-mobile-apps-with-visual-c"></a>使用視覺效果建立跨平臺行動應用程式C++
您可以使用 Visual C++ for Cross-Platform Mobile Development 為 Android、iOS 和 Windows 裝置建置跨平台程式碼。 這是 Visual Studio 2015 中可用的選擇性功能，可以使用 Visual C++ 進行 iOS、Android 和 Windows 的跨平台程式碼開發。  
  
 如需安裝指示和程式碼範例，請參閱[適用于跨平臺行動裝置C++開發的視覺效果](../cross-platform/visual-cpp-for-cross-platform-mobile-development.md)。
