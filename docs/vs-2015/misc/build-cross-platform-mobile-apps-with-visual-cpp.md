# <a name="build-cross-platform-mobile-apps-with-visual-c"></a>使用 Visual C++ 建置跨平台行動裝置應用程式
您可以使用 Visual C++ for Cross-Platform Mobile Development 為 Android、iOS 和 Windows 裝置建置跨平台程式碼。 這是 Visual Studio 2015 中可用的選擇性功能，可以使用 Visual C++ 進行 iOS、Android 和 Windows 的跨平台程式碼開發。  
  
 如需安裝指示和程式碼範例，請參閱 [適用於跨平臺行動裝置應用程式開發的 Visual C++](../cross-platform/visual-cpp-for-cross-platform-mobile-development.md)。
