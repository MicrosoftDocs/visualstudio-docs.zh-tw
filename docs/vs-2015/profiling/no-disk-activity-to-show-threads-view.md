# <a name="no-disk-activity-to-show-threads-view"></a>沒有磁碟活動可顯示 (執行緒檢視)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

無法顯示磁碟 I/O 通道中位於目前時間範圍內的磁碟活動。 如果您縮小範圍或加以捲動，可能就會顯示磁碟活動。 此外，請確定並未隱藏任何磁碟通道。
