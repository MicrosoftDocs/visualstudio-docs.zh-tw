# <a name="spanspan-destructor"></a>span::~span 解構函式
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主題的最新的版本可從[s p a n:: ~ span 解構函式](https://docs.microsoft.com/visualstudio/profiling/span-tilde-span-destructor)。  
  
終結 `span` 物件，並釋放其資源。  
  
## <a name="syntax"></a>語法  
  
```  
~span();  
```  
  
## <a name="requirements"></a>需求  
 **標頭：** cvmarkersobj.h  
  
 **命名空間：** Concurrency::diagnostic
