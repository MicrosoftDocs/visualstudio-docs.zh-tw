# <a name="markerseriesmarkerseries-destructor"></a>marker_series::~marker_series 解構函式

本主題的最新的版本可從[marker_series:: ~ marker_series 解構函式](https://docs.microsoft.com/visualstudio/profiling/marker-series-tilde-marker-series-destructor)。  
  
終結 marker_series 物件並釋放所有配置的資源。

```  
~marker_series();  
```
