### <a name="to-customize-a-report-view"></a>自訂報表檢視  
  
1.  開啟報表檢視。  
  
2.  以滑鼠右鍵按一下報表檢視介面，然後按一下 [新增/移除資料行]。  
  
     [新增/移除資料行] 對話方塊隨即顯示。  
  
3.  選取您要新增的資料行。  
  
4.  按一下箭號按鈕以重新排列資料行。  
  
5.  按一下 [確定 **Deploying Office Solutions**]。
