# <a name="working-with-domain-specific-language-solutions"></a>使用網域指定的語言方案
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

特定領域語言 (DSL) 是使用解決方案所建立 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] 。 本節說明如何設定和自訂解決方案。

## <a name="in-this-section"></a>本章節內容
 [關於網域指定的語言](../modeling/about-domain-specific-languages.md) 討論如何開發 Dsl 以及如何套用它們。

 [特定領域語言工具總覽](../modeling/overview-of-domain-specific-language-tools.md) 說明開發和散佈 DSL 的主要工作。

 [特定領域語言工具消費者介面總覽](../modeling/overview-of-the-domain-specific-language-tools-user-interface.md) 描述與 Dsl 相關聯的視窗。

 [選擇特定領域語言方案範本](../modeling/choosing-a-domain-specific-language-solution-template.md) 說明如何選擇範本以啟動您的 DSL 專案。

 [如何：建立特定領域語言方案](../modeling/how-to-create-a-domain-specific-language-solution.md) 描述設定 DSL 的步驟。

 使用[DSL 定義圖表](../modeling/working-with-the-dsl-definition-diagram.md)說明圖表的各部分。

 使用[網域指定的語言瀏覽器](../modeling/working-with-the-domain-specific-language-explorer.md)說明 explorer 內容

 [如何：變更特定領域語言的命名空間](../modeling/how-to-change-the-namespace-of-a-domain-specific-language.md) 您從 DSL 產生的程式碼有一個命名空間，您可以在起始解決方案時設定命名空間，但之後可以變更它。

 [如何：擴充特定領域語言設計工具](../modeling/how-to-extend-the-domain-specific-language-designer.md) 您可以使用功能表命令自訂 DSL 設計工具。

 [如何自動化轉換所有範本](https://msdn.microsoft.com/b63cfe20-fe5e-47cc-9506-59b29bca768a) 若要避免按一下 [轉換所有範本]，您可以將它設為組建的自動部分。
