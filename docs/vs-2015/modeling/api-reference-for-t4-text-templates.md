# <a name="api-reference-for-t4-text-templates"></a>T4 文字範本 API 參考
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

文字模板化 API 可讓您叫用及自訂 [文字模板](../modeling/code-generation-and-t4-text-templates.md)的轉換。

## <a name="namespaces"></a>命名空間

|命名空間|目的|
|---------------|-------------|
|<xref:Microsoft.VisualStudio.TextTemplating>|包含文字模板轉換功能的類別。 文字模板轉換引擎已整合至 Visual Studio，並將文字模板檔案轉換成產生的文字輸出檔。|
|[模型化](/previous-versions/ee844312(v=vs.140))|提供與 UML 模型和網域特定語言相關的文字轉換工具，例如存取 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] ModelBus。|
|<xref:Microsoft.VisualStudio.TextTemplating.VSHost>|提供中文字模板化服務的存取 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] 。|
