# <a name="api-reference-for-t4-text-templates"></a>T4 文字範本 API 參考
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

文字模板化 API 可讓您叫用及自訂[文字模板](../modeling/code-generation-and-t4-text-templates.md)的轉換。

## <a name="namespaces"></a>命名空間

|命名空間|用途|
|---------------|-------------|
|<xref:Microsoft.VisualStudio.TextTemplating>|包含文字模板轉換功能的類別。 文字模板轉換引擎已整合到 Visual Studio, 並將文字模板檔案轉換成產生的文字輸出檔案。|
|[模型](/previous-versions/ee844312(v=vs.140))|提供與 UML 模型和特定領域語言相關的文字轉換設備, 例如[!INCLUDE[vsprvs](../includes/vsprvs-md.md)] ModelBus 的存取權。|
|<xref:Microsoft.VisualStudio.TextTemplating.VSHost>|提供中[!INCLUDE[vsprvs](../includes/vsprvs-md.md)]文字模板化服務的存取權。|
