# <a name="creating-a-wpf-based-domain-specific-language"></a>建立 WPF 架構之網域指定的語言
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

您可以建立具有 WPF 設計工具的特定領域語言，而不是圖形化設計工具。

 這項功能的相關資訊和範例可在 [Visual Studio 視覺效果和模型 SDK](https://www.microsoft.com/download/details.aspx?id=48148)中找到。
