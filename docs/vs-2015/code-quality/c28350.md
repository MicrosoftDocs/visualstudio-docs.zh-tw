# <a name="c28350"></a>C28350
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28350：批註 \<annotation> 會描述未有條件地適用的狀況。  
  
 通常在檢查 C/c + + 類型的位置套用批註時，會產生此警告。
