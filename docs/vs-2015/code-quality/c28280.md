# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28280：針對符號，找到 ' end ' 但沒有相符的 ' begin '  
  
 注釋語言支援 begin 和 end (`{` 以及 `}` C) 結構中，而配對的功能卻不平衡。 如果使用宏，則可以避免此情況。
