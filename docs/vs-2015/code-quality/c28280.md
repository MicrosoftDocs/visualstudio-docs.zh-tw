# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28280： 對於符號，'end' 找不到相符的 'begin'  
  
 註釋語言支援 begin 和 end (`{`和`}`C 中) 建構，並將其配對有點不對稱。 您可以避免這種情況下，如果使用了巨集。
