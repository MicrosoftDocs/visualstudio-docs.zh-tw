# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28020： 運算式\<expr > 不是在此呼叫，則為 true  
  
 會報告這個警告時\_Satisfies\_所列的運算式不是 true。 通常這表示不正確的參數。  
  
 如果發生這種情況在函式宣告，註解會指出不可能的條件。
