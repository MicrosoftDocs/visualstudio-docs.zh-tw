# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28020： \<expr> 此呼叫的運算式不是 true  
  
 當 \_ 列出的滿足運算式不成立時，就會回報此警告 \_ 。 這通常表示不正確的參數。  
  
 如果在函式宣告上發生這種情況，批註會指出不可能的狀況。
