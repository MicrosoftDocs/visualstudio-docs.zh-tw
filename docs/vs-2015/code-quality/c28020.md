# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28020：此呼叫的運算式 \<expr > 不是 true  
  
 當 \_符合列出的\_ 運算式不是 true 時，就會回報此警告。 這通常表示不正確的參數。  
  
 如果這種情況發生在函式宣告中，注釋就表示不可能的狀況。
