# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28021： > 以 \<西元 > 標注的參數 \<param，必須是指標  
  
 當標注的物件不是指標類型時，就會回報此警告。 此注釋不能與 `void` 或整數類型一起使用。
