# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28021:參數\<param > 所標註\<anno > 必須是指標  
  
 所標註的物件不是指標類型時，會報告這個警告。 此註解不能搭配`void`或整數類資料類型。
