# <a name="c6316"></a>C6316
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6316：不正確的運算子：測試的運算式是常數且非零。 使用位 and 來判斷是否已設定位  
  
 此警告表示在 `|` 應該使用位 and () 時，要使用位或 () `&` 。 位 or 會將位加入至產生的運算式，而位並只選取其兩個運算子之間共通的位。 旗標的測試必須使用位 and 或相等的測試來執行。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#define INPUT_VALUE 2  
void f( int Flags)  
{  
  if (Flags | INPUT_VALUE) // warning  
  {  
    // code  
  }  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
#define ALLOWED 1  
#define INPUT_VALUE 2  
  
void f( int Flags)  
{  
  if ((Flags & INPUT_VALUE) == ALLOWED)  
  {  
    // code  
  }  
}  
```
