# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>測量 Managed 程式碼的複雜度和維護性
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

新式軟體應用程式增加的複雜度也會提高使程式碼可靠且可維護的困難。 在最近幾年，已開發了許多軟體量值（稱為程式碼度量），可協助開發人員瞭解其程式碼需要重新修改或增加測試的位置。

 開發人員可以使用 [!INCLUDE[vsprvsts](../includes/vsprvsts-md.md)] 來產生程式碼度量資料，以測量其 managed 程式碼的複雜性和可維護性。 您可以針對整個方案或單一專案產生程式碼度量資料。

 本節說明如何在中產生和使用程式碼度量 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] 。
