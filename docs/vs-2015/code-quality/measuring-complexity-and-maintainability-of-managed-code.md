# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>測量 Managed 程式碼的複雜度和維護性
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

新式軟體應用程式的複雜性也增加了讓程式碼可靠和維護的難度。 最近幾年來，我們開發了許多軟體量值，也稱為程式碼計量，可協助開發人員瞭解他們的程式碼需要修改或增加測試的位置。

 開發人員可以使用 [!INCLUDE[vsprvsts](../includes/vsprvsts-md.md)] 來產生程式碼計量資料，以測量其 managed 程式碼的複雜度和維護性。 可以針對整個方案或單一專案產生程式碼計量資料。

 本節說明如何產生及使用 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] 中的程式碼計量。
