# <a name="c28109"></a>C28109
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28109：呼叫函數時不能保留變數  
  
 此程式所持有的資源，不應在呼叫此函式時保留。 通常，這表示資源會不慎取得兩次。 當資源在相同的函式中取得和釋放時，程式碼分析工具會回報此警告。  
  
## <a name="example"></a>範例  
 下列程式碼範例會產生此警告：  
  
```  
ExAcquireResourceLite(resource, true);  
...  
ExAcquireResourceLite(resource, true);  
```  
  
 下列程式碼範例可避免此警告：  
  
```  
ExAcquireResourceLite(resource, true);  
```
