# <a name="c6510"></a>C6510
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6510：不正確注釋： ' NullTerminated ' 屬性只能用在其元素為整數或指標類型的緩衝區上：函式 ' \<function> ' \<parameter> 。  
  
 此警告表示 **NullTerminated** 屬性的使用不正確， (結尾為 ' `_z` ' ) 。 您只能在指標或陣列類型上使用這種類型的屬性。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#include <sal.h>  
  
void f(_In_z_ char x)  
{  
    // code ...  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
#include <sal.h>  
  
void f(_In_z_ char * x)  
{  
    // code ...  
}  
```
