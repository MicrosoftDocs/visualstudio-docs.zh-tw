# <a name="c6201"></a>C6201
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6201：的緩衝區溢位 \<variable> ，可能已配置堆疊：索引超出 \<name> 有效的索引範圍 \<min> 到 \<max>  
  
 此警告表示指定堆疊陣列中的整數位移超過該陣列的最大界限。 此瑕疵可能會造成隨機行為或損毀。  
  
 這項瑕疵的一個常見原因是使用陣列的大小做為陣列中的索引。 由於 C/c + + 陣列索引是以零為基底，因此陣列中的最大合法索引會比陣列元素的數目少一個。  
  
## <a name="example"></a>範例  
 下列程式碼會產生此警告，因為陣列索引超出有效範圍：  
  
```  
void f( )  
{  
  int buff[25];  
  for (int i=0; i <= 25; i++) // i exceeds array bound  
  {  
    buff[i]=0; // initialize i  
    // code ...  
  }  
}  
  
```  
  
 若要更正這兩個警告，請使用正確的陣列大小，如下列程式碼所示：  
  
```  
void f( )  
{  
  int buff[25];  
  for (int i=0; i < 25; i++)  
  {  
    buff[i]=0; // initialize i  
    // code ...  
  }  
}  
```
