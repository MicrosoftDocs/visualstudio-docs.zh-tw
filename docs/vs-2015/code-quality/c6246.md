# <a name="c6246"></a>C6246
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6246：區域宣告會 \<variable> 隱藏外部範圍中相同名稱的宣告。 其他資訊：請參閱先前的宣告 \<location> 。  
  
 此警告表示兩個宣告在本機範圍具有相同的名稱。 外部範圍的名稱是由內部範圍內的宣告所隱藏。 任何預期的外部範圍宣告用途都會導致使用區域宣告。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#include <stdlib.h>  
#define UPPER_LIMIT 256  
int DoSomething( );  
  
int f( )  
{  
  int i = DoSomething( );  
  if (i > UPPER_LIMIT)  
  {  
    int i;  
    i = rand( );  
  }  
  return i;  
}  
```  
  
 若要更正這個警告，請使用另一個變數名稱，如下列程式碼所示：  
  
```  
#include <stdlib.h>  
#define UPPER_LIMIT 256  
int DoSomething( );  
  
int f ( )  
{  
  int i = DoSomething( );  
  if (i > UPPER_LIMIT)  
  {  
    int j = rand( );  
    return j;  
  }  
  else  
  {  
    return i;  
  }  
}  
```  
  
 此警告只會識別範圍重迭。
