# <a name="c6317"></a>C6317
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6317：不正確的運算子：邏輯 not （！）無法與一補數（~）交換  
  
 此警告表示邏輯 not （`!`）正套用至可能是位旗標的常數。 邏輯 not 的結果是布林值;將位 and （`&`）運算子套用至布林值是不正確的。 使用一補數（`~`）運算子來翻轉旗標。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#define FLAGS   0x4004  
  
void f(int i)  
{  
  if (i & !FLAGS) // warning  
  {   
    // code  
  }  
}  
```  
  
 若要修正這個警告，請使用下列程式碼：  
  
```  
#define FLAGS   0x4004  
  
void f(int i)  
{  
  if (i & ~FLAGS)  
  {  
    // code  
  }  
}  
```
