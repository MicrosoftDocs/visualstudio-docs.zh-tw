# <a name="c28195"></a>C28195
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28195: 函式宣告為會在變數中取得記憶體，但未執行此動作即結束  
  
 此警告表示所分析之函式的函式原型具有 `__drv_acquiresMemory` 注釋。 `__drv_acquiresMemory`批註指出函式會取得指定之結果位置中的記憶體，但至少有一個路徑中的函式未取得記憶體。 請注意，程式碼分析工具將無法辨識 (涉及位址算術) 的記憶體配置器實際執行，而且將無法辨識配置 (的記憶體，雖然有許多包裝函式) 會被辨識。 在此情況下，程式碼分析工具無法辨識已配置的記憶體，併發出此警告。 若要隱藏誤報，請在函式 `#pragma` 主體的左大括弧前面的行上使用警告 `{`
