# <a name="c1250"></a>C1250
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1250：無法載入外掛程式。  
  
 程式碼分析工具會在外掛程式中發生內部錯誤（而不是在分析的程式碼中）時，報告這個警告。
