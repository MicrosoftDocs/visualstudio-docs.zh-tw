# <a name="c6412"></a>C6412
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6412：寫入緩衝區時可能發生緩衝區溢位。 可寫入的大小是 *write_size* 個位元組，但可能會寫入 *write_index* 個位元組。  
  
 此警告表示用來寫入緩衝區的索引值可能會超過緩衝區的可寫入大小。  
  
 由於程式碼分析工具會在無法減少表示緩衝區大小的複雜運算式，或是用來存取緩衝區的索引時回報此警告，因此可能會在錯誤中報告此警告。  
  
## <a name="example"></a>範例  
 下列程式碼會產生此警告。  
  
```  
char *a = new char[strlen(InputParam)];  
a[10] = 1;  
delete[] a;  
```  
  
 下列程式碼會修正這個錯誤。  
  
```  
int i = strlen(InputParam);  
char *a = new char[i];  
if (i > 10) a[10] = 1;  
delete[] a;  
```
