[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1256： PCH 必須同時為寫入和讀取。  
  
 當工具中有內部錯誤，而不是在分析的程式碼中時，程式碼分析工具會報告此警告。
