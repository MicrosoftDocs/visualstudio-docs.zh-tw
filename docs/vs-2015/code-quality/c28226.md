# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28226：注釋也不能是目前宣告中的 PrimOp ()   
  
 此警告表示命名的注釋已宣告為 PrimOp，而且先前宣告為一般批註。 如果標準宏用於批註，則應該不可能這麼做;正常使用時，不應該看到此警告。
