# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28226： 註釋不能也是 PrimOp （在目前宣告中）  
  
 這則警告表示具名的註解會被宣告為 PrimOp，，和也先前宣告為一般的註解。 這不能進行標準巨集所使用的註解;這個警告不應出現在正常使用。
