# <a name="c6258"></a>C6258
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6258：使用 TerminateThread 不允許適當的執行緒清除。  
  
 此警告表示已偵測到對 TerminateThread 的呼叫。  
  
 TerminateThread 是一項危險的函式，只應用於最極端的情況。 如需 TerminateThread 呼叫相關問題的詳細資訊，請參閱 MSDN Library 中的這個主題： [TerminateThread](https://msdn.microsoft.com/library/ms686717(VS.85).aspx)。  
  
### <a name="to-properly-terminate-threads"></a>適當地終止執行緒  
  
1. 使用 `CreateEvent` 函數建立事件物件。  
  
2. 建立執行緒。  
  
3. 每個執行緒會藉由呼叫 `WaitForSingleObject` 函數來監視事件狀態。  
  
4. 當事件設定為已發出信號的狀態時，每個執行緒都會結束自己的執行（`WaitForSingleObject` 會傳回 `WAIT_OBJECT_0`）。  
  
   如需詳細資訊，請參閱下列其中一個主題：  
  
   [終止執行緒](https://msdn.microsoft.com/library/ms686724(VS.85).aspx)  
  
   [WaitForSingleObject](https://msdn.microsoft.com/library/ms687032.aspx)  
  
   [SetEvent](https://msdn.microsoft.com/library/ms686211(VS.85).aspx)
