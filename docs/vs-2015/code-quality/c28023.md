# <a name="c28023"></a>C28023
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28023：所指派或傳遞的函式 `_Function_class_` 在中至少應該有一個類別 () 的注釋： \<classlist>  
  
 當只有一個函式類別正在使用中，且未宣告適當類型的回呼時，通常會報告此警告。  
  
 此警告會在指派的左邊的函式 (或隱含指派時發出，如果這是函式呼叫) 會標注，表示它是使用批註的驅動程式特定函式類型， `_Function_class_` 或是包含這類批註的 typedef。 在指派右邊的函數沒有 `_Function_class_` 注釋。 右邊的函式應標注為與左側函式相同的類型。 這通常是藉由 \<class1> \<funcname1> 在的目前第一個宣告之前加入宣告來達到最佳效果 \<funcname2> 。
