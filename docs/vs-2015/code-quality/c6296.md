# <a name="c6296"></a>C6296
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6296：對-迴圈定義錯誤：迴圈主體只執行一次  
  
 此警告表示 for 迴圈可能無法如預期運作。 當索引不帶正負號，且迴圈從零開始計算時，其主體只會執行一次。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
