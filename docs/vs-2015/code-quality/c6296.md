# <a name="c6296"></a>C6296
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6296： 定義錯誤的 for 迴圈： 迴圈主體僅執行一次  
  
 這則警告指出 for 迴圈可能無法運作正常。 當索引是不帶正負號和向下計數從零的迴圈時，一次執行其主體。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
