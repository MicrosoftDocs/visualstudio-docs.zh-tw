# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28229：注釋不支援參數  
  
 此警告表示在宣告不含參數的注釋時，與參數搭配使用批註。 如果標準宏用於批註，則應該不可能這麼做;這項警告在一般使用上並不會出現。
