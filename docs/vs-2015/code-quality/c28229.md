# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28229:註釋不支援參數  
  
 這個警告表示註釋宣告不含參數時，已與參數搭配使用註解。 這不能進行標準巨集所使用的註解;這個警告不應該看到典型的使用中。
