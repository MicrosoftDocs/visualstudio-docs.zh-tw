# <a name="c28159"></a>C28159
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28159：請考慮改為使用其他函數。  
  
 系統會針對驅動程式回報此警告，建議您使用與驅動程式所呼叫之函式語義相等的慣用函式呼叫。 這是一般警告訊息;使用批註 `__drv_preferredFunction` (可能有條件式 a `__drv_when` ( # A2 注釋) 來標示不正確的編碼作法。  
  
## <a name="example"></a>範例  
 下列程式碼範例會產生此警告：  
  
```  
char buff[MAX_PATH];  
  
OemToChar(buff, input);  
  
    // if strlen(input) > MAX_PATH  
....// leads to buffer overrun  
```  
  
 下列程式碼範例可避免此警告：  
  
```  
char buff[MAX_PATH];  
  
OemToCharBuff(buff, input, MAX_PATH);  
```
