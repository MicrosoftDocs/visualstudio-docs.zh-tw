# <a name="c28204"></a>C28204
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28204： \<function> ：只有這個多載的其中一個，而 \<filename> () 的其中一個 \<line> 已標注為 \<paramname> ：兩者或兩者都必須加上批註。  
  
 當批註中有錯誤時，就會回報此警告。
