# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28232： \_ Pre \_ 、 \_ Post \_ 或 \_ Deref \_ 未套用至任何注釋  
  
 此警告表示 `_Pre_` ， `_Post_` 或 `_Deref_` 運算子出現在不含後續功能性注釋的批註運算式中; 已忽略修飾詞，但這表示不正確的程式碼批註。
