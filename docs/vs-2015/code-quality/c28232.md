# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28232:\_Pre\_， \_Post\_，或\_Deref\_不會套用至任何註釋  
  
 這個警告指出`_Pre_`， `_Post_`，或`_Deref_`運算子會出現在 如果沒有後續的功能性註解註釋運算式; 修飾詞會被忽略，但這表示不正確地自動程式碼註解。
