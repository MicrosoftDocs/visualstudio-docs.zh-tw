# <a name="c6552"></a>C6552
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6552：無效 `Deref=` 或 `Notref=` ：無法剖析運算式  
  
> [!NOTE]
> 只有使用已淘汰版本的原始程式碼批註語言 (SAL) 的程式碼才會發生這個警告。 建議您移植程式碼，以使用最新版本的 SAL。 如需詳細資訊，請參閱 [使用 SAL 注釋減少 C/c + + 程式碼](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)缺失。
