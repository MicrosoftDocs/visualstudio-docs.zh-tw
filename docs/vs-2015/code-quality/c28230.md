# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主題的最新的版本可從[C28230](https://docs.microsoft.com/visualstudio/code-quality/c28230)。  
  
警告 C28230： 參數的型別有任何成員。  
  
 這則警告指出，註解的引數會嘗試存取`struct`， `class`，或`union`和具名的成員不存在。
