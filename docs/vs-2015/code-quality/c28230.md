# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28230：參數的類型沒有成員。  
  
 此警告表示批註的引數嘗試存取 `struct`、`class`或 `union`，而且命名成員不存在。
