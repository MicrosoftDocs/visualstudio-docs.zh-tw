# <a name="c1255"></a>C1255
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主題的最新的版本可從[C1255](https://docs.microsoft.com/visualstudio/code-quality/c1255)。  
  
警告 C1255： 外掛程式的 PCH 資料長度不正確。  
  
 在工具中，而不會被分析的程式碼發生內部錯誤時，程式碼分析工具會報告這個警告。
