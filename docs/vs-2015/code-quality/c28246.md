# <a name="c28246"></a>C28246
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28246：函數 '\<名稱 > '-參數 '\<參數 > ' 的注釋不符合參數的類型  
  
 建立注釋時，會將 __deref 運算子套用至非指標類型。
