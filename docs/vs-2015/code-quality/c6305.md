# <a name="c6305"></a>C6305
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6305: sizeof 和 countof 的數量可能不符  
  
 此警告表示變數持有`sizeof`結果正在加入或減去指標或`countof`運算式。 這會導致未預期的調整，在指標算術。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f(int *p)   
{   
  int cb=sizeof(int);   
  //code...   
  p +=cb; // warning 6305  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f(int *p)   
{  
  // code...  
  p += 1;   
}  
```
