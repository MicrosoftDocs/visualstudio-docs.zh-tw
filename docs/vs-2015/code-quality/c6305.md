# <a name="c6305"></a>C6305
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6305： sizeof 和 countof 數量之間可能不相符  
  
 此警告表示 `sizeof` 會在指標或運算式中加入或減去包含結果的變數 `countof` 。 這會導致指標算術中發生未預期的縮放。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f(int *p)   
{   
  int cb=sizeof(int);   
  //code...   
  p +=cb; // warning 6305  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f(int *p)   
{  
  // code...  
  p += 1;   
}  
```
