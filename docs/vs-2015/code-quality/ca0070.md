# <a name="ca0070"></a>CA0070

當程式碼分析分析元件並遇到許可權屬性，而且至少下列其中一個條件成立時，就會發生這個錯誤：

- 程式碼分析找不到包含屬性的元件。

- 程式碼分析沒有載入元件的許可權。

- 程式碼分析所載入的元件不包含屬性。

在大部分情況下，您可以藉由確定您擁有最新版的程式碼分析工具來解決此問題。
