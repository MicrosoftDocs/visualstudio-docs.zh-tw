# <a name="ca0070"></a>CA0070

當程式碼分析會分析組件，且遇到權限屬性且至少其中一個下列條件成立時，就會發生此錯誤：

-   程式碼分析中，找不到包含屬性的組件。

-   程式碼分析並沒有載入組件的權限。

-   載入程式碼分析的組件不包含屬性。

在大部分情況下，確定您有最新版的程式碼分析工具可以解決此問題。
