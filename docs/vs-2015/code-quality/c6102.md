# <a name="c6102"></a>C6102
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6102： \<variable> 在行上使用 from 失敗的函式呼叫 \<location> 。  
  
 未設定變數時，會報告此警告，而不是 [C6001](../code-quality/c6001.md) ，因為它 `_Out_` 在先前的函式呼叫上被標示為已失敗的參數。  
  
 問題可能在於之前呼叫的函式沒有完整標註。 它可能需要 `_Always_`、`_Outptr_result_nullonfailure_` (COM 程式碼的 `_COM_Outptr_`) 或相關註釋。
