# <a name="c6306"></a>C6306
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6306：不正確的呼叫 \<function> ：請考慮使用 \<function> 它接受 va_list 做為引數  
  
 此警告表示函式呼叫不正確。 此 `printf` 系列包含數個使用引數清單的函式; 不過，您無法使用引數來呼叫這些函數 `va_list` 。 有一系列對應 `vprintf` 的函式可用於這類呼叫。 呼叫錯誤的列印函式將會造成不正確的輸出。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#include <stdio.h>  
#include <stdarg.h>  
  
void f(int i, ...)  
{  
   va_list v;  
  
   va_start(v, i);  
   //code...  
   printf("%s", v); // warning 6306   
   va_end(v);  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
#include <stdio.h>  
#include <stdarg.h>  
  
void f(int i, ...)  
{  
   va_list v;  
  
   va_start(v, i);  
   //code...  
   vprintf_s("%d",v);  
   va_end(v);  
}  
```
