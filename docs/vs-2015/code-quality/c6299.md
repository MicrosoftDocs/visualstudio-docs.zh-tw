# <a name="c6299"></a>C6299
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6299：將位欄位明確地與布林值型別進行比較會產生非預期的結果  
  
 此警告表示布林值和位欄位相等時，不正確的假設。 指派1個到位欄位會在其單一位中放置 1;不過，此位欄位與1之間的任何比較，都包含將位欄位隱含轉換成帶正負號的整數。這種轉換會將儲存的1轉換為-1，而比較可能會產生非預期的結果。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
struct myBits  
{  
  short flag : 1;  
  short done : 1;  
  //other members  
} bitType;  
  
void f( )  
{  
  if (bitType.flag == 1)   
  {  
  // code...  
  }  
}  
```  
  
 若要更正這個警告，請使用位欄位，如下列程式碼所示：  
  
```  
void f ()  
{  
  if(bitType.flag==bitType.done)  
  {  
    // code...  
  }  
}  
```
