# <a name="c28218"></a>C28218
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主題的最新的版本可從[C28218](https://docs.microsoft.com/visualstudio/code-quality/c28218)。  
  
警告 C28218： 對於函式參數，註釋的參數不符合檔案中找到的  
  
 目前這一行，並在訊息中的一行上的註釋不一致。 這不能進行標準巨集所使用的註解;這個警告不應出現在正常使用。
