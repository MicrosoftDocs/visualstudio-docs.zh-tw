# <a name="c28218"></a>C28218
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28218：針對函式參數，注釋的參數不符合檔案中找到的參數  
  
 目前行上和訊息行中的注釋不一致。 如果標準宏用於批註，則應該不可能這麼做;正常使用時，不應該看到此警告。
