# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28216： \_ Check_return \_ 注釋只適用于特定函式參數的後置條件。  
  
 批註已套用至以外的 `_Check_return_` 內容 `__post` ; 它可能需要 `__post` (或 `__drv_out`) 修飾詞，否則可能會不正確地放置。
