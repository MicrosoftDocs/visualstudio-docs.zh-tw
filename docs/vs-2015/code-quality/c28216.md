# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28216： \_Check_return\_ 注釋僅適用于特定函式參數的後置條件。  
  
 `_Check_return_` 批註已套用至 `__post`以外的內容;它可能需要 `__post` （或 `__drv_out`）修飾詞，否則可能會不正確地放置。
