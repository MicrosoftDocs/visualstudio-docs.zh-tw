# <a name="c6287"></a>C6287
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6287：多餘的程式碼：左邊和右邊的子運算式相同  
  
 此警告表示在運算式中偵測到多餘的元素。  
  
 在不檢查程式碼的情況下，很難判斷這個問題的嚴重性。 本身的重複測試是無害的，但刪除第二個測試的結果可能很嚴重。 您應檢查程式碼，以確保不會省略測試。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f(int x)  
{  
  if ((x == 1) && (x == 1))   
  {  
    //logic   
  }  
  if ((x != 1) || (x != 1))  
  {  
    //logic  
  }  
}  
```  
  
 下列程式碼顯示修正此警告的各種方法：  
  
```  
void f(int x, int y)  
{  
  /* Remove the redundant sub-expression: */  
  if (x == 1)   
  {  
     // logic   
  }  
  if (x != 1)   
  {  
    // logic  
  }  
  /* or test the missing variable: */  
  if ((x == 1) && (y == 1))  
  {  
     // logic  
  }  
  if ((x != 1) || (y != 1))  
  {  
     // logic  
  }  
}  
```
