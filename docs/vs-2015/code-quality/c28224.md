# <a name="c28224"></a>C28224
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28224：批註需要參數  
  
 此警告表示使用了未使用參數的命名注釋，而且至少需要一個參數。 如果標準宏用於批註，則應該不可能這麼做;正常使用時，不應該看到此警告。
