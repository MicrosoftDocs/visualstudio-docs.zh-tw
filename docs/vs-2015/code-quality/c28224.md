# <a name="c28224"></a>C28224
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28224： 註釋需要參數  
  
 這則警告表示具名的註解未使用任何參數，和至少一個參數是必要的。 這不能進行標準巨集所使用的註解;這個警告不應出現在正常使用。
