# <a name="c28210"></a>C28210
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告28210：內容的注釋 `_On_failure_` 不得在明確的前置內容中  
  
 批註 `_On_failure_` 必須明確或隱含地在內容中指出 `__post` ，也就是在函數傳回之後套用。  使用 `_drv_out` 以確保這一點。
