# <a name="c28210"></a>C28210
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主題的最新的版本可從[C28210](https://docs.microsoft.com/visualstudio/code-quality/c28210)。  
  
警告 28210： 註釋`_On_failure_`內容不能在明確 pre context 中  
  
 註釋`_On_failure_`必須明確或隱含地指定在`__post`內容，也就是要套用函式傳回之後。  使用`_drv_out`以確保此行為。
