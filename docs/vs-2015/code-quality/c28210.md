# <a name="c28210"></a>C28210
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告28210： `_On_failure_` 內容的注釋不得在明確的前置內容中  
  
 批註 `_On_failure_` 必須在 `__post` 內容中明確或隱含指示，也就是在函式傳回之後套用。  請使用 `_drv_out` 來確保這一點。
