# <a name="c28163"></a>C28163
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28163: 呼叫函式應該永遠不會從 try / except 區塊內  
  
 類型的函式時，會報告這個警告，應該永遠不會括住`try/except`中找不到區塊`try/except`區塊。 程式碼分析工具找到呼叫的函式是在至少一個路徑`try/except`區塊。
