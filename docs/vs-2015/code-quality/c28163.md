# <a name="c28163"></a>C28163
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28163：永遠不應該從 try/except 區塊中呼叫函數  
  
 當函式的類型不應該包含在區塊中的區塊時，就會回報此警告 `try/except` `try/except` 。 程式碼分析工具找到至少一個路徑，其中呼叫的函式在區塊內 `try/except` 。
