# <a name="c28215"></a>C28215
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28215：無法將 typefix 套用至已有 typefix 的參數  
  
 將 `__typefix` 批註套用至已經有該注釋的參數是錯誤。 `__typefix`批註只在幾個特殊情況下使用，而且在正常使用時，不應該看到此警告。
