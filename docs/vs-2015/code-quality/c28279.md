# <a name="c28279"></a>C28279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28279：若為符號，則會找到 ' begin '，但沒有相符的 ' end '  
  
 注釋語言支援「開始」和「結束」（`{` 和在 C 中 `}`）結構，而配對已取得「不平衡」。 如果使用宏，就可以避免這種情況。
