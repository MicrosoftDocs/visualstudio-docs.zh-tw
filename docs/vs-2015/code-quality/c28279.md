# <a name="c28279"></a>C28279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28279：針對符號，找到 ' begin '，但沒有相符的 ' end '  
  
 注釋語言支援 begin 和 end (`{` 以及 `}` 在 C) 結構中，且配對已取得不平衡。 如果使用宏，則可以避免此情況。
