# <a name="c6303"></a>C6303
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6303：格式字串不符： \<number> 當呼叫中需要字元字串時，以參數形式傳遞寬字元字串 \<function>  
  
 此警告表示格式字串指定需要字元字串。 但是，會傳遞寬字元字串。 這項瑕疵可能會導致某種形式的損毀或損毀。  
  
## <a name="example"></a>範例  
 下列範例程式碼會產生這個警告：  
  
```  
#include <stdio.h>  
  
void f()  
{  
  wchar_t buff[5] = L"hi";  
  
  printf("%s", buff);  
}  
```  
  
 若要更正這個警告，請使用， `%ls` 如下列範例程式碼所示：  
  
```  
#include <stdio.h>  
  
void f()  
{  
  wchar_t buff[5] = L"hi";  
  
  printf("%ls", buff);  
}  
```  
  
 下列範例程式碼使用安全字串操作函式 `printf_s` 來修正這個警告：  
  
```  
#include <stdio.h>  
  
void f()  
{  
  wchar_t buff[5] = L"hi";  
  
  printf_s("%ls",buff);  
}  
```
