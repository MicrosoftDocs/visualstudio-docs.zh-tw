# <a name="c6515"></a>C6515
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6515-不正確注釋： \<name> 屬性只可用於指標類型的值  
  
 此警告表示在指標上使用的屬性已套用至非指標類型。 如需批註屬性的清單，請參閱 [批註屬性](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```cpp  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char pc,  size_t c)  
{  
    // code ...  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char * pc,  size_t c)  
{  
    // code ...  
}  
```
