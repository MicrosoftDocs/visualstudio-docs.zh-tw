# <a name="c28138"></a>C28138
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28138：常數引數應改為變數  
  
 此警告會在需要變數或非常數運算式的函式呼叫中報告，但呼叫包含常數。 如需函數及其參數的詳細資訊，請參閱函式的 WDK 檔。  
  
## <a name="example"></a>範例  
 例如，在下列程式碼範例中，宏的參數 `READ_PORT_UCHAR` 必須是埠位址的指標，而不是做為常數提供的位址。  
  
 下列程式碼範例會產生下列警告訊息：  
  
```  
READ_PORT_UCHAR(0x80001234);  
```  
  
 若要更正這個警告，請使用埠位址的指標。  
  
```  
READ_PORT_UCHAR(PortAddress);  
```  
  
 有幾個較舊的裝置可接受 READ_PORT 和 WRITE_PORT 系列函式的常數參數。 當這些裝置收到此警告時，就可以隱藏或忽略警告。 不過，任何新的裝置都不應該採用固定的硬體位址。
