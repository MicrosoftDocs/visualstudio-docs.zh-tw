# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28211：預期 SAL_coNtext 的靜態內容名稱  
  
 此警告表示 `_Static_context_` 批註的運算元必須是工具定義內容的名稱。 這項警告應該永遠不會發生在正常使用中。
