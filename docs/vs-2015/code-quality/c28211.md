# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28211: sal_context 需要靜態內容名稱  
  
 此警告表示運算元`_Static_context_`註釋必須是工具定義的內容的名稱。 這個警告應該永遠不會發生在正常使用。
