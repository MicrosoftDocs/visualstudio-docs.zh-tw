# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28211： SAL_coNtext 應為靜態內容名稱  
  
 此警告表示批註的運算元 `_Static_context_` 必須是工具定義內容的名稱。 此警告應該不會在正常使用時發生。
