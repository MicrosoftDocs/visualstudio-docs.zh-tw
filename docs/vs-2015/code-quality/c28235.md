# <a name="c28235"></a>C28235
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28235：函式不能獨立為注釋  
  
 此警告表示嘗試使用未宣告為批註內容中批註的函式。 這包括在獨立內容中使用基本作業 (PrimOp) 。 如果標準宏用於批註，則應該不可能發生這種情況。 這項警告在一般使用上並不會出現。
