# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告28222： \_ 是 \_ 、 \_ 否 \_ 或 \_ 可能是 \_ 批註的預期  
  
 此警告表示批註的參數應為其中一個符號 `_Yes_` 、 `_No_` 或 `_Maybe_` ，而且遇到一些其他符號。 這通常表示不正確編碼的注釋宏。
