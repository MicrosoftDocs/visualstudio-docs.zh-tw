# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 28222:\_[是]\_， \_No\_，或\_或許\_註釋  
  
 這則警告指出，註釋的參數必須是其中一個符號`_Yes_`， `_No_`，或`_Maybe_`，並發現其他一些符號。 這通常表示的編碼錯誤註釋巨集。
