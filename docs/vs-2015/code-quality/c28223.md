# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28223： 找不到預期的語彙基元/識別項註釋、 參數  
  
 這則警告表示註釋的參數必須是識別項，並發現其他一些符號。 這通常表示的編碼錯誤註釋巨集。 使用 C 或 c + + 關鍵字，在此位置會導致這個錯誤。
