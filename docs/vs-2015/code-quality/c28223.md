# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28223：找不到注釋的預期標記/識別碼，參數  
  
 此警告表示批註的參數應該是識別碼，而且遇到一些其他符號。 這通常表示不正確編碼的注釋宏。 在此位置使用 C 或 c + + 關鍵字將會導致此錯誤。
