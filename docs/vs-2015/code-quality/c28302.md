# <a name="c28302"></a>C28302
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28302：針對 c + + 參考-參數 <parameter_name>， `_Deref_` 在上找到額外的運算子 \<annotation> 。  
  
 當參數上使用額外的層級時，就會回報此警告 `_Deref_` 。  
  
 處理參考參數時，SAL2 不需要使用額外的層級 `_Deref_` 。 這個特定的注釋是明確的，並且會正確地解讀，但應該予以修正。  
  
 通常只要移除舊版 `__deref` 批註並使用 SAL2 語法，就可以更正此情況。 有時可能需要使用 `_At_` 來參考要標注的特定物件。
