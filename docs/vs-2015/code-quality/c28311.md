# <a name="c28311"></a>C28311
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告28311： \<函式上的 < annotation_name > 注釋 > \<參數 > 是 SAL 的 sal 版本。  
  
 批註是舊版本，應該更新為對等的[SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)。 如果之前已經發出註釋不一致警告，就不會發出這個警告，而且這種狀況每個宣告只會報告一次。 檢查宣告的其他部分，找出更多過時的 SAL。
