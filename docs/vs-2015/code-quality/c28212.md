# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28212：注釋應為指標運算式  
  
 此警告表示批註的編號參數 (不是所標注的函式) 應該是指標或陣列類型，但卻遇到一些其他類型。 必須更正注釋。
