# <a name="c28208"></a>C28208
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28208： \<function> 之前已使用不同的參數清單在 () 上定義函數 \<file> \<line> 。 有些分析工具將會產生不正確的結果  
  
 當函數的已知定義不符合另一個出現專案時，就會回報此警告。
