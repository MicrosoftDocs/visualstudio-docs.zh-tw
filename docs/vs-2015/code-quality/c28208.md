# <a name="c28208"></a>C28208
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28208：函式 \<函式 > 先前以 \<檔案 > （\<行 >）的不同參數清單定義。 有些分析工具會產生不正確的結果  
  
 當函式的已知定義不符合另一個發生的情況時，就會回報此警告。
