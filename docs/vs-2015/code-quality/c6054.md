# <a name="c6054"></a>C6054
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6054：字串 \<variable> 可能不是以零結尾  
  
 此警告表示要求以零結尾的字串傳遞了非零終止的字串。 預期以零結束之字串的函式將會超過緩衝區結尾，以找出零。 此瑕疵可能會導致可利用的緩衝區溢位錯誤或損毀。 程式應確定字串的結尾為零。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```cpp  
  
#include <sal.h>  
  
void func( _In_z_ wchar_t* wszStr );  
  
void g ( )  
{  
    wchar_t wcArray[200];  
    func(wcArray); // Warning C6054  
}  
```  
  
 若要更正這個警告，請在呼叫函式前以 null 終止， `wcArray` `func()` 如下列範例程式碼所示：  
  
```cpp  
  
#include <sal.h>  
  
void func( _In_z_ wchar_t* wszStr );  
  
void g( )  
{  
    wchar_t wcArray[200];   
    wcArray[0]= '\0';  
    func(wcArray);  
}  
```
