# <a name="c28194"></a>C28194
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28194：函式已宣告為變數中的值加上別名，但未執行此動作  
  
 此警告表示所分析之函式的函式原型具有 `__drv_isAliased` 注釋，這表示它會將指定的引數 *別名* (也就是指派值，使其在從函式) 後存活的方式指派。 不過，此函式不會在批註所表示的路徑中，將引數加上別名。 以變數為別名的大部分函式會將其值儲存至全域資料結構。
