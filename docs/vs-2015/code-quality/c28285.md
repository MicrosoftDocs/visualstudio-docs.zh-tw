# <a name="c28285"></a>C28285
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28285： For 函式，參數中有語法錯誤  
  
 當模型檔案中發生可能的錯誤時，程式碼分析工具會報告這個警告。 少數的來源檔案錯誤也可能會造成這類錯誤。
