# <a name="ca0505"></a>CA0505

因為 superceding 屬性定義，將會忽略已被取代的屬性。

當同時指定已被取代，並取代程式碼分析屬性包含在專案中的值，或建置組態檔，就會產生警告 CA505。 若要解決此問題，請移除已被取代的屬性。
