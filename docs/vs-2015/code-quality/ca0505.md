# <a name="ca0505"></a>CA0505

已取代的屬性會被忽略，因為已定義取代屬性。

當指定的已被取代和取代的程式碼分析屬性包含專案或組建設定檔中的值時，就會產生警告 CA505。 若要解決此問題，請移除已被取代的屬性。
