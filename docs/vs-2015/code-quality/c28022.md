# <a name="c28022"></a>C28022
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28022：這個函式上 (es) 的函式類別， \<classlist1> 與 \<classlist2> 用來定義它的 typedef 上的函數類別 (es) 不相符。  
  
 當批註中有錯誤時，就會回報此警告。 Typedef 和函式本身都有 `_Function_class_` 注釋，但不相符。 如果兩者都使用，它們必須相符。
