# <a name="c6281"></a>C6281
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 6281-不正確的作業順序：關係運算子的優先順序高於位運算子  
  
 此警告表示運算子優先順序可能發生錯誤。 這可能會產生不正確的結果。 您應該檢查優先順序，並使用括弧來闡明意圖。 關聯式運算子 (\<, > 、 \<=, > =、= =、！ = ) 的優先順序高於位運算子 ( # B0 ^) 。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#include <stdlib.h>  
#define FORMAT 1  
#define TYPE 2  
  
void f(int input)  
{  
  if (FORMAT & TYPE != input)  
  {  
    // code...  
  }  
}  
```  
  
 下列程式碼會使用括弧來修正這個警告：  
  
```  
#include <stdlib.h>  
#define FORMAT 1  
#define TYPE 2  
  
void f(int input)  
{  
  if ((FORMAT & TYPE) != input)  
  {  
    // code...  
  }  
}  
```
