# <a name="c6059"></a>C6059
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6059：呼叫中的長度參數不正確 \<function> 。 傳遞剩餘字元數，而不是緩衝區大小 \<variable>  
  
 此警告表示對字串串連函式的呼叫可能會針對要串連的字元數傳遞不正確的值。 此瑕疵可能會導致可利用的緩衝區溢位或損毀。 這項瑕疵的常見原因是將緩衝區大小（而不是緩衝區中剩餘的字元數）傳遞給字串操作函數。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#include <string.h>  
#define MAX 25  
  
void f( )  
{  
  char szTarget[MAX];  
  char *szState ="Washington";  
  char *szCity="Redmond, ";  
  
  strncpy(szTarget,szCity, MAX);  
  szTarget[MAX -1] = '\0';  
  strncat(szTarget, szState, MAX); //wrong size   
  // code ...                                   
}  
```  
  
 若要更正這個警告，請使用正確的字元數來串連，如下列程式碼所示：  
  
```  
#include <string.h>  
#define MAX 25  
  
void f( )  
{  
  char szTarget[MAX];  
  char *szState ="Washington";  
  char *szCity="Redmond, ";  
  
  strncpy(szTarget,szCity, MAX);  
  szTarget[MAX -1] = '\0';  
  strncat(szTarget, szState, MAX - strlen(szTarget)); // correct size   
  // code ...                                   
}  
```  
  
 若要使用安全字串操作函式更正這個警告，請參閱下列程式碼：  
  
```  
#include <string.h>  
  
void f( )  
{  
  char *szState ="Washington";  
  char *szCity="Redmond, ";  
  
  size_t nTargetSize = strlen(szState) + strlen(szCity) + 1;  
  char *szTarget= new char[nTargetSize];  
  
  strncpy_s(szTarget, nTargetSize, szCity,strlen(szCity));  
  strncat_s(szTarget, nTargetSize, szState,  
                    nTargetSize - strlen(szTarget));  
  // code ...  
  delete [] szTarget;  
}  
```
