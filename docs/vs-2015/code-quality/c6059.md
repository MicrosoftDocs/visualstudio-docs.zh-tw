# <a name="c6059"></a>C6059
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6059： 對呼叫中不正確的長度參數\<函式 >。 傳遞剩餘的字元數，而非的緩衝區大小\<變數 >  
  
 這則警告表示字串串連函式的呼叫可能會傳遞要串連的字元數的值不正確。 這項缺失可能會導致可利用來攻擊的緩衝區溢位或損毀。 此瑕疵的常見原因傳遞緩衝區大小，而不是剩餘的緩衝區，字串操作函式中的字元數。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#include <string.h>  
#define MAX 25  
  
void f( )  
{  
  char szTarget[MAX];  
  char *szState ="Washington";  
  char *szCity="Redmond, ";  
  
  strncpy(szTarget,szCity, MAX);  
  szTarget[MAX -1] = '\0';  
  strncat(szTarget, szState, MAX); //wrong size   
  // code ...                                   
}  
```  
  
 若要更正這個警告，請使用正確的字元數目進行串連，如下列程式碼所示：  
  
```  
#include <string.h>  
#define MAX 25  
  
void f( )  
{  
  char szTarget[MAX];  
  char *szState ="Washington";  
  char *szCity="Redmond, ";  
  
  strncpy(szTarget,szCity, MAX);  
  szTarget[MAX -1] = '\0';  
  strncat(szTarget, szState, MAX - strlen(szTarget)); // correct size   
  // code ...                                   
}  
```  
  
 若要更正這個警告，使用安全的字串操作函式，請參閱下列程式碼：  
  
```  
#include <string.h>  
  
void f( )  
{  
  char *szState ="Washington";  
  char *szCity="Redmond, ";  
  
  size_t nTargetSize = strlen(szState) + strlen(szCity) + 1;  
  char *szTarget= new char[nTargetSize];  
  
  strncpy_s(szTarget, nTargetSize, szCity,strlen(szCity));  
  strncat_s(szTarget, nTargetSize, szState,  
                    nTargetSize - strlen(szTarget));  
  // code ...  
  delete [] szTarget;  
}  
```
