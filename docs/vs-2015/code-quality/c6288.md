# <a name="c6288"></a>C6288
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6288：不正確的運算子：對 && 的相互包含一律為零。 您是否想要改用 &#124;&#124; ？  
  
 此警告表示在測試運算式中，變數是針對兩個不同的常數進行測試，而結果取決於兩個條件都是 true。 這些案例中的程式碼表示未正確地捕捉程式設計人員的意圖。 檢查程式碼並修正問題是很重要的。否則，您的程式碼將不會以您預期的方式運作。  
  
 這個問題通常是使用所造成 `&&` ，而不是使用 `||` ，但也可能是使用預期的位置所造成 `==` `!=` 。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f(int x)  
{  
  if ((x == 1) && (x == 2)) // warning  
  {  
    // code ...  
  }  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f(int x)  
{  
     if ((x == 1) || (x == 2))   
     {  
          // logic  
     }  
  
    /* or */  
    if ((x != 1) && (x != 2))  
    {  
        // code ...  
    }  
}  
```  
  
 如果運算式有副作用，則分析工具不會發出警告。
