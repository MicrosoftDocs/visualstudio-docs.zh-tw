# <a name="c6292"></a>C6292
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6292： 定義錯誤的 for 迴圈： 從最大值向上計數  
  
 這則警告指出 for 迴圈可能無法運作正常。  
  
 迴圈從最大值向上計數，但具有較低的終止條件時，它就會發生。 僅在發生整數溢位之後，將會終止此迴圈。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f( )  
{  
   int i;  
  
   for (i = 100; i >= 0; i++)  
   {  
      // code ...  
   }  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   int i;  
  
  for (i = 100; i >= 0; i--)  
   {  
      // code ...  
   }  
}  
```
