# <a name="c6292"></a>C6292
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6292：對-迴圈定義錯誤：從最大值計算  
  
 此警告表示 for 迴圈可能無法如預期運作。  
  
 當迴圈從最大值算起，但有較低的終止條件時，就會發生此問題。 此迴圈只會在發生整數溢位之後終止。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f( )  
{  
   int i;  
  
   for (i = 100; i >= 0; i++)  
   {  
      // code ...  
   }  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   int i;  
  
  for (i = 100; i >= 0; i--)  
   {  
      // code ...  
   }  
}  
```
