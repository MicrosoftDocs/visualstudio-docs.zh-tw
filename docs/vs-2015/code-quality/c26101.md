# <a name="c26101"></a>C26101
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C26101：無法針對變數正確使用連鎖作業 \<var> 。  
  
 Windows Api 提供各種連鎖的作業。 注釋 `_Interlocked_` 指定只能透過連鎖作業來存取變數。 當存取與注釋不一致時，就會發出警告 C26101 `_Interlocked_` 。  
  
## <a name="example"></a>範例  
 下列範例會產生警告 C26101，因為 `_Interlocked_` 合約違規。  
  
```  
  
CRITICAL_SECTION cs;  
typedef struct _DATA   
{  
    _Interlocked_ LONG data;  
} DATA;  
  
void Safe(DATA* p)   
{  
    InterlockedIncrement(&p->data); // OK  
}  
  
void Unsafe(DATA* p)   
{  
    p->data += 1; // Warning C26101  
    EnterCriticalSection(&cs);  
    LeaveCriticalSection(&cs);  
}  
  
```
