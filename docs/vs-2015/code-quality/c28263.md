# <a name="c28263"></a>C28263
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28263：發現內建批註的條件式批註中有語法錯誤  
  
 當指定函式的傳回值具有條件值時，程式碼分析工具會回報此警告。 此警告表示批註中的錯誤，而不是在分析的程式碼中。 如果函式宣告是在標頭檔中，則應該更正注釋，使其符合標頭檔。  
  
 指定之函式和參數的結果清單有多個無條件值。  
  
 通常，這表示已使用一個以上的無條件 `_Null_` 或 `__drv_valueIs` 批註來指定結果值。
