# <a name="c6400"></a>C6400
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6400：使用 \<function name> 執行與常數位串不區分大小寫的比較 \<string name> 。 在非英文地區設定中產生非預期的結果  
  
 此警告表示以地區設定相關的方式執行與常數位串的不區分大小寫比較，在此情況下，預期會有與地區設定無關的比較。  
  
 這項瑕疵的一般結果是非英文說話地區設定中不正確的行為。 例如，在土耳其文中，".gif" 將不會符合 "。GIF ";在越南文中，"LogIn" 將不會符合 "LOGIN"。  
  
 您通常應該使用函數來執行字串比較 `CompareString` 。 若要在 Windows XP 上執行與地區設定無關的比較，第一個參數應該是常數 `LOCALE_INVARIANT` 。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#include <windows.h>  
int f(char *ext)  
{  
  // code...  
  return (lstrcmpi(ext, TEXT("gif")) == 0);  
}  
```  
  
 若要更正這個警告，請執行與地區設定無關的測試，以瞭解 char * ext 是否會忽略大寫/小寫的差異，請使用下列程式碼：  
  
```  
#include <windows.h>  
int f(char *ext)  
{  
  // code...  
  return (CompareString(  
                        LOCALE_INVARIANT,  
                        NORM_IGNORECASE,   
                        ext,  
                        -1,  
                        TEXT ("gif"),  
                        -1) == CSTR_EQUAL);  
}  
```
