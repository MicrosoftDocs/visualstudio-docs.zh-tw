# <a name="c28310"></a>C28310
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 28310:上的 < f > 註釋\<函式 >\<參數 > 沒有 SAL 版本。  
  
 原始程式碼中使用的所有 SAL 註釋都應該具有使用 SAL_name 時所套用的註釋版本。 這點應在巨集定義中更正。  
  
 這種狀況每個宣告只會報告一次。 檢查宣告的其他部分，找出更多過時的 SAL。
