# <a name="c28282"></a>C28282
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28282：格式字串必須在前置條件中  
  
 此警告表示 `__drv_formatString` 找不到 () 注釋中的注釋 (函 `_Pre_` 式 `__drv_in` 參數預設為 `_Pre_`) 。 檢查是否在具有 `_Post_` () 注釋的明確區塊中使用注釋 `__drv_out` 。 如果是的話，請從已放入內容的任何封閉區塊中移除批註 `_Post_` 。
