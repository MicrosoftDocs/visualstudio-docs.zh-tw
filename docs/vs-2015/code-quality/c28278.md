# <a name="c28278"></a>C28278
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28278：函式名稱出現，但範圍中沒有原型。  
  
 此警告通常表示需要將 `__deref` `__return` 批註套用至傳回的值。  
  
 當呼叫沒有宣告的函式時，程式碼分析工具會回報此警告，而且可以執行的分析會受到限制，因為宣告包含重要的資訊。  
  
 C 語言允許 (但不鼓勵) 使用未宣告原型的函式。 函式定義或函式宣告 (原型) 應該會在函式的第一次使用之前出現。 此警告表示呼叫了沒有宣告的函式，而且可以執行的分析受限，因為宣告包含重要的資訊。 如果函式宣告包含批註，則函式宣告對於程式碼分析工具更為有用。
