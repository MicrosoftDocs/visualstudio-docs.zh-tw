# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28220：注釋中的參數需要有整數運算式  
  
 此警告表示批註的參數應為整數運算式，而且遇到一些其他類型。 這通常表示不正確編碼的注釋宏。
