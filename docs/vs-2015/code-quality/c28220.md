# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28220： 註釋需要整數運算式中的註釋的參數  
  
 這則警告表示註釋的參數必須是整數運算式，而且發現某些其他類型。 這通常表示的編碼錯誤註釋巨集。
