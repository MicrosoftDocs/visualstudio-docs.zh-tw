# <a name="c28273"></a>C28273
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主題的最新的版本可從[C28273](https://docs.microsoft.com/visualstudio/code-quality/c28273)。  
  
警告 C28273： 對於函式，線索不一致和函式宣告  
  
 這個警告表示註解中，而不要分析的程式碼發生錯誤。 出現在函式定義上的註解會與宣告上不會出現不一致。 兩項註釋應該解析為相符。
