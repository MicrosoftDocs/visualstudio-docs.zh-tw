# <a name="c28273"></a>C28273
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28273：針對函式，線索與函式宣告不一致  
  
 此警告表示批註中的錯誤，而不是正在分析的程式碼中。 出現在函式定義上的批註與出現在宣告中的注釋不一致。 這兩個批註應解析為相符。
