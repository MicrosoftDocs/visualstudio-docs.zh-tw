# <a name="c28241"></a>C28241
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28241：無法辨識參數上的函式注釋  
  
 使用了無法辨識的注釋名稱。 如果標準宏用於批註，則應該不可能這麼做;這項警告在一般使用上並不會出現。
