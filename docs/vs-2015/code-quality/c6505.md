# <a name="c6505"></a>C6505
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6505：不正確注釋： MustCheck 屬性不能用於 void 類型的值  
  
 此警告指出，MustCheck 屬性是在 void 資料類型上使用。 您無法在 void 類型上使用 MustCheck 屬性。 請移除 MustCheck 屬性，或使用其他資料類型。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```cpp  
#include <sal.h>  
_Must_inspect_result_ void f()  
{  
    //Code ...  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```cpp  
#include <sal.h>  
_Must_inspect_result_ char* f()  
{  
    char *str ="Hello World";  
    //Code ...  
    return str;  
}  
```
