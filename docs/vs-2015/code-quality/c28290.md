# <a name="c28290"></a>C28290
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28290：函式的注釋包含比實際參數數目更多的外部  
  
 當函數的注釋包含比實際參數數目更多的外部外部時，程式碼分析工具會回報此警告。
