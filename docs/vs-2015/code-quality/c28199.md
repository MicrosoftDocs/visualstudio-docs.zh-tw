# <a name="c28199"></a>C28199
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28199：使用可能未初始化的記憶體  
  
 此訊息表示變數已取得其位址，但尚未探索到它的指派。  
  
 使用指定的變數時，不會明確初始化，但在某個時間點會取得其位址，這表示它可能會以不可見的方式初始化至程式碼分析工具。  
  
 如果在函式之外初始化變數，此警告可能會被誤認為。  
  
 如果參數具有 `_Out_` 或 `_Inout_` 批註且變數未初始化，則程式碼分析工具會在函式結束時回報此警告。
