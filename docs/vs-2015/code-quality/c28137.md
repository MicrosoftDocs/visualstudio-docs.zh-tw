# <a name="c28137"></a>C28137
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28137：變數引數應該改為) 常數的 (常值  
  
 函式呼叫遺漏必要的 (常值) 常數時，會報告這個警告。 請參閱函式的文件。  
  
## <a name="example"></a>範例  
 例如， [ExAcquireResourceExclusiveLite](https://msdn.microsoft.com/library/windows/hardware/ff544351.aspx) 常式需要參數的值為 TRUE 或 FALSE `Wait` 。 下列程式碼範例會產生這個警告：  
  
```  
ExAcquireResourceExclusiveLite(Resource, Wait);  
```  
  
 下列程式碼範例可避免此警告：  
  
```  
ExAcquireResourceExclusiveLite(Resource, TRUE);  
```
