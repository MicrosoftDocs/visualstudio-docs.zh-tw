# <a name="c6517"></a>C6517
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6517：不正確注釋：無法讀取的緩衝區上不得指定 ' SAL_readableTo ' 屬性： '*Param \\ * (1) '。  
  
> [!NOTE]
> 只有使用已淘汰版本的原始程式碼批註語言 (SAL) 的程式碼才會發生這個警告。 建議您移植程式碼，以使用最新版本的 SAL。 如需詳細資訊，請參閱 [使用 SAL 注釋減少 C/c + + 程式碼](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)缺失。  
  
 這則警告指出 `SAL_readableTo` 屬性沒有必要的讀取存取權限。 在未提供讀取存取權限的情況下，不能使用這個屬性為參數加上附註。  
  
## <a name="example"></a>範例  
 下列程式碼會產生此警告，因為未授與緩衝區的讀取權限：  
  
```cpp  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([Pre( ValidBytesConst=10 )][Pre( Deref=1, Access=Write )] char* buffer );  
  
```  
  
 若要更正這個警告，請授與讀取權限，如下列程式碼所示：  
  
```cpp  
#include <CodeAnalysis\SourceAnnotations.h>  
using namespace vc_attributes;  
void f([Pre( ValidBytesConst=10 )][Pre( Deref=1, Access=Read)] char* buffer );  
```
