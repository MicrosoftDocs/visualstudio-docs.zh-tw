# <a name="c6506"></a>C6506
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6506：不正確注釋： \<名稱 > 屬性僅可用於指標或陣列類型的值  
  
 此警告表示屬性用於指標或陣列類型以外的類型。 Access、污染和 Valid 屬性可以在所有資料類型上使用。 其他屬性，例如 ValidBytesConst、ValidElementsConst、ElementSize 和 NullTerminted 支援指標、成員指標或陣列類型。 如需屬性和支援的資料類型的完整清單，請參閱[Annotation 屬性](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 若要更正這個警告，請使用指標或陣列類型，如下列範例程式碼所示：  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
