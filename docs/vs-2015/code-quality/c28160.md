# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28160：錯誤注釋  
  
 當遇到注釋時，就會回報此警告 `__drv_error` 。 這個批註是用來旗標應修正的程式碼撰寫做法，而且可以 `__drv_when` 搭配注釋使用，以指出特定的參數組合。
