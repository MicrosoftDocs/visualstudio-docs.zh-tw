# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28160：錯誤注釋  
  
 當遇到 `__drv_error` 注釋時，就會回報此警告。 這個批註是用來標示應該修正的編碼做法，而且可以與 `__drv_when` 注釋搭配使用，以指示特定的參數組合。
