# <a name="c28217"></a>C28217
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28217：針對函式，注釋的參數數目不符合檔案中找到的參數  
  
 目前行上和訊息行中的注釋不一致。 如果標準宏用於批註，則應該不可能這麼做;正常使用時，不應該看到此警告。
