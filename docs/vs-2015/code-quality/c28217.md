# <a name="c28217"></a>C28217
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主題的最新的版本可從[C28217](https://docs.microsoft.com/visualstudio/code-quality/c28217)。  
  
警告 C28217： 函式，註釋的參數數目不符合找到的檔案  
  
 目前這一行，並在訊息中的一行上的註釋不一致。 這不能進行標準巨集所使用的註解;這個警告不應出現在正常使用。
