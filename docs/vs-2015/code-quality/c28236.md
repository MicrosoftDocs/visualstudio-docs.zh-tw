# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28236：批註不能用在運算式中  
  
 此警告表示嘗試使用宣告為運算式內容中注釋的函式。 如果標準宏正用於注釋，則不應該這麼做;一般用途不應出現此警告。
