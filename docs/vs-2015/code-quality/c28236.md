# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28236:註釋不能用在運算式中  
  
 這個警告表示嘗試使用函式宣告為註解中的運算式內容。 這不能進行標準巨集所使用的註解;這個警告不應該看到典型的使用中。
