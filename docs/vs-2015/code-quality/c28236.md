# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28236：注釋不能用在運算式中  
  
 此警告表示嘗試使用宣告為運算式內容中之注釋的函式。 如果標準宏用於批註，則應該不可能這麼做;這項警告在一般使用上並不會出現。
