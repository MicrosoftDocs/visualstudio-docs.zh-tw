# <a name="c28287"></a>C28287
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28287:函式的 \_At\_() 註釋中有語法錯誤 (無法辨認的參數名稱)  
  
 程式碼分析工具會報告這個警告的時機`SAL_at`(`__drv_at`) 註解用並無法在目前內容中解譯參數運算式。 這可能包括使用的拼字錯誤的參數或成員名稱或"return"或"this"關鍵字的拼字錯誤。
