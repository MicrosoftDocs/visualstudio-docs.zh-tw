# <a name="c28287"></a>C28287
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28287： For 函數， \_ \_ ( # A1 批註中的語法錯誤 (無法辨識的參數名稱)   
  
 使用 () 注釋時，程式碼分析工具會回報此警告 `SAL_at` `__drv_at` ，而且無法在目前內容中解讀參數運算式。 這可能包括使用拼錯的參數或成員名稱，或 "return" 或 "this" 關鍵字的拼寫錯誤。
