# <a name="c6293"></a>C6293
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6293：對-迴圈定義錯誤：從最小值減少計數  
  
 此警告表示 for 迴圈可能無法如預期運作。 當迴圈從最小值算出，但有更高的終止條件時，就會發生此問題。  
  
 帶正負號或不帶正負號的索引變數與負數遞增會導致迴圈計算負的，直到發生溢位為止。 這會結束迴圈。  
  
## <a name="example"></a>範例  
 下列範例程式碼會產生這個警告：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
