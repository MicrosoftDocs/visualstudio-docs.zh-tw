# <a name="c6293"></a>C6293
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6293：為-loop 定義錯誤：從最小計數減少  
  
 此警告表示 for 迴圈可能無法如預期運作。 當迴圈從最小值向下計算，但具有較高的終止條件時，就會發生此錯誤。  
  
 帶正負號或無符號的索引變數加上負增量，會使迴圈在發生溢位之前計算負值。 這會終止迴圈。  
  
## <a name="example"></a>範例  
 下列範例程式碼會產生這個警告：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
  
```  
  
 若要修正這個警告，請使用下列程式碼：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
