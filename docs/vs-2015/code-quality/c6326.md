# <a name="c6326"></a>C6326
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6326：常數與另一個常數的可能比較  
  
 此警告表示常數與另一個常數的可能比較，這是多餘的程式碼。 您必須檢查以確定已在程式碼中正確地捕捉您的意圖。 在某些情況下，您可以簡化測試條件以達成相同的結果。  
  
## <a name="example"></a>範例  
 下列程式碼會產生此警告，因為比較兩個常數：  
  
```  
#define LEVEL    
const int STD_LEVEL = 5;  
  
const int value =   
#ifdef LEVEL  
10;  
#else   
5;  
#endif  
  
void f()  
{  
  if( value > STD_LEVEL)  
  {  
    // code...  
  }  
  else  
  {  
    // code...  
  }  
}  
```  
  
 下列程式碼示範如何使用 #ifdef 語句來判斷應執行的程式碼，以修正這個警告的其中一種方法：  
  
```  
#define LEVEL    
const int STD_LEVEL = 5;  
  
const int value =   
#ifdef LEVEL  
10;  
#else   
5;  
#endif  
  
void f ()  
{  
#ifdef LEVEL  
  {  
    // code...  
  }  
#else  
  {  
    // code...  
  }  
#endif  
}   
```
