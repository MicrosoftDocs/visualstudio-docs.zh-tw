# <a name="c6326"></a>C6326
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6326：常數與另一個常數的可能比較  
  
 此警告表示可能會比較常數與另一個常數，也就是多餘的程式碼。 您必須檢查以確定您的意圖已正確地在程式碼中捕捉。 在某些情況下，您可以簡化測試條件來達到相同的結果。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告，因為會比較兩個常數：  
  
```  
#define LEVEL    
const int STD_LEVEL = 5;  
  
const int value =   
#ifdef LEVEL  
10;  
#else   
5;  
#endif  
  
void f()  
{  
  if( value > STD_LEVEL)  
  {  
    // code...  
  }  
  else  
  {  
    // code...  
  }  
}  
```  
  
 下列程式碼顯示使用 #ifdef 語句來判斷應該執行哪一種程式碼，以更正此警告的一種方法：  
  
```  
#define LEVEL    
const int STD_LEVEL = 5;  
  
const int value =   
#ifdef LEVEL  
10;  
#else   
5;  
#endif  
  
void f ()  
{  
#ifdef LEVEL  
  {  
    // code...  
  }  
#else  
  {  
    // code...  
  }  
#endif  
}   
```
