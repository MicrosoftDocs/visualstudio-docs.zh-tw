# <a name="ca0068"></a>CA0068

找不到目標群組件 *' AssemblyName '*) 找不到 (符號的偵錯工具資訊。 為了獲得最佳分析結果，請在與目標群組件相同的目錄中，加入 .pdb 檔案，其中包含 *' AssemblyName '* 的偵錯工具資訊。

.Pdb 檔案中的偵錯工具資訊可改善某些程式碼分析檢查的精確度。 遺失的 .pdb 檔案可能會導致更高的誤報（也稱為雜訊）或不正確的分析。 若要啟用 .pdb 產生，請開啟 [專案屬性] 頁面。 在 [ **組建** ] 索引標籤上，尋找 [ **Debug] 資訊** 清單。 位置會因專案類型而異。 請確定偵錯工具的 [ **debug] 資訊** 已設定為 [ **完整** ]，或針對發行組建設定為 [  **僅限 pdb** ]。 它不應該設定為 [ **無**]。
