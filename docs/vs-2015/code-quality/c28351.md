# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28351:註釋\<註釋 > 描述條件中不能使用動態值 （變數）。  
  
 註釋套用時，會報告這個警告，C /C++會檢查類型。  
  
 中的運算式`_When_`應該評估為常數。 `_When_`會被忽略。
