# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28351：批註 \<annotation> 描述了動態值 (變數) 無法在條件中使用的位置。  
  
 在檢查 C/c + + 類型的位置套用批註時，會報告此警告。  
  
 中的運算式 `_When_` 應評估為常數。 `_When_`會被忽略。
