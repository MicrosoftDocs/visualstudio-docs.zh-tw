# <a name="c6282"></a>C6282
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6282:不正確的運算子： 在布林內容中的常數指派。 請考慮使用 '= =' 改為  
  
 這個警告表示測試內容中偵測到指派給變數的常數。 常數指派給變數，以測試內容中幾乎都是不正確的。 取代`=`與`==`，或若要解決這個警告的測試內容中移除指派。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f( int i )  
{  
   while (i = 5)  
   {  
   // code    
   }  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f( int i )  
{  
   while (i == 5)  
   {  
   // code    
   }  
}  
```
