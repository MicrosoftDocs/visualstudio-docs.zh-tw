# <a name="c1252"></a>C1252
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1252：外掛程式之間的迴圈或遺失相依性：需要 GUID  
  
 程式碼分析工具會在有外掛程式相依性的內部錯誤時報告此警告，而不是在分析的程式碼中。
