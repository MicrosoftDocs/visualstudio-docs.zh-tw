# <a name="c28289"></a>C28289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28289： For 函數： ReadableTo 或 Writableto 有限沒有做為參數的限制規格  
  
 當函式/參數注釋 miscoded 為已注明時，程式碼分析工具會報告這個警告。
