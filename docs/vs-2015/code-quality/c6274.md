# <a name="c6274"></a>C6274
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6274：在 \<number> 呼叫中需要字元時傳遞非字元作為參數 \<function>  
  
 此警告表示格式字串指定需要字元 (例如， `%c` 或 `%C` 規格) ，但傳遞的是非整數（如 float、string 或 struct）。 這項瑕疵可能會導致不正確的輸出。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf(buff,"%c",str);  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf(buff,"%c",str[0]);  
}  
```  
  
 下列程式碼會使用安全的字串操作函式， `sprintf_s` 以更正此警告：  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf_s(buff,5,"%c", str[0]);  
}  
```
