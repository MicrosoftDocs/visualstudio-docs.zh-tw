# <a name="c6302"></a>C6302
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6302：格式字串不符： \<number> 當呼叫中需要寬字元字串時，以參數形式傳遞的字元字串 \<function>  
  
 此警告表示格式字串指定需要寬字元字串。 但是，會傳遞字元字串。 這項瑕疵可能會導致某個表單損毀或損毀。  
  
## <a name="example"></a>範例  
 下列範例程式碼會產生此警告，因為字元字串會傳遞給函式 `wprintf` ：  
  
```  
#include<stdio.h>  
  
void f()  
{  
  char buff[5] = "hi";  
  
  wprintf(L"%s", buff);  
}  
```  
  
 下列範例程式碼會使用 `%hs` 來指定具有函式的單一位元組字元字串 `wprintf` ：  
  
```  
#include<stdio.h>  
  
void f()  
{  
  char buff[5] = "hi";  
  
  wprintf(L"%hs", buff);  
}  
```  
  
 下列範例程式碼使用安全字串操作函式 `wprintf_s` 來修正這個警告：  
  
```  
#include<stdio.h>  
  
void f()  
{  
  char buff[5] = "hi";  
  
  wprintf_s(L"%hs", buff);  
}  
```
