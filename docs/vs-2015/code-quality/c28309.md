# <a name="c28309"></a>C28309
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告28309： <parameter_name>：批註運算元必須是整數/列舉/指標類型。 不支援 Void 運算元和 c + + 多載運算子。 浮點數近似為整數。 類型： \<typelist> 。  
  
 您已嘗試在批註運算式中使用 void 或函數，而且程式碼分析無法處理它。  當使用實作為函式的時，通常會發生此錯誤 `operator==` ，但也可能會發生其他情況。 檢查中的型別 \<typelist> ，以取得有問題的線索。
