# <a name="c28164"></a>C28164
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28164：將引數傳遞至函式，而該函式需要物件的指標， (不是指標的指標)   
  
 當指標的指標用於需要物件指標的函式時，就會回報此警告。  
  
 此函式會在此位置採用 PVOID。 通常，這表示在需要 p*xxx*時使用 &p*xxx* 。  
  
 某些 *多* 型函式 (函式，這些函式可以使用採用任何指標類型的 PVOID 引數來評估，並套用至不同類型的值) 在 C 中執行。 不過，這可讓程式設計人員將指標編碼為指標，而不會造成編譯器錯誤，即使這種類型不適當也是如此。  
  
## <a name="example"></a>範例  
 下列程式碼範例會產生此警告：  
  
```  
PFAST_MUTEX pFm;  
...  
KeWaitForSingleObject(&pFm, UserRequest, UserMode, false, NULL);  
```  
  
 下列程式碼範例會避免出現警告：  
  
```  
PFAST_MUTEX pFm;  
...  
KeWaitForSingleObject(pFm, UserRequest, UserMode, false, NULL);  
```
