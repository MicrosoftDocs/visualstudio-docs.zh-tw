# <a name="c6385"></a>C6385
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6385：不正確資料：存取 \<buffer name> ，可讀取的大小為 \<size1> 位元組，但 \<size2> 可能會讀取位元組：行： x、y  
  
 此警告表示指定緩衝區的可讀取範圍可能小於用來讀取它的索引。 嘗試讀取有效範圍以外的資料會導致緩衝區溢位。  
  
## <a name="example"></a>範例  
 下列程式碼會產生這個警告：  
  
```  
void f(int i)  
{  
   char a[20];  
   char j;  
   if (i <= 20)  
   {  
      j = a[i];  
   }  
}  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```  
void f(int i)  
{  
   char a[20];  
   char j;  
   if (i < 20)  
   {  
      j = a[i];  
   }  
}   
```
