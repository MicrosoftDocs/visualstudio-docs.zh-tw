# <a name="c28024"></a>C28024
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28024：指派給的函式指標是以函式類別來標注，該函式類別 \<class> 未包含在函數類別中 (es) \<classlist> 。  
  
 當兩個函式都以函式類別標注，但類別不相符時，就會回報此警告。  
  
 當函式指標的 `_Function_class_` 注釋指定只能指派特定功能類別的函式時，就會發出這個警告。 在函式呼叫的指派或隱含指派中，來源和目標必須屬於相同的函式類別，但函數類別不相符。
