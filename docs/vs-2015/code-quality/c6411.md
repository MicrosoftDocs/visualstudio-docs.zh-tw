# <a name="c6411"></a>C6411
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6411：可能是從緩衝區讀取不正確資料。  
  
 此警告表示用來從緩衝區讀取的索引值可能會超過緩衝區的可讀取大小。 由於程式碼分析工具會在無法減少表示緩衝區大小的複雜運算式，或是用來存取緩衝區的索引時回報此警告，因此可能會在錯誤中報告此警告。  
  
## <a name="example"></a>範例  
 下列程式碼會產生此警告。  
  
```  
char *a = new char[strlen(InputParam)];  
delete[] a;  
a[10];  
```  
  
 下列程式碼會修正這個錯誤。  
  
```  
int i = strlen(InputParam);  
char *a = new char[i];  
if (i > 10) a[10];  
delete[] a;  
```
