# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28221：注釋中的參數必須要有字串運算式  
  
 此警告表示批註的參數必須是字串，而且遇到一些其他類型。 這通常表示不正確編碼的注釋宏，而且不應該在正常使用時看到。
