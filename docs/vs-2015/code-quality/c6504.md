# <a name="c6504"></a>C6504
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6504：不正確注釋：屬性只可用於指標、成員指標或陣列類型的值  
  
 此警告表示在不相容的資料類型上使用屬性。 如需屬性所支援資料類型的詳細資訊，請參閱 [批註屬性](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)。  
  
## <a name="example"></a>範例  
 因為 `_Null_` 屬性不能用於參考資料類型，所以下列程式碼會產生這個警告。  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_ Point& pt)  
{  
    // code ...  
}  
  
```  
  
 若要更正這個警告，請使用下列程式碼：  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_  Point* pt)  
{  
    // code ...  
}  
```  
  
 稍早所示的瑕疵代碼也會產生警告 [C6516](../code-quality/c6516.md) ，因為屬性衝突會導致不正確注釋。
