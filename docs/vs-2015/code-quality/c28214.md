# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28214：屬性參數名稱必須是 p1 .。。p9  
  
 此警告表示當您建立批註宣告時，參數名稱限制為 p1 .。。p9. 此警告應該不會在正常使用時發生。
