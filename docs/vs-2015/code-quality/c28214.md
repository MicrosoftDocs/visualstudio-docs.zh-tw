# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28214:屬性參數名稱必須是 p1...p9  
  
 這則警告指出，當您建構的註釋宣告時，參數名稱僅限 p1...p9。 這個警告應該永遠不會發生在正常使用。
