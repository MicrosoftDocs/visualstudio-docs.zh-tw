# <a name="ca0057"></a>CA0057

違規無法對應到目前的目標和規則集。

如果以專案檔或/imported XML 報表（具有無法對應到目前目標和規則集的違規）來手動叫用 FxCopCmd.exe，就會發生這個錯誤。

只有在使用工具的命令列版本進行分析時，才會發生。
