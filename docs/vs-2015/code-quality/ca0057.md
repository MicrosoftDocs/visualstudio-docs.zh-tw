# <a name="ca0057"></a>CA0057

違規無法對應至目前的目標和規則集。

如果 FxCopCmd.exe 叫用手動方式在專案檔，就會發生此錯誤或 / 匯入 XML 報告中具有無法對應至目前的一組目標和規則的違規。

分析完畢時，才會發生使用命令列工具的版本。
