# <a name="ca0059"></a>CA0059

命令列參數錯誤。

如果遺漏必要的命令列參數，就會發生此錯誤，例如

未指定/out 或/console 參數。 如果有不正確參數，例如錯誤的超時值，也會發生此錯誤。
