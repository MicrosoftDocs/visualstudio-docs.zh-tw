# <a name="c28228"></a>C28228
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28228： 註釋參數： 註解中不可使用類型  
  
 這個警告表示參數是不支援的型別。 註釋只能使用一組有限的類型做為參數。 這不能進行標準巨集所使用的註解;這個警告不應該看到典型的使用中。
