# <a name="c28228"></a>C28228
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28228：批註參數：注釋中不能使用類型  
  
 此警告表示參數的類型不受支援。 批註只能使用一組有限的類型做為參數。 如果標準宏用於批註，則應該不可能這麼做;這項警告在一般使用上並不會出現。
