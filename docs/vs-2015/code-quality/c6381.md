# <a name="c6381"></a>C6381
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6381：關機 API \<function> 需要有效的 dwReason 或 lpMessage  
  
 如果呼叫 InitiateSystemShutdownEx，就會發出這個警告：  
  
- 未通過有效的關機原因， (dwReason) 。 如果 dwReason 參數為零，預設值就是未定義的關機。 依預設，它也是未計畫的關機。 針對此參數，您應該使用其中一個系統關機原因代碼。  
  
- 不傳遞關機訊息 (lpMessage) 。  
  
  建議您在呼叫此 API 時使用適當的參數，以協助系統管理員判斷關機的原因。  
  
## <a name="example"></a>範例  
 下列程式碼會產生此警告，因為 dwReason 為零，而 lpMessage 為 null：  
  
```  
void f()  
{  
  //...  
  BOOL bRet;  
  bRet = InitiateSystemShutdownEx( NULL,  
                                   NULL, // message  
                                   0,          
                                   FALSE,      
                                   TRUE,       
                                   0);  // shutdown reason  
  // ...  
}  
```  
  
 若要更正這個警告，請指定 dwReason 和 lpMessage，如下列程式碼所示：  
  
```  
#include <windows.h>  
void f()  
{  
  //...  
  BOOL bRet;  
  bRet = InitiateSystemShutdownEx( NULL,  
                               "Hardware Failure",  // message    
                               0,          
                               FALSE,      
                               TRUE,       
                               SHTDN_REASON_MAJOR_HARDWARE ); // reason  
  // ...  
}  
```
