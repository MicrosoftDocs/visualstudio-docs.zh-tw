# <a name="c28275"></a>C28275
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28275： Macro_value 的參數 \_ \_ 為 null  
  
 此警告表示模型檔案中有內部錯誤，而不是在分析的程式碼中。 未使用參數呼叫 *macroValue* 函數。
