# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28227: 註釋不能也是 PrimOp (參閱之前的宣告)  
  
 此警告表示命名的注釋已宣告為一般批註，而且先前已宣告為 PrimOp。 如果標準宏用於批註，則應該不可能這麼做;這項警告在一般使用上並不會出現。
