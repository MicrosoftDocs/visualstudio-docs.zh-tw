# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主題的最新的版本可從[C28227](https://docs.microsoft.com/visualstudio/code-quality/c28227)。  
  
警告 C28227: 註釋不能也是 PrimOp (參閱之前的宣告)  
  
 這則警告表示具名的註解會被宣告為一般的註解，並也先前宣告為 PrimOp。 這不能進行標準巨集所使用的註解;這個警告不應該看到典型的使用中。
