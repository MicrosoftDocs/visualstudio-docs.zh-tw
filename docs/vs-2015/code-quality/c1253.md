# <a name="c1253"></a>C1253
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1253：無法載入模型檔。  
  
 當模型檔案中有內部錯誤，而不是在分析的程式碼時，程式碼分析工具會報告此警告。
