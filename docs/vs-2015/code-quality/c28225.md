# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28225： 註釋中找不到正確的必要參數數目  
  
 這個警告表示具名的註釋搭配參數數目不正確。 這不能進行標準巨集所使用的註解;這個警告不應該看到典型的使用中。
