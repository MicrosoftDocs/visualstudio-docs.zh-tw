# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28225：在批註中找不到正確的必要參數數目  
  
 此警告表示已命名的注釋使用了不正確的參數數目。 如果標準宏正用於注釋，則不應該這麼做;一般用途不應出現此警告。
