# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28225：在批註中找不到正確的必要參數數目  
  
 此警告表示使用了不正確參數數目的命名注釋。 如果標準宏用於批註，則應該不可能這麼做;這項警告在一般使用上並不會出現。
