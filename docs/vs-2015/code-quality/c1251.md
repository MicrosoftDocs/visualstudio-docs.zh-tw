# <a name="c1251"></a>C1251
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C1251：無法載入模型。  
  
 當模型檔案中有內部錯誤時，程式碼分析工具會報告這個警告，而不是正在分析的程式碼中。
