# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28213： `_Use_decl_annotations_` 批註必須用來參考先前的宣告，而不需要修改。 \<why>  
  
 如果未使用必要的注釋參考先前的宣告，就會回報此警告 `_Use_decl_annotations_` 。
