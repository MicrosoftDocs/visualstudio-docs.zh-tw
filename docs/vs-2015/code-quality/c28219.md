# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28219:註釋中標註的參數需要列舉的成員  
  
 為註解參數必須是成員的具名`enum`遇到類型，以及其他一些符號; 請使用該成員`enum`型別。 這通常表示的編碼錯誤註釋巨集。
