# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主題的最新的版本可從[C28219](https://docs.microsoft.com/visualstudio/code-quality/c28219)。  
  
警告 C28219： 註釋需要列舉的成員中的註釋的參數  
  
 為註解參數必須是成員的具名`enum`遇到類型，以及其他一些符號; 請使用該成員`enum`型別。 這通常表示的編碼錯誤註釋巨集。
