# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28219：批註中的參數必須是列舉的成員  
  
 注釋的參數應該是命名類型的成員 `enum` ，而且遇到一些其他符號，請使用該類型的成員 `enum` 。 這通常表示不正確編碼的注釋宏。
