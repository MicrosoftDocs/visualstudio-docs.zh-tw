# <a name="c28125"></a>C28125
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28125:Try / except 區塊內，必須從呼叫函式  
  
 驅動程式會呼叫函式必須呼叫從 try / except 區塊內這類[ProbeForRead](https://msdn.microsoft.com/library/windows/hardware/ff559876.aspx)， [ProbeForWrite](https://msdn.microsoft.com/library/windows/hardware/ff559879.aspx)， [MmProbeAndLockPages](https://msdn.microsoft.com/library/windows/hardware/ff554664.aspx)。  
  
## <a name="example"></a>範例  
 下列程式碼範例會產生此警告：  
  
```  
ProbeForRead(addr, len, 4);  
```  
  
 下列程式碼範例可避免此警告：  
  
```  
  
__try  
{  
    ProbeForRead(addr, len, 4);  
}  
__except(EXCEPTION_EXECUTE_HANDLER)  
{  
    Status = GetExceptionCode();  
    ... report error status  
}  
```
