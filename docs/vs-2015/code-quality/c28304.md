# <a name="c28304"></a>C28304
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28304：針對 c + + 參考-參數 <parameter_name>，找到未正確放置的 `_Notref_` 運算子套用至 \<token> 。  
  
 `_Notref_`操作員應該只在涉及 c + + 參考參數的特殊情況下使用，而且只適用于系統提供的宏。 其後必須緊接 `_Deref_` 運算子或功能注釋。
