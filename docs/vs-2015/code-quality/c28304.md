# <a name="c28304"></a>C28304
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28304： 對於 c + + 參考參數 < p >、 不當放置`_Notref_`找不到運算子套用至\<k >。  
  
 `_Notref_`運算子只能在包含 c + + 參考參數的特殊情況下，而且只用於系統提供的巨集。 它必須緊接著`_Deref_`運算子或功能的註解。
