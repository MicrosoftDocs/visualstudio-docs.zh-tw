# <a name="saving-entity-data"></a>儲存實體資料
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

當您透過 WCF Data Services 建立、更新或刪除連接至資料庫的實體時，必須明確將變更存回資料庫。 下列主題將說明如何儲存對 WCF Data Services 應用程式中實體資料的變更。  
  
|主題|描述|  
|-----------|-----------------|  
|[如何：新增、修改和刪除實體](http://msdn.microsoft.com/library/a00f8933-b232-4445-95ba-adc634f055d8)|示範如何透過資料服務傳回的實體資料中儲存的變更。|  
|[如何：執行非同步資料服務查詢](http://msdn.microsoft.com/library/902a2dc1-d0e9-4b00-90a8-becc4cb1f6a7)|示範如何以非同步方式儲存透過資料服務所做的變更。|
