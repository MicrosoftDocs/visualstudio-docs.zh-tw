# <a name="could-not-retrieve-schema-information-for-database-object-ltobject-namegt"></a>無法取出資料庫物件物件名稱的架構資訊 &lt;&gt;
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

當 **伺服器總管/資料庫總管** 中的物件複製到剪貼簿、從資料庫中刪除，然後貼到設計工具上時，通常會出現此訊息。 因為資料庫物件不再存在，所以會出現這則訊息。

### <a name="to-correct-this-error"></a>更正這個錯誤

- 按一下 [確定]**** 以關閉對話方塊。
