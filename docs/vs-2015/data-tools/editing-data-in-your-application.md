# <a name="editing-data-in-your-application"></a>在您的應用程式中編輯資料
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

當您的資料模型填入資料之後，您通常會加入、編輯或刪除部分資料，再將資料模型傳送回資料來源或其他處理序或應用程式。  
  
 下列主題的詳細資料是關於加入、編輯及刪除資料，以及如何使用應用程式各種階段中的資料。
