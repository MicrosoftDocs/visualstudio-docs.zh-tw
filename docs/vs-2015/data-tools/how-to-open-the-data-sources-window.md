# <a name="how-to-open-the-data-sources-window"></a>如何：開啟資料來源視窗
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**Zdroje dat**視窗會顯示在您的專案中的資料來源。 您使用**Zdroje dat**拖曳以建立資料繫結控制項的使用者介面項目從視窗拖曳到您的專案中的表單的視窗。 請參閱[繫結 Windows Form 控制項加入 Visual Studio 中的資料](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md)。  
  
 您可以填入**資料來源**視窗執行**資料來源組態**精靈。 如需詳細資訊，請參閱[資料來源組態精靈](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f)。  
  
### <a name="to-open-the-data-sources-window"></a>若要開啟資料來源視窗  
  
-   在功能表列上選擇 **檢視**，**其他 Windows**， **Zdroje dat** （或選擇 Shift + Alt + D 鍵）。  
  
    > [!IMPORTANT]
    >  只有當專案支援建立和使用資料來源時，您才可以將資料來源加入至該專案。 例如，您無法開啟**Zdroje dat**  視窗中的 Windows 市集應用程式的專案。
