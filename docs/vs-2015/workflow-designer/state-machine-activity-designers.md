# <a name="state-machine-activity-designers"></a>狀態機器活動設計工具
[!INCLUDE[wfd1](../includes/wfd1-md.md)] 包含一些活動設計工具，可讓您在建立工作流程時使用。 本節包含系統提供的活動設計工具，這些活動設計工具用來建立和管理狀態機器工作流程。 下列主題會說明這些設計工具，並且提供如何使用的指引。

## <a name="in-this-section"></a>本節內容
 [StateMachine](../workflow-designer/statemachine-activity-designer.md) 包含狀態和模型工作流程的集合，這些狀態和模型工作流程使用熟悉的狀態機器範例。

 [狀態](../workflow-designer/state-activity-designer.md) 代表狀態機器可以處於的狀態。

 [FinalState](../workflow-designer/finalstate-activity-designer.md) 表示終止狀態機器實例的狀態。

 [轉換](../workflow-designer/transition-activity-designer.md) 表示兩個狀態之間的轉換。
