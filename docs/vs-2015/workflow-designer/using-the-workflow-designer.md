# <a name="using-the-workflow-designer"></a>使用工作流程設計工具
本節詳述如何使用內建的設計工具，以及多個活動設計工具裝載的運算式編輯器，建立新的活動與工作流程專案。

## <a name="in-this-section"></a>本節內容
 [建立工作流程專案](../workflow-designer/creating-a-workflow-project.md) 描述如何透過使用這些範本來建立程式庫的主題。

 How [to：將活動新增至工具箱](../workflow-designer/how-to-add-activities-to-the-toolbox.md)顯示將活動加入至 [工具箱] 的不同方式。

 [如何：使用引數設計](../workflow-designer/how-to-use-the-argument-designer.md) 工具顯示引數設計工具如何讓資料能夠流入和流出活動。

 [如何：使用變數設計](../workflow-designer/how-to-use-the-variable-designer.md) 工具示範如何建立變數，以用於資料系結案例和條件陳述式中。

 [如何：使用運算式編輯器](../workflow-designer/how-to-use-the-expression-editor.md) 顯示如何使用運算式編輯器來輸入和評估運算式。

 [如何：使用匯入設計](../workflow-designer/how-to-use-the-imports-designer.md) 工具顯示如何輸入您在運算式中使用之類型的命名空間。

 [如何：使用](../workflow-designer/how-to-use-breadcrumb-navigation.md) 階層連結流覽顯示如何在活動之間流覽。
