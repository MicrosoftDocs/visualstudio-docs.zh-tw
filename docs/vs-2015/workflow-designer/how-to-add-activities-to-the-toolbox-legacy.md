# <a name="how-to-add-activities-to-the-toolbox-legacy"></a>HOW TO：將活動新增至工具箱 (舊版)
使用以或為目標的舊版來建立工作流程方案時 [!INCLUDE[wfd1](../includes/wfd1-md.md)] [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)] ，可以將自訂活動加入至工作流程專案，並將其設計 **工具** 放在工具箱中，以方便存取。 您也可以從動態連結程式庫 (DLL) ，直接將活動新增至 **工具箱** 。

### <a name="to-add-an-activity-to-the-toolbox-from-a-dll"></a>若要將活動從 DLL 新增至工具箱

1. 以滑鼠右鍵按一下 [ **Windows Workflow**] 下的 [工具箱] 視窗介面，然後按一下 **[選擇專案**]。

2. 在 [ **選擇工具箱專案** ] 對話方塊中，按一下 [系統] 的 [ **元件** ] 索引標籤，然後按一下視窗右下方的 **[流覽** ]。

3. 從包含要加入至 [ **工具箱**] 之自訂活動執行所在的檔案目錄中選取 DLL，然後按一下 [ **開啟**]。

4. 按一下 **[確定** ]，完成將活動加入至 [工具箱]。
