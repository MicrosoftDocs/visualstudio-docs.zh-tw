# <a name="how-to-add-activities-to-the-toolbox-legacy"></a>HOW TO：將活動新增至工具箱 (舊版)
使用以 [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] 或 [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)] 為目標的舊版 [!INCLUDE[wfd1](../includes/wfd1-md.md)] 來建立工作流程方案時，可以將自訂活動加入至工作流程專案，並將其設計**工具**放在 [工具箱] 中，以方便存取。 您也可以從動態連結程式庫（DLL）將活動直接新增至 [**工具箱**]。

### <a name="to-add-an-activity-to-the-toolbox-from-a-dll"></a>若要將活動從 DLL 新增至工具箱

1. 以滑鼠右鍵按一下 [ **Windows Workflow**] 底下的 [工具箱] 視窗介面，然後按一下 **[選擇專案**]。

2. 在 [**選擇工具箱專案**] 對話方塊中，按一下 [**系統活動元件**] 索引標籤，然後按一下視窗右下方的 **[流覽**]。

3. 從檔案目錄中選取包含要加入 [**工具箱**] 之自訂活動執行的 DLL，然後按一下 [**開啟**]。

4. 按一下 **[確定**]，完成將活動新增至 [工具箱] 的作業。
