# <a name="themes-workflow-designer-options-dialog-box-legacy"></a>選項對話方塊、工作流程設計工具、佈景主題 (舊版)
本主題描述如何在舊版 [!INCLUDE[wfd1](../includes/wfd1-md.md)] 中使用 [**主題]、[工作流程設計工具]、[選項**] 對話方塊。 當您需要以 [!INCLUDE[wfd2](../includes/wfd2-md.md)] 或 [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] 為目標時，請使用舊版 [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]。

 佈景主題會定義工作流程的背景色彩、前景色彩、樣式、圖示和其他視覺化項目。 您可以儲存佈景主題以供其他工作流程重複使用。 您可以使用 [主題]、[工作流程設計工具] **、[選項**] 對話方塊，選取預設和已儲存的自訂主題。 若要開啟此對話方塊，請按一下 [**工作流程**] 功能表上的 [**選取自訂主題**]。 或者，以滑鼠右鍵按一下工作流程設計介面，然後按一下 [**選取自訂主題**]。

 下表描述 [**主題]、[工作流程設計工具]、[選項**] 對話方塊的使用者介面（UI）元素。

|UI 項目|描述|
|----------------|-----------------|
|**新增**|按一下以開啟 [[主題設定] 對話方塊（舊版）](../workflow-designer/theme-configuration-dialog-box-legacy.md)以建立新的主題。|
|**修改 。**|按一下以開啟 [[主題設定] 對話方塊（舊版）](../workflow-designer/theme-configuration-dialog-box-legacy.md)以修改選取的主題。|
|**瀏覽**|按一下以瀏覽工作流程主題檔案，其副檔名為 .wtm。|
|**選取主題**|主題清單的主題名稱資料行標題。 從清單中選取主題，按一下 [**修改**] 以編輯，或按一下 **[確定]** 以針對工作流程使用選取的主題。|
|**說明**|主題清單的主題描述資料行標題。|
|**位置**|主題清單的主題位置資料行標題。|
