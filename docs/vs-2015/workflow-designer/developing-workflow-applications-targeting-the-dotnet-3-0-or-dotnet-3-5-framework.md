# <a name="developing-workflow-applications-targeting-the-net-30-or-net-35-framework"></a>以 .NET 3.0 或 .NET 3.5 Framework 為目標開發工作流程應用程式
[!INCLUDE[vs2010](../includes/vs2010-md.md)] 提供 [!INCLUDE[wfd1](../includes/wfd1-md.md)] 來建立以 [!INCLUDE[netfx40_long](../includes/netfx40-long-md.md)]為目標的 [!INCLUDE[wf](../includes/wf-md.md)] 應用程式。 設計工具會使用熟悉的 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] 使用者介面進行工作流程應用程式的圖形建構與偵錯。

 [!INCLUDE[vs2010](../includes/vs2010-md.md)] 也會提供舊版工作流程設計工具的存取，該設計工具可用來以 [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] 或 [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)] 為目標。 本章節包含使用這個舊版設計工具的指引。
