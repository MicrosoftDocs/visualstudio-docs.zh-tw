# <a name="add-parameters-and-add-arguments-dialog-boxes"></a>加入參數及加入引數對話方塊
下表說明設計工具所使用之 [**加入參數**] 對話方塊的使用者介面（UI）元素 <xref:System.Activities.Statements.InvokeMethod> 。

|名稱|描述|
|-|-|
|[方向]|指定參數表示資料傳入方法、從方法傳出資料，或兩者皆是。|
|**型別**|新參數的型別名稱。|
|**ReplTest1**|用來將預設值指派配給新參數的 VB 運算式|

 下表說明設計工具所使用之 [**加入引數**] 對話方塊的使用者介面（UI）元素 <xref:System.Activities.Statements.InvokeDelegate> 。

|名稱|描述|
|-|-|
|**名稱**|引數的名稱。|
|[方向]|指定引數表示資料傳入委派、從委派傳出資料，或兩者皆是。|
|**型別**|新引數的型別名稱。|
|**ReplTest1**|用於此委派引數執行個體的值。|
