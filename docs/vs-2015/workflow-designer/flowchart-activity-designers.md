# <a name="flowchart-activity-designers"></a>Flowchart 活動設計工具
[!INCLUDE[wfd1](../includes/wfd1-md.md)] 包含一些系統提供的活動，可供您在建立工作流程時使用。 本節包含用來管理工作流程中之流程圖的系統提供活動。 下列主題說明會這些活動，並且提供如何使用的指引。

## <a name="in-this-section"></a>本節內容
 **用來建立工作流程的[流程圖](../workflow-designer/flowchart-activity-designer.md)，可定義及管理複雜的流程式控制制。**  [FlowDecision](../workflow-designer/flowdecision-activity-designer.md)根據是否符合指定的條件，將控制流程的分支提供給兩個替代專案的其中一個。

 [FlowSwitch \<T >](../workflow-designer/flowswitch-t-activity-designer.md)當需要兩個以上的替代分支時，會根據比對準則提供控制流程的分支。
