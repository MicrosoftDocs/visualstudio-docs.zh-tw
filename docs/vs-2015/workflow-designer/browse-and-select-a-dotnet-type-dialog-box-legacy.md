# <a name="browse-and-select-a-net-type-dialog-box-legacy"></a>瀏覽並選取 .NET 類型對話方塊 (舊版)
本主題描述如何使用舊版 [!INCLUDE[wfd1](../includes/wfd1-md.md)] 中的 [**流覽並選取 .Net 類型**] 對話方塊。 當您需要以 [!INCLUDE[wfd2](../includes/wfd2-md.md)] 或 [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] 為目標時，請使用舊版 [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]。

 在 [**屬性**] 視窗中，當您選取接受所參考元件中之 .NET Framework 類型的屬性時，屬性的文字方塊結尾會出現省略號 **[...]** 。 按一下 **[...]** 即可開啟 [**流覽並選取 .net 類型**] 對話方塊。 在這個對話方塊中，您可以從參考組件的樹狀檢視中選擇型別。 例如，當您使用活動設計工具時，請在 [**屬性**] 視窗中，按一下 [**基類**省略號 **[...]]** ，從 [參考的元件] 樹狀結構中選取活動的另一個基類。

 下表描述 [**流覽並選取 .Net 類型**] 對話方塊的使用者介面（UI）元素。

|UI 項目|描述|
|----------------|-----------------|
|**類型名稱：**|目前選取之型別的名稱。|
|**Type**|左窗格會顯示 [參考的組件] 的樹狀檢閱。 在左窗格中所選 [參考的組件] 的選擇項目其可用的型別，會在右窗格中顯示。|
