# <a name="bind-to-an-activity39s-property-dialog-box-legacy"></a>系結至活動&#39;的屬性對話方塊（舊版）
本主題描述如何在舊版  **中使用 [** 繫結至活動的屬性[!INCLUDE[wfd1](../includes/wfd1-md.md)]] 對話方塊。 當您需要以 [!INCLUDE[wfd2](../includes/wfd2-md.md)] 或 [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] 為目標時，請使用舊版 [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]。

 相依性屬性執行個體型別可以繫結至另一個活動的公用屬性或事件。 如需活動系結的詳細資訊，請參閱使用相依性[屬性](https://go.microsoft.com/fwlink?LinkID=65007)。

 您可以使用 [**繫結至活動的屬性**] 對話方塊來選取要繫結的屬性。 按一下 [**屬性**] 視窗中所選屬性其文字方塊尾端的省略符號 **[…]** ，或按一下屬性瀏覽器中屬性名稱旁的藍色驚嘆號圖示，開啟這個對話方塊。

 下表說明 [**繫結至活動的屬性**] 對話方塊的使用者介面 (UI) 項目。

|UI 項目|描述|
|----------------|-----------------|
|**系結至現有的成員**|在樹狀檢視窗格中選取您要繫結的成員。 樹狀檢閱下方的窗格會顯示一個訊息，表示成員是否為可繫結程序的有效型別。 按一下 [**確定**] 可繫結至選取的有效成員。|
|**系結到新成員**|建立要繫結的新成員欄位或屬性。 輸入 [**新成員名稱**]。 選取 [**建立欄位**] 或 [**建立屬性**] 來選擇您要建立相依性屬性或公用欄位。 按一下 [**確定**] 建立新成員。|
