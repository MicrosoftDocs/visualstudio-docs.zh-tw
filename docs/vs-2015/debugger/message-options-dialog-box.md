# <a name="message-options-dialog-box"></a>訊息選項對話方塊
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

使用此對話方塊來選取要在 [ [訊息] 視圖](../debugger/messages-view.md)中列出的訊息。 若要顯示此對話方塊，請從**Spy**功能表選擇 [**記錄訊息**]。  
  
## <a name="in-this-section"></a>本節內容  
 [訊息選項對話方塊、視窗索引標籤](../debugger/windows-tab-message-options-dialog-box.md)  
 用來選取要列出的訊息類型。 包含 Finder 工具。  
  
 [訊息選項對話方塊、訊息索引標籤](../debugger/messages-tab-message-options-dialog-box.md)  
 用來選取要進行查看的訊息類型。  
  
 [訊息選項對話方塊、輸出索引標籤](../debugger/output-tab-message-options-dialog-box.md)  
 用來指定要顯示的訊息資料。
