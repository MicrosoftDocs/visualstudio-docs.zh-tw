# <a name="tree-menu-commands"></a>樹狀結構功能表命令
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**樹狀結構**功能表提供命令，以展開或摺疊檢視。  
  
 [展開一層]   
 展開目前選取的項目，在系統檢視中顯示其下的一個層級。  
  
 **展開分支**  
 展開目前選取的項目底下的所有層級。  
  
 [全部展開]   
 會展開並顯示在系統檢視中的所有層級。  
  
 [摺疊]   
 摺疊目前選取的項目底下的所有層級。
