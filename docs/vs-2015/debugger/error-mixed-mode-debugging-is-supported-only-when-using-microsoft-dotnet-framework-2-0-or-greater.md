# <a name="error-mixed-mode-debugging-is-supported-only-when-using-microsoft-net-framework-20-or-greater"></a>錯誤：只有使用 Microsoft .NET Framework 2.0 或更新版本時才支援混合模式偵錯
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

若要混合偵錯機器碼和 Managed 程式碼，您必須要有 [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] 2.0、3.0、 3.5 或 4 版。 混合模式偵錯與舊版[!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)]不支援。  
  
### <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] 升級為 2.0、3.0、3.5 或 4 版。
