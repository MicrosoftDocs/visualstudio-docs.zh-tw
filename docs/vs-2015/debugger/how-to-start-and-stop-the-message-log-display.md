# <a name="how-to-start-and-stop-the-message-log-display"></a>作法：啟動與停止訊息記錄顯示
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

當[訊息檢視](../debugger/messages-view.md)視窗是作用中，**開始記錄**或**停止記錄**選擇出現在 Spy + +**訊息** 功能表中，而對應**開始記錄**或是**停止記錄**工具列按鈕變成無法使用。  
  
### <a name="to-start-the-message-log-display"></a>若要啟動訊息記錄顯示  
  
1. 從**訊息**功能表上，選擇**開始記錄**。  
  
### <a name="to-stop-the-message-log-display"></a>若要停止訊息記錄顯示  
  
1. 從**訊息**功能表上，選擇**停止記錄**。
