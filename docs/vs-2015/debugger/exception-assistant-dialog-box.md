# <a name="exception-assistant-dialog-box"></a>例外狀況助理對話方塊
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**例外狀況助理**擲回執行階段例外狀況時，對話方塊隨即出現。 **例外狀況助理**顯示例外狀況類型、 提供其他資訊和連結至疑難排解提示、 提供搜尋其他線上，說明的方法和可讓使用者執行特定動作，例如檢視例外狀況的詳細資料。  
  
## <a name="task-list"></a>工作清單  
 若要查看主題處理疑難排解您遇到的例外狀況類型，請按一下其中一個提示訊息中顯示**疑難排解祕訣**窗格。  
  
 若要執行與例外狀況相關聯的動作，請按一下執行窗格中顯示的其中一個動作。  
  
 如需如何啟用或停用例外狀況助理的詳細資訊，請參閱[General，Debugging，Options Dialog Box](../debugger/general-debugging-options-dialog-box.md)。  
  
## <a name="user-interface-element-list"></a>使用者介面項目清單  
 **例外狀況類型**  
 顯示擲回的例外狀況類型。  
  
 **其他資訊**  
 顯示例外狀況的其他相關資訊。  
  
 **疑難排解祕訣**  
 顯示疑難排解提示的連結，這些連結可能幫助您找出例外狀況的來源。  
  
 **動作**  
 列出可以執行的動作，例如查看有關例外狀況物件的詳細資訊。  
  
 **取得線上說明**  
 可讓您搜尋其他線上說明。
