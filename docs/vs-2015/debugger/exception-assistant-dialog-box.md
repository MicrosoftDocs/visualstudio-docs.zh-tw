# <a name="exception-assistant-dialog-box"></a>例外狀況助理對話方塊
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

擲回執行時間例外狀況時，[ **例外狀況助理** ] 對話方塊隨即出現。 [ **例外狀況助理** ] 會顯示例外狀況的類型、提供其他資訊和疑難排解提示的連結、提供一種方式來搜尋其他線上說明，並允許使用者執行特定的動作，例如查看例外狀況的詳細資料。  
  
## <a name="task-list"></a>工作清單  
 若要查看針對您遇到的例外狀況類型進行疑難排解的主題，請按一下 [ **疑難排解提示** ] 窗格中顯示的其中一個提示訊息。  
  
 若要執行與例外狀況相關聯的動作，請按一下執行窗格中顯示的其中一個動作。  
  
 如需如何啟用或停用例外狀況助理的詳細資訊，請參閱 [[選項] 對話方塊的 [一般]、[偵錯工具]](../debugger/general-debugging-options-dialog-box.md)。  
  
## <a name="user-interface-element-list"></a>使用者介面項目清單  
 **例外狀況類型**  
 顯示擲回的例外狀況類型。  
  
 **其他資訊**  
 顯示例外狀況的其他相關資訊。  
  
 **疑難排解秘訣**  
 顯示疑難排解提示的連結，這些連結可能幫助您找出例外狀況的來源。  
  
 **動作**  
 列出可以執行的動作，例如查看有關例外狀況物件的詳細資訊。  
  
 **取得協助 Online**  
 可讓您搜尋其他線上說明。
