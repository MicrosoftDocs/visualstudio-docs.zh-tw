# <a name="where-can-i-look-up-win32-error-codes"></a>哪裡可以查看 Win32 錯誤碼？
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

您的預設系統安裝 INCLUDE 目錄裡的 WINERROR.H 包含 Win32 API 函式的錯誤碼定義。  
  
 您可以輸入中的程式碼查閱錯誤碼**監看式**視窗或**快速監看式** 對話方塊。 例如:   
  
```  
0x80000004,hr  
```
