# <a name="windows-tab-message-options-dialog-box"></a>訊息選項對話方塊、視窗索引標籤
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

您可以使用 [ **Windows** ] 索引標籤來設定 [ [訊息] 視圖](../debugger/messages-view.md)中列出的訊息範圍。 若要顯示 [[訊息選項] 對話方塊](../debugger/message-options-dialog-box.md)，請從 [ **Spy** ] 功能表選擇 [**記錄訊息**]。  
  
 [ **Windows** ] 索引標籤上提供下列設定：  
  
 [搜尋工具]****  
 用來選取視窗的拖放工具。  
  
 **隱藏 Spy + +**  
 移除主要的 Spy + + 視窗，並讓 **MessageOptions** 對話方塊出現在其他應用程式的頂端。  
  
 **父視窗**  
 顯示所選視窗及其直屬父視窗的訊息。  
  
 **子視窗**  
 顯示所選視窗及其所有子視窗的訊息，包括嵌套的子視窗。  
  
 [相同執行緒的視窗]****  
 顯示所選視窗和相同執行緒所擁有之所有其他視窗的訊息。  
  
 [相同處理序的視窗]****  
 顯示所選視窗的訊息，以及同一個進程擁有的所有其他視窗。  
  
 [系統中的所有視窗]****  
 顯示所有視窗的訊息。  
  
 [將設定另存成預設]****  
 針對新的訊息串流視窗儲存先前的設定。 當結束 Spy + + 時，也會儲存這些設定。
