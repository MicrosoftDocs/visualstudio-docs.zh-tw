# <a name="vsgdbgvsgdbg-destructor"></a>VsgDbg::~VsgDbg (解構函式)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

終結類別的實例 `VsgDbg` 。 如果正在記錄圖形資訊，圖形記錄檔將會完成並關閉，而在主動捕獲圖形資訊時使用的資源則會放開。  
  
## <a name="syntax"></a>語法  
  
```cpp  
~VsgDbg();  
```
