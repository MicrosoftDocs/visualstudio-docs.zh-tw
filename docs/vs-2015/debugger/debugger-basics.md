# <a name="debugger-basics"></a>偵錯工具基礎
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]
Visual Studio 偵錯工具讓您能夠觀察程式的執行階段行為，並找出邏輯錯誤。 偵錯工具可以與所有 Visual Studio 程式設計語言及其相關程式庫搭配使用。 藉助此偵錯工具，您可以中斷執行，以便逐步執行程式碼和評估應用程式中的變數。
