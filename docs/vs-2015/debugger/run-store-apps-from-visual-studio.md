# <a name="run-store-apps-from-visual-studio"></a>從 Visual Studio 執行市集應用程式
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

市集應用程式在多種裝置上執行，而且 Visual Studio 提供數種方式，讓您執行那些應用程式，並與之互動。 "F5" 體驗已獲增強，可在開發期間輕鬆地偵錯、分析和測試應用程式。
