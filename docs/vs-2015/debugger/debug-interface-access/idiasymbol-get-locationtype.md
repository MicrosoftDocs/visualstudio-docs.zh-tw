# <a name="idiasymbolgetlocationtype"></a>IDiaSymbol::get_locationType
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

本主題的最新的版本可從[idiasymbol:: Get_locationtype](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-locationtype)。  
  
擷取資料符號的位置類型。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT get_locationType (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回值，以從[LocationType 列舉](../../debugger/debug-interface-access/locationtype.md)列舉，指定位置的資料符號類型，例如`static`或`local`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。  
  
> [!NOTE]
>  傳回值為`S_FALSE`表示此屬性不適用於符號。
