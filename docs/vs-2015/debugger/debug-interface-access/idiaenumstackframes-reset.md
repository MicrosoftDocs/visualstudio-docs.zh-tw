# <a name="idiaenumstackframesreset"></a>IDiaEnumStackFrames::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

將列舉序列重設為開頭。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>傳回值  
 傳回 `S_OK`。
