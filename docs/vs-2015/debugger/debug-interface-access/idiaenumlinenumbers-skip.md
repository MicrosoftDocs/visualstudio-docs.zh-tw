# <a name="idiaenumlinenumbersskip"></a>IDiaEnumLineNumbers::Skip
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

略過列舉序列中指定數目的行號。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT Skip (   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>參數  
 celt  
 在列舉順序中要跳過的行號數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功， `S_OK` 則傳回; 否則， `S_FALSE` 如果沒有其他行號可略過，則會傳回。
