# <a name="idiaenumsectioncontribsreset"></a>IDiaEnumSectionContribs::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

將列舉順序重設為開頭。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>傳回值  
 傳回 `S_OK`。
