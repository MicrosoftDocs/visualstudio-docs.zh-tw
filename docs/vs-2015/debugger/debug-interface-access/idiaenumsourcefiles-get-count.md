# <a name="idiaenumsourcefilesgetcount"></a>IDiaEnumSourceFiles::get_Count
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

擷取來源檔案的數目。  
  
## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 pRetVal  
 [out]傳回原始程式檔數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回錯誤碼。
