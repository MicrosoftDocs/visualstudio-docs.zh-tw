# <a name="idiaenumsymbolsreset"></a>IDiaEnumSymbols::Reset

## <a name="syntax"></a>語法  
  
```cpp#  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>傳回值  
 傳回 `S_OK`。
