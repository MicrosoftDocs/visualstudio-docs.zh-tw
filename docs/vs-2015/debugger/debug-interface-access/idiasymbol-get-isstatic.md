# <a name="idiasymbolgetisstatic"></a>IDiaSymbol::get_isStatic
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

本主題的最新的版本可從[idiasymbol:: Get_isstatic](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-isstatic)。  
  
擷取旗標，指定是否函式或 thunk 層已標示為靜態。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT get_isStatic(  
   BOOL *pFlag  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pFlag`  
 [out]會傳回`TRUE`函式或 thunk 層已標示為靜態，否則為，如果傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。  
  
> [!NOTE]
>  傳回值為`S_FALSE`表示此屬性不適用於符號。
