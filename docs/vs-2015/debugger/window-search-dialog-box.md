# <a name="window-search-dialog-box"></a>視窗搜尋對話方塊
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

使用這個對話方塊，即可在 [Windows View](../debugger/windows-view.md)中尋找並選取特定視窗的節點。 若要顯示此對話方塊，請將焦點移至 Windows view 視窗。 然後選擇 [**搜尋**] 功能表中的 [**尋找視窗]** 。  
  
 您可以使用 [ **Finder] 工具** 來選取視窗，也可以輸入視窗控制碼，或依標題和類別來識別視窗。  
  
 可用的設定如下：  
  
 [搜尋工具]****  
 用來選取視窗的拖放工具。  
  
 **隱藏 Spy + +**  
 移除主要的 Spy + + 視窗，並讓 **WindowSearch** 對話方塊出現在其他應用程式的頂端。  
  
 **Handle**  
 要搜尋之視窗的唯一識別碼。  
  
 **Caption**  
 要搜尋之視窗標題中的文字。  
  
 **類別**  
 要搜尋之視窗的類別。  
  
 **向上或向下搜尋方向**  
 搜尋的初始方向。
