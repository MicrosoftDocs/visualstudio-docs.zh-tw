# <a name="messages-menu-commands"></a>訊息功能表命令
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**訊息**功能表提供控制訊息檢視命令。  
  
 **啟動/停止記錄**  
 啟動或停止 [訊息] 檢視。  
  
 [記錄選項]  
 會開啟**MessageOptions**對話方塊中，您可以在其中指定訊息的搜尋準則。  
  
 [清除記錄檔]  
 清除 使用中的 訊息 檢視的內容。
