# <a name="how-to-change-fonts"></a>作法：變更字型
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

您可以變更字型、 字型樣式和字型大小，Spy + + windows。  
  
### <a name="to-change-font-options"></a>若要變更字型選項  
  
1. 從**檢視**功能表上，選擇**字型**。  
  
2. 選擇字型、 字型樣式和字型大小。  
  
3. 按一下 [確定]  。  
  
   選取**預設為儲存字型**會導致所有未來 Spy + + windows 會使用此字型。
