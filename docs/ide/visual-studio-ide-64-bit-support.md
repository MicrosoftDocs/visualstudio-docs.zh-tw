# <a name="visual-studio-ide-64-bit-support"></a>Visual Studio IDE 64 位元支援

Visual Studio 可讓您將應用程式的目標設定為不同的平台，包括 64 位元的平台。 有關 Visual Studio 中的 64 位平臺支援的詳細資訊，請參閱[64 位應用程式](/dotnet/framework/64-bit-apps)。
