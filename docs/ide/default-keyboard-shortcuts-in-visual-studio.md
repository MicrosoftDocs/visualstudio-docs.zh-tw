# <a name="default-keyboard-shortcuts-in-visual-studio"></a>Visual Studio 中的預設鍵盤快速鍵

您可以通過選擇適當的鍵盤快速鍵訪問 Visual Studio 中的各種[命令](reference/visual-studio-commands.md)和視窗。 此頁面會針對您可能已經在安裝 Visual Studio 時選擇的 [一般]**** 設定檔，列出預設命令捷徑。 無論您選擇了哪個設定檔，您可以通過打開 **"選項"** 對話方塊、展開 **"環境"** 節點，然後選擇 **"鍵盤**"[來標識](identifying-and-customizing-keyboard-shortcuts-in-visual-studio.md)命令的快捷方式。 您也可以為指定的命令指派不同的捷徑，以自訂捷徑。

如需常用鍵盤快速鍵的清單和其他產能資訊，請參閱：

- [常用命令的鍵盤快速鍵](../ide/default-keyboard-shortcuts-for-frequently-used-commands-in-visual-studio.md)
- [鍵盤祕訣](../ide/productivity-shortcuts.md)
- [生產力提示](../ide/productivity-features.md)。

有關 Visual Studio 中協助工具的詳細資訊，請參閱[協助工具提示和技巧](../ide/reference/accessibility-tips-and-tricks.md)以及如何[：獨佔使用鍵盤](../ide/reference/how-to-use-the-keyboard-exclusively.md)。
