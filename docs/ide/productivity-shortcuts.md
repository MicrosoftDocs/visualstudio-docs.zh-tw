# <a name="shortcut-tips-for-visual-studio"></a>適用於 Visual Studio 的快速鍵祕訣

您可以使用此文章中的快速鍵，更輕鬆地在 Visual Studio 中導覽。 這些捷徑包括能夠協助您更輕鬆完成工作的鍵盤和滑鼠快速鍵，以及您可以輸入的文字。

如需命令捷徑的完整清單，請參閱[預設鍵盤快速鍵](../ide/default-keyboard-shortcuts-in-visual-studio.md)。

> [!NOTE]
> 此主題適用於 Windows 上的 Visual Studio。 針對 Visual Studio for Mac，請參閱 [Visual Studio for Mac 中的常用鍵盤快速鍵](/visualstudio/mac/keyboard-shortcuts)。

## <a name="window-management"></a>視窗管理

|工作|快速鍵|
|-|-|
|最大化浮動視窗|**按兩下**標題列|
|最大化/最小化視窗|**Win**+**向上鍵** / **Win**+**向下鍵**|
|重新停駐浮動視窗|**Ctrl**+**按兩下**標題列|
|移動/停駐浮動視窗|**Win**+**向左鍵** / **Win**+**向右鍵**|
|關閉使用中的文件|**Ctrl**+**F4**|
|顯示開啟檔案清單|**Ctrl**+**Alt**+**向下鍵**|
|顯示所有浮動視窗|**Ctrl**+**Shift**+**M**|
|顯示捷徑清單|**Win**+**Alt**+**N**|
|開始新執行個體|**Win**+**Shift**+**N**|
|切換視窗|**Win**+**N**|

## <a name="search"></a>搜尋

|工作|快速鍵|
|-|-|
|方案總管搜尋|**Ctrl**+ **;**|
|將焦點放在任何工具視窗的搜尋方塊 (除了編輯器之外)|**Alt**+ **`** ，當工具視窗有焦點時|
|搜尋 Visual Studio|**Ctrl**+**Q**|
|搜尋方塊結果篩選|@opt - 選項<br />@cmd - 命令<br />@mru - 最近使用<br />@doc - 開啟的文件|
|在工具選項中搜尋|**Ctrl**+**E**|

|-|-|
|快速尋找|**Ctrl**+**F**|

|快速尋找上一個結果|**Shift**+**Enter**|

|快速取代 - 取代下一個|**Alt**+**R**|
|快速取代 - 全部取代|**Alt**+**A**|

|檔案中取代|**Ctrl**+**Shift**+**H**|
