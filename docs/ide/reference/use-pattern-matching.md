# <a name="use-pattern-matching"></a>使用模式比對

此重構適用於：

- C#

事項 **：** 使用模式比對。

時機 **：** 您想要使用新的 [c # 模式](https://docs.microsoft.com/dotnet/csharp/whats-new/csharp-9#pattern-matching-enhancements)比對功能。

**原因：** 這種重構會讓您的程式碼更清楚且簡潔。

## <a name="how-to"></a>使用方法

1. 將您的插入號放在語句內。

2. 按下 **Ctrl** + **。** 以觸發 [快速動作與重構] 功能表。

3. 選取 [使用模式比對]。
