# <a name="options-text-editor-u-sql-intellisense"></a>選項、文字編輯器、U-SQL、IntelliSense

使用 [IntelliSense] 選項頁面修改 U-SQL 的一些文字編輯器設定。 若要存取此選項頁面，請選擇 [**工具**  >  **選項**]，然後選擇 [**文字編輯器**]-[文字編輯器]  >    >  ****。

## <a name="intellisense-settings"></a>IntelliSense 設定

選取核取方塊，以啟用 [快速諮詢] 或 [Intellisense]。 當您將滑鼠指標暫留在變數上時，就會顯示快速諮詢。

## <a name="completion-lists"></a>完成清單

- **在輸入字元之後顯示完成清單**

   選取這個選項時，IntelliSense 會在您開始輸入時自動顯示完成清單。 如果您未選取此選項，則可以從 [intellisense] 功能表或按 **Ctrl** 空格鍵來使用 intellisense 完成  +  ****。

- **將關鍵字置於完成清單中**

   選取此選項時，IntelliSense 會在完成清單中包含關鍵字。

- **將程式碼片段置於完成清單中**

   選取此選項時，IntelliSense 會在完成清單中包含程式碼片段。

## <a name="selection-in-completion-list"></a>完成清單中的選取範圍

- **輸入下列字元表示認可**

   此欄位所顯示的字元，會造成認可目前反白顯示的完成清單建議。 您可以從此清單中新增或移除字元。

- **按空格鍵表示認可**

   選取此選項時，您可以藉由按下空白鍵來認可反白顯示的完成清單建議。

- **當輸入完整的單字後按 Enter 時，於行尾新增新行**

   選取時，當您輸入完成清單建議的所有字元之後，系統會自動新增一行，且游標會移動到新的行。
