# <a name="go-to-line"></a>移至行
此對話方塊可讓您移至使用中文件內的特定行。 若要存取此對話方塊，請開啟文件以進行編輯，然後選取 [編輯] 功能表中的 [移至]。  
  
 行號 (1 - `n`)  
 可讓您輸入使用中文件內您要移至的行號。 輸入的數字必須落在 1 與 `n` 之間，即目前文件中的行號。
