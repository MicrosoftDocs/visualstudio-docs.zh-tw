# <a name="sort-usings"></a>為 using 排序

此重構適用於：

- C#

- Visual Basic

事項 **：** 排序 using。

時機 **：** 您想要在檔案頂端排序指示詞， `using` 讓它們依字母順序排列。 

**原因：** 這可讓您更輕鬆地找到 using 指示詞。

## <a name="how-to"></a>使用方法

1. 從功能表列選取 [編輯]。
2. 選取 [ **Intellisense**  >  **排序 using**]。
