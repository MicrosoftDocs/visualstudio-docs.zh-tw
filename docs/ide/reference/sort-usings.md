# <a name="sort-usings"></a>為 Using 排序

此重構適用於：

- C#

- Visual Basic

**內容：** 對使用進行排序。

**何時：** 您希望對檔頂部`using`的指令進行排序，以便它們按字母順序排列。 

**原因：** 它使查找 using 指令變得更加容易。

## <a name="how-to"></a>操作方式

1. 從功能表列選取 [編輯]****。
2. 選擇 **"感知** > **排序使用**"。
