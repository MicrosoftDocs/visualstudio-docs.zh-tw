# <a name="make-class-abstract"></a>將類別設為抽象

此重構適用於：

- C#

- Visual Basic

事項 **：** 將類別設為抽象重構。

時機 **：** 您在不是抽象的類別中撰寫抽象方法。

**原因：**  撰寫程式碼以在撰寫抽象方法之後讓類別變成抽象，可節省您的時間。

## <a name="how-to"></a>使用方法

1. 將您的插入號放在抽象方法上。

2. 按下 **Ctrl** + **。** 以觸發 [快速動作與重構] 功能表。

3. 選取 [ **使類別成為抽象**]。

    ![將類別設為抽象](media/make-class-abstract.png)
