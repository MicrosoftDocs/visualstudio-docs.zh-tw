# <a name="make-member-static"></a>將成員設為靜態

此重構適用於：

- C#

**內容：** 使成員保持靜態。

**何時：** 您希望非靜態成員是靜態的。

**原因：** 靜態成員提高了可讀性：知道特定代碼是隔離的，可以更容易理解、重讀和重用。 

## <a name="how-to"></a>操作方式

1. 把你的護理放在會員名稱上。

2. 按**Ctrl**+**。** （期間）以觸發**快速操作和重構**功能表。

   ![將成員設為靜態](media/make-member-static.png)

3. 選取 [設為靜態]****。
