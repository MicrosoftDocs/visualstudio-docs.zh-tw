# <a name="evaluate-statement-command"></a>評估陳述式命令

評估並顯示指定的陳述式。

## <a name="syntax"></a>語法

```cmd
>Debug.EvaluateStatement text
```

## <a name="arguments"></a>引數

`text`

必要項。 要評估的陳述式。

## <a name="example"></a>範例

```cmd
>Debug.EvaluateStatement args.Length
```
