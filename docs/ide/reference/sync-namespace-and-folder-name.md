# <a name="sync-namespace-and-folder-name"></a>同步命名空間和資料夾名稱

此重構適用於：

- C#

**功能：** 同步命名空間和資料夾名稱。

**時機：** 您想要透過將檔案拖曳到新的資料夾來重新架構您的解決方案各個部分。 

**原因：** 您想要確定您的命名空間與新的資料夾結構保持在最新狀態。

## <a name="how-to"></a>操作說明

1. 將游標放在命名空間名稱中。
2. 在字行任何地方按 **Ctrl**+ **.** ， 以觸發 [快速動作與重構]  功能表。
3. 選取 [將命名空間變更為 \< 資料夾名稱>]  。
