# <a name="sync-namespace-and-folder-name"></a>同步命名空間和資料夾名稱

此重構適用於：

- C#

**內容：** 同步命名空間和資料夾名稱。

**何時：** 您希望通過將檔拖動到新資料夾來重新構建解決方案的某些部分。 

**原因：** 您希望確保命名空間與新資料夾結構保持最新。

## <a name="how-to"></a>操作方式

1. 將游標放在命名空間名稱中。
2. 按**Ctrl**+**。** 以觸發 [快速動作與重構]**** 功能表。
3. 選取 [將命名空間變更為 \< 資料夾名稱>]****。
