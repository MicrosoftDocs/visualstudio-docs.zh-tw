# <a name="add-explicit-cast"></a>新增明確轉換

此程式碼產生適用於：

- C#

**內容:** 允許您根據使用方式自動向表達式添加顯式強制轉換。

**何時:** 您需要向表示式添加顯式強制轉換,並希望自動正確分配它。

**原因:** 您可以手動向表達式添加顯式強制轉換,但此功能會根據代碼上下文自動添加它。

## <a name="how-to-use-it"></a>如何使用它

1. 把你的關心錯誤。
2. 按**Ctrl**+**。** 以觸發 [快速動作與重構]**** 功能表。
3. 選擇 **「添加顯式強制轉換**」 。

   ![在視覺化工作室中加入顯式強制轉換快速操作](media/add-explicit-cast.png)
