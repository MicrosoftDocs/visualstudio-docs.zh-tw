# <a name="code-analysis-project-designer"></a>專案設計工具、程式碼分析

包含您可以選擇在您的程式碼上執行的程式碼分析工具。 此工具會報告您組件的相關資訊，例如違反由 Microsoft .NET 設計方針所制定的程式設計和設計規則的情況。

## <a name="uielement-list"></a>UIElement 清單

 啟用程式碼分析\
 啟用或停用專案的程式碼分析。

 設計規則\
 啟用或停用設計規則。 您也可以展開此項目來啟用或停用個別規則。
