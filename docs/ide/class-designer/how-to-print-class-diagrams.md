# <a name="how-to-print-class-diagrams"></a>HOW TO：列印類別圖表

您可以使用 Visual Studio 的列印功能來列印類別圖表。

## <a name="to-print-a-class-diagram"></a>列印類別圖表

1.  開啟類別圖表。

2.  按一下 [檔案] 功能表中的 [列印]。

整份類別圖表隨即列印。 您可能需要調整 [版面設定] 對話方塊中的設定，才能以適當大小列印。
