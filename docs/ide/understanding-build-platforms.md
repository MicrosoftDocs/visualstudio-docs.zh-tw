# <a name="understand-build-platforms"></a>了解組建平台

您可以儲存套用至不同目標平台之不同版本的方案和專案屬性。 例如，您可以建立以 x86 平台為目標的偵錯組態以及以 x64 平台為目標的偵錯組態。 您可以快速變更使用中平台，以輕鬆地建置多個組態。
