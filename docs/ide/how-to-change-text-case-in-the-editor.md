# <a name="how-to-change-text-case-in-the-editor"></a>如何：變更編輯器中的文字大小寫

您可以使用功能表命令或鍵盤快速鍵，將所選取文字的大小寫轉換成全部大寫或全部小寫。

> [!NOTE]
> 您所看到對話方塊與功能表命令可能會與本文中描述的不同，因為本文是以 [一般]**** 環境設定為基礎。 要更改環境設置，請選擇 **"工具** > **導入和匯出設置**"，然後選擇"**重置所有設置**"。

## <a name="to-change-text-case"></a>變更文字大小寫

1. 選取您要轉換的文字。

2. 要將文本轉換為所有大寫字母，請選擇 **"編輯** > **高級** > **大寫"** 或按**Ctrl**+**Shift**+**U**。

   要將文本轉換為所有小寫，請選擇 **"編輯** > **高級** > **小寫"** 或按**Ctrl**+**U**。 (如果您已安裝 C++ 開發工作負載，此按鍵繫結關係可能已被不同的命令所使用。)

> [!TIP]
> 若要還原為這項變更之前的先前大小寫格式，請選取 [編輯]**** 功能表中的 [復原]****。
