# <a name="how-to-change-text-case-in-the-editor"></a>作法：變更編輯器中的文字大小寫

您可以使用功能表命令或鍵盤快速鍵，將所選取文字的大小寫轉換成全部大寫或全部小寫。

> [!NOTE]
> 您所看到對話方塊與功能表命令可能會與本文中描述的不同，後者是根據 [一般]  開發設定。 若要變更您的開發設定，請選擇 [工具]   > [匯入和匯出設定]  ，然後選擇 [重設所有設定]  。

## <a name="to-change-text-case"></a>變更文字大小寫

1. 選取您想要轉換的文字。

2. 若要將文字轉換成全部大寫，請選擇 [編輯]   > [進階]   > [設成大寫]  ，或按 **Ctrl**+**Shift**+**U**。

   若要將文字轉換成全部小寫，請選擇 [編輯]   > [進階]   > [設成小寫]  ，或按 **Ctrl**+**U**。

> [!TIP]
> 若要還原為這項變更之前的先前大小寫格式，請選取 [編輯]  功能表中的 [復原]  。
