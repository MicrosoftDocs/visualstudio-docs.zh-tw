# <a name="adding-and-removing-project-items"></a>新增和移除專案項目
以滑鼠右鍵按一下方案總管中的專案或方案節點，然後選擇 [新增] 或 [移除]，即可新增或移除專案和專案項目。 移除方案中的專案或移除專案中的項目，並不會從磁碟或來源控制刪除它，除非您選取要永久刪除它的選項。
