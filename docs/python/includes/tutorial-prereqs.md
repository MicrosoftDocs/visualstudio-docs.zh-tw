## <a name="prerequisites"></a>必要條件

::: moniker range="vs-2017"
- 已安裝 Python 工作負載的 Visual Studio 2017。 如需相關指示，請參閱[在 Visual Studio 中使用 Python - 步驟 0](../tutorial-working-with-python-in-visual-studio-step-00-installation.md)。
::: moniker-end

::: moniker range=">=vs-2019"
- 已安裝 Python 工作負載的 Visual Studio 2019。 如需相關指示，請參閱[在 Visual Studio 中使用 Python - 步驟 0](../tutorial-working-with-python-in-visual-studio-step-00-installation.md)。
::: moniker-end

您也可以透過安裝適用於 Visual Studio 的 Python 工具來使用舊版 Visual Studio。 [在 Visual Studio 中安裝 Python 支援](../installing-python-support-in-visual-studio.md)。
