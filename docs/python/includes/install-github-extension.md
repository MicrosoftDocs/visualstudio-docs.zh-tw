若要從 VS 內使用 GitHub 存放庫，您需要安裝 Visual Studio 的 GitHub 延伸模組。 若要這樣做，請執行 Visual Studio 安裝程式，並選取 [修改]，然後選取 [個別元件] 索引標籤。向下捲動至 [程式碼工具] 區段，並選取 [Visual Studio 的 GitHub 延伸模組]，然後選取 [修改]。
