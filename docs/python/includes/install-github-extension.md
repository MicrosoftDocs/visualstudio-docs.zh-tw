若要從 VS 內使用 GitHub 存放庫，您需要安裝 Visual Studio 的 GitHub 延伸模組。 若要這樣做，請執行 Visual Studio 安裝程式，選取 [ **修改**]，然後選取 [ **個別元件** ] 索引標籤。向下滾動至 [程式 **代碼工具** ] 區段，選取 Visual Studio 的 [ **GitHub 擴充**功能]，然後選取 [ **修改**]。

![在 VS 安裝程式中選取 GitHub 擴充功能](../media/installation-github-extension.png)
