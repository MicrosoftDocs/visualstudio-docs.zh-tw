若要從 VS 內使用 GitHub 存放庫，您需要安裝 Visual Studio 的 GitHub 延伸模組。 為此，請運行 Visual Studio 安裝程式，選擇 **"修改**"，然後選擇 **"單個元件**"選項卡。 向下滾動到 **"代碼工具"** 部分，為 Visual Studio 選擇**GitHub 擴展**名，然後選擇"**修改**"。

![在 VS 安裝程式中選取 GitHub 擴充功能](../media/installation-github-extension.png)
