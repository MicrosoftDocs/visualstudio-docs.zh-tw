1. 在 [Visual Studio] 功能表中 **，選取** > [檔案] [**新增** > **專案**]。
2. 在 [**新增專案**] 對話方塊的 [**範本**] 區段中，選取 [  >   **C#Visual** **Web**]。
3. 選取 [ASP.NET Core Web 應用程式]。
4. 輸入新應用程式的名稱（或使用預設名稱），然後選取 **[確定]** 。
5. 選取 [Web 應用程式]。
6. 選取 [**啟用 Docker 支援**] 核取方塊。
7. 選取您想要的容器類型（[Windows] 或 [Linux]），然後選取 **[確定]** 。
