1. 在視覺化工作室功能表中，選擇 **"檔** > **新專案** > **"。**
2. 在 **"新專案**"對話方塊的 **"範本"** 部分中，選擇**視覺化 C#** > **Web**。
3. 選取 **ASP.NET Core Web 應用程式**。
4. 輸入新應用程式的名稱（或使用預設名稱），然後選擇 **"確定**"。
5. 選擇**Web 應用程式**。
6. 選擇啟用**Docker 支援**核取方塊。
7. 選擇所需的容器類型（Windows 或 Linux），然後選擇 **"確定**"。
