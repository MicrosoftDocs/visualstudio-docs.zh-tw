1. 在 Visual Studio 開始 視窗中，選取 **建立新專案**。
1. 選取 [ **ASP.NET Core Web 應用程式**]，然後選取 **[下一步]** 。
1. 輸入新應用程式的名稱（或使用預設名稱），然後選取 [**建立**]。
1. 選取 [Web 應用程式]。
1. 選取或清除 [**針對 HTTPS 設定**] 核取方塊，以選擇是否要使用 SSL 支援。
1. 選取 [**啟用 Docker 支援**] 核取方塊。
1. 選取您想要的容器類型（[Windows] 或 [Linux]），然後選取 [**建立**]。
