1. 從 Visual Studio 的開始視窗中，選擇 [建立新專案]  。
1. 依序選擇 [ASP.NET Core Web 應用程式]  和 [下一步]  。
1. 指定新應用程式的名稱 (或使用預設)，然後選擇 [建立]  。
1. 選擇 [Web 應用程式]  。
1. 使用 [設定 HTTPS]  核取方塊，選擇您是否需要 SSL 支援。
1. 請勾選 [啟用 Docker 支援]  核取方塊。
1. 選取您想要的容器類型 (Windows 或 Linux)，然後按一下 [建立]  。
