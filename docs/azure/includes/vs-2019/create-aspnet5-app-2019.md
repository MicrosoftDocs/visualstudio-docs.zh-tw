1. 在"視覺化工作室啟動"視窗中，選擇 **"創建新專案**"。
1. 選擇**ASP.NET核心 Web 應用程式**，然後選擇 **"下一步**"。
1. 輸入新應用程式的名稱（或使用預設名稱），然後選擇 **"創建**"。
1. 選擇**Web 應用程式**。
1. 通過選擇或清除"**為 HTTPS 配置**"核取方塊，選擇是否需要 SSL 支援。
1. 選擇啟用**Docker 支援**核取方塊。
1. 選擇所需的容器類型（Windows 或 Linux），然後選擇 **"創建**"。
