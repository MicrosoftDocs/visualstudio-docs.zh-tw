# <a name="debugger-project-settings"></a>偵錯工具專案設定
某些專案設定也會影響偵錯。 這些設定可以決定像是偵錯工具要查看哪些目錄、用於啟動程式的命令和命令引數，以及為程式建立的偵錯資訊類型等行為。 您可以在 [ **屬性頁** ] 對話方塊中變更這些設定。

 本節說明可以使用的選項，以及設定這些選項的方式。
