# <a name="thread-search-dialog-box"></a>執行緒搜尋對話方塊
使用此對話方塊，即可尋找並選取的節點中的特定執行緒[執行緒檢視](../debugger/threads-view.md)。 若要顯示此對話方塊中，焦點移至**執行緒**檢視 視窗中。 然後選擇**尋找執行緒**從**搜尋**功能表。

 可用的設定如下：

 **執行緒**来搜尋的執行緒識別碼。

 **模組**模組字串來搜尋。

 **搜尋方向，增加或相應減少**初始方向搜尋。
