# <a name="thread-search-dialog-box"></a>執行緒搜尋對話方塊
使用這個對話方塊，即可在 [ [執行緒] 視圖](../debugger/threads-view.md)中尋找並選取特定執行緒的節點。 若要顯示此對話方塊，請將焦點移至 [ **執行緒** ] 視圖視窗。 然後選擇 [**搜尋**] 功能表中的 [**尋找執行緒**]。

 可用的設定如下：

 **執行緒** 要搜尋的執行緒識別碼。

 **課程模組** 要搜尋的模組字串。

 **向上或向下搜尋方向** 搜尋的初始方向。
