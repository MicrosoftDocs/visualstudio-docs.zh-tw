# <a name="how-to-start-and-stop-the-message-log-display"></a>如何：啟動與停止訊息記錄顯示
當 [[訊息] 視圖](../debugger/messages-view.md)視窗作用中時，[**開機記錄**] 或 [**停止記錄**] 選項會出現在 [Spy + +**訊息**] 功能表上，而對應的 [**開始記錄**] 或 [**停止記錄**] 工具列按鈕會變成作用中

### <a name="to-start-the-message-log-display"></a>若要啟動訊息記錄檔顯示

1. 從 [**訊息**] 功能表中，選擇 [**開始記錄**]。

### <a name="to-stop-the-message-log-display"></a>若要停止訊息記錄檔顯示

1. 從 [**訊息**] 功能表中，選擇 [**停止記錄**]。
