# <a name="symbol-load-information-dialog-box"></a>符號載入資訊對話方塊
當您以滑鼠右鍵按一下 [模組] 視窗並選擇 [符號載入資訊] 時，[符號載入資訊] 對話方塊隨即開啟。

## <a name="uielement-list"></a>UIElement 清單
 **資訊**顯示搜尋的符號檔 （PDB 檔案） 根據 Visual Studio 安裝設定，以及您對符號設定中變更目錄**選項** 對話方塊。

 **符號設定**按一下此按鈕即可變更符號設定**選項** 對話方塊 (**偵錯**類別目錄**符號**頁面)。 在該頁面上，您可以加入或移除符號位置、指定本機快取位置 (如果使用符號伺服器)，以及控制何時搜尋符號位置。
