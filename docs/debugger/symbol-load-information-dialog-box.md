# <a name="symbol-load-information-dialog-box"></a>符號載入資訊對話方塊
當您以滑鼠右鍵按一下 [模組] 視窗並選擇 [符號載入資訊] 時，[符號載入資訊] 對話方塊隨即開啟。

## <a name="uielement-list"></a>UIElement 清單
 **資訊**根據 Visual Studio 安裝設定，以及您在 [**選項**] 對話方塊中對符號設定所做的變更，顯示針對符號檔（PDB 檔）搜尋的目錄。

 **符號設定**按一下此按鈕可變更 [**選項**] 對話方塊中的符號設定（[**調試**類別目錄]、[**符號**] 頁面）。 在該頁面上，您可以加入或移除符號位置、指定本機快取位置 (如果使用符號伺服器)，以及控制何時搜尋符號位置。
