# <a name="messages-menu-commands"></a>訊息功能表命令
[ **訊息** ] 功能表提供用來控制訊息查看的命令。

 **啟動/停止記錄** 啟動或停止訊息查看。

 **記錄選項** 開啟 [ **MessageOptions** ] 對話方塊，您可以在其中指定訊息搜尋準則。

 **清除記錄** 檔清除 [使用中訊息] 視圖的內容。
