# <a name="message-search-dialog-box"></a>訊息搜尋對話方塊
使用此對話方塊，即可尋找並選取特定的訊息中的節點[訊息檢視](../debugger/messages-view.md)。 若要顯示此對話方塊中，將焦點移至 [訊息] 檢視視窗。 然後選擇**尋找訊息**從**搜尋**功能表。

 可用的設定如下：

 **搜尋工具**拖放工具，用來選取視窗。

 **隱藏 Spy + +** 移除主要的 Spy + + 視窗，並將保留**MessageSearch**對話方塊顯示在其他應用程式之上。

 **處理**要搜尋之訊息的唯一識別碼。

 **型別**来搜尋的訊息類型。

 **訊息**来搜尋的訊息識別碼。

 **搜尋方向，增加或相應減少**初始方向搜尋。
