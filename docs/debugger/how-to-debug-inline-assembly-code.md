# <a name="how-to-debug-inline-assembly-code"></a>HOW TO：對內嵌組譯碼進行偵錯

> [!NOTE]
> 根據目前使用的設定與版本，您所看到的對話方塊與功能表命令可能會與 [說明] 中所描述的不同。 若要變更設定，請從 [工具] 功能表中選擇 [匯入和匯出設定]。 如需詳細資訊，請參閱[重設設定](../ide/environment-settings.md#reset-settings)。

偵錯工具提供兩種視窗來對內嵌組譯碼進行偵錯：[反組譯碼] 視窗和 [暫存器] 視窗。

## <a name="debug-inline-assembly-code"></a>對內嵌組譯碼進行偵錯

1.  使用 [反組譯碼] 視窗檢視組譯碼指令。

2.  使用 [暫存器] 視窗檢視暫存器內容。
