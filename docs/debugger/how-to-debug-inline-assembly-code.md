# <a name="how-to-debug-inline-assembly-code"></a>如何：偵錯內嵌組譯程式碼
> [!NOTE]
>  根據目前使用的設定與版本，您所看到的對話方塊與功能表命令可能會與 [說明] 中所描述的不同。 若要變更設定，請從 [工具] 功能表中選擇 [匯入和匯出設定]。 如需詳細資訊，請參閱[將 Visual Studio IDE 個人化](../ide/personalizing-the-visual-studio-ide.md)。  
  
 偵錯工具提供了兩個視窗來偵錯內嵌組譯程式碼，**反組譯碼**視窗和**註冊**視窗。  
  
## <a name="procedure"></a>程序  
  
#### <a name="to-debug-inline-assembly-code"></a>若要偵錯內嵌組譯程式碼  
  
1.  使用**反組譯碼**視窗來檢視組譯碼指令。  
  
2.  使用**註冊**視窗來檢視暫存器的內容。
