# <a name="font-dialog-box-microsoft-spy-help"></a>字型對話方塊 (Microsoft Spy++ 說明)
使用此對話方塊來設定中使用的字元字型[Spy + + 檢視](../debugger/spy-increment-views.md)視窗。 若要顯示此對話方塊中，依序展開**檢視**功能表，然後選擇**字型**。  
  
 可用的設定如下：  
  
 **字型**  
 Spy + + 檢視視窗具有焦點之字元的字型。  
  
 [字型樣式]  
 Spy + + 檢視視窗具有焦點的字型樣式。  
  
 **Size**  
 Spy + + 檢視視窗具有焦點的字型大小。  
  
 [將字型另存為預設]  
 開啟 Spy + + 檢視時，請將目前的字型設定儲存為您的喜好設定中。
