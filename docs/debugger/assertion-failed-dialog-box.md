# <a name="assertion-failed-dialog-box"></a>判斷提示已失敗對話方塊
判斷提示 (Assertion) 陳述式會指定您希望在程式的某個特定點維持為 true 的條件。 如果該條件並未維持 true，表示判斷提示失敗，程式執行將會中斷，並出現這個對話方塊。

|按一下|收件者|
|-----------|--------|
|重試|對判斷提示進行偵錯或取得判斷提示的說明。|
|忽略|忽略判斷提示並且繼續執行程式。|
|中止|停止程式執行並且結束偵錯工作階段。|
