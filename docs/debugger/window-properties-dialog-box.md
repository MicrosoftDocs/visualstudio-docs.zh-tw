# <a name="window-properties-dialog-box"></a>視窗屬性對話方塊
使用此對話方塊可深入瞭解特定視窗。 若要顯示此對話方塊，請將焦點移至 [Windows View](../debugger/windows-view.md) 視窗。 選取樹狀結構中的任何視窗節點，然後從 [ **View** ] 功能表選擇 [**屬性**]。
