# <a name="spy-views"></a>Spy++ 檢視
Spy + + 會顯示您電腦上可用的 windows、訊息、進程和執行緒的樹狀檢視。

## <a name="in-this-section"></a>本節內容
 [Windows View](../debugger/windows-view.md) 顯示可用視窗和控制項的樹狀檢視。

 [訊息視圖](../debugger/messages-view.md) 顯示與視窗、進程或執行緒相關聯的訊息資料流程。

 [進程視圖](../debugger/processes-view.md) 顯示現用進程的樹狀檢視。

 [執行緒視圖](../debugger/threads-view.md) 顯示現用執行緒的樹狀檢視。
