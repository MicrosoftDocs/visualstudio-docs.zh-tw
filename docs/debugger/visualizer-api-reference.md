# <a name="visualizer-api-reference"></a>視覺化檢視 API 參考

視覺化檢視 API 是提供給使用者用以撰寫 Visual Studio 偵錯工具的視覺化檢視。 視覺化檢視是一種小型應用程式，用以擴充 Visual Studio 偵錯工具使用者介面的功能。 視覺化檢視可以顯示 (或選擇性地編輯) 特定類型的資料物件，這是視覺化檢視的設計目的。
