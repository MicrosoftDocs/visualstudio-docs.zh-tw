# <a name="idiaenuminjectedsourcesreset"></a>IDiaEnumInjectedSources::Reset
將列舉型別序列重設到開頭。

## <a name="syntax"></a>語法

```
HRESULT Reset();
```

## <a name="return-value"></a>傳回值
 傳回 `S_OK`。
