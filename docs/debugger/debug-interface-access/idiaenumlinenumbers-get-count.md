# <a name="idiaenumlinenumbersget_count"></a>IDiaEnumLineNumbers::get_Count
抓取行號的數目。

## <a name="syntax"></a>語法

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>參數
 pRetVal

脫銷傳回行號的數目。

## <a name="return-value"></a>傳回值
 如果成功，會傳回 `S_OK`;否則，會傳回錯誤碼。
