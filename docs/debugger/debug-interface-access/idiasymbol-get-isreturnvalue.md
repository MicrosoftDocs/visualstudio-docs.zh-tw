# <a name="idiasymbolgetisreturnvalue"></a>IDiaSymbol::get_isReturnValue
指定變數是否包含傳回的值。

## <a name="syntax"></a>語法

```C++
HRESULT get_isReturnValue(
   BOOL* pRetVal);
```

#### <a name="parameters"></a>參數
 `pRetVal`

[out]指標`BOOL`表示變數是否包含傳回的值。

## <a name="return-value"></a>傳回值
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
