# <a name="idiaenumtablesget_count"></a>IDiaEnumTables::get_Count
抓取資料表的數目。

## <a name="syntax"></a>語法

```C++
HRESULT get_Count (    LONG* pRetVal
);

```

#### <a name="parameters"></a>參數
 `pRetVal`

脫銷傳回資料表的數目。

## <a name="return-value"></a>傳回值
 如果成功，會傳回 `S_OK`;否則，會傳回錯誤碼。
