# <a name="idiasymbolgetobjectfilename"></a>IDiaSymbol::get_objectFileName
擷取物件的檔案名稱。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_objectFilename(   
   BSTR *pRetVal);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]指標`BSTR`存有物件檔案名稱。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
