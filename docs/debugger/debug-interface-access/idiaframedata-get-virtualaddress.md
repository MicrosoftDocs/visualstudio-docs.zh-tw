# <a name="idiaframedataget_virtualaddress"></a>IDiaFrameData::get_virtualAddress
抓取框架的程式碼 (VA) 的虛擬位址。

## <a name="syntax"></a>語法

```C++
HRESULT get_virtualAddress ( 
   ULONGLONG* pRetVal
);
```

#### <a name="parameters"></a>參數
 `pRetVal`

擴展傳回框架程式碼的虛擬位址。

## <a name="return-value"></a>傳回值
 如果成功，則傳回 `S_OK`。 `S_FALSE`如果不支援這個屬性，則傳回。 否則會傳回錯誤碼。
