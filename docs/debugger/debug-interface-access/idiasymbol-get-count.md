# <a name="idiasymbolget_count"></a>IDiaSymbol::get_count
捕獲清單或陣列中的專案數。

## <a name="syntax"></a>語法

```C++
HRESULT get_count ( 
   DWORD* pRetVal
);
```

#### <a name="parameters"></a>參數
 `pRetVal`

擴展傳回清單或陣列中的專案數。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回 `S_FALSE` 錯誤碼。

> [!NOTE]
> 的傳回值 `S_FALSE` 表示該屬性不適用於符號。
