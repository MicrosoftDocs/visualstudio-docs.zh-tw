# <a name="idiasymbolget_count"></a>IDiaSymbol::get_count
抓取清單或陣列中的專案數。

## <a name="syntax"></a>語法

```C++
HRESULT get_count ( 
   DWORD* pRetVal
);
```

#### <a name="parameters"></a>參數
 `pRetVal`

脫銷傳回清單或陣列中的專案數。

## <a name="return-value"></a>傳回值
 如果成功，會傳回 `S_OK`;否則，會傳回 `S_FALSE` 或錯誤碼。

> [!NOTE]
> @No__t_0 的傳回值表示該屬性不適用於符號。
