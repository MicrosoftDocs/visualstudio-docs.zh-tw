# <a name="idiasegmentgetexecute"></a>IDiaSegment::get_execute
擷取指出區段是否可執行檔的旗標。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_execute (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]會傳回`TRUE`區段會標示為可執行檔，否則為，如果傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，會傳回 `S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 否則會傳回錯誤碼。
