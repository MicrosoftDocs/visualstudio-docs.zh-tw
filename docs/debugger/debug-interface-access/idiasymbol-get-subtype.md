# <a name="idiasymbolget_subtype"></a>IDiaSymbol::get_subType
抓取子類型。

## <a name="syntax"></a>語法

```C++
HRESULT get_subType(
   IDiaSymbol** pRetVal);
```

#### <a name="parameters"></a>參數
 `pRetVal`

擴展子類型的指標。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回 `S_FALSE` 錯誤碼。
