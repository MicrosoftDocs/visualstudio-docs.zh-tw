# <a name="idiasymbolgetsubtype"></a>IDiaSymbol::get_subType
擷取的子型別。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_subType(   
   IDiaSymbol** pRetVal);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]子型別指標。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
