# <a name="idialoadcallbacknotifyopenpdb"></a>IDiaLoadCallback::NotifyOpenPDB
在開啟候選 .pdb 檔案時呼叫。

## <a name="syntax"></a>語法

```C++
HRESULT NotifyOpenPDB ( 
   LPCOLESTR pdbPath,
   HRESULT   resultCode
);
```

#### <a name="parameters"></a>參數
 `pdbPath`

在.Pdb 檔案的完整路徑。

 `resultCode`

在此程式碼表示套用至這個檔案的負載成功 (`S_OK`) 或失敗。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回錯誤碼。 通常會忽略傳回碼。
