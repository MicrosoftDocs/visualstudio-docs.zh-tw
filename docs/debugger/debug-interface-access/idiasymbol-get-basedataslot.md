# <a name="idiasymbolget_basedataslot"></a>IDiaSymbol::get_baseDataSlot
抓取基底資料位置。

## <a name="syntax"></a>語法

```C++
HRESULT get_baseDataSlot(
   DWORD* pRetVal);
```

#### <a name="parameters"></a>參數
 `pRetVal`

脫銷`DWORD`包含基底資料位置之的指標。

## <a name="return-value"></a>傳回值
 如果成功，會傳回，否則會傳回 `S_OK` `S_FALSE` 或錯誤碼。
