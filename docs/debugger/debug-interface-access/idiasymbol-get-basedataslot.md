# <a name="idiasymbolgetbasedataslot"></a>IDiaSymbol::get_baseDataSlot
擷取的基底的資料插槽。

## <a name="syntax"></a>語法

```C++
HRESULT get_baseDataSlot(
   DWORD* pRetVal);
```

#### <a name="parameters"></a>參數
 `pRetVal`

[out]指標`DWORD`保存的基底的資料插槽。

## <a name="return-value"></a>傳回值
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
