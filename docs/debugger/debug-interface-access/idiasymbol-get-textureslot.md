# <a name="idiasymbolget_textureslot"></a>IDiaSymbol::get_textureSlot
抓取材質位置。

## <a name="syntax"></a>語法

```C++
HRESULT get_textureSlot(
   DWORD* pRetVal);
```

#### <a name="parameters"></a>參數
 `pRetVal`

擴展保存材質位置之的指標 `DWORD` 。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回 `S_FALSE` 錯誤碼。
