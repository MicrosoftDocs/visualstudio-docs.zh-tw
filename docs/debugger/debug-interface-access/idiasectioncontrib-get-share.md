# <a name="idiasectioncontribgetshare"></a>IDiaSectionContrib::get_share
擷取指出區段是否可以共用記憶體中的旗標。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_share (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回`TRUE`區段，則可在記憶體中; 否則傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
