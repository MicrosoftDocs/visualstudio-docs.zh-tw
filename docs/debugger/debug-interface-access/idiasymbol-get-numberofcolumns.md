# <a name="idiasymbolget_numberofcolumns"></a>IDiaSymbol::get_numberOfColumns
捕獲矩陣中的資料行數目。

## <a name="syntax"></a>語法

```C++
HRESULT get_numberOfColumns(
   DWORD* pRetVal);
```

#### <a name="parameters"></a>參數
 `pRetVal`

擴展的指標 `DWORD` ，其中包含矩陣中的資料行數目。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回 `S_FALSE` 錯誤碼。
