# <a name="idiasymbolgetvirtual"></a>IDiaSymbol::get_virtual
擷取指定函數是否為虛擬的旗標。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_virtual (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]會傳回`TRUE`函式是虛擬，否則為，如果傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。  
  
> [!NOTE]
>  傳回值為`S_FALSE`表示此屬性不適用於符號。
