# <a name="idiasectioncontribgetcode"></a>IDiaSectionContrib::get_code
擷取指出區段是否包含可執行程式碼的旗標。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_code (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回`TRUE`如果章節包含可執行程式碼; 否則會傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
