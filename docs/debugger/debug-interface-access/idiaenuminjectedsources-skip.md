# <a name="idiaenuminjectedsourcesskip"></a>IDiaEnumInjectedSources::Skip
在列舉序列中略過指定數目的插入來源。

## <a name="syntax"></a>語法

```C++
HRESULT Skip ( 
   ULONG celt
);
```

#### <a name="parameters"></a>參數
 celt

在要略過的列舉序列中插入的來源數目。

## <a name="return-value"></a>傳回值
 如果成功，會傳回 `S_OK`;否則，如果沒有其他插入的來源要略過，則會傳回 `S_FALSE`。
