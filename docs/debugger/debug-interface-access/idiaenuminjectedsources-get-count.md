# <a name="idiaenuminjectedsourcesget_count"></a>IDiaEnumInjectedSources::get_Count
抓取插入的來源數目。

## <a name="syntax"></a>語法

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>參數
 pRetVal

擴展傳回插入的來源數目。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回錯誤碼。
