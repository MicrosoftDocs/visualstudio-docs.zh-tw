# <a name="idiaimagedataget_virtualaddress"></a>IDiaImageData::get_virtualAddress
抓取映射虛擬記憶體中的位置。

## <a name="syntax"></a>語法

```C++
HRESULT get_virtualAddress ( 
   ULONGLONG* pRetVal
);
```

#### <a name="parameters"></a>參數
 `pRetVal`

脫銷傳回映射的虛擬位址。

## <a name="return-value"></a>傳回值
 如果成功，會傳回 `S_OK`;否則，會傳回錯誤碼。
