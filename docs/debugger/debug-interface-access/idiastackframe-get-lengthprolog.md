# <a name="idiastackframegetlengthprolog"></a>IDiaStackFrame::get_lengthProlog
擷取的初構程式碼區塊中的位元組數目。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_lengthProlog (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回的初構程式碼的位元組數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回 `S_OK`。 傳回`S_FALSE`不支援的屬性。 反之則傳回錯誤碼。
