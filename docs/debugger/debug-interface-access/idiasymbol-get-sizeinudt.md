# <a name="idiasymbolgetsizeinudt"></a>IDiaSymbol::get_sizeInUdt
擷取的使用者定義型別之成員的大小。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_sizeInUdt(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]指標`DWORD`指定成員的大小。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
