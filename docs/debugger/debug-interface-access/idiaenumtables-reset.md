# <a name="idiaenumtablesreset"></a>IDiaEnumTables::Reset
將列舉序列重設為開頭。

## <a name="syntax"></a>語法

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>傳回值
 傳回 `S_OK`。
