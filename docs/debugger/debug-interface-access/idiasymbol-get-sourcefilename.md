# <a name="idiasymbolgetsourcefilename"></a>IDiaSymbol::get_sourceFileName
擷取編譯原始程式檔的檔案名稱。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_sourceFileName (   
   BSTR* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回編譯原始程式檔的檔案名稱。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。  
  
> [!NOTE]
>  傳回值為`S_FALSE`表示此屬性不適用於符號。
