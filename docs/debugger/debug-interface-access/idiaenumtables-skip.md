# <a name="idiaenumtablesskip"></a>IDiaEnumTables::Skip
略過指定的數目的列舉型別序列中的資料表。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT Skip (   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>參數  
 `celt`  
 [in]略過列舉序列中的資料表數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`有沒有更多的資料表，以略過。
