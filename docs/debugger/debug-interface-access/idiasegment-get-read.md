# <a name="idiasegmentgetread"></a>IDiaSegment::get_read
擷取旗標，指出是否可以讀取區段。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_read (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]會傳回`TRUE`可以讀取區段; 否則會傳回`FALSE`。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`。 傳回`S_FALSE`不支援這個屬性，則為。 反之則傳回錯誤碼。
