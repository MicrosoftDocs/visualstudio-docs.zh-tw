# <a name="idiastackframegetlocalsbase"></a>IDiaStackFrame::get_localsBase
擷取框架的區域變數的基底位址。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_localsBase (   
   ULONGLONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]傳回區域變數的基底位址。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回 `S_OK`。 傳回`S_FALSE`不支援的屬性。 反之則傳回錯誤碼。
