# <a name="idiastackframeget_localsbase"></a>IDiaStackFrame::get_localsBase
抓取框架的本機變數基底位址。

## <a name="syntax"></a>語法

```C++
HRESULT get_localsBase ( 
   ULONGLONG* pRetVal
);
```

#### <a name="parameters"></a>參數
 `pRetVal`

脫銷傳回本機變數的基底位址。

## <a name="return-value"></a>傳回值
 如果成功，會傳回 `S_OK`。 如果不支援此屬性，則會傳回 `S_FALSE`。 否則會傳回錯誤碼。
