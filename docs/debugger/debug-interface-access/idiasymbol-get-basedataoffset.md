# <a name="idiasymbolget_basedataoffset"></a>IDiaSymbol::get_baseDataOffset
抓取基底資料位移。

## <a name="syntax"></a>語法

```C++
HRESULT get_baseDataOffset(
   DWORD* pRetVal);
```

#### <a name="parameters"></a>參數
 `pRetVal`

擴展 `DWORD` 保存基底資料位移之的指標。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回 `S_FALSE` 錯誤碼。
