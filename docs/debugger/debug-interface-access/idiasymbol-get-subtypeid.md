# <a name="idiasymbolget_subtypeid"></a>IDiaSymbol::get_subTypeId
抓取子類型識別碼。

## <a name="syntax"></a>語法

```C++
HRESULT get_subTypeId(
   DWORD* pRetVal);
```

#### <a name="parameters"></a>參數
 `pRetVal`

脫銷`DWORD`包含子類型識別碼之的指標。

## <a name="return-value"></a>傳回值
 如果成功，會傳回，否則會傳回 `S_OK` `S_FALSE` 或錯誤碼。
