# <a name="idiaenumsourcefilesget_count"></a>IDiaEnumSourceFiles::get_Count
抓取原始檔的數目。

## <a name="syntax"></a>語法

```C++
HRESULT get_Count ( 
   LONG* pRetVal
);
```

#### <a name="parameters"></a>參數
 pRetVal

脫銷傳回原始檔的數目。

## <a name="return-value"></a>傳回值
 如果成功，會傳回，否則會傳回 `S_OK` 錯誤碼。
