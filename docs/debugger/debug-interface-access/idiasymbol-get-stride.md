# <a name="idiasymbolgetstride"></a>IDiaSymbol::get_stride
擷取矩陣或 strided 陣列的步距。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_stride(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]指標`DWORD`，持有分散。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
