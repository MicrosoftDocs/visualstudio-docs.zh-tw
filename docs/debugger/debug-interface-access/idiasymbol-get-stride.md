# <a name="idiasymbolgetstride"></a>IDiaSymbol::get_stride
擷取 strided 陣列之矩陣的分散。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_stride(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>參數  
 `pRetVal`  
 [out]指標`DWORD`保存分散。  
  
## <a name="return-value"></a>傳回值  
 如果成功，傳回`S_OK`; 否則傳回`S_FALSE`或錯誤碼。
