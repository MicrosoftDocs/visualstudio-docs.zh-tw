# <a name="idiasymbolget_guid"></a>IDiaSymbol::get_guid
抓取符號的全域唯一識別碼 (GUID) 。

## <a name="syntax"></a>語法

```C++
HRESULT get_guid ( 
   GUID* pRetVal
);
```

#### <a name="parameters"></a>參數
 `pRetVal`

擴展傳回符號的 GUID。

## <a name="return-value"></a>傳回值
 如果成功， `S_OK` 則傳回; 否則傳回 `S_FALSE` 錯誤碼。

> [!NOTE]
> 的傳回值 `S_FALSE` 表示該符號無法使用該屬性。
