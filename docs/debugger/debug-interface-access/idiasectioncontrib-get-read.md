# <a name="idiasectioncontribget_read"></a>IDiaSectionContrib::get_read
抓取指出是否可以讀取區段的旗標。

## <a name="syntax"></a>語法

```C++
HRESULT get_read ( 
   BOOL* pRetVal
);
```

#### <a name="parameters"></a>參數
 `pRetVal`

脫銷`TRUE`如果可以讀取區段，則傳回，否則傳回 `FALSE` 。

## <a name="return-value"></a>傳回值
 如果成功，則傳回 `S_OK`。 `S_FALSE`如果不支援此屬性，則傳回。 否則會傳回錯誤碼。
