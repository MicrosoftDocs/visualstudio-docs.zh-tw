# <a name="idiaenumsegmentsgetcount"></a>IDiaEnumSegments::get_Count
擷取區段的數目。  
  
## <a name="syntax"></a>語法  
  
```C++  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>參數  
 pRetVal  
 [out，retval]傳回區段的數目。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回錯誤碼。
