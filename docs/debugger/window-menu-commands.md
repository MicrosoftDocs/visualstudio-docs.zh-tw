# <a name="window-menu-commands"></a>視窗功能表命令
[ **視窗]** 功能表提供管理 Spy + + 視窗的命令。

 **Cascade** 以串聯模式重新排列所有視窗。

 **水準並排** 顯示重新排列水準磚模式中的所有視窗。

 **垂直並排** 顯示以垂直磚模式重新排列所有視窗。

 **排列圖示** 重新排列檢視圖示（如果已將視窗最小化）。

 **關閉/關閉全部** 關閉 Spy + + 中的一個或所有視窗。

 重新整理重新整理目前選取的視圖。

 **視窗** 允許選取特定視窗。
