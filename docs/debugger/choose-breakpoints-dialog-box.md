# <a name="choose-breakpoints-dialog-box"></a>[選擇中斷點] 對話方塊
當您嘗試在 [**新增中斷點**] 對話方塊中，從不明確的輸入設定中斷點時，會開啟 [**選擇中斷點**] 對話方塊。 模糊輸入的常見範例是多載函數名稱。

 **中斷點**列出可以根據不明確的輸入來設定的可能中斷點。 您可以選取一或多個要設定的中斷點。

 **全部**選取 [**中斷點**] 方塊中列出的所有中斷點。

 **無**清除 [**中斷點**] 方塊中列出的所有中斷點。
