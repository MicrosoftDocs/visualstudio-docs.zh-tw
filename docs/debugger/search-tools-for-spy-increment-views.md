# <a name="search-tools-for-spy-views"></a>Spy++ 檢視的搜尋工具
使用這些對話方塊來指定 Spy + + Views 視窗將顯示的資料：

## <a name="in-this-section"></a>本節內容
 [尋找視窗對話方塊](../debugger/find-window-dialog-box.md) 用來從特定視窗中查看屬性或訊息。

 [[視窗搜尋] 對話方塊](../debugger/window-search-dialog-box.md)用來尋找 Windows view 中特定視窗的節點。

 [訊息搜尋對話方塊](../debugger/message-search-dialog-box.md) 用來尋找訊息查看中特定訊息的節點。

 [進程搜尋對話方塊](../debugger/process-search-dialog-box.md) 用來尋找進程視圖中特定進程的節點。

 [執行緒搜尋對話方塊](../debugger/thread-search-dialog-box.md) 用來尋找執行緒視圖中特定執行緒的節點。
