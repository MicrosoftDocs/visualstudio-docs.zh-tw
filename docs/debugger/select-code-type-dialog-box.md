# <a name="select-code-type-dialog-box"></a>選取程式碼類型對話方塊
若要開啟這個對話方塊，請開啟 [附加至處理序] 對話方塊，然後按一下 [選取] 按鈕。

 **自動判斷要偵錯的程式碼類型**會選取適當的偵錯工具，根據執行的程式碼的類型。

 **對這些程式碼類型進行偵錯：** 從提供的清單選擇要進行偵錯的程式碼類型。
