# <a name="view-menu-commands"></a>檢視功能表命令
**檢視**功能表提供的命令控制 Spy + + 使用者介面。

 **字型**會開啟**字型**對話方塊中，您可以在其中選取目前的 Spy + + 檢視中使用的字型。

 **工具列**隱藏或顯示工具列。

 **狀態列**隱藏或顯示 [狀態] 列。

 **屬性**會開啟**屬性**現用檢視表 對話方塊。
