# <a name="tree-menu-commands"></a>樹狀結構功能表命令
**樹狀目錄** 功能表提供用來展開或折迭視圖的命令。

 **展開一個層級** 在系統檢視中顯示目前選取的專案，方法是在其下顯示一個層級。

 **展開分支** 展開目前選取之專案底下的所有層級。

 **全部展開** 展開並顯示系統視圖中的所有層級。

 折迭折迭目前選取之專案底下的所有層級。
