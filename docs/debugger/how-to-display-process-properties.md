# <a name="how-to-display-process-properties"></a>如何：顯示處理序屬性
若要深入瞭解 [[進程] 視圖](../debugger/processes-view.md)中所列的進程，請使用 [[處理屬性] 對話方塊](../debugger/process-properties-dialog-box.md)。

### <a name="to-open-a-process-properties-dialog-box"></a>若要開啟進程屬性對話方塊

1. 從 [**進程] 視圖**中顯示的樹狀結構選取進程。

2. 從 **[檢視]** 功能表中選擇 **[屬性]**。

   [屬性] 對話方塊不是強制回應，因此您可以按一下 [視圖] 視窗中的另一個專案，對話方塊就會顯示所選取專案的相關資訊。
