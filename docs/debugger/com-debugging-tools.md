# <a name="com-debugging-tools"></a>COM 偵錯工具

本主題列出可幫助您進行測試和檢驗 COM 程式的工具。

OLE/COM 物件檢視器

OLEVIEW 顯示了安裝在電腦中的 ActiveX 和 OLE 物件，以及它們所支援的介面。 它也可讓您編輯登錄，並檢視型別程式庫。

MFC 範例

MFC ActiveX 範例 OCLIENT 和 SUPERPAD 也可以用來測試您的應用程式。 這些範例的完整原始程式碼都包含在內。
