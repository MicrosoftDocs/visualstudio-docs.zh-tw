# <a name="find-window-dialog-box"></a>尋找視窗對話方塊
使用這個對話方塊來選取視窗，並顯示其屬性或訊息。 若要顯示此對話方塊，請從 **Spy** 功能表選擇 [**尋找視窗]** 。

 可用的設定如下：

 **Finder 工具** 用來選取視窗的拖放工具。

 **隱藏 Spy + +** 移除主要的 Spy + + 視窗，並讓 [ **尋找視窗** ] 對話方塊出現在其他應用程式的頂端。

 **控制碼** 要尋找之視窗的唯一識別碼。

 **標題** 視窗標題的文字。

 **類別** 視窗的類別。

 **樣式** 視窗的樣式。

 **Rect** 視窗的周框。

 **顯示內容或訊息** 決定是否按下 [確定] 會顯示 [ [視窗屬性] 對話方塊](../debugger/window-properties-dialog-box.md) ，或是開啟 [ [訊息] 視圖](../debugger/messages-view.md)。
