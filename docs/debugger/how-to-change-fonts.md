# <a name="how-to-change-fonts"></a>如何：變更字型
您可以變更 Spy + + 視窗的字型、字型樣式和字型大小。

### <a name="to-change-font-options"></a>變更字型選項

1. 選擇 [ **View** ] 功能表中的 [ **字型**]。

2. 選擇字型、字型樣式和字型大小。

3. 按一下 [確定]  。

   選取 [ **將字型儲存為預設值** ] 會導致所有未來的 Spy + + 視窗使用此字型。
