# <a name="message-options-dialog-box"></a>訊息選項對話方塊
使用此對話方塊來選取哪一個訊息會列在[訊息檢視](../debugger/messages-view.md)。 若要顯示此對話方塊中，選擇**記錄檔訊息**從**Spy**功能表。

 用來選取要列出的訊息類型。 包含搜尋工具。  
  
 [訊息選項對話方塊、訊息索引標籤](../debugger/messages-tab-message-options-dialog-box.md)  
 用來選取檢視的訊息類型。  
  
 [訊息選項對話方塊、輸出索引標籤](../debugger/output-tab-message-options-dialog-box.md)  
 用來指定要顯示的訊息資料。
