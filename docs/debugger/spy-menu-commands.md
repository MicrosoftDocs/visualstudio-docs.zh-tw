# <a name="spy-menu-commands"></a>Spy 功能表命令
**Spy**功能表提供使用 spy + + 視窗和 views 的命令。

 **Windows** 開啟新的 Windows view。

 **進程** 開啟新的進程視圖。

 **執行緒** 開啟新的執行緒視圖。

 **記錄訊息** 開啟新的訊息視圖，並開啟該視圖的 [ [訊息選項] 對話方塊](../debugger/message-options-dialog-box.md) 。

 **尋找視窗** 顯示 [ **FindWindow** ] 對話方塊，以使用滑鼠來選取視窗。

 **Exit**結束關閉 Spy + + 公用程式。
