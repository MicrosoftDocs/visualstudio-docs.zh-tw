# <a name="spy-menu-commands"></a>Spy 功能表命令
**Spy**功能表提供命令以使用 Spy + + Windows 和檢視表。

 **Windows**開啟新的 Windows 檢視。

 **處理序**會開啟新的處理序檢視。

 **執行緒**會開啟新的 [執行緒] 檢視。

 **記錄訊息**會開啟新的 [訊息] 檢視，並開啟[訊息選項對話方塊](../debugger/message-options-dialog-box.md)檢視。

 **尋找視窗**顯示器**FindWindow**對話方塊來選取視窗中的，使用滑鼠。

 **結束**關閉 Spy + + 公用程式。
