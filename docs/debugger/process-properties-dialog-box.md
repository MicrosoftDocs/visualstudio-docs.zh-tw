# <a name="process-properties-dialog-box"></a>處理序屬性對話方塊
若要深入了解特定的處理序中使用此對話方塊。 若要顯示此對話方塊中，焦點移至[處理序檢視](../debugger/processes-view.md)視窗。 在樹狀目錄中，選取任何處理序節點，然後選擇**屬性**從**檢視**功能表。

 Windows 98 和 Windows 95 在作業系統上，只能修改後的版本**一般** 索引標籤會顯示。

## <a name="in-this-section"></a>本節內容
 [一般索引標籤、 處理序屬性對話方塊](../debugger/general-tab-thread-properties-dialog-box.md)使用此對話方塊中深入了解選取的處理序。

 [記憶體索引標籤，處理序屬性對話方塊](../debugger/memory-tab-process-properties-dialog-box.md)使用此索引標籤以顯示處理程序如何使用記憶體。

 [分頁檔索引標籤、 處理序屬性對話方塊](../debugger/page-file-tab-process-properties-dialog-box.md)檢查分頁檔的處理程序中使用此索引標籤。

 [空間索引標籤，處理序屬性對話方塊](../debugger/space-tab-process-properties-dialog-box.md)檢查處理序的位址空間中使用此索引標籤。
