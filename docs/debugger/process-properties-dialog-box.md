# <a name="process-properties-dialog-box"></a>處理序屬性對話方塊
使用此對話方塊可深入瞭解特定的進程。 若要顯示此對話方塊，請將焦點移至 [ [進程] 視圖](../debugger/processes-view.md) 視窗。 選取樹狀結構中的任何進程節點，然後從 [ **View** ] 功能表選擇 [**屬性**]。

 在 Windows 98 和 Windows 95 作業系統上，只會顯示 [ **一般** ] 索引標籤的修改版本。

## <a name="in-this-section"></a>本節內容
 [進程屬性對話方塊、一般](../debugger/general-tab-thread-properties-dialog-box.md) 索引標籤使用這個對話方塊，即可進一步瞭解所選的進程。

 [進程屬性對話方塊、記憶體](../debugger/memory-tab-process-properties-dialog-box.md) 索引標籤使用此索引標籤可顯示進程如何使用記憶體。

 [進程屬性對話方塊、分頁檔](../debugger/page-file-tab-process-properties-dialog-box.md) 索引標籤使用此索引標籤來檢查進程的分頁檔案。

 [進程屬性對話方塊、空間](../debugger/space-tab-process-properties-dialog-box.md) 索引標籤使用此索引標籤來檢查進程的位址空間。
