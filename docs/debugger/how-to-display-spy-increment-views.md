# <a name="how-to-display-spy-views"></a>如何：顯示 Spy++ 檢視
Spy + + 啟動時，它會開啟在[Windows 檢視](../debugger/windows-view.md)，所有的 windows 和您的電腦上可用控制項的樹狀檢視。 此根視窗上設定 「 Windows 1 」。 也有三個其他 Spy + + 檢視可用：[訊息檢視](../debugger/messages-view.md)，[處理序檢視](../debugger/processes-view.md)，並[執行緒檢視](../debugger/threads-view.md)。 您可以顯示所要的檢視，從**Spy**功能表。 如需詳細資訊，請參閱 < [Spy + + 檢視](../debugger/spy-increment-views.md)。
