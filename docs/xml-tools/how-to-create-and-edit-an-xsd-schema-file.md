# <a name="how-to-create-and-edit-an-xsd-schema-file"></a>如何：建立和編輯 XSD 架構檔案

本主題描述如何建立新 XML 結構描述 (XSD) 檔案，然後將內容加入至 XSD 檔案。

## <a name="to-create-a-new-xml-schema-file"></a>建立新的 XML 結構描述檔

1. 在 Visual Studio 中，開啟 **[檔案**] 功能表，然後選取 [**新增** ** >  檔案**]。 或者，使用**Ctrl** +**N**鍵盤快速鍵。

2. 在 [**新增**檔案] 對話方塊中，選取 [ **XML 架構**]，然後選取 [**開啟**]。

   新檔案隨即建立。 [開始] 視圖和空白的 [ **XML 架構瀏覽器**] 視窗會出現在 Visual Studio 中。

## <a name="to-edit-a-file"></a>編輯檔案

1. 選取 [使用 XML 編輯器]，即可在 [開始] 視圖上**查看和編輯基礎 XML 架構**檔案連結。

   [XML 編輯器] 隨即出現，並開啟新的檔案。

2. 從[訂單架構](../xml-tools/sample-xsd-file-simple-schema.md)複製 XML 架構範例程式碼並貼上，以取代預設新增至新 XSD 檔案的程式碼。

   **XML 架構瀏覽器**會以新的架構集更新。
