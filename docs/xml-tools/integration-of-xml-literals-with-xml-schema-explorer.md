# <a name="integration-of-xml-literals-with-xml-schema-explorer"></a>XML 常值與 XML 架構瀏覽器的整合

Visual Basic 支援 XML 常值，這表示您可以將 XML 片段直接合併至 Visual Basic 程式碼中。 如需詳細資訊，請參閱 [XML 常值總覽](/dotnet/visual-basic/programming-guide/language-features/xml/xml-literals-overview)。

## <a name="how-to"></a>如何

如果 Visual Basic 專案中的 XSD 檔案包含 XML 常值，您可以在 **Xml 架構瀏覽器** 中查看 xml 架構集。 若要查看與 XML 常值相關聯的架構集，請以滑鼠右鍵按一下 XML 常值或 XML 命名空間匯入中的 XML 節點，然後選取 [ **在架構瀏覽器中顯示**]。

![Visual Basic 專案視窗的螢幕擷取畫面，其中顯示 XML 節點上的內容功能表，並醒目提示 [架構瀏覽器] 命令中的快照。](../xml-tools/media/vbxmlliteralswithxmlschemaexplorer1.gif)

這會與您的 Visual Basic 檔並存開啟 **XML 架構瀏覽器** 。

![Visual Basic 專案視窗的螢幕擷取畫面，其中顯示已在右窗格中開啟 XML 架構瀏覽器和方案總管。](../xml-tools/media/vbxmlliteralswithxmlschemaexplorer2.gif)
