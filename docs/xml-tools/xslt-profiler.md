# <a name="xslt-profiler"></a>XSLT 分析工具

XSLT 分析工具是一種效能分析工具，可協助您開發並偵錯 XSLT 文件。 您可以測量並評估 XSLT 程式碼中的效能相關問題，藉由建立詳細的 XSLT 效能報告。 XSLT 分析工具包括 XSL 和 XSLT 樣式表最佳化的實用提示，這些對要求最大效能的 XSLT 架構之應用程式而言是很重要的。

XSLT Profiler 是 Visual Studio 的一部分，而且都可從**XML**功能表。

![XSLT 分析工具](../xml-tools/media/profile-xslt-menu.png)

> [!NOTE]
> 只有 Enterprise 版的 Visual Studio 中使用 XSLT Profiler。
