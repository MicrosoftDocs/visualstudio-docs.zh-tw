# <a name="how-to-build-sharepoint-solutions"></a>如何：建立 SharePoint 方案

建立 SharePoint 方案與在 Visual Studio 中建立其他應用程式並無不同。 如需建立應用程式的詳細資訊，請參閱[Visual Studio 中的編譯和組建](../ide/compiling-and-building-in-visual-studio.md)。 如需 SharePoint 方案特有的一些考慮的討論，請參閱[如何：將 sharepoint 方案部署和發行至本機 sharepoint 網站](../sharepoint/how-to-deploy-and-publish-a-sharepoint-solution-to-a-local-sharepoint-site.md)。

## <a name="to-build-sharepoint-solutions"></a>若要建立 SharePoint 方案

1. 決定您要進行的是「debug」還是「發行」組建。 如需詳細資訊，請參閱[偵錯工具專案設定](../debugger/debugger-project-settings.md)。

2. 選擇適當的選項，以建立或重建您的方案或專案。 如需詳細資訊，請參閱[在 Visual Studio 中編譯和建立](../ide/compiling-and-building-in-visual-studio.md)。

3. 在功能表列上，選擇 [**組建**] [組建  >  **方案**]。
