# <a name="how-to-add-a-custom-property"></a>如何：加入自訂屬性
  您可以使用 **屬性編輯器**，將自訂屬性加入至模型。 您可以在程式碼中存取這些屬性，以便在執行時間取得連接字串和其他資料等資訊。

### <a name="to-add-a-custom-property"></a>若要加入自訂屬性

1. 在 [ **BDC Explorer**] 中，選擇代表您要套用自訂屬性之模型元素的節點。

2. 在功能表列上，選擇 [**視圖**  >  **屬性視窗]**。

3. 在 [ **屬性** ] 視窗中，選擇 [ **自訂屬性** ] 屬性，然後選擇省略號按鈕 (![ASP.NET Mobile Designer 橢圓形](../sharepoint/media/mwellipsis.gif "ASP.NET Mobile 設計工具橢圓形")) 。

     [ **屬性編輯器** ] 對話方塊隨即出現。

4. 在 [ **名稱** ] 欄的文字方塊中，指定屬性的名稱。

5. 在 [自訂] 屬性的 [ **類型** ] 欄位中，選擇適當的資料類型。

6. 針對 [自訂] 屬性的 [ **值** ] 欄位，指定一個值，然後選擇 [ **確定]** 按鈕。
