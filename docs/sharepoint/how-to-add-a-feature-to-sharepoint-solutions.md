# <a name="how-to-add-a-feature-to-sharepoint-solutions"></a>如何：將功能新增至 SharePoint 方案
  當您建立 SharePoint 方案時，會將一個功能和一個封裝新增至方案。 您可以新增更多功能專案來組織不同類型的 SharePoint 檔案。

### <a name="to-add-a-feature-through-solution-explorer"></a>透過方案總管新增功能

1. 在 **方案總管**中，開啟 [ **功能**] 的快捷方式功能表，然後選擇 [ **加入功能**]。
