# <a name="visual-studio-architecture-tooling-guidance"></a>Visual Studio 架構工具指南

此架構工具指引是在 Visual Studio 2010 中使用模型化工具的實用指引的可下載套件。 當中呈現如何以整合的方式使用所有模型工具的高階觀點，並討論使用該些工具最常見的案例。

內容是由 Visual Studio ALM Ranger 所建立，這個特殊群組具有來自 Visual Studio 產品小組、Microsoft 服務、Microsoft 最有價值專家（Mvp）和 Visual Studio Community 潛在客戶的成員。
