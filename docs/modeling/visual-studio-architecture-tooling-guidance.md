# <a name="visual-studio-architecture-tooling-guidance"></a>Visual Studio 架構工具指南

此架構工具指南是有關在 Visual Studio 2010 中使用模型工具的實用指導方針的可下載套件。 它提供如何以整合方式使用所有模型工具的高階觀點，並討論使用工具的最常見案例。

內容是由 Visual Studio ALM Ranger 所建立，這是一個特殊群組，其中包含 Visual Studio 產品小組、Microsoft 服務、Microsoft 最有價值專家 (Mvp) 和 Visual Studio Community 潛在客戶的成員。
