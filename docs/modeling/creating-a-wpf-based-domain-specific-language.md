# <a name="create-a-wpf-based-domain-specific-language"></a>建立 WPF 架構的特定領域語言

您可以建立具有 WPF 設計工具的特定領域語言，而不是圖形化設計工具。

如需這項功能的資訊和範例，請參閱 Visual Studio 的視覺效果和模型工具網站，網址為[http://go.microsoft.com/fwlink/?LinkId=186128](https://code.msdn.microsoft.com/Visualization-and-Modeling-313535db)
