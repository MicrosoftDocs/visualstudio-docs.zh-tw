# <a name="api-reference-for-t4-text-templates"></a>T4 文字範本 API 參考

文字模板化 API 可讓您叫用及自訂[文字模板](../modeling/code-generation-and-t4-text-templates.md)的轉換。

## <a name="namespaces"></a>命名空間

|命名空間|用途|
|-|-|
|<xref:Microsoft.VisualStudio.TextTemplating?displayProperty=fullName>|包含文字模板轉換功能的類別。 文字模板轉換引擎已整合到 Visual Studio, 並將文字模板檔案轉換成產生的文字輸出檔案。|
|[VisualStudio. TextTemplating. 模型化](/previous-versions/ee844312(v=vs.140))|提供與 UML 模型和特定領域語言相關的文字轉換設備, 例如 Visual Studio ModelBus 的存取。|
|<xref:Microsoft.VisualStudio.TextTemplating.VSHost?displayProperty=fullName>|提供 Visual Studio 中文字模板化服務的存取權。|
