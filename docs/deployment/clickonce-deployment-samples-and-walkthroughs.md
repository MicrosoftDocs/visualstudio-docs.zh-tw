# <a name="clickonce-deployment-samples-and-walkthroughs"></a>ClickOnce 部署範例和逐步解說
本章節包含範例應用程式、 程式碼範例和逐步解說，說明語法、 結構和技術來部署 Windows Form、 WPF 和主控台應用程式。

 範例程式碼是為了提供指示，，因此不應用於已部署的解決方案，而不需要修改。 特別是，安全性必須納入大於的考量。
