# <a name="msbuild-concepts"></a>MSBuild 概念

MSBuild 提供了一個基本的 XML 架構，您可以使用該架構來控制生成平臺構建軟體的方式。 若要指定組建中的元件以及建置它們的方式，請使用下列這四個 MSBuild 組件︰屬性、項目、工作和目標。

| [MS 生成項](../msbuild/msbuild-items.md) | 推出項目和項目集合。 項目是建置系統的輸入內容，通常代表檔案。 |
| [MS 構建目標](../msbuild/msbuild-targets.md) | 解釋如何以特定順序將各項工作集合在一起成為群組，並能夠在命令列上呼叫建置流程的區段。 |
| [MS 生成任務](../msbuild/msbuild-tasks.md) | 演示如何創建可執行代碼單元，MSBuild 可用於執行原子生成操作。 |
