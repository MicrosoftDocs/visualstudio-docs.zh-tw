# <a name="msb3552-resource-file-filename-cannot-be-found"></a>MSB3552：找不到資源檔 ' filename '

找不到要求資源檔時，就會發生此錯誤。 檢查檔案名的路徑和拼寫。 這些檔案通常會指定為 `EmbeddedResource` 專案檔中的專案。
