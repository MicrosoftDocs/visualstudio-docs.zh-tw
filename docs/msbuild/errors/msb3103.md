# <a name="msb3103-invalid-resx-file"></a>MSB3103：不正確 Resx 檔案

當 .resx 資源檔有問題時，就會發生這個錯誤。 當您看到此錯誤時，會顯示另一個錯誤，其中包含更多詳細資料。 查看其他錯誤以取得詳細資訊。
