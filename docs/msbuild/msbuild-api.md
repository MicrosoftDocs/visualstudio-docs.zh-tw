# <a name="use-the-msbuild-api"></a>使用 MSBuild API

MSBuild 提供公開的 API 介面，讓您的程式可以執行建置並檢查專案。

您可以在 [Microsoft.Build 命名空間](https://msdn.microsoft.com/library/mt476050(v=vs.110).aspx)找到 MSBuild API 的文件。
