# <a name="wpf-targets-files"></a>WPF .targets 檔案

Windows 演示基礎 （WPF） 通過添加一組特定于 WPF 的任務來擴展 MSBuild，這些任務被合併到特殊的 *.target*檔中*Microsoft.WinFX.target*。 此檔合併了在 WPF 中構建 MSBuild 專案所需的 MSBuild 任務集。
