# <a name="wpf-targets-files"></a>WPF .targets 檔案

Windows Presentation Foundation (WPF) 藉由新增一組將 WPF 特定的工作合併到特殊 *的 .targets* *檔案中，* 來擴充 MSBuild。 此檔案結合了在 WPF 中建立 MSBuild 專案所需的一組 MSBuild 工作。
