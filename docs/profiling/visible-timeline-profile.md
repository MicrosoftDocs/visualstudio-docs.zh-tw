# <a name="visible-timeline-profile"></a>可見時間表分析
執行緒封鎖檢視的可見時間表分析提供統計資訊和報表連結。 當您放大、縮小、水平捲動、隱藏通道或顯示通道時，作用中圖例中的數字會隨之變更，反映目前在檢視中的項目。 若要檢視有關圖例中項目的報表，按一下該項目。
