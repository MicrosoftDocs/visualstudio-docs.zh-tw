# <a name="unblock-stack"></a>解除封鎖堆疊
如果目前選取的執行緒元素代表封鎖的區段，稍後會在目前處理序中的另一個執行緒將其解除封鎖之後執行，此索引標籤上會顯示執行解除封鎖的執行緒呼叫堆疊。
