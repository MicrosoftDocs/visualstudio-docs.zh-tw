# <a name="instruction-pointers-ips-view"></a>指令指標 (IP) 檢視
[指令指標] 檢視可讓您查看取樣程式碼剖析期間所執行的特定指標。  
  
 在檢測程式碼剖析期間，無法使用此檢視。  
  
 對於取樣程式碼剖析資料，IP 檢視會識別收集取樣時直接執行之函式中的指令。  
  
 對於 .NET 記憶體資料，[程式行] 檢視會識別用於配置記憶體的指令。
