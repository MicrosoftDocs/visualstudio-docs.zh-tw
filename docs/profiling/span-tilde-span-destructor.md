終結 `span` 物件，並釋放其資源。

## <a name="syntax"></a>語法

```cpp
~span();
```

## <a name="requirements"></a>需求
 **標頭：** *cvmarkersobj.h*

 **命名空間：** Concurrency::diagnostic
