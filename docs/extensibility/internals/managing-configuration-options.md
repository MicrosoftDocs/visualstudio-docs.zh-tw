# <a name="managing-configuration-options"></a>管理組態選項
創建新項目類型時,必須管理專案和解決方案配置設置,以確定專案如何生成、打包、部署和運行。 以下主題討論專案和解決方案配置。

## <a name="in-this-section"></a>本節內容
- [概觀](../../extensibility/internals/configuration-options-overview.md)

 描述[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]中的專案如何支援多個配置。

- [屬性頁](../../extensibility/internals/property-pages.md)

 說明使用者可以使用屬性頁查看和更改專案配置相關屬性和獨立屬性。

- [方案組態](../../extensibility/internals/solution-configuration.md)

 提供有關解決方案配置中儲存的內容以及解決方案配置如何指導 **"開始"和"****生成**"命令的行為的資訊。

- [專案組態物件](../../extensibility/internals/project-configuration-object.md)

 說明專案設定物件如何管理向 UI 顯示配置資訊。

- [建置的專案組態](../../extensibility/internals/project-configuration-for-building.md)

 說明解決方案配置對話方塊如何管理特定解決方案的解決方案**配置**清單。

- [管理部署的專案組態](../../extensibility/internals/project-configuration-for-managing-deployment.md)

 定義部署行為,並且兩種方式[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]支援支援部署的專案。

- [輸出的專案組態](../../extensibility/internals/project-configuration-for-output.md)

 解釋每個配置可以支援的生成過程以及可以使用輸出項的介面和方法。

## <a name="related-sections"></a>相關章節
- [專案類型](../../extensibility/internals/project-types.md)

 提供專案概述,作為[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]集成開發環境 (IDE) 的基本構建基塊。 連結指向其他主題,這些主題解釋了專案如何控制構建和編譯代碼。
