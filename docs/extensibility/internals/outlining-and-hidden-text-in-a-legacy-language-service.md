# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>舊版語言服務中的大綱和隱藏文字
大綱，讓能夠摺疊成單一行的文字行的順序。 例如，在C++可以摺疊成一行，顯示方法簽章的所有方法。 隱藏的文字是可顯示或隱藏的文字行的順序。
