# <a name="what39s-new-in-the-source-control-plug-in-api-version-12"></a>原始&#39;檔控制外掛程式 API 版本1.2 中的新功能
本節將重點放在原始檔控制外掛程式 API 版本1.2 中的先進原始碼控制功能。 下列列出版本1.2 中的新功能：

## <a name="changes"></a>變更
- [消除 ~ SAK](../../extensibility/internals/elimination-of-tilde-sak-files.md)檔案說明原始檔控制外掛程式 API 1.2 如何與暫存檔案互動。

- [跨多個專案連接進行設定的應用](../../extensibility/internals/application-of-settings-across-multiple-project-connections.md)說明原始檔控制外掛程式可以如何使用批次作業，在多個專案或多個連接內容之間執行相同的原始檔控制作業。

- [為方案建立父容器檔案夾](../../extensibility/internals/creating-parent-container-folders-for-solutions.md)說明原始檔控制外掛程式 API 版本1.2 如何讓使用者為方案中的所有 Web 專案指定單一根來源控制目的地。

- [本機專案資料夾與原始檔控制存放區的選擇性比較](../../extensibility/internals/optional-comparison-of-local-project-folder-to-source-control-store.md)說明原始檔控制外掛程式 API 1.2 如何比較檔案的版本。

- [從移除原始檔控制資訊。Proj 和。Sln](../../extensibility/internals/removal-of-source-control-information-from-dot-proj-and-dot-sln-files.md)檔案說明原始檔控制外掛程式 API 版本1.2 如何儲存專案和解決方案的相關資訊。
