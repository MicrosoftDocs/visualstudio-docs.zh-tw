# <a name="what39s-new-in-the-source-control-plug-in-api-version-12"></a>來源控制外掛程式 API 版本 1.2 中&#39;新增功能
本節重點介紹原始程式碼管理版本 1.2 中的高級原始程式碼控制功能。 下面列出版本 1.2 中的新功能:

## <a name="changes"></a>變更
- [清除 _SAK 檔案](../../extensibility/internals/elimination-of-tilde-sak-files.md)解釋原始程式碼管理外掛程式 API 1.2 如何與暫存檔互動。

- [跨一個多個項目連線設定設定](../../extensibility/internals/application-of-settings-across-multiple-project-connections.md)說明原始程式管理外掛程式如何使用批次處理操作跨多個專案或多個連接上下文執行相同的原始程式碼管理操作。

- [為解決方案建立父容器資料夾](../../extensibility/internals/creating-parent-container-folders-for-solutions.md)說明原始碼管理外掛程式 API 版本 1.2 使用戶能夠為解決方案中的所有 Web 專案指定單根根源控制元件目標。

- [本地端項目資料夾與源碼管理儲存的可選比較](../../extensibility/internals/optional-comparison-of-local-project-folder-to-source-control-store.md)解釋原始程式碼管理外掛程式 API 1.2 如何比較檔案版本。

- [從中刪除原始程式碼管理資訊。普羅傑和.Sln 檔案](../../extensibility/internals/removal-of-source-control-information-from-dot-proj-and-dot-sln-files.md)解釋原始程式碼管理外掛程式 API 版本 1.2 如何儲存有關專案和解決方案的資訊。
