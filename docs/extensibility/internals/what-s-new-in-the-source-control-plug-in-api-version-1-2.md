# <a name="what39s-new-in-the-source-control-plug-in-api-version-12"></a>什麼&#39;的新功能的原始檔控制外掛程式 API 版本 1.2
本章節強調在原始檔控制外掛程式 API 版本 1.2 進階的原始程式碼控制功能。 以下列出在 1.2 版的新功能：

## <a name="changes"></a>變更
- [消除 ~ SAK 檔案](../../extensibility/internals/elimination-of-tilde-sak-files.md)說明原始檔控制外掛程式 API 1.2 與暫存檔案之間的互動方式。

- [應用程式的設定跨多個專案連接](../../extensibility/internals/application-of-settings-across-multiple-project-connections.md)說明原始檔控制外掛程式可如何使用批次作業執行相同的原始檔控制作業，跨多個專案或多個連接內容。

- [建立解決方案的父容器資料夾](../../extensibility/internals/creating-parent-container-folders-for-solutions.md)說明原始檔控制外掛程式 API 版本 1.2 如何讓使用者指定的方案中的所有 Web 專案的單一根來源控制目的地。

- [選擇性比較的本機專案資料夾，以原始檔控制存放](../../extensibility/internals/optional-comparison-of-local-project-folder-to-source-control-store.md)說明原始檔控制外掛程式 API 1.2 如何比較檔案的版本。

- [從原始檔控制資訊移除。Proj 和。Sln 檔案](../../extensibility/internals/removal-of-source-control-information-from-dot-proj-and-dot-sln-files.md)說明原始檔控制外掛程式 API 版本 1.2 儲存專案和方案的相關資訊的方式。
