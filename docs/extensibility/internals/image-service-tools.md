# <a name="image-service-tools"></a>影像服務工具
VS SDK 包括多個工具,可協助擴展生成器使用 Visual Studio 圖像服務。

- 「[來自資源清單」](../../extensibility/internals/manifest-from-resources.md)工具取得影像資源清單 (.png 或 .xaml 檔案)並產生影像清單檔。

- 清單[到代碼「](../../extensibility/internals/manifest-to-code.md)工具」來採用影像清單檔並生成包裝檔,用於C++、C#、VB或 .vsct 檔。

- [圖像庫檢視器](../../extensibility/internals/image-library-viewer.md)可以載入、編輯和搜尋影像清單。
