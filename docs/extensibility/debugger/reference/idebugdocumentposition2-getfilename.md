# <a name="idebugdocumentposition2getfilename"></a>IDebugDocumentPosition2::GetFileName
獲取包含文件位置的源檔的檔名。

## <a name="syntax"></a>語法

```cpp
HRESULT GetFileName( 
   BSTR* pbstrFileName
);
```

```csharp
int GetFileName( 
   out string pbstrFileName
);
```

## <a name="parameters"></a>參數
`pbstrFileName`\
[出]返回源檔的檔名。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。

## <a name="remarks"></a>備註
 源檔可能並不總是具有檔名(例如,磁碟上可能不存在源檔)。
