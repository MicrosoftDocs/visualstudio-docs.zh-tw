# <a name="idebugprocess2getport"></a>IDebugProcess2::GetPort
獲取進程正在運行的埠。

## <a name="syntax"></a>語法

```cpp
HRESULT GetPort( 
   IDebugPort2** ppPort
);
```

```csharp
int GetPort( 
   out IDebugPort2 ppPort
);
```

## <a name="parameters"></a>參數
`ppPort`\
[出]返回表示啟動進程的埠的[IDebugPort2](../../../extensibility/debugger/reference/idebugport2.md)物件。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。
