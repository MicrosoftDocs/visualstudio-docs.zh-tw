# <a name="idebugdocumenttext2getsize"></a>IDebugDocumentText2::GetSize
抓取檔中這個位置的文字大小。

## <a name="syntax"></a>語法

```cpp
HRESULT GetSize( 
   ULONG* pcNumLines,
   ULONG* pcNumChars
);
```

```csharp
int GetSize( 
   ref uint pcNumLines,
   ref uint pcNumChars
);
```

## <a name="parameters"></a>參數
`pcNumLines`\
擴展傳回文字的行數。

`pcNumChars`\
擴展傳回文字的字元數。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回錯誤碼。

## <a name="remarks"></a>備註

 [僅限 c + +]如果不需要特定的值，請針對參數傳遞 Null。

 [僅限 c #]這兩個參數都必須指定。
