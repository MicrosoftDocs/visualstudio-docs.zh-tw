# <a name="idebugsettingscallback2getmetricguid"></a>IDebugSettingsCallback2::GetMetricGuid
擷取指定其名稱的度量的唯一識別碼。

## <a name="syntax"></a>語法

```cpp
HRESULT GetMetricGuid(
   LPCWSTR pszType,
   REFGUID guidSection,
   LPCWSTR pszMetric,
   GUID*   pguidValue
);
```

```csharp
private int GetMetricGuid(
   string   pszType,
   ref Guid guidSection,
   string   pszMetric,
   out Guid pguidValue
);
```

## <a name="parameters"></a>參數
`pszType`\
[in]度量的類型。

`guidSection`\
[in]區段的唯一識別碼。

`pszMetric`\
[in]計量名稱。

`pguidValue`\
[out]傳回度量的唯一識別碼。

## <a name="return-value"></a>傳回值
 如果成功，則傳回`S_OK`; 否則傳回錯誤碼。
