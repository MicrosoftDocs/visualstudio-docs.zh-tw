# <a name="idebugsettingscallback2geteemetricfile"></a>IDebugSettingsCallback2::GetEEMetricFile
檢索給定名稱或指標的運算式賦值器指標檔。

## <a name="syntax"></a>語法

```cpp
HRESULT GetEEMetricFile(
   REFGUID guidLang,
   REFGUID guidVendor,
   LPCWSTR pszMetric,
   BSTR*   pbstrValue
);
```

```csharp
private int GetEEMetricFile(
   ref Guid   guidLang,
   ref Guid   guidVendor,
   string     pszMetric,
   out string pbstrValue
);
```

## <a name="parameters"></a>參數
`guidLang`\
[在]程式設計語言的唯一標識符。

`guidVendor`\
[在]供應商的唯一標識碼。

`pszMetric`\
[在]指標的名稱。

`pbstrValue`\
[出]將指標文件的內容作為字串返回。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。
