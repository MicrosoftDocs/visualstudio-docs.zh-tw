# <a name="idebugcoreserver3queryislocal"></a>IDebugCoreServer3::QueryIsLocal
判斷伺服器是否為本機呼叫端。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT QueryIsLocal(  
   void  
);  
```  
  
```csharp  
int QueryIsLocal();  
```  
  
## <a name="return-value"></a>傳回值  
 傳回`S_OK`表示本機伺服器。 傳回`S_FALSE`如果伺服器正在執行 msvsmon.exe，通常用於遠端偵錯執行個體中。
