# <a name="idebugbinder3getmemoryobject"></a>IDebugBinder3::GetMemoryObject
此方法檢索表示此物件綁定到的記憶體的記憶體物件。

## <a name="syntax"></a>語法

```cpp
HRESULT GetMemoryObject(
   IDebugField*   pField,
   UINT64         uConstant,
   IDebugObject** ppObject
);
```

```csharp
int GetMemoryObject(
   IDebugField      pField,
   long             uConstant,
   out IDebugObject ppObject
);
```

## <a name="parameters"></a>參數
`pField`\
[在]指定要取得記憶體物件的欄位。

`uConstant`\
[在]表示常量值的記憶體地址或值。

`ppObject`\
[出]表示此物件綁定到的記憶體的[IDebugObject。](../../../extensibility/debugger/reference/idebugobject.md)

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。
