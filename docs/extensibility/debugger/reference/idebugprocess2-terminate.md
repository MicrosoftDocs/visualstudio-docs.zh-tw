# <a name="idebugprocess2terminate"></a>IDebugProcess2::Terminate
終止處理序。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Terminate(   
   void   
);  
```  
  
```csharp  
int Terminate();  
```  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`; 否則傳回錯誤碼。  
  
## <a name="remarks"></a>備註  
 處理序終止時，會終止該處理序內的所有程式;無都可以執行任何更多的程式碼。
