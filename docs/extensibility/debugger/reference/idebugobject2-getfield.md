# <a name="idebugobject2getfield"></a>IDebugObject2::GetField
獲取此對象的類型。

## <a name="syntax"></a>語法

```cpp
HRESULT GetField(
 IDebugField** ppField
);
```

```csharp
int GetField(
   out IDebugField ppField
);
```

## <a name="parameters"></a>參數
`ppField`\
[出]返回[IDebugField](../../../extensibility/debugger/reference/idebugfield.md)物件(如果不是空值)。

## <a name="return-value"></a>傳回值
 如果成功,返回S_OK;否則,返回錯誤代碼。

## <a name="remarks"></a>備註
 欄位描述物件的類型。
