# <a name="idebugreference2getsize"></a>IDebugReference2::GetSize
取得參考值的大小（以位元組為單位）。 保留供未來使用。

## <a name="syntax"></a>語法

```cpp
HRESULT GetSize ( 
   DWORD* pdwSize
);
```

```csharp
int GetSize ( 
   out uint pdwSize
);
```

## <a name="parameters"></a>參數
`pdwSize`\
擴展傳回參考值的大小（以位元組為單位）。

## <a name="return-value"></a>傳回值
 一律傳回 `E_NOTIMPL`。
