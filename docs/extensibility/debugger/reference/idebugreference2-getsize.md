# <a name="idebugreference2getsize"></a>IDebugReference2::GetSize
取得大小，以位元組為單位，值的參考。 保留供未來使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetSize (   
   DWORD* pdwSize  
);  
```  
  
```csharp  
int GetSize (   
   out uint pdwSize  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pdwSize`  
 [out]傳回大小，以位元組為單位，值的參考。  
  
## <a name="return-value"></a>傳回值  
 一律傳回 `E_NOTIMPL`。
