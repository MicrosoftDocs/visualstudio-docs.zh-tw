# <a name="idebugmodoptgetmodopts"></a>IDebugModOpt::GetModOpts
擷取一份選擇性修飾詞。

## <a name="syntax"></a>語法

```cpp
HRESULT GetModOpts(
   ULONG  celt,
   BSTR*  rgelt,
   ULONG* pceltFetched
);
```

```csharp
int GetModOpts(
   uint         celt,
   out string[] rgelt,
   ref uint     pceltFetched
);
```

#### <a name="parameters"></a>參數
 `celt`

 [in]要傳回的項目數目。

 `rgelt`

 [out]傳回陣列，其中包含的選項。

 `pceltFetched`

 [in、 out]在傳回的項目數`rgelt`陣列。

## <a name="return-value"></a>傳回值
 如果成功，則傳回`S_OK`; 否則傳回錯誤碼。
