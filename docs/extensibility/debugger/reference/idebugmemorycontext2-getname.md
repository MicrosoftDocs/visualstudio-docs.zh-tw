# <a name="idebugmemorycontext2getname"></a>IDebugMemoryContext2::GetName
取得此內容的使用者可顯示名稱。

## <a name="syntax"></a>語法

```cpp
HRESULT GetName( 
   BSTR* pbstrName
);
```

```csharp
int GetName(
   out string pbstrName
);
```

## <a name="parameters"></a>參數
`pbstrName`\
擴展傳回記憶體內容的名稱。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回錯誤碼。

## <a name="remarks"></a>備註
 通常不會使用記憶體內容的名稱。
