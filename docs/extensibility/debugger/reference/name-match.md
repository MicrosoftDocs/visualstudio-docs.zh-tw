# <a name="name_match"></a>NAME_MATCH
選擇匹配名稱的案例選項。

## <a name="syntax"></a>語法

```cpp
typedef enum { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
} NAME_MATCH;
```

```csharp
public enum NameMatchOptions { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
}
```

## <a name="fields"></a>欄位
 `nmNone`\
 未指定任何選項。

 `nmCaseSensitive`\
 指示要匹配的名稱區分大小寫。

 `nmCaseInsensitive`\
 指示要匹配的名稱不區分大小寫。
