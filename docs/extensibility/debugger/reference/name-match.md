# <a name="namematch"></a>NAME_MATCH
選取 比對名稱的大小寫選項。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```  
  
## <a name="members"></a>成員  
 nmNone  
 未指定任何選項。  
  
 nmCaseSensitive  
 表示要比對的名稱會區分大小寫。  
  
 nmCaseInsensitive  
 表示要比對的名稱不區分大小寫。
