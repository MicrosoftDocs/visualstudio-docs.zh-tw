# <a name="idebugmodule3isusercode"></a>IDebugModule3::IsUserCode
擷取有關模組是否代表使用者程式碼。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsUserCode(  
   BOOL* pfUser  
);  
```  
  
```csharp  
int IsUserCode(  
   out int pfUser  
);  
```  
  
#### <a name="parameters"></a>參數  
 `pfUser`  
 [out]非零值 (`TRUE`) 如果模組會代表使用者程式碼，則為零 (`FALSE`) 如果沒有。  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK`，否則會傳回錯誤碼。
