# <a name="idebugprocess2candetach"></a>IDebugProcess2::CanDetach
判斷工作階段的偵錯管理員 (SDM) 可以中斷連結程序。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT CanDetach(  
   void  
);  
```  
  
```csharp  
int CanDetach();  
```  
  
## <a name="return-value"></a>傳回值  
 如果成功，則傳回`S_OK.`傳回`S_FALSE`如果偵錯工具無法從處理序中斷連結。 反之則傳回錯誤碼。
