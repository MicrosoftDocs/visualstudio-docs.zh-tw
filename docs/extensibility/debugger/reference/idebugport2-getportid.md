# <a name="idebugport2getportid"></a>IDebugPort2::GetPortId
取得埠識別碼。

## <a name="syntax"></a>語法

```cpp
HRESULT GetPortId( 
   GUID* pguidPort
);
```

```csharp
int GetPortId( 
   out Guid pguidPort
);
```

## <a name="parameters"></a>參數
`pguidPort`\
擴展傳回識別埠的 GUID。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回錯誤碼。
