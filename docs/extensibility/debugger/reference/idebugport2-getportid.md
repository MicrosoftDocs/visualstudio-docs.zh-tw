# <a name="idebugport2getportid"></a>IDebugPort2::GetPortId
獲取埠標識碼。

## <a name="syntax"></a>語法

```cpp
HRESULT GetPortId( 
   GUID* pguidPort
);
```

```csharp
int GetPortId( 
   out Guid pguidPort
);
```

## <a name="parameters"></a>參數
`pguidPort`\
[出]返回標識埠的 GUID。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。
