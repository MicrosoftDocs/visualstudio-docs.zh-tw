# <a name="idebugobject2createalias"></a>IDebugObject2::CreateAlias
為此物件創建唯一的 ID 或別名或返回現有別名。

## <a name="syntax"></a>語法

```cpp
HRESULT CreateAlias(
   IDebugAlias** ppAlias
);
```

```csharp
int CreateAlias(
   out IDebugAlias ppAlias
);
```

## <a name="parameters"></a>參數
`ppAlias`\
[出]新的(或現有)別名。

## <a name="return-value"></a>傳回值
 如果成功,返回S_OK;否則,返回錯誤代碼。

## <a name="remarks"></a>備註
 別名是在物件在記憶體中時表示特定對象的標籤。
