# <a name="idebugport2getportname"></a>IDebugPort2::GetPortName
取得埠名稱。

## <a name="syntax"></a>語法

```cpp
HRESULT GetPortName( 
   BSTR* pbstrName
);
```

```csharp
int GetPortName( 
   out string pbstrName
);
```

## <a name="parameters"></a>參數
`pbstrName`\
脫銷傳回埠的名稱。

## <a name="return-value"></a>傳回值
 如果成功，會傳回 `S_OK`;否則，會傳回錯誤碼。
