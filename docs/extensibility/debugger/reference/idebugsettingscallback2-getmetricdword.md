# <a name="idebugsettingscallback2getmetricdword"></a>IDebugSettingsCallback2::GetMetricDword
擷取值，指定其名稱的度量資訊。

## <a name="syntax"></a>語法

```cpp
HRESULT GetMetricDword(
   LPCWSTR pszType,
   REFGUID guidSection,
   LPCWSTR pszMetric,
   DWORD*  pdwValue
);
```

```csharp
private int GetMetricDword(
   string   pszType,
   ref Guid guidSection,
   string   pszMetric,
   out uint pdwValue
);
```

## <a name="parameters"></a>參數
 `pszType`\

 [in]度量的類型。

 `guidSection`\

 [in]區段的唯一識別碼。

 `pszMetric`\

 [in]計量名稱。

 `pdwValue`\

 [out]傳回的度量值。

## <a name="return-value"></a>傳回值
 如果成功，則傳回`S_OK`; 否則傳回錯誤碼。
