# <a name="reference-visual-studio-debugging-apis"></a>參考 (Visual Studio 偵錯 API)

Visual Studio 偵錯 SDK 參考會說明執行階段架構和功能的偵錯工具和其元件的基本模型。 文件會較少的做出假設，盡可能以相關元件和介面的項目，它將告訴您，除了它會透過介面定義的行為。

[參考] 區段的目標是提供支援為各種語言、 執行的時間和作業系統的一組越好。
