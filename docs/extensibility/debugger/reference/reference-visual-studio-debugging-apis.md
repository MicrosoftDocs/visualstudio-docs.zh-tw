# <a name="reference-visual-studio-debugging-apis"></a>參考 (Visual Studio 偵錯 API)

Visual Studio 調試 SDK 引用概述了調試器及其元件的運行時體系結構和功能的基本模型。 除了通過介面定義的行為外,本文檔對它描述的元件和介面做出盡可能少的假設。

參考部分的目標是盡可能為各種語言、運行時間和作業系統提供支援。
