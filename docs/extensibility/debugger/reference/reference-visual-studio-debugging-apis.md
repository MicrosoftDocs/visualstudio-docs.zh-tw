# <a name="reference-visual-studio-debugging-apis"></a>參考 (Visual Studio 偵錯 API)

Visual Studio 偵錯工具 SDK 參考概述偵錯工具及其元件的執行時間架構和功能的基本模型。 除了透過介面所定義的行為之外，檔也會盡可能地為它所描述的元件和介面提出最少的假設。

參考章節的目標是要盡可能將支援提供給各種語言、執行時間和作業系統。
