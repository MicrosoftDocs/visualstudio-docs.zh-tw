# <a name="idebugprogramhost2gethostid"></a>IDebugProgramHost2::GetHostId
獲取承載此程序的進程的進程標識符。

## <a name="syntax"></a>語法

```cpp
HRESULT GetHostId( 
   AD_PROCESS_ID* pdwId
);
```

```csharp
int GetHostId( 
   AD_PROCESS_ID[] pdwId
);
```

## <a name="parameters"></a>參數
`pdwId`\
[進出]使用進程識別碼資訊填充[AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md)結構。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。
