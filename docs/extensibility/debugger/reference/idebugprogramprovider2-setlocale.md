# <a name="idebugprogramprovider2setlocale"></a>IDebugProgramProvider2::SetLocale
建立用於任何特定於區域設置的資源的區域設置。

## <a name="syntax"></a>語法

```cpp
HRESULT SetLocale(
   WORD wLangID
);
```

```csharp
int SetLocale(
   ushort wLangID
);
```

## <a name="parameters"></a>參數
`wLangID`\
[在]要建立的語言 ID。 例如,英語為 1033。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。
