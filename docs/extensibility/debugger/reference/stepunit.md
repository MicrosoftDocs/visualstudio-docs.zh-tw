# <a name="stepunit"></a>STEPUNIT
指定間距單位逐步執行。

## <a name="syntax"></a>語法

```cpp
enum enum_STEPUNIT { 
   STEP_STATEMENT   = 0,
   STEP_LINE        = 1,
   STEP_INSTRUCTION = 2
};
typedef DWORD STEPUNIT;
```

```csharp
enum enum_STEPUNIT { 
   STEP_STATEMENT   = 0,
   STEP_LINE        = 1,
   STEP_INSTRUCTION = 2
};
```

## <a name="members"></a>成員
 STEP_STATEMENT 陳述式的步驟。

 STEP_LINE 所列的步驟。

 STEP_INSTRUCTION 指示的步驟。

## <a name="remarks"></a>備註
 作為引數[步驟](../../../extensibility/debugger/reference/idebugprocess3-step.md)方法。
