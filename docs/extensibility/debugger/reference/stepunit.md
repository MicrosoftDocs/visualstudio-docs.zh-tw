# <a name="stepunit"></a>STEPUNIT
指定逐步執行的步驟單位。

## <a name="syntax"></a>語法

```cpp
enum enum_STEPUNIT { 
   STEP_STATEMENT   = 0,
   STEP_LINE        = 1,
   STEP_INSTRUCTION = 2
};
typedef DWORD STEPUNIT;
```

```csharp
enum enum_STEPUNIT { 
   STEP_STATEMENT   = 0,
   STEP_LINE        = 1,
   STEP_INSTRUCTION = 2
};
```

## <a name="fields"></a>欄位
 `STEP_STATEMENT`\
 逐步執行語句。

 `STEP_LINE`\
 依行的步驟。

 `STEP_INSTRUCTION`\
 逐步指示。

## <a name="remarks"></a>備註
 以引數形式傳遞至 [Step](../../../extensibility/debugger/reference/idebugprocess3-step.md) 方法。

 標頭： msdbg。h

 命名空間： VisualStudio

 元件： Microsoft.VisualStudio.Debugger.Interop.dll
