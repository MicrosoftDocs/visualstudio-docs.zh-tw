# <a name="idebugprogram2enummodules"></a>IDebugProgram2::EnumModules
檢索此程式已載入並正在執行的模組的清單。

## <a name="syntax"></a>語法

```cpp
HRESULT EnumModules( 
   IEnumDebugModules2** ppEnum
);
```

```csharp
int EnumModules( 
   out IEnumDebugModules2 ppEnum
);
```

## <a name="parameters"></a>參數
`ppEnum`\
[出]返回包含模組清單的[IEnumDebugModule2](../../../extensibility/debugger/reference/ienumdebugmodules2.md)物件。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。

## <a name="remarks"></a>備註
 模組是 DLL 或程式集,通常列在**模組**調試視窗中。
