# <a name="idebugdefaultport2queryislocal"></a>IDebugDefaultPort2::QueryIsLocal
這個方法會判斷此連接埠是否在本機電腦上。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT QueryIsLocal(  
   void  
);  
```  
  
```csharp  
int QueryIsLocal();  
```  
  
## <a name="return-value"></a>傳回值  
 傳回`S_OK`此連接埠是否在本機 （在與呼叫端相同的電腦） 或`S_FALSE`連接埠是否位於另一部電腦上。
