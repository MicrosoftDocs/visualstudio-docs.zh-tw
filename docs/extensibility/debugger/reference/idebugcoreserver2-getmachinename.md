# <a name="idebugcoreserver2getmachinename"></a>IDebugCoreServer2::GetMachineName
獲取核心伺服器運行的計算機的名稱。

## <a name="syntax"></a>語法

```cpp
HRESULT GetName( 
   BSTR* pbstrName
);
```

```csharp
int GetName( 
   out string pbstrName
);
```

## <a name="parameters"></a>參數
`pbstrName`\
[出]傳回包含電腦名稱的字串。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。
