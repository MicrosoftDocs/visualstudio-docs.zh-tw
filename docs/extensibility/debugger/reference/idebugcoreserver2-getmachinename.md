# <a name="idebugcoreserver2getmachinename"></a>IDebugCoreServer2::GetMachineName
取得核心伺服器執行所在的電腦名稱稱。

## <a name="syntax"></a>語法

```cpp
HRESULT GetName( 
   BSTR* pbstrName
);
```

```csharp
int GetName( 
   out string pbstrName
);
```

## <a name="parameters"></a>參數
`pbstrName`\
擴展傳回包含電腦名稱稱的字串。

## <a name="return-value"></a>傳回值
 如果成功，則傳回， `S_OK` 否則傳回錯誤碼。
