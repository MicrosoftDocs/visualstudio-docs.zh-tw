# <a name="idebugdocumenttextevents2oninserttext"></a>IDebugDocumentTextEvents2::onInsertText
通知調試包文本已插入到文檔中。

## <a name="syntax"></a>語法

```cpp
HRESULT onInsert( 
   TEXT_POSITION pos,
   DWORD         dwNumToInsert
);
```

```csharp
int onInsert( 
   enum_TEXT_POSITION pos,
   uint               dwNumToInsert
);
```

## <a name="parameters"></a>參數
`pos`\
[在]指示文本插入位置[TEXT_POSITION](../../../extensibility/debugger/reference/text-position.md)結構。

`dwNumToInsert`\
[在]指定插入的文字的字元數。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。
