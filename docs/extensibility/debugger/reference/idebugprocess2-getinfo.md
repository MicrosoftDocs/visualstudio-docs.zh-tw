# <a name="idebugprocess2getinfo"></a>IDebugProcess2::GetInfo
獲取流程的說明。

## <a name="syntax"></a>語法

```cpp
HRESULT GetInfo(
   PROCESS_INFO_FIELDS  Fields,
   PROCESS_INFO*        pProcessInfo
);
```

```csharp
int GetInfo(
   enum_PROCESS_INFO_FIELDS  Fields,
   PROCESS_INFO[]            pProcessInfo
);
```

## <a name="parameters"></a>參數
`Fields`\
[在][PROCESS_INFO_FIELDS](../../../extensibility/debugger/reference/process-info-fields.md)枚舉中的值的組合,指定要填充`pProcessInfo`參數 的欄位。

`pProcessInfo`\
[出]一[個PROCESS_INFO](../../../extensibility/debugger/reference/process-info.md)結構,其中填寫了流程的描述。

## <a name="return-value"></a>傳回值
 如果成功,返回`S_OK`;否則,返回錯誤代碼。
