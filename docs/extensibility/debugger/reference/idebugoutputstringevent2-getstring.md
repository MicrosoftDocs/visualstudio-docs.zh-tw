# <a name="idebugoutputstringevent2getstring"></a>IDebugOutputStringEvent2::GetString
取得可顯示的訊息。

## <a name="syntax"></a>語法

```cpp
HRESULT GetString( 
   BSTR* pbstrString
);
```

```csharp
int GetString( 
   out string pbstrString
);
```

## <a name="parameters"></a>參數
`pbstrString`\
[out]傳回可顯示的訊息。

## <a name="return-value"></a>傳回值
 如果成功，則傳回`S_OK`; 否則傳回錯誤碼。
