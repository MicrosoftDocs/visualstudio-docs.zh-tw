# <a name="modules"></a>模組
偵錯工具就架構而言，*模組*:  
  
-   是一個實體容器的程式碼，例如可執行檔或 DLL。  
  
-   可以重新載入其符號，並描述本身。 模組描述會顯示在 IDE 的 [模組] 視窗中。  
  
-   由[IDebugModule2](../../extensibility/debugger/reference/idebugmodule2.md)來描述模組的偵錯引擎所建立的介面。
