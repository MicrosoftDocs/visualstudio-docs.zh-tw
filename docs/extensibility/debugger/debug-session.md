# <a name="debug-session"></a>偵錯工作階段
偵錯工具就架構而言，*工作階段*:

-   是集合的所有處理序，相關或不相關，已啟動或附加到 IDE 的單一執行個體。

-   是由工作階段的偵錯管理員 (SDM) 實作。 如需詳細資訊，請參閱 <<c0> [ 工作階段進行偵錯管理員](../../extensibility/debugger/session-debug-manager.md)。
