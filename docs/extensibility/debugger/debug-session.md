# <a name="debug-session"></a>Debug 會話
就偵錯工具架構而言， *會話*：

- 這是從 IDE 的單一實例啟動或附加的所有進程（相關或不相關）的集合。

- 是由會話 debug manager (SDM) 所執行。 如需詳細資訊，請參閱 [會話偵錯工具管理員](../../extensibility/debugger/session-debug-manager.md)。
