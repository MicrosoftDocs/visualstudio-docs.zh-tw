# <a name="debug-session"></a>除錯工作階段
在除錯器架構結構方面,*工作階段*:

- 是從IDE的單個實例啟動或附加到的所有進程(相關或不相關的)集。

- 由工作階段除錯管理員 (SDM) 實現。 有關詳細資訊,請參閱[工作階段除錯管理員](../../extensibility/debugger/session-debug-manager.md)。
