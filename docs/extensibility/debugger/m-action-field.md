# <a name="m_action-field"></a>m_action欄位
表示要在物件中執行的代碼的<xref:System.Threading.Tasks.Task>委託。

 **命名空間:**<xref:System.Threading.Tasks?displayProperty=fullName>

 **程式集**:mscorlib(在*mscorlib.dll*中)

 由於您無法從 .NET 框架訪問此內部成員,因此在通用中間語言 (CIL) 中提供了以下語法。

## <a name="syntax"></a>語法

```csharp
.field assembly object m_action
```

## <a name="remarks"></a>備註
 這是構造函數`action`<xref:System.Threading.Tasks.Task.%23ctor%2A>中的 參數。
