# <a name="visual-language-dictionary"></a>視覺化語言字典

Visual Studio 使用常見的視覺語言，來說明其圖示和其他影像的構成要素的概念。 設計新的圖示時，請使用此字典來識別您想要在自己的映射中傳達的概念。

## <a name="general-guidelines"></a>一般指導方針

- 這些視覺概念必須以一致的字典定義來使用。 否則，您可能會對使用者造成混淆，而這些使用者預期一般專案具有一致的意義。

- 選擇您的圖示中使用的最相關或重要概念，讓影像清晰易懂且容易區分。

- 在每個影像中，只使用一或兩個概念。

- 如需設計圖示的詳細策略，請參閱 [圖示設計](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign)。
