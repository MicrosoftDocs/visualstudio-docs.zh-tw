# <a name="color-value-reference-for-visual-studio"></a>Visual Studio 的色彩值參考

本主題是建立新的色彩標記的參考。 當您建立新的 UI 時，您可以從現有的 UI 使用類似的色彩，以選擇每個主要主題的色彩，以及高對比主題的該元素的系統色彩名稱。

如需建立新的色彩標記的執行資訊，請參閱 [色彩主題工具](../../extensibility/internals/color-theming-tools.md)。

若要搭配現有的 UI 使用色彩權杖，請參閱 [Visual Studio 的共用色彩](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md)。

使用本參考指南時，請記住：

- **一律將前景和背景色調一起複製。** 請不要從某個 UI 元素選擇前景，並從另一個 UI 元素選擇背景，因為您喜歡色調。

- **這些是十六進位碼，不是色彩標記。** 您無法使用這些 UI 元素的實際色彩標記名稱，因為您無法控制未來可能變更權杖的方式。 相反地，請在您自己的類別內建立權杖，並使用資料表中顯示的色調值。 您可以在 [Visual Studio 的共用色彩](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md)中找到現有 UI 的色彩標記。

- **只會顯示高對比樣本以進行比較。** 使用高對比 UI 的系統色彩名稱，請勿在這些樣本上使用滴管或色彩選擇器。 本主題中的高對比樣本反映 Windows 中的高對比 #1 主題。 其他高對比主題的色彩看起來會不同。

- **為了獲得最佳結果，請使用與範例類似的 UI 來配對色彩組。**
