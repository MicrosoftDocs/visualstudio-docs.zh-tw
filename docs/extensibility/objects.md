# <a name="objects"></a>物件
下列參考資訊說明用來建立核心文字編輯器實例的物件。

## <a name="in-this-section"></a>本節內容
- [VSCodeWindow 物件](../extensibility/vscodewindow-object.md) 描述程式碼視窗物件，這是可括住一或多個文字視圖的特殊文件視窗。

- [VSCodeWindowManager 物件](../extensibility/vscodewindowmanager-object.md) 描述語言服務所執行且負責管理裝飾的程式碼視窗管理員物件。

- [VSTextBuffer 物件](../extensibility/vstextbuffer-object.md) 描述文字緩衝區物件，它代表 Unicode 文字的資料流程、檔案的資料。

- [VSTextView 物件](../extensibility/vstextview-object.md) 描述文字視圖，此視窗可讓使用者查看及編輯文字緩衝區的 Unicode 文字。
