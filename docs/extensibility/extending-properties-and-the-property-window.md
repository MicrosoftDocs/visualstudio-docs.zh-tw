# <a name="extend-properties-and-the-property-window"></a>延伸屬性與屬性視窗
屬性**視窗**用於查看和編輯所選物件的設計時屬性和事件。

 在 中選擇[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]物件 時,該物件的公共屬性將顯示在 **「屬性」** 視窗中。 有關詳細資訊,請參閱[屬性視窗概述](../extensibility/internals/properties-window-overview.md)。
