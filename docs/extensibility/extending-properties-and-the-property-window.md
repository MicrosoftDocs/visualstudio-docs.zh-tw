# <a name="extend-properties-and-the-property-window"></a>擴充屬性和屬性視窗
[ **屬性** ] 視窗是用來查看和編輯所選取物件的設計階段屬性和事件。

 當您在中選取物件時 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] ，該物件的公用屬性會出現在 [ **屬性** ] 視窗中。 如需詳細資訊，請參閱 [屬性視窗總覽](../extensibility/internals/properties-window-overview.md)。
