# <a name="monitoring-gpu-utilization"></a>監視 GPU 使用率

若要監視遠端 Linux 電腦的 GPU 使用量：

1. 在 [伺服器總管] 中，展開 [遠端電腦]。
2. **以滑鼠右鍵按一下** 您想要監視的遠端電腦。

    ![顯示 AI 工具資料夾樹狀結構伺服器總管的螢幕擷取畫面。 已選取 [Azure 示範]，並開啟具有 [顯示熱度圖] 命令的內容功能表。](media/monitor-gpu/gpu-heatmap-0.png)

3. 按一下 [顯示熱度圖]。

    ![伺服器總管中 [AI 工具熱度圖] 視窗的螢幕擷取畫面，其中顯示 Azure DLVM 虛擬機器中 Gpu 的摘要、詳細資料和熱度圖資料。](media/monitor-gpu/heatmap.png)
