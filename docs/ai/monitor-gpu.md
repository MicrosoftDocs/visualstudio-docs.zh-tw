# <a name="monitoring-gpu-utilization"></a>監視 GPU 使用率

若要監視遠端 Linux 電腦的 GPU 使用量：

1. 在 [伺服器總管]**** 中，展開 [遠端電腦]****。
2. **以滑鼠右鍵按一下**您想要監視的遠端電腦。

    ![GPU 熱度圖](media/monitor-gpu/gpu-heatmap-0.png)

3. 按一下 [顯示熱度圖]****。

    ![GPU 熱度圖](media/monitor-gpu/heatmap.png)
