## <a name="what-is-the-difference-between-using-a-microsoft-account-vs-work-account"></a>使用 Microsoft 帳戶與公司帳戶有什麼差別？

Microsoft 帳戶 (MSA) 是個人擁有的個人電子郵件帳戶，可在組織未使用 Microsoft 365 或 Azure Active Directory 的受控租用戶時存取 Microsoft 服務。 工作帳戶是由組織所擁有，且用來存取 Microsoft 365 或 Azure Active Directory 租用戶上的服務。 如需有關 MSA 與工作帳戶之間差異的完整資訊，請參閱[了解使用 MSA (Microsoft 帳戶) 與工作帳戶 (Azure Active Directory 或 Microsoft 365) 的差異，以及這如何影響組織](https://aka.ms/MSAvsAAD)。
