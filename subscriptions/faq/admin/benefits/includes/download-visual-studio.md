## <a name="how-do-my-developers-download-visual-studio"></a>我的開發人員應如何下載 Visual Studio？
開發人員獲指派的訂閱層級，可以指出其有權存取的 Visual Studio IDE 層級。 例如 Visual Studio Professional 訂閱無權存取 Visual Studio Enterprise。 請確定您獲派正確的層級。 

### <a name="for-the-latest-release"></a>若是最新版本
1.  請要求訂閱者登入位於 <https://my.visualstudio.com/benefits> 的訂閱入口網站。 
0.  在 [工具] 區段中找出 [Visual Studio IDE] 磚，然後按一下 [權益] 磚底部的 [下載] 連結。 
0.  訂閱者將被重新導向至 Visual Studio 的 [下載結果] 頁面，以下載 IDE。 
0.  請遵循下載及安裝指示。
0.  使用指派訂閱的電子郵件，就能在安裝後立即登入 IDE，而無須提供產品金鑰。 [了解如何索取產品金鑰](https://docs.microsoft.com/visualstudio/subscriptions/find-keys)，在離線環境中使用。

### <a name="find-older-versions"></a>尋找舊版
您的訂閱者也可以在位於 <https://my.visualstudio.com> 的訂閱入口網站中，從[下載頁面](https://my.visualstudio.com/downloads)搜尋及下載舊版。
