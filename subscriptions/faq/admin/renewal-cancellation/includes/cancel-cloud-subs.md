## <a name="how-do-i-cancel-monthly-and-annual-subscriptions"></a>如何? 取消每月和年度訂閱？
若要取消從 [Visual Studio Marketplace](https://marketplace.visualstudio.com)購買的每月和年度訂閱，您需要登入系統 [管理入口網站](https://manage.visualstudio.com) ，並將合約的訂用帳戶數目設定為零。

請遵循下列步驟來減少訂閱數目：
1.  登入 https://manage.visualstudio.com
2.  如果您是一個以上合約的系統管理員，請從下拉式清單中選擇所需的合約。
3.  選取左上角的 [ **總覽** ] 圖示，以顯示訂閱的相關資訊。
4.  找出您要取消之訂用帳戶的專案，然後選取 [ **變更數量**]。 這會帶您前往 Visual Studio Marketplace 您將可以在其中更新購買數量。 
5.  將 quantity 設定為零 (0) 。 您的訂用帳戶將會維持在其排程的計費日期之前，但不會在帳單日期到期時續約。
