## <a name="how-much-does-it-cost"></a>費用是多少？

使用這些點數並不會有任何費用。 其已包含在您的 Visual Studio 訂閱中。 當您用完分配給該月份的點數之後，您將無法繼續加以使用，直到其於下個月重設為止。
