## <a name="can-i-use-my-monthly-azure-credits-to-run-production-applications"></a>我是否可以使用每月 Azure 點數來執行生產應用程式？

Visual Studio 訂閱者的每月 Azure 點數只能用於特定的個別開發/測試用途。 若要執行生產工作負載，您將必須升級到付費 Azure 訂用帳戶，例如[隨用隨付](https://azure.microsoft.com/offers/ms-azr-0003p/)定價。
