## <a name="what-are-the-monthly-azure-credits"></a>什麼是每月 Azure 點數？

Azure 每月點數可讓您免費探索及試用各種 Azure 服務。  範例包括： 
- 託管靜態網站
- 將 ASP.NET 網站發佈到 Azure
- 部署容器化網站
- 將網站部署到虛擬機器
- 使用無伺服器後端來整合您的資料與服務
- 部署 Python Web 應用程式

如需如何使用 Azure 點數的詳細資訊，請瀏覽 [Visual Studio 訂閱者的每月 Azure 點數](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/)。
