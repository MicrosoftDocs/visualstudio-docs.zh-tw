## <a name="what-are-the-licensing-terms-for-visual-studio-subscriptions"></a>Visual Studio 訂閱的授權條款有哪些？ 

所有 Visual Studio 訂用帳戶都是以每個使用者為基礎來授權。  每位授權的使用者都可以在任意數目的裝置上安裝和使用軟體，以設計、開發、測試及示範其程式。  Visual Studio 訂用帳戶也允許授權使用者評估軟體並模擬客戶環境，以診斷與您的程式相關的問題。  以這種方式使用軟體的每個額外人員也必須有授權。 

如需詳細資訊，請參閱 [Microsoft Visual Studio 授權](https://aka.ms/vslicensing)技術白皮書。
