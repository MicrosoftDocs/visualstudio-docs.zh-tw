## <a name="how-do-i-download-find-a-key-and-activate-the-visual-studio-ide"></a>如何下載、尋找金鑰及啟用 Visual Studio IDE？
您的訂閱層級會指出您有權存取的 Visual Studio IDE 層級。 例如 Visual Studio Professional 訂閱無權存取 Visual Studio Enterprise。 若不確定您可使用的層級，請參閱入口網站的[訂閱頁面](https://my.visualstudio.com/subscriptions)，查看您可使用的訂閱。 

### <a name="how-to-download-and-activate"></a>如何下載及啟用 
1. 登入訂閱入口網站。  
0. 若是最新版，請在 [工具] 區段中，找出 [Visual Studio IDE] 磚，然後按一下 [權益] 磚底部的 [下載] ****   連結。  
0. 您將被重新導向至 Visual Studio 的 [下載結果] 頁面，以下載 IDE。  
0. 請遵循下載及安裝指示。 
0. 使用指派訂閱的電子郵件，就能在安裝後立即登入 IDE，**而無須提供產品金鑰**。 了解如何[索取產品金鑰](https://docs.microsoft.com/visualstudio/subscriptions/find-keys)，在離線環境中使用。

### <a name="find-an-older-version"></a>尋找舊版 
您也可以在訂閱入口網站中，從 [下載頁面](https://my.visualstudio.com/Downloads?q=visual%20studio&pgroup=)搜尋及下載舊版。
