## <a name="where-can-i-find-older-versions-of-visual-studio"></a>我可以在哪裡找到 Visual Studio 的舊版本？ 

如果您要尋找 Visual Studio 2019 的先前版本，您可以在 [Visual Studio 2019 發行版本](https://docs.microsoft.com/visualstudio/releases/2019/history?branch=updates%2Fhistory) 文章中找到這些版本的下載連結和指示。

如果您需要舊版的 Visual Studio，例如 Visual Studio 2017，請參閱 [Visual Studio 較舊的下載](https://visualstudio.microsoft.com/vs/older-downloads/)。
