## <a name="renewing-subscriptions-through-microsoft-store"></a>透過 Microsoft Store 更新訂閱 

您應在到期日之前更新您的訂用帳戶，以確保能夠存取包含的開發人員工具和軟體。 續約價格僅適用于過去從 Microsoft store 購買的現有 Visual Studio 訂閱客戶，並在到期日的五個月內。 如果您的訂用帳戶已過期超過五個月，您就不符合續約價格的資格。 

如果您的訂用帳戶尚未過期，您會在訂用帳戶 [入口網站](https://my.visualstudio.com/subscriptions)中查看您的訂用帳戶，以找出到期日。 

## <a name="subscriptions-available"></a>可用的訂閱 

零售 Visual Studio 可透過 Microsoft Store 續訂的訂閱包括： 

* [Visual Studio 企業版](https://www.microsoft.com/en-us/p/visual-studio-enterprise-subscription/DG7GMGF0DST4/0003?rtc=1&activetab=pivot:overviewtab) 
* [Visual Studio Professional](https://www.microsoft.com/p/visual-studio-professional-subscription/dg7gmgf0dst3?activetab=pivot%3aoverviewtab) 
* [Visual Studio Test Professional](https://www.microsoft.com/p/visual-studio-test-professional-subscription/dg7gmgf0dst6?activetab=pivot%3aoverviewtab) 

每個訂用帳戶都提供進行初始購買或更新現有訂用帳戶的選項。 續約價格提供極優惠的折扣，但僅限於現有的訂閱。  

## <a name="how-to-renew"></a>如何續約 

如果您有標準零售訂用帳戶，該訂用帳戶會從 [Microsoft Store](https://www.microsoft.com/store) 購買，並應在該處續約。  

1. 請造訪商店，找出您目前擁有的訂用帳戶層級 (您只能更新相同的訂用帳戶層級) 。 
1. 按一下 [下載–續約] 並新增至購物車。 如果您想要變更訂用帳戶層級，您必須按一下 [下載-新增]。  
1. 退房。在 [確認] 頁面上，您將會獲得啟用訂用帳戶所需的產品金鑰。 請參閱下列指示。 

## <a name="how-to-activate"></a>啟用方式  

透過 Microsoft Store 更新 Visual Studio 訂用帳戶之後，您可以從付款確認頁面或在購買後收到的確認電子郵件開始啟用程式。 

1. 流覽至訂用帳戶 [入口網站的啟用頁面](https://my.visualstudio.com/subscriptions/activate)。 
1. 使用需要存取 Visual Studio 訂用帳戶的電子郵件登入。 
1. 在 [啟用您的 Visual Studio 訂用帳戶] 快顯視窗中，輸入您的產品金鑰。 您會在付款確認頁面或確認電子郵件中找到此金鑰。 

訂用帳戶現在應該會顯示在 [訂用帳戶] [頁面](https://my.visualstudio.com/subscriptions) 上，而且您應該可以存取您的所有訂用帳戶權益。
