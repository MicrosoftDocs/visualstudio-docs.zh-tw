## <a name="how-to-become-a-super-admin-for-visual-studio-subscriptions-agreements-managed-by-partners"></a>如何成為夥伴所管理 Visual Studio 訂用帳戶協定的超級管理員

透過大量授權購買 Visual Studio 訂閱的組織會使用系統管理員和超級管理員來管理訂用帳戶。  只有超級管理員可以將系統管理許可權指派給使用者。  若要成為系統管理員或超級管理員，請洽詢您組織中的超級系統管理員。  如果您的組織中沒有超級系統管理員，而且您是透過合作夥伴或轉銷商購買訂用帳戶，請與您的合作夥伴/轉銷商聯繫，以取得至少一個超級系統管理員經營的協助。 建議每個合約隨時都有至少兩個超級系統管理員。  

如果您已經指派訂用帳戶，請登入 https://my.visualstudio.com/subscriptions 並按一下 [ **連絡人我** 的系統管理員]。 您可以在我們的「 [我](https://docs.microsoft.com/visualstudio/subscriptions/contact-my-admin) 的系統管理員」文章中取得更多資訊。
