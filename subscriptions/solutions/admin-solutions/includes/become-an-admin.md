## <a name="how-to-become-an-administrator-or-super-admin-for-visual-studio-subscriptions"></a>如何成為 Visual Studio 訂閱的系統管理員或超級管理員

透過大量授權購買 Visual Studio 訂閱的組織會使用系統管理員和超級管理員來管理訂用帳戶。  只有超級管理員可以將系統管理許可權指派給使用者。  若要成為系統管理員或超級管理員，請洽詢您組織中的超級系統管理員。  

如果您已經指派訂用帳戶，請登入 https://my.visualstudio.com/subscriptions 並按一下 [ **連絡人我** 的系統管理員]。 如需詳細資訊，請參閱我們的「我的系統 [管理員](https://docs.microsoft.com/visualstudio/subscriptions/contact-my-admin) 」文章。
