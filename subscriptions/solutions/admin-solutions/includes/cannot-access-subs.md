## <a name="why-cant-i-access-my-subscription"></a>為何無法存取我的訂用帳戶？

有幾個原因可能會導致您無法存取您的訂用帳戶：

**您的訂用帳戶可能已過期。**  如果您是透過 Microsoft Store 購買訂用帳戶，您可以透過新訂用帳戶成本的折扣續訂。  以下是一些可協助您更新訂用帳戶的快速連結：
- [Visual Studio 企業版](https://www.microsoft.com/p/visual-studio-enterprise-subscription/dg7gmgf0dst4?activetab=pivot%3aoverviewtab)
- [Visual Studio Professional](https://www.microsoft.com/p/visual-studio-professional-subscription/dg7gmgf0dst3?activetab=pivot%3aoverviewtab)
- [Visual Studio Test Professional](https://www.microsoft.com/p/visual-studio-test-professional-subscription/dg7gmgf0dst6?activetab=pivot%3aoverviewtab)

**您的訂用帳戶可能已被您的系統管理員移除或重新指派。**  如果您的訂用帳戶是由您的公司或學校提供給您，則可能由系統管理員 (系統管理員) 管理。  如果您先前可以存取訂用帳戶，則您的系統管理員可能已將其刪除，或將其重新指派給其他人。  請 [洽詢您的系統管理員](https://docs.microsoft.com/visualstudio/subscriptions/contact-my-admin) 以取得協助。

**您的系統管理員可能沒有指派給您的訂用帳戶。**  如果您的公司或學校管理組織的訂用帳戶，您可能未獲指派訂用帳戶。  如果您有一個指派的帳戶，您應該會有確認訂用帳戶指派的電子郵件訊息。  請確定您使用的電子郵件地址與您收到該通知的電子郵件地址相同，以登入您的訂用帳戶。 請 [洽詢您的系統管理員](https://docs.microsoft.com/visualstudio/subscriptions/contact-my-admin) 以取得協助。 

**您可能使用錯誤的認證登入。**  如果您確信您有訂用帳戶，但您無法登入，或是您可以登入但未顯示訂用帳戶，您可能使用與訂用帳戶相關聯的電子郵件地址來登入。  如果您有多個訂用帳戶，也可能只是顯示錯誤的訂用帳戶。  使用 [權益] 頁面頂端的訂用帳戶選擇器，在訂用帳戶之間切換。  若要查看與您的登入電子郵件地址相關聯的所有訂用帳戶，並選取您想要使用的訂用帳戶，請前往 [ [訂閱]](https://my.visualstudio.com/subscriptions)索引標籤。
