## <a name="im-unable-to-download-or-activate-software-excluding-visual-studio-family"></a>我無法下載或啟用 Visual Studio 系列以外的軟體 () 

您可以使用的軟體下載取決於您所使用的 Visual Studio 訂用帳戶。 您可以透過幾種方式找到下載和產品金鑰。  

## <a name="download-a-list-of-available-software"></a>下載可用軟體的清單 
您可以[下載](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx)   所有訂用帳戶類型的廣泛可用軟體標題清單。 若要檢查指派給您的類型，請在訂用帳戶 [入口網站](https://my.visualstudio.com/subscriptions)中查看您的訂用帳戶。  

## <a name="how-to-find-software-downloads"></a>如何尋找軟體下載 
1. 登入訂用帳戶 [入口網站](https://my.visualstudio.com/benefits)。  
1. 移至 [精選下載頁面](https://my.visualstudio.com/downloads/featured)。  
1. 您將可以在這裡看到熱門下載清單或搜尋特定產品。 注意：如果您有多個與相同登入位址相關聯的訂用帳戶，您會看到最高階訂用帳戶隨附的下載，不論您要查看的訂用帳戶為何。  
4. 按一下 [取得金鑰] 以查看您要下載的產品的所有可用金鑰。 如果下載需要金鑰才能開始使用，請按一下 [宣告金鑰] 來宣告產品金鑰。 

## <a name="product-keys-page"></a>產品金鑰頁面 
您也可以流覽 [[產品金鑰](https://my.visualstudio.com/productkeys)]   頁面，其中所有索引鍵都可在匯總中使用。 您可以編輯 [附注] 資料行，以識別您索取金鑰的時機和原因。 

## <a name="more-information-about-downloads"></a>有關下載的詳細資訊 
[深入瞭解](https://docs.microsoft.com/visualstudio/subscriptions/find-keys) 尋找和索取產品金鑰和下載。
