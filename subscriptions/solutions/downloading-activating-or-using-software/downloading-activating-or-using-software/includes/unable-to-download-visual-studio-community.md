## <a name="im-unable-to-download-or-activate-visual-studio-community"></a>我無法下載或啟用 Visual Studio Community
Visual Studio Community 是免費的輕量版 IDE，而且您不需要付費的 Visual Studio 訂用帳戶就能使用它。 您可以 [直接](https://visualstudio.microsoft.com/vs/community/) 從 Visual Studio 網站下載。
