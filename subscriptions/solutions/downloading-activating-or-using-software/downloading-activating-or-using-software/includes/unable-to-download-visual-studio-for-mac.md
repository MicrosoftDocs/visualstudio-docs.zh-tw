## <a name="im-unable-to-download-or-activate-visual-studio-for-mac"></a>我無法下載或啟用 Visual Studio for Mac

身為 Visual Studio 訂閱者，您可以存取 Visual Studio IDE。 Visual Studio for Mac 是個別的下載內容，您可以在訂用帳戶入口網站的 [ [下載] 頁面](https://my.visualstudio.com/Downloads) 中找到。  

## <a name="find-and-download-visual-studio-for-mac"></a>尋找並下載 Visual Studio for Mac 
1. 登入訂用帳戶 [入口網站](https://my.visualstudio.com/benefits)，並查看可用 Visual Studio for Mac 下載的 [篩選視圖](https://my.visualstudio.com/Downloads?q=Visual%20Studio%20for%20mac&pgroup=) 。 我們只提供最新的產品版本。 
2. 請遵循下載及安裝指示。 
1. 使用您的訂用帳戶指派的電子郵件，您將能夠在安裝後立即登入 IDE，不需要產品金鑰。

## <a name="no-download-available"></a>沒有下載可用 
您可以使用的軟體下載取決於您所使用的訂用帳戶。 如果您沒有看到您要尋找的下載內容，則可能無法供您使用。 若要檢查您有哪些訂用帳戶可供使用，請登入入口網站的 [訂用帳戶] [頁面](https://my.visualstudio.com/subscriptions) 。 您可以[下載](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx)適用   于所有訂用帳戶類型的廣泛清單可用軟體專案。
