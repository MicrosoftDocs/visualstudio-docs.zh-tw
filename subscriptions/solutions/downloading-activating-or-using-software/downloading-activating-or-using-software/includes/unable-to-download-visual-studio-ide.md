## <a name="im-unable-to-download-or-activate-visual-studio-enterprise-or-professional"></a>我無法下載或啟用 Visual Studio Enterprise 或專業版

您的訂閱層級會指出您有權存取的 Visual Studio IDE 層級。 例如 Visual Studio Professional 訂閱無權存取 Visual Studio Enterprise。 若不確定您可使用的層級，請參閱入口網站的[訂閱頁面](https://my.visualstudio.com/subscriptions)，查看您可使用的訂閱。    

## <a name="how-to-download-and-activate"></a>如何下載及啟用 

1. 確定您已登入訂用帳戶 [入口網站](https://my.visualstudio.com/benefits)。  
1. 如需最新版本，請在 [工具] 區段中找到 Visual Studio IDE] 磚，然後按一下權益磚底部的 [下載] 連結。 
1. 您將被重新導向至 Visual Studio 的 [下載結果] 頁面，以下載 IDE。
1. 請遵循下載及安裝指示。
1. 使用您的訂用帳戶指派的電子郵件，您將能夠在安裝後立即登入 IDE，不需要產品金鑰。 如果您是在離線環境中，請[瞭解如何索取產品金鑰](https://docs.microsoft.com/visualstudio/subscriptions/find-keys)。 

## <a name="find-an-older-version"></a>尋找舊版

您也可以從訂用帳戶入口網站的 [ [下載] 頁面](https://my.visualstudio.com/Downloads?PId=6545) 搜尋和下載先前的版本。
