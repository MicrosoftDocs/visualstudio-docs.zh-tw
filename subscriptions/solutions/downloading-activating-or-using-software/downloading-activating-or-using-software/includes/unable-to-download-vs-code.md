## <a name="im-unable-to-download-or-activate-visual-studio-code"></a>我無法下載或啟用 Visual Studio Code

Visual Studio Code 是免費的程式碼編輯器，而且您不需要付費的 Visual Studio 訂用帳戶就能使用它。 您可以 [直接](https://code.visualstudio.com/download) 從 Visual Studio 網站下載。
