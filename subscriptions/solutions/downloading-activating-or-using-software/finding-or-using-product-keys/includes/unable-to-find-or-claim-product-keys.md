## <a name="im-unable-to-find-or-claim-a-product-key"></a>我找不到或索取產品金鑰。

您可以使用的產品金鑰取決於您所使用的 Visual Studio 訂用帳戶。  

## <a name="product-keys-page"></a>產品金鑰頁面 

[[產品金鑰](https://my.visualstudio.com/productkeys)]   頁面會列出匯總中可供您使用的所有金鑰。 您可以編輯 [附注] 資料行，以識別您索取金鑰的時機和原因。 根據產品而定，有不同的金鑰類型。 您將會立即看到一些需要宣告的金鑰和其他金鑰。 

如果您沒有看到金鑰或產品，可能是您的訂用帳戶層級無法使用產品。 您可以 [下載](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx) 所有訂用帳戶類型的廣泛可用軟體標題清單。 若要檢查指派給您的類型，請在訂用帳戶 [入口網站](https://my.visualstudio.com/subscriptions)中查看您的訂用帳戶。  

## <a name="more-information"></a>詳細資訊 

[深入](https://docs.microsoft.com/visualstudio/subscriptions/find-keys) 瞭解產品金鑰類型，以及索取產品金鑰。
