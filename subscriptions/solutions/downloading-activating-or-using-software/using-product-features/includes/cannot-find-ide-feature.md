## <a name="what-if-you-cant-find-the-ide-feature-you-want-to-use"></a>如果您找不到想要使用的 IDE 功能該怎麼辦？ 

您可以使用的特定 IDE 版本和功能將取決於您的訂用帳戶層級。 Visual Studio 會在訂用帳戶 [入口網站](https://my.visualstudio.com/benefits)的 [權益] 頁面上顯示為 [工具] 類別中的圖格。 

## <a name="compare-supported-visual-studio-features"></a>比較支援的 Visual Studio 功能 

在注意到您的訂用帳戶隨附的 Visual Studio 版本之後，請 [比較支援的功能](https://visualstudio.microsoft.com/vs/compare/) ，以查看您所使用的 IDE 中是否有可用的功能。 如果您使用的 IDE 版本不包含功能，您將需要下載正確的 IDE。 

## <a name="upgrade-your-subscription-level"></a>升級您的訂用帳戶層級  

如果功能是另一個訂用帳戶層級的特定功能，您將需要變更您的訂用帳戶層級。  

如果您的 Visual Studio 訂用帳戶是由您的公司或學校提供給您，則您可能會被訂用帳戶管理員指派給您。 使用訂用帳戶 [入口網站](https://my.visualstudio.com/benefits)中導覽右邊的 [連絡人我的系統管理員] 按鈕，與您組織的系統管理員聯繫。  

如果您直接購買訂用帳戶，或沒有系統管理員，您必須在所需的層級購買新的訂用帳戶。  

## <a name="how-to-contact-your-admin"></a>如何連絡您的管理員 

由於許多組織的大小和複雜性，您可能不知道系統管理員的身分，但訂用帳戶入口網站中有一項功能可協助您聯絡系統管理員。 

1. 登入訂用帳戶 [入口網站](https://my.visualstudio.com/benefits)。  
1. 按一下右上方的 [連絡人我的系統管理員] 按鈕。 
1. 輸入給管理員的訊息。 
1. 如果您想要收到傳送給系統管理員的訊息複本，請按一下 [傳送複本給我] 核取方塊。 最後，按一下 [傳送]。
