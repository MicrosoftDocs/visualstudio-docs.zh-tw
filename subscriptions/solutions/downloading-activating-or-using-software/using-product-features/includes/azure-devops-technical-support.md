## <a name="azure-devops-technical-support"></a>Azure DevOps 技術支援  

Visual Studio 訂閱支援提供 Visual Studio 訂用帳戶的銷售、訂用帳戶、帳戶和計費的協助。 如果您有 Azure DevOps 的技術問題或問題，您的問題必須導向 [Azure DevOps 支援](https://azure.microsoft.com/support/devops/)。 以下是一些可協助您尋找解答的資源。

## <a name="documentation--azure-devops"></a>檔 |Azure DevOps 

請參閱教學課程、疑難排解指南、實用的資源，以及 [Azure DevOps 檔](https://docs.microsoft.com/azure/devops/?view=azure-devops&preserve-view=true)中的其他主題範圍。

## <a name="developer-community--azure-devops"></a>開發人員群體 |Azure DevOps

搜尋線上 [開發人員社群](https://developercommunity.visualstudio.com/spaces/21/index.html) ，並從支援社區的論壇取得專家解答。 您可以搜尋關鍵字，並在不登入的情況下讀取、投票、批註。 如果您在搜尋時找不到需要的內容，您可以從導覽功能表建議功能或回報問題。 

## <a name="stack-overflow--azure-devops"></a>堆疊溢位 |Azure DevOps

針對任何人進行編碼的開放團體。 搜尋 [公開的 Q&，並以 azure devops 標記](https://stackoverflow.com/questions/tagged/azure-devops?tab=Newest) ，以取得已提出問題的解答，或註冊帳戶並提出新問題。
