## <a name="windows-developer-technical-support"></a>Windows 開發人員技術支援  

Visual Studio 訂閱支援提供 Visual Studio 訂用帳戶的銷售、訂用帳戶、帳戶和計費的協助。 如果您有 Windows 技術或開發人員帳戶的問題，您的問題必須導向 [windows 開發人員支援](https://developer.microsoft.com/windows/support/?utm_source=developermscom)。 建議您先嘗試這些其他資源。 

## <a name="windows-technical-documentation"></a>Windows 技術文件 
複習教學課程、疑難排解指南、實用的資源，以及 [Windows 技術檔](https://docs.microsoft.com/windows/)中的其他主題。

## <a name="windows-developer-account"></a>Windows 開發人員帳戶
您的 Visual Studio 訂用帳戶可能包含 Windows 開發人員帳戶。 造訪訂用帳戶 [入口網站權益頁面](https://my.visualstudio.com/Benefits) ，然後按一下權益磚，以啟用您的帳戶。 

閱讀更多有關 [Windows 開發人員帳戶權益](https://docs.microsoft.com/visualstudio/subscriptions/vs-windows-dev)的資訊。 

[比較 Visual Studio 訂](https://visualstudio.microsoft.com/vs/benefits/#azure?cat=visual-studio-enterprise-subscription) 用帳戶權益，以瞭解您的訂用帳戶所含的內容。
