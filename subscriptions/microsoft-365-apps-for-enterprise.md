# <a name="microsoft-365-apps-for-enterprise-in-visual-studio-subscriptions"></a>Visual Studio 訂用帳戶中的 Microsoft 365 企業應用程式
在選取的 Visual Studio 訂用帳戶中，可使用適用于企業的 Microsoft 365 應用程式（先前稱為 Office 365 Pro Plus）。 

建立您最好的作品。 使用永遠都會更新的應用程式在任何地方工作。 Office 365 Pro Plus 是一個進階應用程式 (例如 Word、Excel、PowerPoint、OneNote、Outlook、Publisher 及 Access) 隨附的訂用帳戶 (Publisher 與 Access 只能在 PC 上使用)。 應用程式可以安裝在多部裝置上，包括電腦、Mac、iPad、iPhone、Android 平板電腦與 Android 手機。 擁有訂用帳戶，您就可以取得最新版的應用程式，並且在有更新時自動收到更新。

> [!NOTE]
> 當您的 Visual Studio 訂用帳戶中包含時，您就有權使用適用于企業的 Microsoft 365 應用程式供生產環境使用。  

## <a name="activation-steps"></a>啟用步驟
為企業訂用帳戶啟用 Microsoft 365 應用程式的流程很簡單。  請遵循下列步驟來啟用您的訂用帳戶，並安裝您的應用程式：

1. 登入您的 Visual Studio 訂用帳戶，網址為[https://my.visualstudio.com](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)
1. 找出 [適用于企業的 Microsoft 365 應用程式] 磚，然後按一下 [**啟用**]。
   > [!div class="mx-imgBorder"]
   > ![適用于企業的 Microsoft 365 應用程式磚](_img/microsoft-365-apps-for-enterprise/tile-activate.png "按一下 [啟用] 以開始使用您的訂用帳戶。")

1. 在 [歡迎使用，讓我們知道您] 對話方塊中，提供要求的個人連絡人資訊，然後按 **[下一步]**。
   > [!div class="mx-imgBorder"]
   > ![Microsoft 365 企業客戶資訊的應用程式](_img/microsoft-365-apps-for-enterprise/get-to-know-you.png "輸入您的連絡人資訊")

1. 在 [建立您的使用者識別碼] 對話方塊中，為您的租使用者建立使用者名稱和密碼，然後按一下 [**建立我的帳戶**]。
   > [!div class="mx-imgBorder"]
   > ![Microsoft 365 建立企業使用者識別碼的應用程式](_img/microsoft-365-apps-for-enterprise/create-your-user-id.png "建立您的使用者識別碼和密碼")

1. 接下來，系統會要求您提供程式碼來證明您不是機器人。  您可以讓程式碼以文字訊息的形式傳送給您。  輸入代碼，然後按一下 [下一步]****。 
   > [!div class="mx-imgBorder"]
   > ![適用于企業的 Microsoft 365 應用程式證明您不是機器人](_img/microsoft-365-apps-for-enterprise/prove-youre-not-a-robot.png "要求程式碼並輸入它以繼續")

1. 成功建立您的帳戶之後，您會看到一個對話方塊，其中顯示 Office 入口網站的登入頁面，以及您的新使用者識別碼。  請記下這項資訊以供稍後使用，然後按一下 [**開始安裝**] 以開始安裝您的 Microsoft 365 應用程式。
   > [!div class="mx-imgBorder"]
   > ![適用于企業的 Microsoft 365 應用程式儲存此資訊](_img/microsoft-365-apps-for-enterprise/save-this-info.png "儲存新的使用者識別碼，以及 Office 入口網站的連結。")

1. 您將會自動登入 Office 入口網站，而且您現在已經準備好安裝您的應用程式。  按一下 [**安裝**] 按鈕以開始使用。
   > [!div class="mx-imgBorder"]
   > ![適用于企業安裝的 Microsoft 365 應用程式](_img/microsoft-365-apps-for-enterprise/install-your-office-apps.png "按一下 [安裝] 按鈕以安裝您的應用程式。")
1. 依照提示安裝您選取的應用程式。
