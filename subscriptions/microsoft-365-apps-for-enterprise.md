# <a name="microsoft-365-apps-for-enterprise-in-visual-studio-subscriptions"></a>Visual Studio 訂用帳戶中適用于企業的 Microsoft 365 應用程式
您可以在選取的 Visual Studio 訂用帳戶中，取得適用于 enterprise (的 Microsoft 365 應用程式先前的 Office 365 Pro Plus)  

建立您最好的作品。 使用永遠都會更新的應用程式在任何地方工作。 適用于企業的 Microsoft 365 應用程式是隨附于 Word、Excel、PowerPoint、OneNote、Outlook、Publisher 及 Access (Publisher 和 access 的高階應用程式) 的訂閱。 應用程式可以安裝在多部裝置上，包括電腦、Mac、iPad、iPhone、Android 平板電腦與 Android 手機。 擁有訂用帳戶，您就可以取得最新版的應用程式，並且在有更新時自動收到更新。

> [!NOTE]
> 當您的 Visual Studio 訂用帳戶中包含時，系統會授權您使用適用于企業的 Microsoft 365 應用程式以供生產環境使用。  

## <a name="activation-steps"></a>啟用步驟
針對企業訂用帳戶啟用 Microsoft 365 應用程式的程式很簡單。  遵循下列步驟來啟用您的訂用帳戶，並安裝您的應用程式：

1. 登入您的 Visual Studio 訂用帳戶，網址為： [https://my.visualstudio.com](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)
1. 找出 Microsoft 365 enterprise 磚的應用程式，然後選取 [ **啟用**]。
   > [!div class="mx-imgBorder"]
   > ![適用于企業的 Microsoft 365 應用程式磚](_img/microsoft-365-apps-for-enterprise/tile-activate.png "選取 [啟動] 以開始使用您的訂用帳戶。")

1. 在 [歡迎使用，讓我們知道您的資訊] 對話方塊中，提供要求的個人連絡人資訊，然後選取 **[下一步]**。
   > [!div class="mx-imgBorder"]
   > ![適用于企業客戶資訊的 Microsoft 365 應用程式](_img/microsoft-365-apps-for-enterprise/get-to-know-you.png "輸入您的連絡人資訊")

1. 在 [建立您的使用者識別碼] 對話方塊中，為您的租使用者建立使用者名稱和密碼，然後選取 [ **建立我的帳戶**]。
   > [!div class="mx-imgBorder"]
   > ![Microsoft 365 建立企業使用者識別碼的應用程式](_img/microsoft-365-apps-for-enterprise/create-your-user-id.png "建立您的使用者識別碼和密碼")

1. 接下來，系統會要求您提供程式碼，以證明您不是機器人。  您可以讓程式碼以文字訊息的形式傳送給您。  輸入程式碼，然後選取 **[下一步]**。 
   > [!div class="mx-imgBorder"]
   > ![適用于企業的 Microsoft 365 應用程式證明您不是機器人](_img/microsoft-365-apps-for-enterprise/prove-youre-not-a-robot.png "要求程式碼並輸入以繼續")

1. 成功建立您的帳戶之後，您會看到一個對話方塊，顯示 Office 入口網站的登入頁面，以及新的使用者識別碼。  請記下此資訊以供稍後使用，然後選取 [ **啟動安裝程式** ] 開始安裝 Microsoft 365 應用程式。
   > [!div class="mx-imgBorder"]
   > ![適用于企業的 Microsoft 365 應用程式請儲存此資訊](_img/microsoft-365-apps-for-enterprise/save-this-info.png "儲存新的使用者識別碼和 Office 入口網站的連結。")

1. 您將會自動登入 Office 入口網站，而且您現在已經準備好安裝應用程式。  選取 [ **安裝** ] 按鈕以開始使用。
   > [!div class="mx-imgBorder"]
   > ![適用于企業安裝的 Microsoft 365 應用程式](_img/microsoft-365-apps-for-enterprise/install-your-office-apps.png "選取 [安裝] 按鈕以安裝您的應用程式。")
1. 遵循提示來安裝您選取的應用程式。
