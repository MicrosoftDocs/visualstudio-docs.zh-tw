# <a name="technical-support-benefits-included-with-visual-studio-subscriptions"></a>Visual Studio 訂用帳戶中包含的技術支援權益

依賴技術專家協助您解決問題並回答問題。

當您有問題或需要有關技術問題的協助時，有多種可用的支援資源。

> [!NOTE]
> 訂閱並不一定會包含全部的權益。  如需是否符合資格的詳細資訊，請參閱下列文章。  

- [Azure 諮詢對談](vs-azure-advisory-chat.md)：即時取得使用 Azure 的協助。

- [Azure 社群](vs-azure-community.md)：直接透過「Azure 社群」從精選論壇取得協助。 

- [技術支援](vs-tech-support.md)：我們提供專家協助，幫助您解決非生產環境中的損壞修正問題。

- [優先支援](vs-priority-support.md)：在精選論壇清單中詢問您的問題。 若開發人員社群未在兩天內回應問題，Microsoft 將會回應問題 (僅支援使用 Microsoft 帳戶)。

- [線上指引對談](vs-concierge-chat.md)：與我們交談，協助您尋找技術文件及產品資訊，或使用您的訂用帳戶權益。
