# <a name="microsoft-365-developer-subscription-in-visual-studio-subscriptions"></a>微軟 365 開發人員訂閱在可視化工作室訂閱
從 2019 年 9 月開始,Office 365 開發人員計畫的所有**新成員**(包括 Visual Studio 專業版和企業版訂戶)都可以註冊 Microsoft 365 E5 開發人員訂閱(不包括 Windows)。 這將替換我們從 2018 年 3 月開始提供的 Office 365 E3 開發人員訂閱。 它現在包括EMS,我們的客戶要求我們在用戶語音。  有關詳細資訊,請查看最近的[開發人員部落格](https://developer.microsoft.com/graph/blogs/new-microsoft-365-e5-subscription-with-ems-now-available-for-developers/)。

選定的 Visual Studio 專業版或企業標準訂閱現在可以訪問 Microsoft 365 開發人員訂閱權益,只要其訂閱處於活動狀態。

## <a name="what-is-different-about-e5-vs-e3"></a>E5 與 E3 有何不同?
Microsoft 365 E5 開發人員訂閱提供了新的價值,並擴展了對關鍵的 Microsoft 365 工作負載和功能的訪問。 使用 Microsoft 365 E5 沙箱訂閱的開發人員可以獲取他們在 E3 中享受的所有內容,並可以利用:

- 使用 Power BI 的進階分析
- 企業移動性 + 安全 (EMS), 用於合規性和資訊保護
- Office 365 進階威脅防護
- 建立進階識別與存取管理解決方案的 Azure 活動目錄

要查看 Microsoft 365 E5 開發人員訂閱中包含的內容,請參閱[Microsoft 365 開發人員計畫常見問題解答](https://docs.microsoft.com/office/developer-program/microsoft-365-developer-program-faq#whats-included-in-the-developer-subscription)。


## <a name="activation-steps"></a>啟用步驟
1. 登入 [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)。

0. 在「工具」類別中找到 Microsoft 365 權益磁貼,然後單擊 **「啟動**」。


    > [!div class="mx-imgBorder"]
    >![微軟 365 開發人員磁貼](_img/vs-m365/vs-m365-tile.png)
    
0. 在歡迎螢幕上,選擇您所在的地區/國家/地區,輸入您的公司名稱,然後單擊複選框以接受使用條款。  然後按一下 [下一步]  。

0. 在「程式首選項」螢幕上,填寫各種欄位以指示您的偏好和感興趣區域。 完成後,按一下 **「啟動訂閱**」。

0. 在 **「設定開發人員訂閱**」對話框中,選擇您所在的國家/地區、創建使用者名和密碼並設置密碼。  然後按一下 **[繼續]**。

0. 指定您所在的國家/地區代碼支援 SMS 消息的電話號碼,然後按一下「**發送代碼**」。  您將透過簡訊收到代碼。 

0. 輸入您在簡訊中收到的代碼,然後按下「**設定**」 。

0. 您將看到 Office 開發人員中心頁面,並顯示訂閱資訊。 最好藉此機會將訂閱分配給使用者。  按下 **「轉到訂閱**」。

0. 登入您的帳戶。 

0. 取消簡介對話方塊。

0. 按下 **「已獲得」** 以關閉 Office 365 應用對話方塊。 

0. 按下 **「管理員」** 按鈕打開 Microsoft 365 管理中心。 

0. 選擇左側導航窗格中的**使用者**,然後選擇 **「活動使用者**」。。  

0. 將顯示活動使用者的清單。  如果您剛剛設置訂閱,您將是唯一列出的訂閱。  按一下 [新增使用者]****。

0. 系統可能會要求您個人化您的登錄和電子郵件。  選擇最適合的選項,然後單擊 **「下一步**」。。 

0. 在「**新增新使用者」** 頁上,您將看到一個網格,可用於添加都將為其分配許可證的使用者。  如果需要添加更多使用者,請單擊 「**添加其他使用者**」。。

新增使用者後,您可以選擇執行創建團隊和安裝應用程式等操作。  您現在可以執行任何或所有這些事情,或者隨時返回管理中心來設置這些操作。

## <a name="frequently-asked-questions"></a>常見問題集
### <a name="q-how-do-current-users-upgrade-to-e5"></a>問:當前使用者如何升級到 E5?
答:目前,只有新的 Office 365 開發人員計劃成員將自動獲得 E5 產品/ 請放心,我們正在努力使所有 Office 365 開發人員計畫成員從 E3 移動到 E5。 請繼續關注未來幾個月的更多資訊。

### <a name="q-can-i-activate-a-microsoft-365-developer-subscription-using-an-existing-tenant-or-do-i-need-to-create-a-new-one"></a>問:我能否使用現有租戶啟動 Microsoft 365 開發人員訂閱,還是需要創建新訂閱?
答:要啟動 Office 開發人員訂閱,需要新的租戶。  

### <a name="q-can-i-choose-e3-or-e5"></a>問:我可以選擇 E3 或 E5 嗎?
截至 2019 年 9 月,新的開發人員計畫成員僅提供 E5,這是我們的旗艦版 Office 365。  此版本的 Office 365 比 E3 具有更大的價值,通常成本可能遠高於我們以前的 E3 訂閱。

具有 E3 的現有成員不需要 E5 的額外功能,他們可以選擇保留其當前的 E3 訂閱並繼續續訂它。 如果任何現有成員不想遷移,則不要求它們從 E3 移動到 E5。 但是,每個成員只能有一個訂閱。

當我們為現有開發人員計劃成員提供遷移選項時,我們將提供有關如何選擇以及如何遷移的詳細資訊。

### <a name="q-how-do-i-know-which-version-i-have"></a>問:如何知道我擁有哪個版本?
現在,您可以在 Office 365 開發人員計畫儀表板上的訂閱磁貼中驗證您的版本。

在磁貼中尋找 E3 或 E5 指示。

### <a name="q-if-microsoft-365-developer-subscription-doesnt-include-windows-how-do-i-get-windows"></a>問:如果 Microsoft 365 開發人員訂閱不包括 Windows,如何獲取 Windows?
答:包含 Microsoft 365 權益的可視化工作室訂閱還包括 Windows 作為訂閱者下載的一部分。  訪問[下載](https://my.visualstudio.com/downloads)以下載 Windows。 (如果您在訂閱中未看到已啟用下載,請[與管理員聯絡](contact-my-admin.md)。
