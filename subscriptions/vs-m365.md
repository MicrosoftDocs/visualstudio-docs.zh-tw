# <a name="microsoft-365-in-visual-studio-subscriptions"></a>Visual Studio 訂用帳戶中的 Microsoft 365
從2019年9月開始，Office 365 開發人員計畫（包括 Visual Studio Professional 和企業訂閱者）的所有**新**成員都可以註冊 Microsoft 365 E5 開發人員訂用帳戶（未包含 Windows）。 這會取代從3月2018開始提供的 Office 365 E3 開發人員訂閱。 它現在包含 EMS，我們的客戶在使用者心聲上要求我們提供這項功能。  如需詳細資訊，請參閱最新的[開發人員 Blog](https://developer.microsoft.com/graph/blogs/new-microsoft-365-e5-subscription-with-ems-now-available-for-developers/) 。

選取的 Visual Studio Professional 或企業標準訂用帳戶現在可以存取 Microsoft 365 權益，只要其訂用帳戶處於作用中狀態即可。

## <a name="what-is-different-about-e5-vs-e3"></a>E5 與 E3 有何不同？
Microsoft 365 E5 開發人員訂用帳戶提供新價值，並擴充對重要 Microsoft 365 工作負載和功能的存取。 具有 Microsoft 365 E5 沙箱訂閱的開發人員可以取得他們在 E3 中喜愛的所有內容，並可利用：

- 使用 Power BI 的先進分析
- 適用于合規性與資訊保護的 Enterprise Mobility + Security （EMS）
- Office 365 先進的威脅防護
- 建立先進的身分識別和存取管理解決方案 Azure Active Directory

若要查看 Microsoft 365 E5 開發人員訂用帳戶中包含的內容，請參閱[比較 Microsoft 365 enterprise 方案](https://nam06.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.microsoft.com%2Fen-us%2Fmicrosoft-365%2Fcompare-all-microsoft-365-plans&data=04%7C01%7Cmikeald%40microsoft.com%7Cab7e48abb3a04d87278308d716e495b8%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C637003043338656615%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C-1&sdata=NO9irwnRgVZme0nuZj50X7mI0G3bEeUs2ZdwYuxEr%2B8%3D&reserved=0)和[Microsoft 365 大量授權](https://nam06.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.microsoft.com%2Fen-us%2Flicensing%2Fproduct-licensing%2Fmicrosoft-365-enterprise&data=04%7C01%7Cmikeald%40microsoft.com%7Cab7e48abb3a04d87278308d716e495b8%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C637003043338666614%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C-1&sdata=WziZov9B702H%2BxTchPPAmJmIpAsc6zbetlWOtFpANSk%3D&reserved=0)。

如需這兩個版本的詳細說明，請參閱我們[已更新的檔](/office/developer-program/office-365-developer-program-faq)。

## <a name="activation-steps"></a>啟用步驟
1. 登入 [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)。

0. 在 [工具] 類別中找出 [Microsoft 365 權益] 磚，然後按一下 [**啟用**]。

0. 在 [歡迎使用] 畫面上，選擇您的地區/國家/地區、輸入您的公司名稱，然後按一下核取方塊以接受使用規定。  然後按一下 [下一步]。

0. 在 [程式喜好設定] 畫面上，填寫各種欄位，以指出您的喜好設定和感興趣的區域。 當您完成時，按一下 [**啟用訂閱**]。

0. 在 [**設定您的開發人員訂**用帳戶] 對話方塊中，選擇您的國家/地區、建立使用者名稱和網域，並設定您的密碼。  然後按一下 [**繼續**]。

0. 指定您的國家/地區代碼支援 SMS 訊息的電話號碼，然後按一下 [**傳送程式碼**]。  您會透過 SMS 訊息收到程式碼。 

0. 輸入您在文字訊息中收到的驗證碼，然後按一下 [**設定**]。

0. 您會看到 [Office 開發人員中心] 頁面，其中會顯示您的訂用帳戶資訊。 將訂用帳戶指派給您的使用者是個不錯的主意。  按一下 [**移至訂**用帳戶]。

0. 登入您的帳戶。 

0. 取消 [簡介] 對話方塊。

0. 按一下 [已**到達**]，關閉 [Office 365 應用程式] 對話方塊。 

0. 按一下 [**管理**] 按鈕以開啟 [Microsoft 365 系統管理中心]。 

0. 選取左側導覽窗格中的 [**使用者**]，然後選擇 [作用中**使用者**]。  

0. 您的作用中使用者清單隨即出現。  如果您只是要設定訂用帳戶，您將會是唯一列出的訂閱。  按一下 [**新增使用者**]。

0. 系統可能會要求您將登入和電子郵件個人化。  選擇最適合您的選項，然後按 **[下一步]** 。 

0. 在 [**新增使用者**] 頁面上，您會看到一個格線，您可以用來新增所有已獲指派授權的使用者。  如果您需要新增更多使用者，請按一下 [ **+ 新增其他使用者**]。

新增使用者之後，您可以選擇建立小組和安裝應用程式等。  您可以立即執行任何或所有這些專案，或隨時返回系統管理中心進行設定。

## <a name="frequently-asked-questions"></a>常見問題集
### <a name="q-how-do-current-users-upgrade-to-e5"></a>問：目前使用者如何升級至 E5？
答：目前只有新的 Office 365 開發人員計畫成員會自動取得 E5 供應專案。 請放心，我們正致力於讓所有 Office 365 開發人員計畫成員從 E3 移至 E5。 在未來的幾個月內，敬請期待更多的資訊。

### <a name="q-can-i-choose-e3-or-e5"></a>問：我可以選擇 E3 或 E5 嗎？
從2019年9月起，新的開發人員計畫成員只提供 E5，也就是我們的旗艦版 Office 365。  這個版本的 Office 365 提供比 E3 更高的價值，而且通常會比先前的 E3 訂閱更具成本。

具有 E3 的現有成員若不需要 E5 的額外功能，將可選擇保留其目前的 E3 訂閱並繼續進行更新。 如果您不想要遷移，則不需要任何現有成員從 E3 移至 E5。 不過，每個成員只能有一個訂用帳戶。

當我們有現有開發人員計畫成員的遷移選項時，我們將提供有關如何選擇及如何進行遷移的詳細資訊。

### <a name="q-how-do-i-know-which-version-i-have"></a>問：如何? 知道我擁有哪個版本？
您現在可以在位於 Office 365 開發人員計畫儀表板的 [訂用帳戶] 磚中，驗證您的版本。

在您的磚中尋找 E3 或 E5 指示。


### <a name="q-if-microsoft-365-doesnt-include-windows-how-do-i-get-windows"></a>問：如果 Microsoft 365 不包含 Windows，如何取得 Windows？
答：包含 Microsoft 365 權益的 Visual Studio 訂用帳戶也包含 Windows 做為訂閱者下載的一部分。  請造訪[下載](https://my.visualstudio.com/downloads)以下載 Windows。 （如果您未在訂用帳戶中看到 [已啟用下載]，請[洽詢您的系統管理員](contact-my-admin.md)。）
