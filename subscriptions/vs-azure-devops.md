# <a name="azure-devops-benefits-for-visual-studio-subscribers"></a>Visual Studio 訂閱者的 Azure DevOps 權益
身為有效的 Visual Studio 訂閱者，您可以使用 Azure DevOps 一定範圍內的功能，這些功能是隨附於訂用帳戶的權益。 您可以使用所屬每個 Azure DevOps 組織中的這些相同功能，無論是您建立了組織，或由他人新增了功能。

## <a name="sign-in"></a>登入

   > [!div class="mx-imgBorder"]
   > ![Azure DevOps 圖格](_img/vs-azure-devops/vs-azure-devops-tile.png "按一下 [開始使用] 以登入 Azure DevOps。")

   
當您使用啟用 Visual Studio 訂用帳戶所用的身分識別，或使用您的[其他身分識別](vs-alternate-identity.md)登入 Azure DevOps 時，我們會自動辨識。  如此一來，您也可以新增工作或學校帳戶 (您登入 Visual Studio、Microsoft 365 或公司或學校網路時所使用的帳戶) ，可讓您使用個人帳戶和公司或學校帳戶來存取 Azure DevOps。

[註冊免費的 Azure DevOps 組織](https://visualstudio.microsoft.com/team-services/)

## <a name="frequently-asked-questions"></a>常見問題集
### <a name="q-as-a-visual-studio-enterprise-subscriber-do-i-get-additional-parallel-jobs-for-tfs-and-azure-pipelines"></a>問：做為 Visual Studio Enterprise 訂閱者，是否會取得 TFS 和 Azure 管線的額外平行作業？
答：可以。 Visual Studio Enterprise 訂閱者可以在 Team Foundation Server 2017 或更新版本中取得一項平行作業，並在每個 Azure DevOps Services 組織中建立一個自我裝載的平行作業，其中是成員。
